# 需求文档 - stock_analysis

## 一、背景

快速获取单只 A 股股票的核心指标和市场表现，帮助投资者了解股票行情和成交活跃度。

## 二、适用场景

- ✅ 查询单只股票的行情数据
- ✅ 了解股票的成交活跃度
- ✅ 快速判断股票涨跌趋势
- ❌ 港股/美股分析
- ❌ 深度财务分析
- ❌ 技术指标分析

## 三、业务规则

### 3.1 活跃度判断

| 换手率 | 结论 |
|--------|------|
| > 10% | 非常活跃 |
| > 5% | 较活跃 |
| > 2% | 正常 |
| <= 2% | 不活跃 |

### 3.2 涨跌判断

| 涨跌幅 | 结论 |
|--------|------|
| > 5% | 大涨 |
| > 2% | 上涨 |
| < -5% | 大跌 |
| < -2% | 下跌 |
| 其他 | 持平 |

## 四、输入输出

### 输入

- 股票代码（如 000001）或名称（如 平安银行）
- 日期（可选，默认最近交易日）

### 输出

- 价格指标（开盘、最高、最低、收盘、涨跌幅）
- 成交指标（成交量、成交额）
- 分析结论（活跃度、涨跌趋势）

## 五、验收标准

- [ ] 能根据股票代码/名称获取行情数据
- [ ] 活跃度判断符合上述规则
- [ ] 涨跌判断符合上述规则
- [ ] 支持指定日期查询
