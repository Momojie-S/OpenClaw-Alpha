# 需求文档 - 策略回测

## 一、背景

验证投资策略的历史表现，评估风险收益指标。

## 二、适用场景

- ✅ 测试均线交叉策略
- ✅ 评估策略收益率和风险指标
- ✅ 分析历史交易记录
- ❌ 多股票组合回测
- ❌ 参数优化
- ❌ 实时策略

## 三、业务规则

### 3.1 可用策略

| 策略 | 买入信号 | 卖出信号 |
|------|----------|----------|
| ma_cross | 快均线上穿慢均线 | 快均线下穿慢均线 |
| rsi | RSI 下穿超卖线 | RSI 上穿超买线 |
| bollinger | 价格上穿下轨 | 价格下穿上轨 |

### 3.2 评估指标

| 指标 | 说明 |
|------|------|
| 总收益率 | 策略总收益 / 初始资金 |
| 年化收益率 | 按年计算的收益率 |
| 夏普比率 | 风险调整后收益 |
| 最大回撤 | 从高点到低点的最大跌幅 |
| 胜率 | 盈利交易占比 |

## 四、验收标准

- [ ] 能运行均线交叉策略
- [ ] 能运行 RSI 策略
- [ ] 能运行布林带策略
- [ ] 能输出风险收益指标
