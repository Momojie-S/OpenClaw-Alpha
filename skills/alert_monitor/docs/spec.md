# 需求文档 - 预警监控

## 一、背景

需要一个统一的预警系统，整合多个 skill 的风险检测能力，自动扫描持仓风险和市场异动，生成预警报告。

## 二、适用场景

- ✅ 持仓风险扫描（业绩、价格、资金、解禁）
- ✅ 市场异动监控（北向资金、板块热度）
- ✅ 定时风险报告
- ✅ 自选股风险追踪
- ❌ 实时交易信号
- ❌ 量化风险模型
- ❌ 高频监控

## 三、业务规则

### 3.1 持仓风险预警

| 预警类型 | 触发条件 | 优先级 |
|---------|---------|:------:|
| 业绩暴雷 | 业绩首亏/预减/增亏 | 高 |
| 连续下跌 | 连续 ≥3 天且累计跌幅 ≥10% | 中 |
| 资金流出 | 连续 ≥3 天净流出且累计 ≥1 亿 | 中 |
| 解禁压力 | 解禁占流通市值 ≥5% | 中 |
| 跌破止损 | 跌破用户设定止损线 | 高 |

### 3.2 市场异动预警

| 预警类型 | 触发条件 | 优先级 |
|---------|---------|:------:|
| 北向大幅流入 | 单日净流入 ≥50 亿 | 中 |
| 北向大幅流出 | 单日净流出 ≥30 亿 | 高 |
| 板块热度急升 | 热度环比 > +30% | 中 |
| 板块热度骤降 | 热度环比 < -30% | 中 |

### 3.3 配置化规则

用户可在配置文件中：
- 添加监控标的（自选股/持仓）
- 设置成本价和止损线
- 开启/关闭特定预警类型
- 调整预警阈值

## 四、验收标准

- [ ] 能扫描配置文件中的持仓风险
- [ ] 能检测业绩暴雷、连续下跌、资金流出风险
- [ ] 能检测解禁风险
- [ ] 能监控北向资金异动
- [ ] 能监控板块热度异动
- [ ] 支持简报/完整两种报告模式
- [ ] 报告可保存到文件
