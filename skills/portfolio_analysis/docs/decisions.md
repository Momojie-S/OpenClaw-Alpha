## 决策

### 分离三个 Processor

原因：
- 持仓结构、相关性、风险贡献是独立需求
- 分离后可独立调用
- 计算复杂度不同

### HHI 阈值选择

参考监管标准：
- < 1500：分散
- 1500-2500：中等
- > 2500：集中

### 输入格式

支持两种输入：
- 字符串格式：简单快速
- JSON 文件：复杂持仓

## 踩坑

### 行业分类数据源

- 问题：AKShare 无申万行业分类
- 解决：使用 Tushare `index_classify()`
