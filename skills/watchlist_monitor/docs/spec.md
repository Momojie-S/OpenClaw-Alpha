# 需求文档 - watchlist_monitor

## 一、背景

维护自选股列表，批量获取行情，快速分析整体表现。

## 二、适用场景

- ✅ 维护关注股票列表
- ✅ 批量查看自选股行情
- ✅ 分析自选股整体表现
- ❌ 实时推送
- ❌ 复杂技术分析
- ❌ 持仓盈亏计算

## 三、业务规则

### 3.1 自选股管理

- 添加：支持单只或多只股票，支持从文件导入
- 移除：支持单只股票移除
- 清空：需确认（`--yes`）

### 3.2 行情展示

| 字段 | 说明 |
|------|------|
| 代码 | 股票代码 |
| 名称 | 股票名称 |
| 现价 | 最新价格 |
| 涨跌幅 | 当日涨跌 |
| 成交额 | 成交金额（亿） |
| 换手率 | 当日换手率 |

### 3.3 分析统计

- 总数、上涨数、下跌数
- 平均涨跌幅
- 表现最好/最差
- 异动提醒

## 四、输入输出

### 输入

- 操作类型：添加、移除、列表、查看、分析
- 股票代码（添加/移除时）

### 输出

- 自选股列表
- 行情数据
- 分析报告

## 五、验收标准

- [ ] 支持添加/移除/清空自选股
- [ ] 支持批量获取行情
- [ ] 支持统计分析和异动提醒
