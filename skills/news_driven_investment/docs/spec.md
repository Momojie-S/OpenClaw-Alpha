# 需求文档 - 新闻驱动投资分析

## 一、背景

从财经新闻中发现投资机会，自动关联产业分析和标的筛选。

## 二、适用场景

- ✅ 新闻热点追踪
- ✅ 概念题材挖掘
- ✅ 动态标的筛选
- ❌ 纯技术分析
- ❌ 长期价值投资

## 三、业务规则

### 3.1 分析流程

1. 获取新闻 → 2. 识别关键词 → 3. 调用分析工具 → 4. 综合分析

### 3.2 数据传递

中间结果保存在 workspace，供后续分析：
- `keywords.json` - 提取的关键词
- `analysis.md` - 分析报告
- `candidates.json` - 候选标的

### 3.3 可调用的 Skill

| Skill | 用途 |
|-------|------|
| industry_trend | 产业热度分析 |
| stock_screener | 标的筛选 |
| northbound_flow | 资金流向 |
| market_sentiment | 市场情绪 |

## 四、验收标准

- [ ] 能获取财经新闻
- [ ] 能按关键词筛选
- [ ] 能按日期筛选
- [ ] 能获取个股新闻
