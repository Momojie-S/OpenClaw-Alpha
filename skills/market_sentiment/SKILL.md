# 市场情绪分析

快速了解 A 股市场整体情绪，识别极端行情信号。

## 使用说明

### 脚本运行

所有脚本需在项目根目录下运行，使用 `uv run --env-file .env` 加载环境变量：

```bash
# 市场情绪分析
uv run --env-file .env python -m skills.market_sentiment.scripts.sentiment_processor.sentiment_processor [--date YYYY-MM-DD]

# 市场宽度分析
uv run --env-file .env python -m skills.market_sentiment.scripts.breadth_processor.breadth_processor [--symbol all|sz50|hs300|zz500] [--days 30]

# 市场择时分析
uv run --env-file .env python -m skills.market_sentiment.scripts.timing_processor.timing_processor [--date YYYY-MM-DD]

# 股债性价比分析
uv run --env-file .env python -m skills.market_sentiment.scripts.equity_bond_ratio_processor.equity_bond_ratio_processor [--date YYYY-MM-DD] [--lookback-days 252]
```

**如果脚本运行失败**：
1. 检查是否在项目根目录
2. 检查 `.env` 文件是否存在且包含 `TUSHARE_TOKEN`
3. 将错误信息记录到进度文件，不要丢失上下文

### 运行记录

**进度文件位置**：`.openclaw_alpha/market-sentiment/{YYYY-MM-DD}/progress.md`

每次运行脚本后，记录：
- 运行时间
- 脚本命令
- 运行结果（成功/失败）
- 关键输出（市场温度、情绪状态）
- 错误信息（如有）

## 分析步骤

### Step 1: 获取市场情绪

**输入**：日期（可选，默认今天）

**动作**：
```bash
uv run --env-file .env python -m skills.market_sentiment.scripts.sentiment_processor.sentiment_processor
```

**输出**：
- 控制台：市场情绪摘要
- 文件：`.openclaw_alpha/market-sentiment/{date}/sentiment.json`（完整数据）

**分析要点**：
- 关注 `temperature` 字段：0-100 分，分数越高越热
- 查看 `status` 字段：极度恐慌/偏冷/正常/偏热/极度亢奋
- 注意 `signals` 字段：极端信号列表

### Step 2: 解读情绪指标

**涨跌停统计**：
- `limit.up`：涨停家数
- `limit.down`：跌停家数
- `limit.break_board`：炸板家数

**涨跌家数**：
- `trend.up`：上涨家数
- `trend.down`：下跌家数
- `trend.flat`：平盘家数

**资金流向**：
- `flow.main_net_inflow`：主力净流入金额
- `flow.main_net_inflow_pct`：主力净流入占比

### Step 3: 市场择时分析

**输入**：日期（可选，默认今天）

**动作**：
```bash
uv run --env-file .env python -m skills.market_sentiment.scripts.timing_processor.timing_processor
```

**输出**：
```
{
  "date": "2026-03-08",
  "sentiment_index": 35.2,
  "zone": "低迷",
  "signals": {
    "left": {
      "signal": "买入",
      "description": "情绪低迷，左侧买入信号"
    },
    "right": {
      "signal": "持有",
      "description": "情绪平稳，持有观望"
    }
  },
  "recommendation": "市场情绪偏低，适合分批建仓；左侧信号：情绪低迷，左侧买入信号"
}
```

**分析要点**：
- **情绪综合指数**：整合温度、涨跌停、涨跌家数、资金流向、市场宽度的综合指标
- **情绪分区**：极度低迷（<25）、低迷（25-40）、中性（40-60）、高涨（60-75）、极度高涨（>75）
- **左侧信号**：逆向择时，在极端情绪时反向操作
- **右侧信号**：趋势跟踪，在情绪趋势确认后跟随

### Step 4: 股债性价比分析

**输入**：日期（可选）、回溯天数（可选，默认252天）

**动作**：
```bash
uv run --env-file .env python -m skills.market_sentiment.scripts.equity_bond_ratio_processor.equity_bond_ratio_processor
```

**输出**：
```
{
  "date": "2026-03-06",
  "hs300_pe": 14.86,
  "risk_premium": 4.95,
  "percentile": 28.7,
  "signal": "卖出",
  "recommendation": "风险溢价较低（28.7%分位），股票相对昂贵；建议降低股票仓位，增加债券或现金"
}
```

**分析要点**：
- **风险溢价** = 1/沪深300 PE - 10年期国债收益率
- **历史分位数**：风险溢价相对于过去一年的位置
- **信号**：分位数 > 80% 买入，< 20% 卖出

### Step 5: 分析市场宽度（可选）

**输入**：指数类型、分析天数

**动作**：
```bash
uv run --env-file .env python -m skills.market_sentiment.scripts.breadth_processor.breadth_processor --symbol all --days 30
```

**输出**：
```

==================================================

收盘点位: 4124.19

【宽度指标】
  20日宽度比率: 85.1%
  60日宽度比率: 93.6%
  120日宽度比率: 94.2%

【新高新低统计】
  20日: 新高 541 / 新低 95
  60日: 新高 383 / 新低 26

【健康度评估】
  等级: 健康
  得分: 85.1

  ⚠️ 底背离信号：指数下跌但市场宽度改善，关注机会
```

**分析要点**：
- 宽度比率 > 70%：市场健康，大部分股票走强
- 宽度比率 < 30%：市场危险，大部分股票走弱
- 顶背离：指数上涨但宽度收窄，注意风险
- 底背离：指数下跌但宽度改善，关注机会

### Step 4: 识别极端信号（如有）

**常见信号**：
| 信号 | 含义 | 操作建议 |
|------|------|----------|
| 过热预警 | 涨停数 > 200 或温度 > 90 | 可能短期见顶，谨慎追高 |
| 恐慌底部 | 跌停数 > 100 或温度 < 10 | 可能出现反弹，关注机会 |
| 主力大幅流出 | 主力净流入占比 < -1% | 资金撤退，注意风险 |
| 分化严重 | 涨停 > 50 且 跌停 > 20 | 结构性行情，精选个股 |
| 顶背离 | 指数上涨但宽度收窄 | 可能见顶，注意风险 |
| 底背离 | 指数下跌但宽度改善 | 可能见底，关注机会 |

## 情绪评分说明

### 市场温度计算

市场温度由三个维度加权计算：

| 维度 | 权重 | 说明 |
|------|------|------|
| 涨停数 | 30% | 涨停数越多，温度越高 |
| 涨跌家数 | 40% | 上涨家数占比越高，温度越高 |
| 资金流向 | 30% | 主力流入越多，温度越高 |

### 情绪状态判断

| 温度范围 | 状态 | 市场含义 |
|----------|------|----------|
| 80-100 | 极度亢奋 | 市场过热，注意风险 |
| 60-80 | 偏热 | 情绪高涨，保持警惕 |
| 40-60 | 正常 | 市场平稳，正常操作 |
| 20-40 | 偏冷 | 情绪低迷，可能有机会 |
| 0-20 | 极度恐慌 | 市场恐慌，关注底部 |

## 择时信号说明

### 情绪综合指数计算

综合指数由五个维度加权计算：

| 维度 | 权重（有宽度） | 权重（无宽度） | 说明 |
|------|:-------------:|:-------------:|------|
| 市场温度 | 25% | 35% | 整体市场情绪 |
| 涨跌停比 | 20% | 25% | 极端行情强度 |
| 涨跌家数比 | 15% | 20% | 市场广度 |
| 资金流向 | 15% | 20% | 主力资金方向 |
| 市场宽度 | 25% | 0% | 趋势健康度 |

### 情绪分区判断

| 分区 | 范围 | 市场含义 | 操作建议 |
|------|------|----------|----------|
| 极度低迷 | 0-25 | 市场恐慌 | 关注底部，左侧买入 |
| 低迷 | 25-40 | 情绪偏低 | 分批建仓 |
| 中性 | 40-60 | 正常波动 | 正常操作 |
| 高涨 | 60-75 | 情绪偏高 | 逐步减仓 |
| 极度高涨 | 75-100 | 市场过热 | 降低仓位，等待回调 |

### 左侧择时信号

左侧择时采用逆向策略，在情绪极端时反向操作：

| 信号 | 触发条件 | 操作 |
|------|---------|------|
| 强买入 | 情绪指数 < 20 | 大幅加仓 |
| 买入 | 情绪指数 20-30 | 分批建仓 |
| 持有 | 情绪指数 30-70 | 正常操作 |
| 卖出 | 情绪指数 70-80 | 逐步减仓 |
| 强卖出 | 情绪指数 > 80 | 大幅减仓 |

### 右侧择时信号

右侧择时采用趋势跟踪，在情绪趋势确认后跟随：

| 信号 | 触发条件 | 操作 |
|------|---------|------|
| 买入 | 情绪单日涨幅 > 15% | 跟随买入 |
| 偏多 | 情绪单日涨幅 5-15% | 偏多操作 |
| 持有 | 情绪变化 -5% 到 5% | 观望 |
| 偏空 | 情绪单日跌幅 5-15% | 偏空操作 |
| 卖出 | 情绪单日跌幅 > 15% | 卖出避险 |

## 市场宽度说明

### 宽度比率计算

宽度比率 = 创新高股票数 / (创新高股票数 + 创新低股票数)

| 周期 | 说明 |
|------|------|
| 20日 | 短期趋势 |
| 60日 | 中期趋势 |
| 120日 | 长期趋势 |

### 健康度判断

| 宽度比率 | 健康度 | 市场含义 |
|----------|--------|----------|
| > 70% | 健康 | 大部分股票走强，趋势健康 |
| 50-70% | 正常 | 市场分化，需要精选 |
| 30-50% | 偏弱 | 大部分股票走弱，谨慎操作 |
| < 30% | 危险 | 极端弱势，等待机会 |

### 背离信号

**顶背离**：指数上涨但市场宽度收窄
- 含义：上涨由少数股票推动，整体市场疲弱
- 操作：注意风险，可能即将见顶

**底背离**：指数下跌但市场宽度改善
- 含义：下跌由少数股票拖累，整体市场转强
- 操作：关注机会，可能即将见底

## 数据源

- **涨停股池**：AKShare（免费）
- **炸板股池**：AKShare（免费）
- **大盘资金流向**：AKShare（免费）
- **涨跌家数**：Tushare daily（120 积分）
- **新高新低统计**：AKShare（免费）

## 使用场景

1. **早盘快速了解市场**：运行脚本，查看当日市场情绪
2. **盘中监控**：定期检查市场情绪变化
3. **极端行情识别**：判断是否出现极端情绪信号
4. **趋势健康判断**：分析市场宽度，判断趋势可持续性
5. **结合板块分析**：配合 `industry_trend` skill，了解热点板块和市场情绪的关系

## 注意事项

1. **数据延迟**：部分数据可能有 15 分钟延迟
2. **跌停数据**：第一版跌停数可能为 0（AKShare 接口限制）
3. **非交易日**：返回最近交易日的数据
4. **宽度数据**：需要至少 10 个交易日才能计算趋势

## 深入分析

获取市场情绪后，可以结合其他 skill 进行深入分析：
- **industry_trend**：查看热点板块
- **stock_analysis**：分析龙头个股
- **option_analysis**：通过期权情绪验证市场观点
