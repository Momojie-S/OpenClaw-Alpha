# 市场情绪分析

快速了解 A 股市场整体情绪，识别极端行情信号。

## 使用说明

### 脚本运行

所有脚本需在项目根目录下运行，使用 `uv run --env-file .env` 加载环境变量：

```bash
# 市场情绪分析
uv run --env-file .env python -m skills.market_sentiment.scripts.sentiment_processor.sentiment_processor [--date YYYY-MM-DD]

# 市场宽度分析
uv run --env-file .env python -m skills.market_sentiment.scripts.breadth_processor.breadth_processor [--symbol all|sz50|hs300|zz500] [--days 30]
```

**如果脚本运行失败**：
1. 检查是否在项目根目录
2. 检查 `.env` 文件是否存在且包含 `TUSHARE_TOKEN`
3. 将错误信息记录到进度文件，不要丢失上下文

### 运行记录

**进度文件位置**：`.openclaw_alpha/market-sentiment/{YYYY-MM-DD}/progress.md`

每次运行脚本后，记录：
- 运行时间
- 脚本命令
- 运行结果（成功/失败）
- 关键输出（市场温度、情绪状态）
- 错误信息（如有）

## 分析步骤

### Step 1: 获取市场情绪

**输入**：日期（可选，默认今天）

**动作**：
```bash
uv run --env-file .env python -m skills.market_sentiment.scripts.sentiment_processor.sentiment_processor
```

**输出**：
- 控制台：市场情绪摘要
- 文件：`.openclaw_alpha/market-sentiment/{date}/sentiment.json`（完整数据）

**分析要点**：
- 关注 `temperature` 字段：0-100 分，分数越高越热
- 查看 `status` 字段：极度恐慌/偏冷/正常/偏热/极度亢奋
- 注意 `signals` 字段：极端信号列表

### Step 2: 解读情绪指标

**涨跌停统计**：
- `limit.up`：涨停家数
- `limit.down`：跌停家数
- `limit.break_board`：炸板家数

**涨跌家数**：
- `trend.up`：上涨家数
- `trend.down`：下跌家数
- `trend.flat`：平盘家数

**资金流向**：
- `flow.main_net_inflow`：主力净流入金额
- `flow.main_net_inflow_pct`：主力净流入占比

### Step 3: 分析市场宽度（可选）

**输入**：指数类型、分析天数

**动作**：
```bash
uv run --env-file .env python -m skills.market_sentiment.scripts.breadth_processor.breadth_processor --symbol all --days 30
```

**输出**：
```

【宽度指标】

  ⚠️ 底背离信号：指数下跌但市场宽度改善，关注机会
```

**分析要点**：
- 宽度比率 > 70%：市场健康，大部分股票走强
- 宽度比率 < 30%：市场危险，大部分股票走弱
- 顶背离：指数上涨但宽度收窄，注意风险
- 底背离：指数下跌但宽度改善，关注机会

### Step 4: 识别极端信号（如有）

**常见信号**：
| 信号 | 含义 | 操作建议 |
|------|------|----------|
| 过热预警 | 涨停数 > 200 或温度 > 90 | 可能短期见顶，谨慎追高 |
| 恐慌底部 | 跌停数 > 100 或温度 < 10 | 可能出现反弹，关注机会 |
| 主力大幅流出 | 主力净流入占比 < -1% | 资金撤退，注意风险 |
| 分化严重 | 涨停 > 50 且 跌停 > 20 | 结构性行情，精选个股 |
| 顶背离 | 指数上涨但宽度收窄 | 可能见顶，注意风险 |
| 底背离 | 指数下跌但宽度改善 | 可能见底，关注机会 |

## 情绪评分说明

### 市场温度计算

市场温度由三个维度加权计算：

| 维度 | 权重 | 说明 |
|------|------|------|
| 涨停数 | 30% | 涨停数越多，温度越高 |
| 涨跌家数 | 40% | 上涨家数占比越高，温度越高 |
| 资金流向 | 30% | 主力流入越多，温度越高 |

### 情绪状态判断

| 温度范围 | 状态 | 市场含义 |
|----------|------|----------|
| 80-100 | 极度亢奋 | 市场过热，注意风险 |
| 60-80 | 偏热 | 情绪高涨，保持警惕 |
| 40-60 | 正常 | 市场平稳，正常操作 |
| 20-40 | 偏冷 | 情绪低迷，可能有机会 |
| 0-20 | 极度恐慌 | 市场恐慌，关注底部 |

## 市场宽度说明

### 宽度比率计算

宽度比率 = 创新高股票数 / (创新高股票数 + 创新低股票数)

| 周期 | 说明 |
|------|------|
| 20日 | 短期趋势 |
| 60日 | 中期趋势 |
| 120日 | 长期趋势 |

### 健康度判断

| 宽度比率 | 健康度 | 市场含义 |
|----------|--------|----------|
| > 70% | 健康 | 大部分股票走强，趋势健康 |
| 50-70% | 正常 | 市场分化，需要精选 |
| 30-50% | 偏弱 | 大部分股票走弱，谨慎操作 |
| < 30% | 危险 | 极端弱势，等待机会 |

### 背离信号

**顶背离**：指数上涨但市场宽度收窄
- 含义：上涨由少数股票推动，整体市场疲弱
- 操作：注意风险，可能即将见顶

**底背离**：指数下跌但市场宽度改善
- 含义：下跌由少数股票拖累，整体市场转强
- 操作：关注机会，可能即将见底

## 数据源

- **涨停股池**：AKShare（免费）
- **炸板股池**：AKShare（免费）
- **大盘资金流向**：AKShare（免费）
- **涨跌家数**：Tushare daily（120 积分）
- **新高新低统计**：AKShare（免费）

## 使用场景

1. **早盘快速了解市场**：运行脚本，查看当日市场情绪
2. **盘中监控**：定期检查市场情绪变化
3. **极端行情识别**：判断是否出现极端情绪信号
4. **趋势健康判断**：分析市场宽度，判断趋势可持续性
5. **结合板块分析**：配合 `industry_trend` skill，了解热点板块和市场情绪的关系

## 注意事项

1. **数据延迟**：部分数据可能有 15 分钟延迟
2. **跌停数据**：第一版跌停数可能为 0（AKShare 接口限制）
3. **非交易日**：返回最近交易日的数据
4. **宽度数据**：需要至少 10 个交易日才能计算趋势

## 深入分析

获取市场情绪后，可以结合其他 skill 进行深入分析：
- **industry_trend**：查看热点板块
- **stock_analysis**：分析龙头个股
- **option_analysis**：通过期权情绪验证市场观点
