# 决策记录 - market_sentiment

## 调研

### 数据源选择

**涨跌家数**：
- AKShare：无直接接口
- Tushare：通过 daily 接口统计

**决策**：Tushare（需要 120 积分）

### 市场宽度

**AKShare 无直接接口**：
- 需要获取全市场股票行情
- 自己计算创新高/新低

## 决策

### 市场温度权重

**权重分配**：
- 涨停数 30%：反映赚钱效应
- 涨跌家数 40%：反映市场广度
- 资金流向 30%：反映主力态度

### 择时信号分类

**左侧 vs 右侧**：
- 左侧：逆向操作，极端情绪反向
- 右侧：趋势跟踪，情绪变化跟随

**原因**：不同投资者风格不同

### 为什么分多个 Processor？

**原因**：
1. 不同分析维度独立
2. 用户可能只需要部分功能
3. 减少不必要的数据获取
