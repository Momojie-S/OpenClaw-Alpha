# 需求文档 - market_sentiment

## 一、背景

市场情绪反映投资者整体心理状态，是判断市场短期走势的重要参考。极端情绪往往预示反转机会。

## 二、适用场景

- ✅ 查看当日涨跌停统计
- ✅ 了解主力资金流向
- ✅ 判断市场整体情绪状态
- ✅ 分析市场宽度指标
- ✅ 左侧/右侧择时
- ❌ 个股选择
- ❌ 技术指标分析
- ❌ 长期投资策略

## 三、业务规则

### 3.1 市场温度计算

| 维度 | 权重 |
|------|------|
| 涨停数 | 30% |
| 涨跌家数 | 40% |
| 资金流向 | 30% |

### 3.2 情绪状态判断

| 温度 | 状态 |
|------|------|
| 80-100 | 极度亢奋 |
| 60-80 | 偏热 |
| 40-60 | 正常 |
| 20-40 | 偏冷 |
| 0-20 | 极度恐慌 |

### 3.3 市场宽度

**宽度比率** = 创新高 / (创新高 + 创新低)

| 宽度 | 健康度 |
|------|--------|
| > 70% | 健康 |
| 50-70% | 正常 |
| 30-50% | 偏弱 |
| < 30% | 危险 |

### 3.4 择时信号

**左侧（逆向）**：
- 情绪 < 20：强买入
- 情绪 20-30：买入
- 情绪 > 80：强卖出
- 情绪 70-80：卖出

**右侧（趋势）**：
- 情绪单日涨幅 > 15%：买入
- 情绪单日跌幅 > 15%：卖出
