# 题材炒作分析 - 需求文档

> 版本: v1.0
> 创建时间: 2026-03-10

---

## 一、背景与目标

### 背景
分析题材股（如"小龙虾"概念）时，现有的个股分析 skill 存在以下问题：
1. 技术指标在涨停/跌停时失效（MACD、布林带失真）
2. 缺少题材炒作专用指标（封单、连板、情绪周期）
3. 缺少板块联动分析（龙头识别）
4. 风险提示不够针对性

### 目标
为题材炒作场景提供专用分析能力：
- 判断题材情绪周期
- 识别龙头/跟风/补涨股
- 提供针对性的炒作风险提示

---

## 二、适用场景

### 适用
- 分析题材股、概念股
- 追踪连板股、涨停板
- 判断题材炒作阶段
- 识别龙头股

### 不适用
- 涨停预测
- 实时推送
- 涨停原因分析（无免费 API）

## 三、业务概念

### 3.1 情绪周期

题材炒作会经历不同的情绪周期：

| 周期 | 特征 | 典型表现 |
|------|------|----------|
| **启动** | 涨停数增加、情绪升温 | 涨停家数 > 前日，炸板率 < 30%，昨日涨停正收益 |
| **加速** | 涨停数持续增加、连板高度提升 | 涨停家数持续增加，2 板以上股增多 |
| **高潮** | 涨停数最多、市场情绪最高 | 涨停家数达到近期峰值，炸板率开始上升 |
| **分歧** | 涨停数减少、市场分化 | 炸板率 > 50%，昨日涨停表现分化 |
| **退潮** | 涨停数大幅减少、亏钱效应 | 涨停家数锐减，昨日涨停亏损率高 |

### 3.2 情绪指标

| 指标 | 说明 |
|------|------|
| 涨停家数 | 当日涨停股数量 |
| 炸板率 | 炸板股 / 曾涨停股 |
| 连板高度 | 最高连板数 |
| 昨日涨停表现 | 昨日涨停股今日平均涨跌幅 |

### 3.3 龙头识别

| 类型 | 特征 |
|------|------|
| **龙头** | 连板最高、封板最早、市值较小 |
| **跟风** | 板块内跟随涨停，但封板较晚 |
| **补涨** | 板块滞涨股，后发涨停 |

### 3.4 炒作风险

| 风险 | 判断条件 |
|------|----------|
| 情绪过热 | 涨停家数 > 100，连板高度 > 5 板 |
| 炸板风险 | 炸板率 > 50%，封单减弱 |
| 监管风险 | 连续大涨、偏离基本面 |

## 四、功能需求

### 4.1 情绪周期判断

**输入**：交易日期

**输出**：
- 情绪周期（启动/加速/高潮/分歧/退潮）
- 情绪指标（涨停家数、炸板率、连板高度、昨日涨停表现）
- 周期判断理由

### 4.2 龙头识别

**输入**：概念板块名称

**输出**：
- 板块内涨停股列表
- 龙头股（TOP 1）
- 跟风股列表
- 补涨股列表
- 识别理由（连板高度、封板时间、市值）

### 4.3 炒作风险提示

**输入**：股票代码

**输出**：
- 情绪过热警示（是/否）
- 炸板风险等级（高/中/低）
- 监管风险提示（是/否）
- 涨幅偏离提醒（是/否）

## 五、验收标准

### 5.1 情绪周期判断
- [ ] 能输出当日的情绪周期
- [ ] 情绪指标计算正确
- [ ] 周期判断有明确的理由说明

### 5.2 龙头识别
- [ ] 能识别板块内涨停股
- [ ] 能识别龙头股（连板最高 + 封板最早）
- [ ] 能识别跟风股和补涨股
- [ ] 识别结果有明确的依据说明

### 5.3 炒作风险提示
- [ ] 能判断情绪过热
- [ ] 能评估炸板风险
- [ ] 能提示监管风险

## 六、范围边界

### 做
- 情绪周期判断
- 龙头识别
- 炒作风险提示

### 不做

- 实时推送
- 涨停原因分析（无免费 API）
- 封单金额（AKShare 无此字段）
