# 需求文档 - stock_fund_flow

## 一、背景

个股资金流向反映主力资金（大资金）的动向，是判断股票走势的重要参考。用户需要快速了解单只股票的资金流入流出情况。

## 二、适用场景

- ✅ 查看单只股票的主力资金动向
- ✅ 分析资金流入/流出趋势
- ✅ 判断资金与价格的关联性
- ❌ 板块资金流向（使用 fund_flow_analysis）
- ❌ 实时推送
- ❌ 资金预测

## 三、业务规则

### 3.1 资金流向分类

| 类型 | 说明 |
|------|------|
| 超大单 | 单笔成交额 > 100 万 |
| 大单 | 单笔成交额 20-100 万 |
| 中单 | 单笔成交额 5-20 万 |
| 小单 | 单笔成交额 < 5 万 |

**主力资金** = 超大单 + 大单

### 3.2 趋势判断

| 趋势 | 判断条件 |
|------|----------|
| 持续流入 | 近 N 日 70%+ 净流入 |
| 持续流出 | 近 N 日 70%+ 净流出 |
| 震荡 | 其他情况 |

### 3.3 资金与价格关联

| 关联性 | 判断条件 |
|--------|----------|
| 资金推动 | 70%+ 同向（资金和价格同涨同跌）|
| 资金背离 | 30%- 同向（资金与价格背离）|
| 无明显关联 | 其他 |

### 3.4 信号输出

**买入信号**：
- 连续 3 天流入 + 累计流入超过阈值

**卖出信号**：
- 连续 3 天流出 + 累计流出超过阈值

## 四、验收标准

- [ ] 支持查看今日/近5日/近10日/近20日资金流向
- [ ] 支持趋势判断（持续流入/流出/震荡）
- [ ] 支持资金与价格关联分析
- [ ] 支持 `--signal-only` 输出历史信号
