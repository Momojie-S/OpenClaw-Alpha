# 需求文档 - 融资融券分析

## 一、背景

监控 A 股市场融资融券数据，分析市场杠杆水平和个股异动，辅助判断市场风险状态。

## 二、适用场景

- ✅ 查看市场整体杠杆水平
- ✅ 发现融资/融券异动个股
- ✅ 判断市场风险状态
- ❌ 实时推送
- ❌ 分时监控

## 三、业务规则

### 3.1 杠杆水平判断

| 水平 | 融资余额环比 | 含义 |
|------|-------------|------|
| **高** | > +2% | 杠杆增加，风险累积，注意回调风险 |
| **正常** | -2% ~ +2% | 正常波动，无特殊信号 |
| **低** | < -2% | 杠杆减少，情绪降温，可能见底 |

### 3.2 指标含义

| 指标 | 说明 | 多空含义 |
|------|------|----------|
| 融资余额 | 当前融资未还金额 | 余额↑ = 看多情绪强 |
| 融资买入额 | 当日融资买入金额 | 买入↑ = 新增多头 |
| 融券余量 | 当前融券未还股数 | 余量↑ = 做空增加 |
| 融券卖出量 | 当日融券卖出股数 | 卖出↑ = 新增空头 |

## 四、验收标准

- [ ] 能查看沪深两市融资融券汇总
- [ ] 能查看融资余额 Top N 个股
- [ ] 能查看融券余额 Top N 个股
- [ ] 杠杆水平判断符合上述规则
