# 需求文档 - 限售解禁风险监控

## 一、背景

限售股解禁是重要的市场风险因素。当大量限售股解禁时，会对股价造成抛压。本 skill 帮助监控解禁风险，提前识别潜在风险股票。

## 二、适用场景

- ✅ 查询即将解禁的股票
- ✅ 查询单只股票的解禁排期
- ✅ 筛选高解禁风险股票
- ❌ 已解禁超过1年的历史数据查询

## 三、业务规则

### 3.1 核心指标

**占流通市值比例** = 解禁股市值 / 流通市值

| 比例 | 风险等级 | 说明 |
|------|:--------:|------|
| ≥ 50% | 极高 | 解禁量超过现有流通股的一半 |
| ≥ 30% | 高 | 大量解禁 |
| ≥ 20% | 中 | 中等解禁压力 |
| < 20% | 低 | 解禁压力较小 |

### 3.2 解禁类型风险

| 类型 | 风险程度 | 说明 |
|------|:--------:|------|
| 首发原股东限售股份 | 高 | 成本低，减持意愿强 |
| 定向增发机构配售股份 | 中 | 机构可能有锁定期约定 |
| 股权激励限售股份 | 低 | 员工持股，通常不会大量减持 |
| 其他类型 | 中 | 需具体分析 |

## 四、验收标准

- [ ] 能查询未来 N 天即将解禁的股票
- [ ] 能按占流通市值比例排序
- [ ] 能查询单只股票的解禁排期
- [ ] 能筛选高解禁风险股票
