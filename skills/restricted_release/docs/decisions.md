## 决策

### 核心指标选择

选择"占流通市值比例"作为核心风险指标：
- 解禁市值绝对值不能反映真实影响
- 流通市值小的股票，相同解禁量影响更大
- 比例更能反映解禁对股价的潜在冲击

### 三种模式设计

| 模式 | 用途 |
|------|------|
| upcoming | 日常监控，查看近期解禁 |
| queue | 持仓检查，查询单股解禁排期 |
| high-risk | 风险筛选，找出高风险股票 |

### 排序字段

- `value`：按解禁市值排序（关注大额解禁）
- `ratio`：按占流通市值比例排序（关注高比例解禁）

## 踩坑

### 解禁日期可能变化

- 现象：实际解禁日期与预告不同
- 原因：公司可能申请延期或提前解禁
- 解决：提醒用户关注公司公告

### 解禁 ≠ 减持

- 现象：解禁后股价不跌反涨
- 原因：解禁只是允许减持，不代表一定会减持
- 解决：在说明中强调，结合其他因素分析
