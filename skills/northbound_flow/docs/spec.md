# 需求文档 - northbound_flow

## 一、背景

北向资金（外资通过沪深港通流入 A 股）被视为"聪明钱"，其动向对市场有重要参考价值。用户需要追踪外资流向，辅助投资决策。

## 二、适用场景

- ✅ 查看当日外资净流入情况
- ✅ 了解外资买入/卖出的个股
- ✅ 分析外资近期流向趋势
- ❌ 内资流向
- ❌ 行业资金分布

## 三、业务规则

### 3.1 资金状态判断

| 净流入范围 | 状态 |
|------------|------|
| > 50 亿 | 大幅流入 |
| 10 ~ 50 亿 | 流入 |
| -10 ~ 10 亿 | 平衡 |
| -50 ~ -10 亿 | 流出 |
| < -50 亿 | 大幅流出 |

### 3.2 趋势判断

| 条件 | 趋势 |
|------|------|
| 净流入天数 >= 总天数 * 70% | 持续流入 |
| 净流出天数 >= 总天数 * 70% | 持续流出 |
| 其他 | 震荡 |

### 3.3 信号输出

**买入信号**：
- 连续 3 天流入 + 单日大幅流入（> 50 亿）
- 流出转流入（趋势转折向上）

**卖出信号**：
- 连续 3 天流出 + 单日大幅流出（< -50 亿）
- 流入转流出（趋势转折向下）

## 四、验收标准

- [ ] 支持查看每日净流入（沪股通 + 深股通）
- [ ] 支持查看个股流向 Top N
- [ ] 支持趋势分析
- [ ] 支持 `--signal-only` 输出历史信号
