# 决策记录 - northbound_flow

## 调研

### 数据源选择

**AKShare vs Tushare**：
- AKShare：免费，数据来自东方财富
- Tushare：需要 token，数据更规范

**决策**：AKShare 为主，Tushare 备用

### 数据来源

- 每日净流入：东方财富
- 个股持股：东方财富

## 决策

### 信号逻辑

**买入信号**：
1. 连续 3 天流入 + 大幅流入（> 50 亿）：趋势确认
2. 流出转流入：趋势转折

**卖出信号**：
1. 连续 3 天流出 + 大幅流出（< -50 亿）：趋势确认
2. 流入转流出：趋势转折

**阈值选择**：
- 连续天数：3 天（经验值）
- 大幅流入/流出：50 亿（约占日均成交的 20%）

### 为什么分多个 action？

**原因**：
1. 用户可能只想看每日净流入
2. 不同 action 输出格式不同
3. 减少不必要的数据获取

## 踩坑

### 数据延迟

**现象**：盘中数据有延迟

**原因**：东方财富数据有 15 分钟延迟

**解决**：在 SKILL.md 中说明

### 非交易日

**现象**：非交易日返回空数据

**原因**：接口不返回非交易日数据

**解决**：返回最近交易日的数据

### 单位混淆

**现象**：金额单位不一致

**原因**：不同接口单位不同

**解决**：统一转换为亿元
