# 需求文档 - 资金流向分析

## 一、背景

快速了解市场资金的流动方向，发现资金青睐的热点板块。

## 二、适用场景

- ✅ 查看资金净流入最多的行业/概念
- ✅ 发现资金青睐的热点板块
- ✅ 多时间周期资金动向分析
- ❌ 个股资金流向
- ❌ 实时推送
- ❌ 资金流向预测

## 三、业务规则

### 3.1 分析思路

1. **短期热点**：看今日资金流向
2. **持续关注**：看 5日/10日资金流向
3. **强弱对比**：对比行业和概念
4. **趋势判断**：资金流入 + 涨幅居前 = 强势

### 3.2 时间周期

支持：今日、3日、5日、10日、20日

## 四、验收标准

- [ ] 能查看行业资金流向
- [ ] 能查看概念资金流向
- [ ] 能按净额/涨幅排序
- [ ] 能筛选大额资金流入板块
