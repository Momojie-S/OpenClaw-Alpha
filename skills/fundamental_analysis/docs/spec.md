# 需求文档 - fundamental_analysis

## 一、背景

获取股票的财务指标（PE/PB/ROE 等），评估公司基本面情况和投资价值。

## 二、适用场景

- ✅ 单只股票基本面分析
- ✅ 了解公司财务健康度
- ✅ 评估投资价值
- ❌ 港股/美股
- ❌ 深度财报分析
- ❌ 投资建议

## 三、业务规则

### 3.1 综合评分权重

| 维度 | 权重 | 说明 |
|------|------|------|
| 估值 | 20% | PE/PB 估值水平 |
| 盈利能力 | 30% | ROE 盈利质量 |
| 成长性 | 25% | 营收/利润增长 |
| 财务健康 | 25% | 偿债能力 + 流动性 + 盈利质量 |

### 3.2 估值评级

| 指标 | 低估 | 合理 | 高估 |
|------|------|------|------|
| PE | < 15 | 15-25 | > 25 |
| PB | < 1.5 | 1.5-3 | > 3 |

### 3.3 ROE 评级

| ROE | 评级 |
|-----|------|
| > 15% | 优秀 |
| 10-15% | 良好 |
| 5-10% | 一般 |
| < 5% | 较差 |

### 3.4 成长性评级

| 条件 | 评级 |
|------|------|
| 营收>20% 且 利润>20% | 高增长 |
| 营收>0 且 利润>0 | 稳定增长 |
| 营收<0 或 利润<0 | 下滑 |
| 营收<-10% 或 利润<-10% | 大幅下滑 |

### 3.5 资产负债率评级

| 资产负债率 | 评级 |
|-----------|------|
| < 40% | 健康 |
| 40-60% | 正常 |
| 60-70% | 关注 |
| > 70% | 风险 |

**注**：金融行业（银行、保险）>90% 为正常。

### 3.6 综合评级

| 分数 | 评级 |
|------|------|
| ≥ 80 | 优秀 |
| 65-79 | 良好 |
| 50-64 | 一般 |
| 35-49 | 较差 |
| < 35 | 危险 |

## 四、输入输出

### 输入

- 股票代码（6 位数字）
- 是否包含历史数据（可选）

### 输出

- 估值指标（PE、PB）
- 盈利能力（ROE、EPS）
- 成长性（营收增长、利润增长）
- 财务健康（资产负债率、流动比率）
- 综合评分和评级

## 五、验收标准

- [ ] 能获取 PE/PB/ROE 等核心指标
- [ ] 评分计算符合上述规则
- [ ] 评级判断符合上述规则
