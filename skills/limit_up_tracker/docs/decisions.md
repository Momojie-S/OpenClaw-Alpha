# 决策记录 - 涨停追踪

## 调研

### 数据源选择

| 数据源 | 接口 | 积分要求 | 结论 |
|--------|------|---------|------|
| AKShare | `stock_zt_pool_em()` | 免费 | ✅ 采用 |
| Tushare | `limit_list_d` | 5000 | ⚠️ 积分高，备用 |

### AKShare 接口对比

| 类型 | 接口 | 说明 |
|------|------|------|
| 涨停 | `stock_zt_pool_em()` | 涨停股池 |
| 跌停 | `stock_zt_pool_dtgc_em()` | 跌停股池 |
| 炸板 | `stock_zt_pool_zbgc_em()` | 炸板股池 |
| 昨日涨停 | `stock_zt_pool_previous_em()` | 昨日涨停今日表现 |

## 决策

### 为什么用 AKShare

- 免费、无积分要求
- 接口丰富，覆盖所有涨停类型
- 数据更新及时

### 连板统计设计

- 在 Fetcher 层计算，而非数据源层
- 4 连板以上归为 "4+"（避免过多分类）
