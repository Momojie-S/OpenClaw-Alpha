# 需求文档 - 涨停追踪

## 一、背景

涨停板是 A 股市场最重要的情绪指标之一，追踪涨停股有助于把握市场热点和风险信号。

## 二、适用场景

- ✅ 查看今日涨停股列表
- ✅ 筛选连板龙头
- ✅ 监控炸板风险
- ✅ 追踪昨日涨停今日表现
- ✅ 查看跌停股
- ❌ 涨停预测
- ❌ 实时推送
- ❌ 复杂技术分析

## 三、业务概念

### 3.1 涨停类型

| 类型 | 说明 |
|------|------|
| 涨停 | 当日涨幅触及涨停板 |
| 跌停 | 当日跌幅触及跌停板 |
| 炸板 | 涨停后打开（封板不牢） |
| 昨日涨停 | 昨日涨停股今日表现 |

### 3.2 连板统计

- 首板：当日首次涨停
- N 板：连续 N 个交易日涨停
- 连板数：连续涨停天数

### 3.3 封板强度指标

| 指标 | 含义 | 判断 |
|------|------|------|
| 封板时间 | 首次封板时间 | 越早封板越强 |
| 炸板次数 | 当日打开次数 | 越少越稳 |
| 封板资金 | 流通市值 | 越大封板越稳 |

## 四、验收标准

- [ ] 能获取指定日期的涨停股列表
- [ ] 能按连板数筛选（如 3 连板以上）
- [ ] 能获取炸板股（涨停后打开）
- [ ] 能获取跌停股
- [ ] 能查看昨日涨停今日表现
- [ ] 输出包含封板时间、炸板次数、流通市值等字段
