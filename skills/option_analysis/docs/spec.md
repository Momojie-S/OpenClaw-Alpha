# 需求文档 - 期权分析

## 一、背景

通过期权市场数据判断投资者情绪，辅助宏观层分析。

## 二、适用场景

- ✅ 判断市场多空情绪（PCR + 隐含波动率）
- ✅ 监控期权市场成交/持仓概况
- ❌ 期权定价
- ❌ 复杂策略组合

## 三、业务规则

### 3.1 PCR 情绪判断

| PCR 范围 | 情绪判断 | 含义 |
|----------|----------|------|
| > 1.2 | 极度悲观 | 市场过度恐慌，可能反转向上 |
| 0.8-1.2 | 中性 | 多空平衡 |
| < 0.7 | 极度乐观 | 市场过度亢奋，可能反转向下 |

### 3.2 IV 波动判断

| IV 范围 | 波动判断 | 含义 |
|---------|----------|------|
| > 30% | 高波动 | 市场恐慌，可能见底 |
| 15-30% | 正常 | 市场平稳运行 |
| < 15% | 低波动 | 市场平静，可能变盘 |

## 四、验收标准

- [ ] 能查询期权 PCR
- [ ] 能查询隐含波动率
- [ ] 能输出情绪判断
- [ ] 能查看市场概况
