# 需求文档 - stock_screener

## 一、背景

从 5000+ 只 A 股中快速筛选符合条件的股票，帮助投资者发现投资机会。

## 二、适用场景

- ✅ 按涨幅/换手率/市值等条件筛选
- ✅ 使用预设策略快速选股
- ✅ 寻找符合条件的投资标的
- ❌ 技术指标筛选
- ❌ 财务指标筛选
- ❌ 回测验证

## 三、业务规则

### 3.1 预设策略

| 策略 | 条件 |
|------|------|
| `volume_breakout` | 涨幅 > 3%，换手率 > 5%，成交额 > 2 亿 |
| `pullback` | 涨幅 -5% ~ 0%，换手率 < 3% |
| `leader` | 涨幅 > 5%，成交额 > 10 亿，市值 > 100 亿 |
| `small_active` | 换手率 > 8%，市值 < 50 亿 |
| `blue_chip` | 市值 > 500 亿，换手率 < 2% |

### 3.2 筛选条件

| 参数 | 说明 | 单位 |
|------|------|------|
| 涨幅 | `--change-min`, `--change-max` | % |
| 换手率 | `--turnover-min`, `--turnover-max` | % |
| 成交额 | `--amount-min`, `--amount-max` | 亿元 |
| 市值 | `--cap-min`, `--cap-max` | 亿元 |
| 价格 | `--price-min`, `--price-max` | 元 |

### 3.3 排序选项

| 参数 | 说明 |
|------|------|
| `--sort change` | 按涨跌幅排序（默认） |
| `--sort turnover` | 按换手率排序 |
| `--sort amount` | 按成交额排序 |
| `--sort cap` | 按市值排序 |

## 四、输入输出

### 输入

- 预设策略名称（`--strategy`）
- 或自定义筛选条件

### 输出

- 筛选结果列表（代码、名称、涨幅、换手率、成交额、市值）
- 最多返回 100 只股票

## 五、验收标准

- [ ] 支持预设策略筛选
- [ ] 支持自定义条件筛选
- [ ] 支持多种排序方式
- [ ] 返回结果不超过 100 只
