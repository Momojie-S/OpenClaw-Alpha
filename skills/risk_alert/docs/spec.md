# 需求文档 - 风险监控

## 一、背景

帮助投资者识别股票风险信号，在投资决策前检查潜在风险，避免踩雷。

## 二、适用场景

- ✅ 检查个股风险信号（业绩、价格、资金）
- ✅ 批量扫描多只股票风险
- ✅ 识别业绩暴雷风险
- ✅ 监控连续下跌和资金流出
- ❌ 实时风险预警
- ❌ 量化风险模型
- ❌ 组合风险管理

## 三、业务规则

### 3.1 风险类型

| 类型 | 数据源 | 说明 |
|------|--------|------|
| 业绩风险 | 业绩预告 | 预告类型判断 |
| 价格风险 | 日线数据 | 连续下跌检测 |
| 资金风险 | 资金流向 | 持续流出检测 |

### 3.2 业绩风险判断

基于业绩预告类型：

| 预告类型 | 风险等级 |
|---------|---------|
| 首亏、预减、增亏、续亏 | 高 |
| 不确定、略减 | 中 |
| 预增、略增、扭亏、减亏、续盈 | 无 |

### 3.3 价格风险判断

基于近 N 日日线数据：

| 规则 | 条件 | 风险等级 |
|------|------|---------|
| 大幅下跌 | 单日跌幅 ≥9% | 高 |
| 连续下跌 | 连续 ≥3 天且累计跌幅 ≥10% | 中 |

### 3.4 资金风险判断

基于近 N 日资金流向数据：

| 规则 | 条件 | 风险等级 |
|------|------|---------|
| 大额流出 | 单日主力净流出 ≥5 亿 | 高 |
| 持续流出 | 连续 ≥3 天且累计流出 ≥1 亿 | 中 |

### 3.5 综合评级规则

- **高风险**：存在 ≥1 个高风险信号
- **中风险**：存在 ≥1 个中风险信号，无高风险
- **低风险**：存在 ≥1 个低风险信号，无中高风险
- **正常**：无任何风险信号

## 四、验收标准

- [ ] 能检查单只股票风险
- [ ] 能批量扫描多只股票
- [ ] 能从文件/自选股读取股票列表
- [ ] 风险等级判断符合上述规则
- [ ] 输出风险建议
