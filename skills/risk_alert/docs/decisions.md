## 决策

### 风险类型选择

选择三种风险类型的原因：
- **业绩风险**：业绩暴雷是A股最大风险之一
- **价格风险**：连续下跌反映市场看空
- **资金风险**：主力资金流出预示下跌

暂未实现的风险类型：
- 大股东减持：数据源不稳定
- 股权质押：需补充数据源

### 阈值设置

| 阈值 | 值 | 依据 |
|------|-----|------|
| 大幅下跌 | ≥9% | 接近跌停 |
| 连续下跌天数 | ≥3 天 | 短期趋势确认 |
| 累计跌幅 | ≥10% | 显著下跌 |
| 大额流出 | ≥5 亿 | 大资金撤离 |
| 累计流出 | ≥1 亿 | 持续流出 |

## 踩坑

### AKShare 业绩预告接口不稳定

- 现象：`stock_em_yjyg()` 偶尔超时
- 原因：东方财富接口网络不稳定
- 解决：添加重试机制

### 停牌股票无数据

- 现象：停牌股票获取日线数据失败
- 原因：停牌期间无交易数据
- 解决：捕获异常，返回"数据不足"
