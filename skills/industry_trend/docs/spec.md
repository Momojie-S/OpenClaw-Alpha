# 需求文档 - industry_trend

## 一、背景

产业热度追踪帮助投资者识别市场热点板块，把握板块轮动机会，发现趋势反转信号。

## 二、适用场景

- ✅ 查看当前热门行业/概念板块
- ✅ 发现趋势反转信号
- ✅ 板块轮动分析
- ✅ 拥挤度分析
- ❌ 个股选择
- ❌ 技术指标分析
- ❌ 长期投资策略

## 三、业务规则

### 3.1 热度指数计算

| 维度 | 权重 |
|------|------|
| 涨跌幅 | 30% |
| 换手率 | 25% |
| 资金净流入 | 25% |
| 涨跌家数比 | 20% |

### 3.2 趋势信号

| 信号 | 条件 |
|------|------|
| 加热中 | 热度环比 > +20% 且 涨幅 > 0 且 资金流入 > 0 |
| 降温中 | 热度环比 < -20% 或 (涨幅 < -3% 且 资金流出) |
| 稳定 | 其他 |

### 3.3 拥挤度

| 拥挤度 | 含义 |
|--------|------|
| < 30% | 低拥挤，可以布局 |
| 30-70% | 正常 |
| > 70% | 高拥挤，注意风险 |

## 四、验收标准

- [ ] 支持一级行业、二级行业、三级行业热度分析
- [ ] 支持概念板块热度分析
- [ ] 支持趋势信号判断
- [ ] 支持拥挤度分析
