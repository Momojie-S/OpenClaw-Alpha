# 决策记录 - 龙虎榜追踪

## 调研

### 数据源对比

| 维度 | AKShare | Tushare |
|------|---------|---------|
| 成本 | 免费 | 需要 2000 积分 |
| 稳定性 | 中等 | 高 |
| 字段完整性 | 完整 | 完整 |

**决策**：优先使用 AKShare（免费），Tushare 作为备选。

## 决策

### 买卖方类型识别规则

通过营业部名称关键字匹配：
- 包含"机构专用" → 机构
- 包含"沪股通/深股通" → 北向
- 其他 → 游资

**局限性**：无法识别具体游资席位（如"宁波解放南"）。

### 金额单位

- 龙虎榜净买入：亿元
- 个股持仓变化：万元

与数据源保持一致，不做单位转换。

## 踩坑

### AKShare 接口命名

- 现象：接口名包含 `_em` 后缀
- 原因：表示数据来源东方财富
- 注意：使用时需确认接口名称正确

### 数据延迟

- 现象：龙虎榜数据 T+1 发布
- 原因：交易所披露规则
- 解决：默认返回最近交易日数据
