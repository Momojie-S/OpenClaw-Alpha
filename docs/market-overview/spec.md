# 综合分析 Skill 需求文档

> 版本: v1.0
> 创建时间: 2026-03-08

---

## 一、问题定义

### 1.1 用户痛点

用户想要了解"市场今天怎么样"，需要：
- 分别运行多个 skill（index_analysis、market_sentiment、industry_trend、northbound_flow...）
- 手动汇总各 skill 的结果
- 自己做出综合判断

### 1.2 解决方案

提供一个 **一键式市场分析** skill，自动串联各层次分析，输出综合报告。

---

## 二、目标用户

| 用户类型 | 需求 |
|---------|------|
| 散户投资者 | 快速了解市场整体情况 |
| 投资顾问 | 生成每日市场简报 |
| 研究人员 | 一键获取市场概览数据 |

---

## 三、业务概念

### 3.1 分析框架

市场分析按以下层次展开：

| 层次 | 分析内容 | 数据来源 Skill |
|------|----------|---------------|
| 宏观 | 大盘指数走势、市场温度 | index_analysis |
| 情绪 | 涨跌停、资金流向、情绪温度 | market_sentiment |
| 中观 | 热门板块、资金流向 | industry_trend, fund_flow_analysis |
| 外资 | 北向资金动向 | northbound_flow |

### 3.2 报告结构

综合报告包含以下部分：

1. **市场概览** - 指数涨跌、温度、趋势
2. **情绪分析** - 涨跌停、资金、情绪评分
3. **板块热点** - Top 板块、资金流向
4. **外资动向** - 北向资金、买入卖出 Top
5. **综合结论** - 整体判断、关注点

### 3.3 综合判断规则

| 判断 | 条件 |
|------|------|
| 强势上涨 | 指数涨 > 1% + 情绪 > 60 + 外资流入 |
| 震荡上涨 | 指数涨 0~1% + 情绪正常 |
| 震荡 | 涨跌互现 |
| 震荡下跌 | 指数跌 0~1% + 情绪正常 |
| 弱势下跌 | 指数跌 > 1% + 情绪 < 40 + 外资流出 |

---

## 四、功能需求

### 4.1 核心功能

| 功能 | 描述 |
|------|------|
| 一键分析 | 运行一个命令，获取完整市场报告 |
| 可选深度 | 支持快速版（仅宏观）和完整版（全层次） |
| 日期指定 | 支持分析历史日期 |

### 4.2 输出格式

**控制台**：精简的文本报告（Markdown 格式）

**文件**：JSON 格式的完整数据（`.openclaw_alpha/market_overview/{date}/report.json`）

---

## 五、非功能需求

| 需求 | 说明 |
|------|------|
| 性能 | 快速版 < 10 秒，完整版 < 30 秒 |
| 容错 | 部分 skill 失败不影响整体报告 |
| 可读性 | 控制台输出简洁清晰 |

---

## 六、验收标准

- [ ] 能一键生成市场综合报告
- [ ] 报告包含宏观、情绪、中观、外资四个层次
- [ ] 控制台输出 Markdown 格式
- [ ] 文件输出 JSON 格式
- [ ] 支持快速版和完整版
- [ ] 支持指定日期

---

## 七、范围边界

### 做

- 串联现有 skill，生成综合报告
- 提供一键式命令入口
- 整合分析结果，给出综合判断

### 不做

- 不新增数据获取能力（复用现有 skill）
- 不做实时推送
- 不做预测分析
- 不保存历史报告（用户自行管理）
