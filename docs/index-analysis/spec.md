# 指数分析 Skill - 需求文档

> 版本: v1.0
> 创建时间: 2026-03-07

---

## 一、背景与目标

### 1.1 背景

当前 OpenClaw-Alpha 已具备：
- 宏观层面：市场情绪分析、北向资金追踪
- 中观层面：产业热度追踪
- 微观层面：个股分析、选股筛选、龙虎榜追踪

缺少核心的**指数分析能力**。指数是大盘的核心指标，是投资者每天关注的首要信息。

### 1.2 目标

提供主要指数的行情数据和技术分析，帮助用户：
- 快速了解大盘整体走势
- 判断市场趋势和转折点
- 分析指数强弱对比
- 为板块和个股分析提供宏观背景

### 2.2 行情数据展示

**基础数据**（每个指数）：
- 当前点位
- 涨跌幅
- 涨跌点数
- 开高低收
- 成交量
- 成交额

**对比数据**：
- 与前一日对比
- 与5日/20日均线对比

### 2.3 技术分析指标

**趋势判断**：
| 趋势 | 判断规则 |
|------|----------|
| 强势上涨 | 价格 > MA5 > MA20 且 涨幅 > 1% |
| 震荡上涨 | MA5 > MA20 且 涨幅在 0~1% |
| 震荡 | MA5 与 MA20 接近，涨跌幅 < 1% |
| 震荡下跌 | MA5 < MA20 且 跌幅在 0~1% |
| 弱势下跌 | 价格 < MA5 < MA20 且 跌幅 > 1% |

**支撑压力位**：
- MA5 支撑/压力
- MA20 支撑/压力
- 近期高点/低点

**市场温度**（基于指数表现）：
| 温度 | 条件 |
|------|------|
| 过热 | 3个以上指数涨幅 > 2% |
| 温热 | 2个指数涨幅 > 1% |
| 正常 | 涨跌互现 |
| 偏冷 | 2个指数跌幅 > 1% |
| 过冷 | 3个以上指数跌幅 > 2% |

### 2.4 指数强弱分析

**横向对比**：
- 按涨跌幅排序
- 按成交额排序
- 识别最强/最弱指数

**阶段表现**：
- 5日累计涨跌幅
- 20日累计涨跌幅

**控制台输出**（精简，给大模型分析）：

```json
      "trend": "震荡",

      "ma20": 3310.5
    }
  ]
}
```

## 三、验收标准

### 3.1 功能验收

- [ ] 能获取6个核心指数的当日行情
- [ ] 能计算MA5、MA20均线
- [ ] 能判断每个指数的趋势状态
- [ ] 能计算市场温度
- [ ] 能识别最强/最弱指数
- [ ] 支持指定日期查询

### 3.2 数据验收

- [ ] 数据来源为东方财富（AKShare）
- [ ] 涨跌幅计算正确
- [ ] 均线计算正确（最近5/20个交易日）

### 3.3 输出验收

- [ ] 控制台输出精简，适合大模型分析
- [ ] 文件输出完整，包含历史数据

## 四、范围边界

### 4.1 包含

- 6个核心指数的行情数据
- 基础技术分析（均线、趋势）
- 指数强弱对比
- 市场温度判断

### 4.2 不包含

- 分时图数据
- 高级技术指标（MACD、KDJ、RSI）
- 指数成份股分析
- 指数期货数据
- 海外指数（道琼斯、纳斯达克等）
