# 行业板块实时行情 - stock_board_industry_name_em

> 更新日期：2026-03-03
> 来源地址：https://quote.eastmoney.com/center/boardlist.html#industry_board

## 官方说明

**接口**: stock_board_industry_name_em

**描述**: 东方财富-沪深京板块-行业板块

**限量**: 单次返回当前时刻所有行业板块的实时行情数据

### 输入参数

无

### 输出参数

| 名称 | 类型 | 描述 |
|------|------|------|
| 排名 | int64 | - |
| 板块名称 | object | - |
| 板块代码 | object | - |
| 最新价 | float64 | - |
| 涨跌额 | float64 | - |
| 涨跌幅 | float64 | 注意单位: % |
| 成交量 | float64 | 注意单位: 手 |
| 成交额 | float64 | 注意单位: 元 |
| 振幅 | float64 | 注意单位: % |
| 换手率 | float64 | 注意单位: % |
| 上涨家数 | int64 | 板块内上涨股票数量 |
| 下跌家数 | int64 | 板块内下跌股票数量 |
| 领涨股票 | object | 板块内涨幅最大的股票名称 |
| 领涨股票-涨跌幅 | float64 | 注意单位: % |

### 接口示例

```python
import akshare as ak

stock_board_industry_name_em_df = ak.stock_board_industry_name_em()
print(stock_board_industry_name_em_df)
```

## 额外说明

### 数据来源

- 来源：东方财富网
- 数据类型：实时行情数据
- 更新频率：交易日实时更新

### 注意事项

1. **无参数**：该接口不需要任何输入参数，直接返回所有行业板块数据
2. **实时性**：返回当前时刻的实时行情，收盘后返回收盘数据
3. **板块分类**：使用东方财富的行业分类标准，与申万行业分类有所不同
