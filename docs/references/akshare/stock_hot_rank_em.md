# 股票热度排名 - stock_hot_rank_em

> 更新日期：2026-03-03
> 来源地址：http://guba.eastmoney.com/rank/

---

## 官方说明

**接口**: stock_hot_rank_em

**描述**: 东方财富网站-股票热度

**限量**: 单次返回当前交易日前 100 个股票的人气排名数据

### 输入参数

无

### 输出参数

| 名称 | 类型 | 描述 |
|------|------|------|
| 当前排名 | int64 | 人气排名 |
| 代码 | object | 股票代码（带市场标识） |
| 股票名称 | object | - |
| 最新价 | float64 | - |
| 涨跌额 | float64 | - |
| 涨跌幅 | float64 | 注意单位: % |

### 接口示例

```python
import akshare as ak

stock_hot_rank_em_df = ak.stock_hot_rank_em()
print(stock_hot_rank_em_df)
```

## 额外说明

### 数据来源

- 来源：东方财富股吧
- 数据类型：股票人气热度
- 更新频率：交易日实时更新
- 数据量：固定返回前100名

### 注意事项

1. **固定数量**：只返回排名前100的股票，不能获取更多
2. **市场标识**：代码字段包含市场前缀（SZ/SH），不同于纯数字代码
3. **热度计算**：热度基于东方财富股吧的用户关注度，包括浏览、评论、发帖等行为
4. **应用场景**：适合追踪市场关注度最高的股票，发现短期热点

### 与其他热度数据的区别

| 特性 | stock_hot_rank_em | 其他热度接口 |
|------|-------------------|-------------|
| 数据量 | 固定100条 | 可变 |
| 数据源 | 东方财富股吧 | 各平台不同 |
| 更新频率 | 实时 | 实时/日频 |
| 市场覆盖 | A股全市场 | 因接口而异 |
