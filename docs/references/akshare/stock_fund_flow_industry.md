## 官方说明

**接口**: stock_fund_flow_industry

**描述**: 同花顺-数据中心-资金流向-行业资金流

**限量**: 单次获取指定 symbol 的行业资金流数据

### 输入参数

| 名称 | 类型 | 描述 |
|------|------|------|
| symbol | str | symbol="即时"; choice of {"即时", "3日排行", "5日排行", "10日排行", "20日排行"} |

### 输出参数

| 名称 | 类型 | 描述 |
|------|------|------|
| 序号 | int32 | - |
| 行业 | object | 行业名称 |
| 行业指数 | float64 | 行业指数点位 |
| 行业-涨跌幅 | object | 注意单位: % |
| 流入资金 | float64 | 注意单位: 亿 |
| 流出资金 | float64 | 注意单位: 亿 |
| 净额 | float64 | 注意单位: 亿 |
| 公司家数 | float64 | 行业内上市公司数量 |
| 领涨股 | object | 行业内领涨股票名称 |
| 领涨股-涨跌幅 | object | 注意单位: % |
| 当前价 | float64 | 领涨股当前价格 |

### 接口示例

```python
import akshare as ak

# 获取即时行业资金流
stock_fund_flow_industry_df = ak.stock_fund_flow_industry(symbol="即时")
print(stock_fund_flow_industry_df)

# 获取3日资金流排行
stock_fund_flow_industry_df = ak.stock_fund_flow_industry(symbol="3日排行")
print(stock_fund_flow_industry_df)
```

## 额外说明

### 数据来源

- 来源：同花顺数据中心
- 数据类型：行业资金流向
- 更新频率：交易日实时更新
- 行业分类：同花顺行业分类标准

### 注意事项

1. **多个时间维度**：支持查看即时、3日、5日、10日、20日的资金流向排行
2. **净额计算**：净额 = 流入资金 - 流出资金，正值表示净流入
3. **单位**：流入/流出/净额单位为亿元
4. **行业分类**：使用同花顺的行业分类，与申万、东财分类可能不同

### 与东方财富资金流的区别

| 特性 | stock_fund_flow_industry | 东方财富资金流 |
|------|--------------------------|----------------|
| 数据源 | 同花顺 | 东方财富 |
| 行业分类 | 同花顺分类 | 东财分类 |
| 时间维度 | 即时/3/5/10/20日 | 通常仅当日 |
| 主力细分 | ✅ 流入/流出分开 | ✅ 主力/散户分开 |

### 使用场景

1. **追踪主力资金**：查看哪些行业受到主力资金关注
2. **行业轮动分析**：观察不同时间周期的资金流向变化
3. **市场情绪判断**：通过资金净流入/流出判断市场偏好
4. **领涨股发现**：查看资金流入行业的领涨股票
