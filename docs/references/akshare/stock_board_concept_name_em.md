## 官方说明

**接口**: stock_board_concept_name_em

**描述**: 东方财富网-行情中心-沪深京板块-概念板块

**限量**: 单次返回当前时刻所有概念板块的实时行情数据

### 输入参数

无

### 输出参数

| 名称 | 类型 | 描述 |
|------|------|------|
| 排名 | int64 | - |
| 板块名称 | object | - |
| 板块代码 | object | - |
| 最新价 | float64 | - |
| 涨跌额 | float64 | - |
| 涨跌幅 | float64 | 注意单位: % |
| 成交量 | float64 | 注意单位: 手 |
| 成交额 | float64 | 注意单位: 元 |
| 振幅 | float64 | 注意单位: % |
| 换手率 | float64 | 注意单位: % |
| 上涨家数 | int64 | 板块内上涨股票数量 |
| 下跌家数 | int64 | 板块内下跌股票数量 |
| 领涨股票 | object | 板块内涨幅最大的股票名称 |
| 领涨股票-涨跌幅 | float64 | 注意单位: % |

```python
import akshare as ak
```

## 额外说明

### 数据来源

- 来源：东方财富网
- 数据类型：实时行情数据
- 更新频率：交易日实时更新

### 注意事项

1. **无参数**：该接口不需要任何输入参数，直接返回所有概念板块数据
2. **概念数量**：返回的概念板块数量约为400-500个，会随市场热点变化
3. **板块命名**：概念板块名称可能会包含特殊字符，如"昨日连板_含一字"
4. **实时性**：返回当前时刻的实时行情，收盘后返回收盘数据

### 常见概念板块类型

- 行业概念：如"华为海思概念股"、"苹果概念"
- 主题概念：如"国企改革"、"一带一路"
- 市场概念：如"昨日涨停"、"次新股"
- 技术概念：如"CPO概念"、"MLOps概念"
