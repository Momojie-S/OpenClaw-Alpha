## 官方说明

**接口**: stock_fund_flow_concept

**描述**: 同花顺-数据中心-资金流向-概念资金流

**限量**: 单次获取指定 symbol 的概念资金流数据

### 输入参数

| 名称 | 类型 | 描述 |
|------|------|------|
| symbol | str | symbol="即时"; choice of {"即时", "3日排行", "5日排行", "10日排行", "20日排行"} |

### 输出参数

| 名称 | 类型 | 描述 |
|------|------|------|
| 序号 | int32 | - |
| 行业 | object | 概念名称 |
| 行业指数 | float64 | 概念指数点位 |
| 行业-涨跌幅 | float64 | 注意单位: % |
| 流入资金 | float64 | 注意单位: 亿 |
| 流出资金 | float64 | 注意单位: 亿 |
| 净额 | float64 | 注意单位: 亿 |
| 公司家数 | float64 | 概念内上市公司数量 |
| 领涨股 | object | 概念内领涨股票名称 |
| 领涨股-涨跌幅 | float64 | 注意单位: % |
| 当前价 | float64 | 注意单位: 元 |

### 接口示例

```python
import akshare as ak

# 获取即时概念资金流
stock_fund_flow_concept_df = ak.stock_fund_flow_concept(symbol="即时")
print(stock_fund_flow_concept_df)

# 获取3日资金流排行
stock_fund_flow_concept_df = ak.stock_fund_flow_concept(symbol="3日排行")
print(stock_fund_flow_concept_df)
```

## 额外说明

### 数据来源

- 来源：同花顺数据中心
- 数据类型：概念资金流向
- 更新频率：交易日实时更新
- 概念分类：同花顺概念分类标准

### 注意事项

1. **多个时间维度**：支持查看即时、3日、5日、10日、20日的资金流向排行
2. **净额计算**：净额 = 流入资金 - 流出资金，正值表示净流入
3. **单位**：流入/流出/净额单位为亿元
4. **概念数量**：概念板块数量通常多于行业板块，可能包含数百个概念
5. **动态性**：概念板块会随市场热点变化而增减

### 与行业资金流的区别

| 特性 | stock_fund_flow_concept | stock_fund_flow_industry |
|------|-------------------------|--------------------------|
| 分类维度 | 概念板块 | 行业板块 |
| 板块数量 | 较多（数百个） | 较少（几十个） |
| 稳定性 | 动态变化 | 相对稳定 |
| 应用场景 | 捕捉市场热点 | 行业轮动分析 |

### 使用场景

1. **追踪热点概念**：查看哪些概念受到主力资金关注
2. **主题投资**：发现资金流入的新兴主题
3. **概念轮动**：观察不同概念在不同时间周期的资金流向变化
4. **事件驱动**：结合市场事件分析相关概念的资金反应

### 常见概念类型

- **产业概念**：如"华为海思概念股"、"苹果概念"
- **政策概念**：如"国企改革"、"自贸区"
- **技术概念**：如"AI眼镜"、"基因测序"
- **事件概念**：如"猴痘概念"（突发公共卫生事件）
