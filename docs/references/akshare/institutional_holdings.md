## stock_institute_hold - 机构持股一览表

### 功能
获取指定季度的所有机构持股汇总数据

### 参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|:----:|------|
| symbol | str | 是 | 报告期，格式 `YYYYQ`，Q 为季度 (1/2/3/4)，如 "20241" 表示 2024 年一季报 |

### 字段

| 字段 | 说明 |
|------|------|
| 证券代码 | 股票代码 |
| 证券简称 | 股票名称 |
| 机构数 | 持股机构数量 |
| 机构数变化 | 较上期变化 |
| 持股比例 | 机构持股占比 (%) |
| 持股比例增幅 | 较上期变化 |
| 占流通股比例 | 占流通股比例 (%) |
| 占流通股比例增幅 | 较上期变化 |

### 示例

```python
import akshare as ak

# 获取 2024 年一季报机构持股
df = ak.stock_institute_hold(symbol='20241')
print(df.head())
```

## stock_institute_hold_detail - 机构持股详情

### 功能
获取指定股票在指定季度的机构持股明细

### 参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|:----:|------|
| stock | str | 是 | 股票代码，如 "000001" |
| quarter | str | 是 | 报告期，格式 `YYYYQ`，如 "20241" |

### 字段

| 字段 | 说明 |
|------|------|
| 持股机构类型 | 机构类型（基金/社保/QFII 等） |
| 持股机构代码 | 机构代码 |
| 持股机构简称 | 机构简称 |
| 持股机构全称 | 机构全称 |
| 持股数 | 上期持股数（万股） |
| 最新持股数 | 本期持股数（万股） |
| 持股比例 | 上期持股比例 (%) |
| 最新持股比例 | 本期持股比例 (%) |
| 占流通股比例 | 上期占流通股比例 (%) |
| 最新占流通股比例 | 本期占流通股比例 (%) |
| 持股比例增幅 | 持股比例变化 |
| 占流通股比例增幅 | 占流通股比例变化 |

### 示例

```python
import akshare as ak

# 获取平安银行 2024 年一季报机构持股明细
df = ak.stock_institute_hold_detail(stock='000001', quarter='20241')
print(df.head())
```

### 参数

| 参数 | 类型 | 必填 | 说明 |

| symbol | str | 是 | 股票代码，如 "000001" |

### 字段

| 字段 | 说明 |
|------|------|
| 基金名称 | 基金名称 |
| 基金代码 | 基金代码 |
| 持仓数量 | 持股数量（股） |
| 占流通股比例 | 占流通股比例 (%) |
| 持股市值 | 持股市值（元） |
| 占净值比例 | 占基金净值比例 (%) |
| 截止日期 | 报告截止日期 |

```python

```

## 使用场景

### 机构持仓追踪 Skill

可以基于这些接口开发机构持仓追踪功能：

1. **季度机构持仓汇总** - `stock_institute_hold`
2. **个股机构明细** - `stock_institute_hold_detail`
3. **基金持股分析** - `stock_fund_stock_holder`
4. **股东变动追踪** - `stock_main_stock_holder`, `stock_circulate_stock_holder`

### 数据更新频率

- 季报：每季度结束后约 1 个月
- 中报/年报：披露后更新

### 时滞性说明

持仓数据有 1-3 个月滞后，需要结合实时资金流向（北向资金、主力资金）使用。
