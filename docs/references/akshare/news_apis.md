# AKShare 新闻接口

> 更新日期：2026-03-07
> 来源地址：https://akshare.akfamily.xyz/data/news.html

---

## 可用接口

### 1. stock_info_global_cls - 财联社全球资讯

**推荐度**：⭐⭐⭐

**函数签名**：
```python
ak.stock_info_global_cls(symbol: str = "全部") -> DataFrame
```

**参数**：
| 参数 | 类型 | 说明 |
|------|------|------|
| symbol | str | "全部" 或 "重点" |

**返回字段**：
| 字段 | 说明 |
|------|------|
| 标题 | 新闻标题 |
| 内容 | 新闻正文 |
| 发布日期 | 日期（如 2026-03-07） |
| 发布时间 | 时间（如 16:05:06） |

**特点**：
- 实时更新
- 快速响应
- 覆盖面广

---

### 2. stock_news_em - 东方财富个股新闻

**推荐度**：⭐⭐

**函数签名**：
```python
ak.stock_news_em(symbol: str = "603777") -> DataFrame
```

**参数**：
| 参数 | 类型 | 说明 |
|------|------|------|
| symbol | str | 股票代码（如 "000001"） |

**返回字段**：
| 字段 | 说明 |
|------|------|
| 关键词 | 搜索关键词 |
| 新闻标题 | 标题 |
| 新闻内容 | 正文 |
| 发布时间 | 时间（如 2026-02-28 02:39:05） |
| 文章来源 | 来源媒体 |
| 新闻链接 | 原文链接 |

---

### 3. news_cctv - CCTV 新闻

**推荐度**：⭐

**函数签名**：
```python
ak.news_cctv(date: str = "20240424") -> DataFrame
```

**参数**：
| 参数 | 类型 | 说明 |
|------|------|------|
| date | str | 日期（YYYYMMDD 格式） |

**特点**：
- 内容权威
- 速度较慢（需要爬取多个页面）
- 非交易日可能无数据

---

### 4. news_economic_baidu - 百度经济数据日历

**推荐度**：-

**说明**：这是经济数据发布日历，不是新闻接口。

## 使用建议

1. **优先使用 `stock_info_global_cls`** - 快速、稳定、实时
2. **个股分析用 `stock_news_em`** - 按股票代码获取
3. **避免频繁调用** - AKShare 接口可能有频率限制
