# 申万行业成分构成(分级) - index_member_all

> 更新日期：2026-03-03
> 来源地址：https://tushare.pro/document/2?doc_id=335

---

## 官方说明

**接口**：index_member_all

**描述**：按三级分类提取申万行业成分，可提供某个分类的所有成分，也可按股票代码提取所属分类，参数灵活

**限量**：单次最大2000行，总量不限制

**权限**：用户需 **2000 积分**可调取

### 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|:----:|------|
| l1_code | str | N | 一级行业代码 |
| l2_code | str | N | 二级行业代码 |
| l3_code | str | N | 三级行业代码 |
| ts_code | str | N | 股票代码 |
| is_new | str | N | 是否最新（默认为"Y是"） |

### 输出参数

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|:--------:|------|
| l1_code | str | Y | 一级行业代码 |
| l1_name | str | Y | 一级行业名称 |
| l2_code | str | Y | 二级行业代码 |
| l2_name | str | Y | 二级行业名称 |
| l3_code | str | Y | 三级行业代码 |
| l3_name | str | Y | 三级行业名称 |
| ts_code | str | Y | 成分股票代码 |
| name | str | Y | 成分股票名称 |
| in_date | str | Y | 纳入日期 |
| out_date | str | Y | 剔除日期 |
| is_new | str | Y | 是否最新Y是N否 |

### 接口示例

```python
import tushare as ts

pro = ts.pro_api('your token')

# 获取某个一级行业的所有成分股
df = pro.index_member_all(l1_code='801010')

# 获取某个二级行业的所有成分股
df = pro.index_member_all(l2_code='801016')

# 获取某只股票所属的行业分类
df = pro.index_member_all(ts_code='000001.SZ')
```

## 额外说明

### 积分要求

- **需要 2000 积分**（不是 120 积分）

### 接口名称注意

- 实际接口名是 `index_member_all`（不是 `index_member`）
- 这是一个分级接口，支持按三级分类查询

### 参数灵活使用

1. **按行业查成分股**：
   - 使用 `l1_code` 获取一级行业的所有成分股
   - 使用 `l2_code` 获取二级行业的所有成分股
   - 使用 `l3_code` 获取三级行业的所有成分股

2. **按股票查所属行业**：
   - 使用 `ts_code` 获取某只股票所属的所有行业分类

3. **获取最新成分股**：
   - `is_new='Y'` 只返回当前仍在成分中的股票
   - `is_new='N'` 返回已被剔除的股票

### 注意事项

1. **单次限制**：单次最大返回2000行
2. **行业变更**：通过 `in_date` 和 `out_date` 可以追踪成分股的变更历史
3. **多级行业**：一只股票可能同时属于多个层级的行业分类
4. **时间维度**：同一只股票在不同时间可能属于不同行业

### 使用场景

- 查询某个行业的所有成分股
- 查询某只股票的行业归属
- 追踪行业成分股的历史变更
- 构建行业指数组合
- 行业轮动分析

### 与 index_classify 的关系

1. 先使用 `index_classify` 获取行业代码
2. 再使用 `index_member_all` 查询该行业的成分股
