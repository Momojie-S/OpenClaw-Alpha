# 个股分析 - 设计文档

## 技术选型

### 数据源选择

| 数据源 | 行情数据 | 估值数据 | 积分要求 | 选择 |
|--------|----------|----------|----------|------|
| Tushare | ✅ daily | ✅ daily_basic (2000积分) | 120起 | 优先（行情） |
| AKShare | ✅ stock_zh_a_hist | ✅ 多个接口 | 无 | 备选 |

**设计决策**：
- 第一版只提供行情数据（价格、成交、涨跌）
- 估值数据（PE、PB）需要 2000 积分，放到后续版本
- 优先使用 Tushare（积分要求低），AKShare 作为备选

## 架构设计

### 模块结构

```
skills/stock_analysis/
├── SKILL.md                    # Skill 文档
└── scripts/
    ├── __init__.py
    ├── stock_fetcher/          # 个股数据获取
    │   ├── __init__.py

        └── stock_analysis_processor.py
```

### 数据流

```
用户输入（股票标识 + 日期）
    ↓
StockFetcher
    ├── 股票代码解析（名称→代码）
    └── 获取行情数据
    ↓
StockAnalysisProcessor
    ├── 计算分析结论
    └── 输出结构化结果
```

## 接口契约

### StockFetcher

**输入参数**：
- `identifier`: 股票代码或名称（如 "000001" 或 "平安银行"）
- `date`: 日期（YYYY-MM-DD），默认最近交易日

**输出格式**：
```python
{
    "code": "000001.SZ",
    "name": "平安银行",
    "date": "2026-03-06",
    "price": {
        "open": 11.20,
        "high": 11.55,
        "low": 11.18,
        "close": 11.50,
        "pre_close": 11.25,
        "change": 0.25,
        "pct_change": 2.22
    },
    "volume": {
        "volume": 123456789,      # 股
        "amount": 1420000000,     # 元
        "turnover_rate": 5.2      # %
    },
    "market_cap": {
        "total": 223000000000,    # 元
        "circulation": 195000000000  # 元
    }
}
```

### StockAnalysisProcessor

**输入参数**：
- `identifier`: 股票代码或名称
- `date`: 日期，默认最近交易日

**输出格式**：
```python
{
    "stock": {...},       # 基本信息
    "price": {...},       # 价格数据
    "volume": {...},      # 成交数据
    "market_cap": {...},  # 市值数据
    "analysis": {         # 分析结论
        "activity": "较活跃",
        "trend": "上涨"
    }
}
```

## 关键算法

### 股票代码解析

1. 判断输入是代码还是名称
2. 如果是名称，通过股票列表查询代码
3. 处理代码格式（补充交易所后缀）

### 活跃度判断

| 换手率 | 结论 |
|--------|------|
| > 10% | 非常活跃 |
| > 5% | 较活跃 |
| > 2% | 正常 |
| <= 2% | 不活跃 |

### 涨跌判断

| 涨跌幅 | 结论 |
|--------|------|
| > 5% | 大涨 |
| > 2% | 上涨 |
| < -5% | 大跌 |
| < -2% | 下跌 |
| 其他 | 持平 |

## 数据源实现

### Tushare 实现

**接口**：
- `daily`: 获取日线行情（价格、成交量）
- 积分要求：120

**代码格式**：
- 需要 `ts_code` 格式：`000001.SZ` / `600000.SH`

### AKShare 实现

**接口**：
- `stock_zh_a_hist`: 获取 A 股历史行情
- 无积分要求

**代码格式**：
- 6 位代码：`000001` / `600000`

## 边界处理

- 股票不存在：返回错误信息
- 日期无交易：返回最近交易日数据
- 停牌股票：提示停牌状态
- 网络错误：重试 3 次
