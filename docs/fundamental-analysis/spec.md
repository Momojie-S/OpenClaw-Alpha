> 创建日期：2026-03-07

## 一、业务背景

### 1.1 问题描述

投资者在选择股票时，除了技术面分析，还需要了解公司的基本面情况，包括：

- **估值是否合理**（PE、PB 指标）
- **盈利能力如何**（ROE、净利润率）
- **财务健康度**（资产负债率、现金流）
- **成长性**（营收/利润增长率）

目前缺乏一个统一的工具来快速获取这些关键指标。

### 1.2 目标用户

- 个人投资者：快速了解个股基本面
- 投资顾问：为客户提供基本面分析支持

### 1.3 核心价值

- **快速**：一次查询获取所有关键指标
- **全面**：覆盖估值、盈利、成长、财务健康
- **直观**：提供评级和趋势判断

## 二、业务概念

### 2.1 核心指标体系

| 维度 | 指标 | 说明 | 重要性 |
|------|------|------|:------:|
| **估值** | PE (TTM) | 市盈率（滚动12个月） | ⭐⭐⭐ |
| | PB | 市净率 | ⭐⭐⭐ |
| | 总市值 | 公司市场价值 | ⭐⭐ |
| **盈利** | ROE | 净资产收益率 | ⭐⭐⭐ |
| | EPS | 每股收益 | ⭐⭐ |
| | 销售净利率 | 净利润/营业收入 | ⭐⭐ |
| | 销售毛利率 | 毛利/营业收入 | ⭐⭐ |
| **成长** | 营收增长率 | 同比增长 | ⭐⭐⭐ |
| | 净利润增长率 | 同比增长 | ⭐⭐⭐ |
| **财务健康** | 资产负债率 | 负债/总资产 | ⭐⭐⭐ |
| | 流动比率 | 流动资产/流动负债 | ⭐⭐ |
| | 每股经营现金流 | 现金流健康度 | ⭐⭐ |

### 2.2 评级标准（参考）

#### 估值评级

| 指标 | 低估 | 合理 | 高估 |
|------|------|------|------|
| PE | < 15 | 15-25 | > 25 |
| PB | < 1.5 | 1.5-3 | > 3 |

**注**：不同行业标准不同，以上为通用参考。

#### ROE 评级

| ROE | 评级 |
|-----|------|
| > 15% | 优秀 |
| 10-15% | 良好 |
| 5-10% | 一般 |
| < 5% | 较差 |

#### 资产负债率评级

| 资产负债率 | 评级 |
|-----------|------|
| < 40% | 健康 |
| 40-60% | 正常 |
| 60-70% | 关注 |
| > 70% | 风险 |

## 三、使用场景

### 场景 1：单只股票基本面快照

**用户需求**：快速了解某只股票的基本面情况

**输入**：股票代码（如 `000001`）

**输出**：
- 最新一期财务指标（估值、盈利、成长、财务健康）
- 指标评级（低估/合理/高估）
- 关键提示（如"ROE 连续下降"）

### 场景 2：基本面筛选

**用户需求**：按基本面条件筛选股票

**输入**：
- 筛选条件（如 PE < 20, ROE > 10%）
- 行业（可选）
- 排序字段（如按 ROE 降序）
- Top N（如返回前 10 只）

**输出**：符合条件且排序后的股票列表

### 场景 3：历史趋势分析

**用户需求**：查看某只股票基本面指标的历史变化

**输入**：
- 股票代码
- 时间范围（如近一年）
- 指标（如 PE、ROE）

**输出**：
- 指标时间序列数据
- 趋势判断（上升/下降/稳定）

## 四、验收标准

### 4.1 功能验收

- [ ] 能获取单只股票的最新财务指标（PE、PB、ROE 等 10+ 个指标）
- [ ] 能提供指标评级（估值、ROE、负债率）
- [ ] 能获取估值指标的历史趋势（PE、PB 近一年）
- [ ] 能按基本面条件筛选股票（PE、ROE、市值等）
- [ ] 能按指定指标排序并返回 Top N

### 4.2 质量验收

- [ ] 数据来源稳定（AKShare 接口）
- [ ] 响应时间 < 5 秒（单只股票）
- [ ] 错误提示友好（无效代码、无数据等）

## 五、边界与约束

### 5.1 做什么

- ✅ 获取 A 股上市公司财务指标
- ✅ 提供指标评级和趋势判断
- ✅ 按基本面条件筛选

### 5.2 不做什么

- ❌ 港股、美股基本面（本期不支持）
- ❌ 深度财报分析（如杜邦分析、现金流拆解）
- ❌ 行业对比分析（后续扩展）
- ❌ 投资建议（仅提供数据和评级）

### 5.3 数据约束

- 财务数据按报告期更新（季报/年报）
- 估值数据每日更新
- 部分新股可能缺少历史数据
