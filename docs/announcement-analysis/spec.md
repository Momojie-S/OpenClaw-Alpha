# 公告解读 Skill - 需求文档

## 一、问题背景

投资者需要及时了解上市公司公告，包括重大事项、财务报告、风险提示等。手动查看公告效率低，需要工具帮助：

- 快速获取当日或指定日期的公告列表
- 按类型筛选（重大事项、财务报告、风险提示等）
- 按股票代码或关键词搜索
- 识别重要公告类型

## 二、目标用户

- 关注公告信息的投资者
- 需要快速了解市场动态的分析师
- 追踪特定股票公告的持有人

## 三、业务概念

### 3.1 公告类型

| 类型 | 说明 | 重要性 |
|------|------|--------|
| 重大事项 | 重大投资、收购、合作等 | ⭐⭐⭐ |
| 财务报告 | 季报、半年报、年报 | ⭐⭐⭐ |
| 风险提示 | 退市风险、业绩预警等 | ⭐⭐⭐ |
| 资产重组 | 重组、并购、剥离等 | ⭐⭐ |
| 融资公告 | 增发、配股、发债等 | ⭐⭐ |
| 持股变动 | 大股东增减持 | ⭐⭐ |
| 信息变更 | 公司更名、变更经营范围等 | ⭐ |

### 3.2 公告信息

每条公告包含：
- 股票代码、名称
- 公告标题
- 公告类型
- 发布日期
- 公告链接（可查看原文）

## 四、功能需求

### 4.1 公告列表

**输入**：
- 日期（默认今日）
- 公告类型（默认全部）
- 股票代码（可选）
- 关键词（可选）

**输出**：
- 公告列表（按重要性排序）
- 最多返回 Top N（默认 20）

### 4.2 按类型筛选

支持按公告类型筛选：
- 全部
- 重大事项
- 财务报告
- 融资公告
- 风险提示
- 资产重组
- 信息变更
- 持股变动

### 4.3 按股票搜索

支持按股票代码或名称搜索该股票的公告。

### 4.4 关键词搜索

支持在公告标题中搜索关键词。

## 五、验收标准

- [ ] 能获取指定日期的公告列表
- [ ] 能按公告类型筛选
- [ ] 能按股票代码筛选
- [ ] 能按关键词搜索
- [ ] 输出格式清晰易读

## 六、不做

- 公告正文解析（需要 LLM，复杂度高）
- 公告历史存档（数据量大）
- 公告订阅推送（需要推送机制）
