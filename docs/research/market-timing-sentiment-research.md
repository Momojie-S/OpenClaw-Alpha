# 市场择时与情绪因子研究

> 研究日期：2026-03-08
> 研究目的：探索基于情绪因子的市场择时方法，增强 OpenClaw-Alpha 的择时能力

---

## 背景

### 当前能力

OpenClaw-Alpha 已具备的情绪相关指标：

| Skill | 指标 | 数据源 |
|-------|------|--------|
| market_sentiment | 涨跌停、换手率、市场温度 | AKShare |
| margin_trading | 融资余额、融券余额 | AKShare |
| northbound_flow | 北向资金净流入 | AKShare |
| option_analysis | PCR（看跌看涨比率） | AKShare |
| index_analysis | 指数估值（PE/PB） | Tushare |

### 问题

1. **指标分散**：各情绪指标分散在不同 skill 中
2. **缺乏综合**：没有统一的情绪综合指数
3. **择时信号缺失**：没有基于情绪的择时信号

---

## 情绪择时方法

### 1. 左侧择时（逆向）

**原理**：在情绪极端时逆向操作

| 信号 | 触发条件 | 操作 |
|------|---------|------|
| 极度悲观 | 情绪指数接近 0 | 买入 |
| 极度乐观 | 情绪指数接近 100（A股存量博弈下 60-65） | 卖出 |

**特点**：
- 适合长期投资者
- 可能抄底抄在半山腰
- 需要耐心等待

### 2. 右侧择时（趋势）

**原理**：在情绪趋势确认后跟随

| 信号 | 触发条件 | 操作 |
|------|---------|------|
| 情绪反转向上 | 1-2 日内情绪涨幅 > 10，或超过近 5 日均值 +2σ | 买入 |
| 情绪反转向下 | 1-2 日内情绪跌幅 > 10，或超过近 5 日均值 -2σ | 卖出 |

**特点**：
- 反应较慢，但更安全
- 适合趋势跟踪者
- 可能错过部分收益

### 3. 情绪分区策略

**原理**：根据情绪状态调整配置风格

| 情绪状态 | 推荐配置 | 原因 |
|---------|---------|------|
| 低迷区 | 大盘价值、高股息 | 防御为主，估值安全 |
| 中性区 | 均衡配置 | 风险收益平衡 |
| 高涨区 | 成长、科技、消费 | 追逐高弹性 |

## 情绪衡量指标

### 定性指标

- **投资社区评论**：普遍唱空时可能接近底部
- **媒体报道**：鲜有人看多时可能接近顶部

### 定量指标

| 指标 | 说明 | 极端值含义 |
|------|------|-----------|
| 偏股型基金发行规模 | 发行清淡时预示底部 | 低 → 买点 |
| 两市成交量/换手率 | 磨底期萎缩，过热后回调 | 低 → 买点，高 → 风险 |
| 股债性价比（风险溢价） | PE 倒数 - 10Y 国债收益率 | 高 → 买点 |
| 涨停股数量占比 | 市场活跃度 | 极高 → 过热 |
| 创近一年新高股票占比 | 市场强度 | 极低 → 底部 |
| 融资余额 | 杠杆资金 | 快速下降 → 底部信号 |
| 融资买入额占比 | 杠杆活跃度 | 极高 → 过热 |
| 北向资金净流入 | 外资态度 | 大幅流出 → 可能见底 |
| 股指期货升贴水 | 市场预期 | 深度贴水 → 底部信号 |

## OpenClaw-Alpha 增强方案

### 方案 1：情绪综合指数（推荐）

**设计**：
1. 整合现有情绪指标
2. 标准化处理后加权合成
3. 生成择时信号

**指标权重建议**：

| 指标 | 权重 | 来源 |
|------|:----:|------|
| 涨跌停比 | 15% | market_sentiment |
| 换手率 | 10% | market_sentiment |
| 融资余额变化 | 15% | margin_trading |
| 北向资金流向 | 15% | northbound_flow |
| 期权 PCR | 10% | option_analysis |
| 市场温度 | 20% | market_sentiment |
| 指数估值分位 | 15% | index_analysis |

**输出**：
- 情绪综合指数（0-100）
- 择时信号（买入/持有/卖出）
- 情绪分区（低迷/中性/高涨）

### 方案 2：股债性价比指标

**设计**：
1. 计算沪深300 PE 倒数
2. 减去 10 年期国债收益率
3. 与历史分位数比较

**信号**：
- 风险溢价 > 75%分位 → 买入信号
- 风险溢价 < 25%分位 → 卖出信号

### 方案 3：情绪择时 Processor

**设计**：
- 读取各 skill 的输出数据
- 综合计算情绪指数
- 生成择时报告

**命令示例**：
```bash
uv run --env-file .env python skills/market_sentiment/scripts/timing_processor/timing_processor.py
```

**输出示例**：
```json
{
  "sentiment_index": 35.2,
  "zone": "低迷区",
  "signals": {
    "left": "买入",
    "right": "持有"
  },
  "recommendation": "当前情绪低迷，左侧买入信号出现，适合分批建仓",
  "components": {
    "涨跌停比": 0.8,
    "融资余额变化": -2.5,
    "北向资金": -15.2
  }
}
```

## 注意事项

1. **情绪指标滞后性**：部分指标（如基金发行）滞后 1-2 周
2. **极端行情失效**：情绪指标在极端行情下可能失效
3. **需要历史数据**：计算分位数需要足够历史数据
4. **信号噪音**：短期情绪波动可能产生噪音信号
