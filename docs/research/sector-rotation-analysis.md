# 行业轮动分析研究

> 研究日期：2026-03-08
> 研究目的：探索行业轮动分析方法，为分析框架补充新维度

---

## 核心概念

### 三个重要标尺

行业轮动策略的三个关键维度：

| 标尺 | 说明 | 理想状态 |
|------|------|----------|
| **景气度** | 行业盈利能力和成长性 | 高景气度 |
| **趋势** | 价格走势强度 | 高趋势（上涨） |
| **拥挤度** | 市场参与程度 | 低拥挤度 |

**黄金组合**：高景气度 + 高趋势 + 低拥挤度 = 最优配置

## 分析方法

### 1. 宏观经济周期分析

经济周期与行业表现的关系：

| 阶段 | 经济状态 | 优势板块 |
|------|----------|----------|
| 1 | 失速 | 下游消费 |
| 2 | 复苏 | 中游制造 |
| 3 | 过热 | 上游原料 |
| 4 | 顶峰 | 上游原料 |
| 5 | 滞胀 | 下游消费 + 中游制造 |
| 6 | 萧条 | 支持服务 + 中游制造 |

### 2. 行业景气度分析

**核心指标**：
- ROE（净资产收益率）趋势
- 行业利润增速
- 分析师评级变化

### 3. 资金流向分析

**观察维度**：
- 主力资金净流入/流出
- 北向资金偏好
- 融资融券余额变化

### 4. 估值水平分析

**核心指标**：
- PE（市盈率）分位数
- PB（市净率）分位数
- 股息率

## 量化指标

### 价量因子

| 指标 | 说明 | 用途 |
|------|------|------|
| 涨跌幅 | 短/中/长期价格变化 | 动量判断 |
| 成交量 | 市场活跃度 | 趋势确认 |
| 成交额 | 资金参与程度 | 热度评估 |
| 换手率 | 流通性 | 拥挤度判断 |

### 动量指标

| 指标 | 计算方式 | 用途 |
|------|----------|------|
| 动量得分 | 不同周期收益率加权 | 强势行业选择 |
| RSI | 相对强弱指数 | 超买超卖判断 |

### 景气度指标

| 指标 | 说明 | 数据来源 |
|------|------|----------|
| ROE | 净资产收益率 | 财务数据 |
| 利润增速 | 净利润同比 | 财务数据 |
| 分析师评级 | 买入/增持比例 | 研报数据 |

### 拥挤度指标

| 指标 | 说明 | 计算 |
|------|------|------|
| 换手率分位 | 相对历史位置 | 历史百分位 |
| 成交额占比 | 板块成交额/全市场 | 比值 |
| 涨停股占比 | 板块内涨停股数量 | 比值 |

## 策略框架

### 动量策略

```
强者恒强逻辑：
1. 计算各行业动量得分
2. 选择 Top N 强势行业
3. 在强势行业中选择优质个股
```

### 均值回归策略

```
弱者反转逻辑：
1. 识别超跌行业（RSI < 30）
2. 等待反转信号
3. 左侧布局或右侧确认
```

### 综合评分策略

```
多因子模型：
1. 景气度因子（ROE趋势、利润增速）
2. 趋势因子（动量得分、技术指标）
3. 拥挤度因子（换手率分位、成交额占比）
4. 估值因子（PE分位、PB分位）
5. 资金因子（主力资金、北向资金）

综合得分 = Σ（因子值 × 因子权重）
```
