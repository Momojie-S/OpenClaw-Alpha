# 智能定投策略研究

> 研究日期：2026-03-08
> 研究目的：设计基于估值指标的智能定投方案，实现"低估值多投、高估值少投"的自动化投资策略

---

## 背景与目标

### 定投的本质

定期定额投资（Dollar Cost Averaging，DCA）是一种长期投资策略：

- **优势**：降低择时风险、平滑成本、纪律性强
- **劣势**：高位也定额买入，错过低位加仓机会

### 智能定投的价值

**传统定投**：每月固定投入 1000 元

**智能定投**：
- 估值低（PE 分位 20%）→ 投入 2000 元（2 倍）
- 估值中（PE 分位 50%）→ 投入 1000 元（1 倍）
- 估值高（PE 分位 80%）→ 投入 500 元（0.5 倍）

**效果**：低位多投、高位少投，进一步降低平均成本

## 估值指标体系

### 核心指标

| 指标 | 说明 | 数据源 | 已实现 |
|------|------|--------|--------|
| 股债性价比 | E/P - 10Y 国债收益率 | market_sentiment | ✅ |
| 宽基指数 PE 分位 | 沪深300/中证500 PE 分位 | index_analysis | ✅ |
| 市场情绪指数 | 综合情绪评分 | market_sentiment | ✅ |

### 股债性价比（FED 模型）

**公式**：`股债性价比 = 沪深300 E/P - 10Y 国债收益率`

**判断标准**：

| 价值范围 | 市场状态 | 定投倍数 |
|----------|----------|----------|
| > 3% | 极度低估 | 2.0x |
| 2% - 3% | 低估 | 1.5x |
| 0% - 2% | 合理 | 1.0x |
| -1% - 0% | 高估 | 0.5x |
| < -1% | 极度高估 | 0x（暂停） |

### PE 分位法

**公式**：`PE 分位 = 当前 PE 在历史 PE 中的百分位`

**判断标准**：

| PE 分位 | 市场状态 | 定投倍数 |
|---------|----------|----------|
| 0% - 20% | 极度低估 | 2.0x |
| 20% - 40% | 低估 | 1.5x |
| 40% - 60% | 合理 | 1.0x |
| 60% - 80% | 高估 | 0.5x |
| 80% - 100% | 极度高估 | 0x（暂停） |

## 策略设计

### 方案一：股债性价比驱动

**适用**：宽基指数定投（沪深300、中证500）

**逻辑**：
1. 获取当前股债性价比
2. 根据性价比区间确定倍数
3. 计算本期定投金额

**示例**：
```
基准金额：1000 元/月
当前股债性价比：2.5%
判断：低估 → 倍数 1.5x
本期定投：1000 × 1.5 = 1500 元
```

### 方案二：多因子综合评分

**适用**：更精细的定投决策

**因子体系**：

| 因子 | 权重 | 数据源 |
|------|------|--------|
| 股债性价比 | 40% | market_sentiment |
| 沪深300 PE 分位 | 20% | index_analysis |
| 中证500 PE 分位 | 20% | index_analysis |
| 市场情绪指数 | 20% | market_sentiment |

**综合评分**：
```
score = Σ (因子得分 × 权重)
```

**定投倍数**：
```
score < 30 → 2.0x
30 ≤ score < 45 → 1.5x
45 ≤ score < 55 → 1.0x
55 ≤ score < 70 → 0.5x
score ≥ 70 → 0x
```

## 功能实现方案

### 新增 Skill：smart_dip

**定位**：智能定投策略计算器

**核心能力**：
1. 股债性价比驱动定投
2. 多因子综合评分定投
3. 历史回测（定投效果验证）

**复用能力**：
- `equity_bond_ratio_processor` - 股债性价比

- `timing_processor` - 市场情绪

**输入**：
- `--base-amount`：基准金额（默认 1000）

```json

  "date": "2026-03-08",

  "base_amount": 1000,
  "valuation": {
    "equity_bond_ratio": 2.5,

  "recommendation": {
```

## 风险提示

1. **估值指标滞后性**：PE 分位基于历史数据，可能无法及时反映市场变化
2. **极端行情**：市场暴跌时可能错过加仓机会（因子尚未反映）
3. **策略失效**：历史有效不代表未来有效
4. **数据质量**：依赖外部数据源，可能存在延迟或错误
