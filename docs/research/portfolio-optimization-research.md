# 投资组合优化研究

> 研究日期：2026-03-08
> 研究目的：探索投资组合优化方法，为 portfolio_analysis skill 增强能力

---

## 背景

现有的 portfolio_analysis skill 提供：
- 持仓结构分析
- 盈亏计算
- 基本的风险指标（持仓集中度等）

可以增强的方向：
- **风险分散分析** - 持仓相关性、行业/风格暴露
- **优化建议** - 基于风险收益的仓位调整建议

## 经典理论

### 马科维茨均值-方差模型

**核心思想**：在给定风险水平下最大化收益，或在给定收益水平下最小化风险。

**有效前沿**：风险-收益平面上最优投资组合的集合。

**输入**：
- 各资产预期收益率
- 各资产收益率协方差矩阵

**输出**：
- 各资产最优权重

### 风险平价模型

**核心思想**：各资产对组合总风险的贡献相等。

**风险贡献**：
```
RC_i = w_i * (Σw)_i / σ_p
```
其中：
- w_i = 资产 i 权重
- Σ = 协方差矩阵
- σ_p = 组合标准差
- (Σw)_i = 协方差矩阵第 i 行与权重向量的乘积

**优势**：
- 不依赖收益预测，只基于风险
- 更稳健，避免极端集中

### 最小方差组合

**核心思想**：在所有可能组合中选择风险最小的组合。

**求解**：
```
min w'Σw
s.t. Σw_i = 1, w_i >= 0
```

## 实现思路

### 数据需求

| 数据 | 来源 | 难度 |
|------|------|------|
| 股票历史价格 | AKShare | ✅ 已有 |
| 行业分类 | AKShare | ✅ 已有 |
| 股票相关性 | 计算 | ✅ 可实现 |
| 市场因子 | 计算 | 中等 |

### 分析模块

| 模块 | 功能 | 优先级 |
|------|------|--------|
| **持仓相关性分析** | 计算持仓股票的相关系数矩阵 | P1 |
| **风险贡献分解** | 各股票对组合风险的贡献 | P1 |
| **有效前沿绘制** | 可视化最优投资组合 | P2 |
| **优化建议** | 给出仓位调整建议 | P2 |

### 持仓相关性分析

**输入**：持仓股票代码列表
**输出**：
- 相关系数矩阵（热力图）
- 高相关股票对（相关性 > 0.7）
- 分散化评分（基于平均相关性）

**示例输出**：
```json
{
  "high_correlation_pairs": [
    {"stock1": "600519", "stock2": "000858", "correlation": 0.85}
  ],
  "diversification_score": 0.65,
  "suggestion": "持仓中有 3 对高相关股票，建议减少同行业持仓"
}
```

### 风险贡献分解

**输入**：持仓股票代码 + 当前权重
**输出**：
- 各股票风险贡献占比
- 风险集中度（Top 3 风险贡献占比）
- 风险平价建议权重

**示例输出**：
```json
{
  "risk_contributions": [
    {"stock": "600519", "weight": 0.3, "risk_contribution": 0.45},
    {"stock": "000858", "weight": 0.25, "risk_contribution": 0.35}
  ],
  "risk_concentration": 0.80,
  "risk_parity_weights": [
    {"stock": "600519", "suggested_weight": 0.18},
    {"stock": "000858", "suggested_weight": 0.20}
  ]
}
```

## 实现挑战

| 挑战 | 说明 | 解决方案 |
|------|------|----------|
| 数据量 | 需要足够长的历史数据 | 至少 60 个交易日 |
| 计算复杂度 | 协方差矩阵计算 | 使用 NumPy 向量化 |
| 新股处理 | 上市时间短的新股 | 标记为"数据不足" |
| 市场环境变化 | 历史相关性不稳定 | 使用滚动窗口计算 |
