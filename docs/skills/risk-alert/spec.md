# 风险监控 Skill 需求文档

> 版本: v1.0
> 创建时间: 2026-03-07

---

## 一、背景

投资者需要及时了解持仓股票的风险状况，避免业绩暴雷、异常下跌等问题造成损失。现有的分析工具多关注机会，缺少系统化的风险识别能力。

## 二、目标

为投资者提供股票风险监控能力：
- 及时发现业绩风险（预告下降、亏损）
- 识别价格异常（连续下跌、异常波动）
- 监控资金流向（持续流出、外资减持）
- 追踪重要事件（大股东减持、质押风险）

## 三、业务概念

### 3.1 风险类型

| 风险类型 | 说明 | 严重性 |
|---------|------|--------|
| 业绩风险 | 业绩预告下降或亏损 | 高 |
| 价格风险 | 连续下跌或异常波动 | 中 |
| 资金风险 | 资金持续流出 | 中 |
| 事件风险 | 大股东减持、高质押 | 低 |

### 3.2 业绩风险

**业绩预告类型**：
| 类型 | 说明 | 风险等级 |
|------|------|---------|
| 预亏 | 预计亏损 | 高 |
| 预降 | 预计利润下降 | 高 |
| 不确定 | 业绩不确定 | 中 |
| 预增/预盈 | 预计增长/盈利 | 无风险 |

**监控规则**：
- 业绩预告类型为"预亏"或"预降" → 触发业绩风险

### 3.3 价格风险

**监控指标**：
- 连续下跌天数
- 累计跌幅
- 单日跌幅

**监控规则**：
| 规则 | 条件 | 风险等级 |
|------|------|---------|
| 连续下跌 | 连续 ≥3 天下跌且累计跌幅 ≥10% | 中 |
| 大幅下跌 | 单日跌幅 ≥9% | 高 |
| 异常波动 | 涨跌幅异常（交易所定义） | 中 |

### 3.4 资金风险

**监控指标**：
- 主力资金净流入/流出
- 连续流出天数
- 累计流出金额

**监控规则**：
| 规则 | 条件 | 风险等级 |
|------|------|---------|
| 持续流出 | 连续 ≥3 天净流出且累计流出 ≥1亿 | 中 |
| 大额流出 | 单日净流出 ≥5亿 | 高 |

### 3.5 事件风险

**监控指标**：
- 大股东减持比例
- 股权质押比例
- 质押平仓风险

**监控规则**：
| 规则 | 条件 | 风险等级 |
|------|------|---------|
| 减持风险 | 大股东减持 ≥1% | 低 |
| 高质押 | 质押比例 ≥50% | 高 |
| 平仓风险 | 质押比例 ≥70% | 高 |

### 3.6 风险评级

综合多个风险信号，给出整体风险评级：

| 评级 | 条件 |
|------|------|
| 高风险 | 存在 ≥1 个高风险信号 |
| 中风险 | 存在 ≥1 个中风险信号，无高风险 |
| 低风险 | 存在 ≥1 个低风险信号，无中高风险 |
| 正常 | 无任何风险信号 |

### 3.7 使用场景

1. **全市场扫描**：扫描全市场，发现存在风险的股票
2. **个股检查**：检查指定股票的风险状况
3. **持续监控**：每日运行，跟踪风险变化

## 四、验收标准

### Phase 1（基础风险监控）

- [ ] 能获取业绩预告数据
- [ ] 能识别预亏/预降的股票
- [ ] 能识别连续下跌的股票（≥3天，累计跌幅≥10%）
- [ ] 能统计主力资金流向
- [ ] 能识别资金持续流出的股票
- [ ] 能输出风险评级和建议

### Phase 2（扩展风险监控）

- [ ] 能获取大股东减持数据
- [ ] 能获取股权质押数据
- [ ] 能识别高质押风险股票

### Phase 3（智能分析）

- [ ] 风险趋势分析（风险是否在加剧）
- [ ] 行业风险对比（同行业风险对比）
- [ ] 历史回溯（过去 N 天的风险变化）

## 五、不做

- 不做投资建议（只提供风险信息）
- 不做实时监控（按日频率）
- 不做空仓/减仓信号（只提供风险事实）
- 不做复杂的量化模型（保持规则简单可解释）

输入：无（扫描全市场）

输出：
```

==========================

1. 000001 平安银行 - 业绩预降, 连续下跌5天
2. 600000 浦发银行 - 大股东减持2%, 资金流出3亿
3. 000002 万科A - 质押比例65%

中风险股票 (5只):
1. 000333 美的集团 - 资金连续流出3天
...

低风险股票 (8只):
...

正常股票 (3000+只)
```

输入：000001 平安银行

输出：
```

=========================

风险信号:

[高] 连续下跌 - 连续下跌 5 天，累计跌幅 12%
[中] 资金流出 - 近 3 天资金净流出 3.5 亿

建议:
- 关注业绩公告详情
- 评估持仓比例
- 考虑风险控制措施
```

## 七、数据依赖

- 业绩预告数据
- 日线行情数据
- 资金流向数据
- 大股东增减持数据（Phase 2）
- 股权质押数据（Phase 2）
