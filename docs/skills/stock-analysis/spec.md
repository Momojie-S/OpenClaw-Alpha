# 个股分析 - 需求文档

## 一、背景

用户在研究股票时，需要一个快速了解单只股票核心指标的工具。目前需要手动打开多个网站查看，效率较低。

## 二、目标

提供一个个股分析功能，输入股票代码或名称，输出该股票的核心指标和简单分析结论。

## 三、业务概念

### 3.1 输入

- **股票标识**：股票代码（如 000001）或股票名称（如 平安银行）
- **日期**：分析日期，默认为最近交易日

### 3.2 核心指标

| 指标 | 说明 | 用途 |
|------|------|------|
| 价格 | 当前价格、涨跌幅、涨跌额 | 了解当日表现 |
| 成交 | 成交量、成交额、换手率 | 判断交易活跃度 |
| 估值 | 市盈率(PE)、市净率(PB) | 评估估值水平 |
| 规模 | 总市值、流通市值 | 了解公司规模 |

### 3.3 分析结论

基于指标给出简单的分析提示：

| 维度 | 判断规则 | 结论 |
|------|----------|------|
| 估值 | PE < 行业平均 - 20% | 低估 |
| 估值 | PE > 行业平均 + 20% | 高估 |
| 估值 | 其他 | 合理 |
| 活跃度 | 换手率 > 10% | 非常活跃 |
| 活跃度 | 换手率 > 5% | 较活跃 |
| 活跃度 | 换手率 > 2% | 正常 |
| 活跃度 | 换手率 <= 2% | 不活跃 |
| 涨跌 | 涨跌幅 > 5% | 大涨 |
| 涨跌 | 涨跌幅 > 2% | 上涨 |
| 涨跌 | 涨跌幅 < -5% | 大跌 |
| 涨跌 | 涨跌幅 < -2% | 下跌 |
| 涨跌 | 其他 | 持平 |

## 四、验收标准

- [ ] 能通过股票代码查询（如 000001）
- [ ] 能通过股票名称查询（如 平安银行）
- [ ] 返回价格相关指标（收盘价、涨跌幅、涨跌额）
- [ ] 返回成交相关指标（成交量、成交额、换手率）
- [ ] 返回估值指标（PE、PB）
- [ ] 返回市值指标（总市值、流通市值）
- [ ] 根据规则给出分析结论

## 五、范围边界

### 包含

- A股股票（沪深两市）
- 核心指标展示
- 简单分析结论

### 不包含

- 港股、美股
- 技术指标（MACD、KDJ等）
- 深度财务分析
- 历史趋势分析
- 买卖建议
