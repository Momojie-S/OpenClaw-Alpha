# 资金流向分析 Skill - 需求文档

## 一、背景

投资者需要了解市场资金的流动方向，把握热点板块和资金动向。现有的 skill 中：
- `northbound_flow` 只看外资
- `lhb_tracker` 只看龙虎榜
- `market_sentiment` 有整体资金流向但不深入

缺少一个综合性的资金流向分析工具，帮助用户从多维度了解资金动向。

## 二、目标

提供行业和概念板块的资金流向分析，帮助用户：
1. 快速了解哪些行业/概念受资金青睐
2. 发现资金流向的趋势变化
3. 对比不同时间周期的资金动向

## 三、业务概念

### 3.1 资金流向数据

| 字段 | 含义 |
|------|------|
| 行业/概念 | 板块名称 |
| 行业指数 | 板块指数点位 |
| 涨跌幅 | 板块当日涨跌幅 |
| 流入资金 | 板块内股票的买入资金总额 |
| 流出资金 | 板块内股票的卖出资金总额 |
| 净额 | 流入资金 - 流出资金（正值为净流入） |
| 公司家数 | 板块内股票数量 |
| 领涨股 | 板块内涨幅最大的股票 |

### 3.2 时间周期

支持多时间周期分析：
- 今日
- 3日排行
- 5日排行
- 10日排行
- 20日排行

## 四、功能需求

### 4.1 行业资金流

**输入**：时间周期（默认今日）
**动作**：获取行业资金流向排名
**输出**：行业资金流向列表（按净额排序）

### 4.2 概念资金流

**输入**：时间周期（默认今日）
**动作**：获取概念板块资金流向排名
**输出**：概念资金流向列表（按净额排序）

### 4.3 筛选功能

**输入**：
- 类型（行业/概念，默认行业）
- 时间周期（默认今日）
- 净额过滤（最小净额）
- Top N（默认 10）

**输出**：符合条件的资金流向列表

## 五、验收标准

- [ ] 能获取行业资金流向排名
- [ ] 能获取概念资金流向排名
- [ ] 支持多时间周期（今日/3日/5日/10日/20日）
- [ ] 支持按净额筛选
- [ ] 支持返回 Top N
- [ ] 控制台输出精简结果
- [ ] 文件保存完整数据

## 六、边界

**不做**：
- 个股资金流向（东方财富接口不稳定）
- 历史资金流向趋势图
- 实时推送
- 资金流向预测
