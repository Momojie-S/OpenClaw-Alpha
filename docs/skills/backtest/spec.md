# 策略回测 - 需求文档

> 版本: v1.0
> 创建时间: 2026-03-08

---

## 一、业务背景

### 1.1 问题陈述

投资者需要验证投资策略的有效性，但缺乏简单易用的回测工具。

**现状**：
- 投资分析框架已完成行业轮动、基本面分析等能力
- 用户可以获取市场数据和分析结果
- 但无法验证"如果按这个策略操作，历史表现会如何"

### 1.2 目标用户

- 量化投资初学者
- 想验证自己投资想法的个人投资者
- 需要快速验证策略有效性的分析师

---

## 二、业务需求

### 2.1 核心场景

**场景 1：均线交叉策略回测**
- 用户：想测试"5 日均线上穿 20 日均线买入"的策略
- 动作：输入股票代码、日期范围、均线参数
- 输出：历史收益率、夏普比率、最大回撤

**场景 2：行业轮动策略回测**
- 用户：想测试"买入热门行业龙头股"的策略
- 动作：输入行业列表、调仓频率
- 输出：收益对比基准、交易记录

### 2.2 验收标准

- [ ] 支持单股票回测（MVP）
- [ ] 支持均线交叉策略
- [ ] 输出关键指标（收益率、夏普比率、最大回撤）
- [ ] 保存完整交易记录到文件
- [ ] 控制台输出精简结果

## 三、业务概念

### 3.1 回测指标

| 指标 | 说明 | 重要性 |
|------|------|--------|
| **总收益率** | 策略总收益 / 初始资金 | 高 |
| **年化收益率** | 按年计算的收益率 | 高 |
| **夏普比率** | 风险调整后收益 | 高 |
| **最大回撤** | 从高点到低点的最大跌幅 | 高 |
| **胜率** | 盈利交易次数 / 总交易次数 | 中 |
| **交易次数** | 总买卖次数 | 中 |

### 3.2 策略类型

| 类型 | 说明 | MVP 优先级 |
|------|------|-----------|
| **均线交叉** | 快慢均线金叉/死叉 | P0（必须） |
| **行业轮动** | 定期调仓到热门行业 | P1（可选） |
| **多因子选股** | 综合评分选股 | P2（未来） |

## 四、边界与限制

### 4.1 MVP 范围

**包含**：
- 单股票回测
- 均线交叉策略
- 关键指标输出
- 交易记录保存

**不包含**：
- 多股票组合回测
- 参数优化
- 可视化图表
- 实时策略推送

### 4.2 A股特殊处理

| 问题 | MVP 处理方式 |
|------|-------------|
| 涨跌停 | 暂不处理（后续优化） |
| 交易费用 | 按默认费率计算 |
| 除权除息 | 使用前复权数据 |
| 停牌 | 数据中自动过滤 |

---

## 五、非功能需求

### 5.1 性能要求

- 1 年历史数据回测 < 5 秒
- 支持最多 3 年历史数据

### 5.2 易用性要求

- 命令行参数简单直观
- 有合理默认值（不传参数也能运行）
- 输出格式清晰易读
