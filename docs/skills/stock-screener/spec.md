# 选股筛选器 - 需求文档

## 一、问题背景

在 A 股市场有 5000+ 只股票，投资者需要一种高效的方式筛选出符合特定条件的股票。传统的筛选方式要么依赖交易软件（功能复杂、操作繁琐），要么依赖网站（广告多、速度慢）。

**用户痛点**：
- 手动翻股票太慢，效率低
- 不清楚哪些股票符合自己的选股标准
- 需要快速找到符合条件的标的进行进一步分析

## 二、目标用户

- 价值投资者：寻找低估值的优质股票
- 成长投资者：寻找高增长的潜力股票
- 趋势投资者：寻找技术形态好的股票
- 量化投资者：按指标组合筛选

## 三、业务概念

### 3.1 筛选条件

支持以下维度的筛选条件：

| 维度 | 条件 | 说明 |
|------|------|------|
| **涨跌幅** | 涨幅范围 | 如：涨幅 > 3% |
| **换手率** | 换手率范围 | 如：换手率 3% ~ 10% |
| **成交额** | 成交额范围 | 如：成交额 > 1 亿 |
| **市值** | 市值范围 | 如：市值 50 ~ 500 亿 |
| **价格** | 价格范围 | 如：价格 10 ~ 50 元 |
| **板块** | 所属板块 | 如：属于新能源板块 |

### 3.2 筛选模式

提供两种筛选模式：

| 模式 | 说明 | 适用场景 |
|------|------|----------|
| **快速筛选** | 预设条件组合，一键筛选 | 新手用户、快速选股 |
| **自定义筛选** | 自由组合多个条件 | 进阶用户、精细筛选 |

### 3.3 预设筛选策略

| 策略名 | 条件组合 | 说明 |
|--------|----------|------|
| **放量突破** | 涨幅 > 3%，换手率 > 5%，成交额 > 2 亿 | 寻找放量上涨的强势股 |
| **缩量回调** | 涨幅 -5% ~ 0%，换手率 < 3% | 寻找缩量回调的潜力股 |
| **龙头股** | 涨幅 > 5%，成交额 > 10 亿，市值 > 100 亿 | 寻找市场龙头 |
| **小盘活跃** | 换手率 > 8%，市值 < 50 亿 | 寻找活跃的小盘股 |
| **低估蓝筹** | 市值 > 500 亿，换手率 < 2% | 寻找低估值蓝筹股 |

### 3.4 输出结果

筛选结果包含以下信息：

| 字段 | 说明 |
|------|------|
| 股票代码 | 如 000001 |
| 股票名称 | 如 平安银行 |
| 涨跌幅 | 当日涨跌幅（%） |
| 换手率 | 当日换手率（%） |
| 成交额 | 当日成交额（亿元） |
| 最新价 | 最新价格（元） |
| 总市值 | 总市值（亿元） |
| 匹配条件 | 命中哪些筛选条件 |

## 四、验收标准

- [ ] 支持按涨跌幅、换手率、成交额、市值、价格筛选
- [ ] 支持多条件组合（AND 逻辑）
- [ ] 提供 5 种预设筛选策略
- [ ] 输出结果按涨幅排序（可配置）
- [ ] 支持 Top N 输出
- [ ] 结果保存到文件

## 五、边界

### 5.1 不做

- 不做技术指标筛选（MACD、RSI 等）- 属于技术分析 skill
- 不做财务指标筛选（PE、PB、ROE 等）- 需要高积分 API
- 不做实时推送 - 属于监控 skill
- 不做回测验证 - 属于策略回测 skill

### 5.2 限制

- 只支持当日数据筛选（不支持历史筛选）
- 筛选结果最多返回 100 只股票
- 非交易日返回最近交易日数据
