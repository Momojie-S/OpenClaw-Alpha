# 需求文档：产业热度追踪

## 一、背景

A 股市场板块轮动频繁，投资者需要快速识别当前热点板块和趋势变化，以便把握市场机会、规避风险。

## 二、目标用户

- 个人投资者：想了解当前市场热点
- 投资顾问：需要快速获取板块热度信息，为客户提供分析建议

## 三、业务概念

### 3.1 板块分类

支持以下分类：

| 分类 | 说明 | 数量 |
|------|------|------|
| 一级行业 | 申万一级行业（如电子、医药生物） | 31 |
| 二级行业 | 申万二级行业（如半导体、化学制药） | 134 |
| 三级行业 | 申万三级行业（如数字芯片设计、CXO） | 346 |
| 概念板块 | 市场主流概念（如AI、低空经济、机器人） | 动态 |

### 3.2 热度指数

热度指数反映板块的市场关注度，由以下维度加权计算：

| 维度 | 权重 | 说明 | 数据来源 |
|------|------|------|----------|
| 涨跌幅 | 30% | 价格变动幅度 | 指数日线 |
| 换手率 | 25% | 交易活跃程度 | 指数日线 |
| 成交额占比 | 25% | 板块成交额 / 全市场成交额 | 指数日线 |
| 涨跌家数比 | 20% | 板块内上涨股票数 / 总股票数 | 板块成分 |

### 3.3 趋势信号

根据热度指数变化，判断板块趋势：

| 信号 | 判断规则 |
|------|----------|
| 加热中 | 热度环比 > +20% 且 涨幅 > 0 |
| 降温中 | 热度环比 < -20% 或 涨幅 < -3% |
| 稳定 | 其他情况 |

### 3.4 数据周期

- **当日数据**：最近一个交易日
- **环比数据**：对比上一交易日

## 四、功能需求

### 4.1 核心功能

1. **获取板块列表**
   - 输入：分类类型（一级行业/二级行业/三级行业/概念）
   - 输出：该分类下所有板块的名称和代码

2. **计算板块热度**
   - 输入：分类类型、日期（默认当日）
   - 输出：该分类下所有板块的热度指数、各维度得分、趋势信号

3. **热度排名**
   - 输入：分类类型、Top N（默认 10）
   - 输出：按热度指数排序的板块列表

### 4.2 输出格式

**精简输出（控制台）**：
```json
[
  {
    "rank": 1,
    "name": "电子",
    "heat_index": 85.2,
    "trend": "加热中",
    "change_pct": 3.5
  },
  ...
]
```

**完整输出（文件）**：
```json
{
  "date": "2026-03-06",
  "boards": [
    {
      "name": "电子",
      "code": "...",
      "heat_index": 85.2,
      "scores": {
        "change": 28.5,
        "turnover": 22.1,
        "volume_ratio": 20.3,
        "up_ratio": 14.3
      },
      "metrics": {
        "change_pct": 3.5,

        "down_count": 15

      "trend": "加热中",
      "heat_change": 25.3
    },

  ]

```

## 五、验收标准

- [ ] 能输出板块热度排名（Top N）
- [ ] 热度指数计算符合上述公式（权重可配置）
- [ ] 趋势信号判断符合上述规则
- [ ] 能对比上一交易日（环比）
- [ ] 控制台输出精简结果（Top N）
- [ ] 文件保存完整数据（所有板块）

## 六、不做

- 历史回溯（只支持当日和环比）
- 自定义权重（首次迭代不支持，后续可扩展）
