# 融资融券分析 - 需求文档

> 版本: v1.0
> 创建时间: 2026-03-08

---

## 一、业务背景

融资融券是 A 股市场的重要杠杆工具，可以反映市场情绪和风险水平：

- **融资**（做多）：借钱买股票，表示看多
- **融券**（做空）：借股票卖出，表示看空

通过监控融资融券数据，可以：
1. 判断市场整体杠杆水平
2. 发现个股融资/融券异动
3. 预警系统性风险

---

## 二、分析场景

### 场景 1：市场杠杆监控

**用户问题**：市场杠杆水平如何？是否过热？

**解决方式**：
- 查看沪深两市融资余额变化
- 计算融资余额环比变化
- 判断杠杆水平（高/正常/低）

### 场景 2：个股融资异动

**用户问题**：哪些股票被大幅融资买入？

**解决方式**：
- 筛选融资余额 Top N
- 筛选融资买入额环比增长 Top N

### 场景 3：融券做空监控

**用户问题**：哪些股票被做空？

**解决方式**：
- 筛选融券余额 Top N
- 监控融券余额变化

## 三、业务概念

### 3.1 融资融券指标

| 指标 | 说明 | 多空含义 |
|------|------|----------|
| 融资余额 | 当前融资未还金额 | 余额↑ = 看多情绪强 |
| 融资买入额 | 当日融资买入金额 | 买入↑ = 新增多头 |
| 融券余额 | 当前融券未还市值 | 余额↑ = 看空情绪强 |
| 融券余量 | 当前融券未还股数 | 余量↑ = 做空增加 |

### 3.2 杠杆水平判断

| 水平 | 融资余额环比 | 含义 |
|------|-------------|------|
| 高 | > +2% | 杠杆增加，风险累积 |
| 正常 | -2% ~ +2% | 正常波动 |
| 低 | < -2% | 杠杆减少，情绪降温 |
