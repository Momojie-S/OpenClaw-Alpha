# 自选股监控 Skill - 需求文档

## 一、背景

投资者通常会关注一批股票，但不是全部持有：
- 想持续跟踪某些股票的价格走势
- 等待合适买入时机的标的
- 已卖出但想继续关注的股票

本 skill 帮助用户维护自选股列表，快速获取批量行情和分析。

## 二、目标用户

- 有自选股需要跟踪的投资者
- 等待买入时机的用户
- 需要快速查看多只股票行情的用户

## 三、核心功能

### 3.1 自选股管理

**添加自选股**：
```
--add 000001,600000,002475
--add-file watchlist.txt  # 每行一个股票代码
```

**移除自选股**：
```
--remove 000001
```

**查看自选股列表**：
```
--list
```

**清空自选股**：
```
--clear
```

### 3.2 数据存储

自选股列表存储在文件中，路径：
```
.openclaw_alpha/watchlist-monitor/watchlist.json
```

**文件格式**：
```json
{
  "stocks": [
    {"code": "000001", "added_at": "2026-03-07", "note": "银行龙头"},
    {"code": "600000", "added_at": "2026-03-06", "note": ""}
  ],
  "updated_at": "2026-03-07T08:00:00"
}
```

### 3.3 行情监控

**批量获取行情**：
```
--watch           # 获取所有自选股的实时行情
--watch --top-n 5 # 只显示涨跌幅最大的5只
```

**输出内容**：

| 字段 | 说明 |
|------|------|
| 代码 | 6位股票代码 |
| 名称 | 股票名称 |
| 现价 | 当前价格 |
| 涨跌幅 | 当日涨跌百分比 |
| 涨跌额 | 当日涨跌金额 |
| 成交额 | 当日成交金额（亿） |
| 换手率 | 当日换手率 |
| 状态 | 涨停/跌停/正常 |

### 3.4 快速分析

**简报分析**：
```
--analyze         # 对所有自选股进行简报分析
```

**输出内容**：
- 市场统计：涨跌数量、平均涨跌幅
- 表现最好/最差的股票
- 异动股票（涨停、跌停、放量）

### 3.5 价格提醒（可选，Phase 2）

设置价格提醒规则：
```
--alert 000001 --above 15 --below 10
```

## 四、验收标准

- [ ] 支持添加、移除、查看、清空自选股
- [ ] 自选股列表持久化存储
- [ ] 批量获取所有自选股的实时行情
- [ ] 正确显示涨跌幅、成交额等关键指标
- [ ] 提供简单的统计汇总

## 五、不做

- 不做实时推送通知（Phase 2 考虑）
- 不做复杂的技术分析（用 stock_analysis）
- 不做选股筛选（用 stock_screener）
- 不做持仓盈亏计算（用 portfolio_analysis）
