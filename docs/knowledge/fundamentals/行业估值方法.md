# 行业估值方法

## 定义

针对不同行业特点选择合适的估值方法，因为不同行业的商业模式、资产结构、增长特性差异巨大，单一估值指标无法通用。

---

## 估值方法分类

### 相对估值法

**原理**：与可比公司或历史水平对比。

**特点**：
- 简单直观，数据易得
- 依赖市场有效性
- 适合成熟行业

**常用指标**：[[估值指标|PE/PB/PS/PCF]]、PEG、EV/EBITDA

### 绝对估值法

**原理**：基于未来现金流折现计算内在价值。

**特点**：
- 理论严谨，不依赖市场
- 对假设敏感（增长率、折现率）
- 适合稳定现金流公司

**常用方法**：DCF（现金流折现）

---

## 行业估值特点

### 银行/金融

**特点**：
- 高杠杆，资产重
- 利润受息差和坏账影响
- 受监管资本充足率约束

**首选方法**：PB

**原因**：
- 资产价值是核心
- 利润波动大，PE 不稳定
- PB < 1 意味着市值低于净资产

**辅助指标**：
- PE（盈利稳定时）
- 股息率（银行股分红高）
- ROE（资产质量）

**注意事项**：
- 关注不良贷款率
- 资本充足率影响扩张能力
- 息差收窄时需谨慎

---

### 消费/制造业

**特点**：
- 盈利相对稳定
- 受经济周期影响
- 品牌和渠道是核心资产

**首选方法**：PE

**原因**：
- 利润是核心指标
- 可比公司多，相对估值有效
- 历史分位参考价值大

**辅助指标**：
- PB（资产重时）
- ROE（盈利能力）
- 自由现金流（扩张后期）

**注意事项**：
- 区分高端 vs 大众消费
- 关注渠道库存
- 原材料成本波动影响

---

### 科技/成长股

**特点**：
- 早期多亏损，高增长
- 研发投入大
- 资产轻，无形资产多

**首选方法**：PS + PEG

**原因**：
- 亏损期 PE 无意义
- 收入增长是核心指标
- PEG 考虑增长率，避免高 PE 陷阱

**PEG 判断**：
| PEG | 含义 |
|-----|------|
| < 1.0 | 低估（增速高于估值） |
| = 1.0 | 合理（估值匹配增速） |
| > 1.0 | 高估（估值高于增速） |
| > 2.0 | 明显高估 |

**辅助指标**：
- 毛利率（商业模式健康度）
- 用户增长（互联网）
- 研发投入占比

**注意事项**：
- 增长率预测不准确会误导
- 区分真成长 vs 烧钱换增长
- 现金流能否支撑烧钱

---

### 公用事业/基础设施

**特点**：
- 现金流稳定
- 资产重，折旧大
- 分红稳定

**首选方法**：PCF（市现率）

**原因**：
- 现金流比利润更稳定
- 折旧是非现金支出，EBITDA 更合适
- 适合 DCF 估值

**辅助指标**：
- 股息率（分红稳定性）
- EV/EBITDA（剔除折旧影响）
- PE（盈利稳定时）

**注意事项**：
- 受政策监管影响
- 利率变化影响 DCF 折现
- 通胀影响实际收益

---

### 周期股（有色/煤炭/化工/航运）

**特点**：
- 利润随价格剧烈波动
- 周期顶部利润高，底部亏损
- 资产重，产能是核心

**首选方法**：PB（周期底部）+ PE（周期顶部反向）

**周期估值陷阱**：
| 周期阶段 | 利润 | PE | PB | 信号 |
|---------|------|----|----|------|
| 底部 | 低/亏损 | 高/无效 | 低 | **买入信号** |
| 上升期 | 增长 | 下降 | 上升 | 持有 |
| 顶部 | 高 | 低 | 高 | **卖出信号** |
| 下降期 | 下降 | 上升 | 下降 | 观望 |

**原因**：
- 底部 PB 低（接近重置成本）
- 顶部 PE 低是反向指标（利润见顶）

**辅助指标**：
- 产品价格趋势
- 产能利用率
- 库存周期

**注意事项**：
- 不要在 PE 低时买（顶部陷阱）
- 不要在 PE 高时卖（底部机会）
- 关注供需格局变化

---

### 房地产

**特点**：
- 资产极重
- 销售确认滞后
- 受政策影响大

**首选方法**：NAV（净资产价值法）

**NAV 公式**：
```
NAV = (土地储备 × 预期售价 - 建安成本 - 税费) / 总股本
```

**辅助指标**：
- PB（资产价值）
- 销售额/市值（去化速度）
- 净负债率

**注意事项**：
- 土地储备价值波动大
- 政策调控影响估值
- 现金流风险

## 进阶估值方法

### DCF（现金流折现）

> 详见 [[DCF估值法]]

**定义**：将未来自由现金流折现到当前，计算企业内在价值。

**公式**：
```
DCF = Σ CFₙ / (1 + r)ⁿ

其中：
CFₙ = 第 n 年自由现金流
r = 折现率（通常用 WACC）
n = 预测年限
```

**适用场景**：
- 现金流稳定的公司（公用事业、消费龙头）
- 成熟期公司
- 并购估值

**关键假设**：
| 假设 | 影响 | 设定难度 |
|------|------|---------|
| 未来现金流 | 直接影响估值 | 高 |
| 折现率（WACC） | 影响折现程度 | 中 |
| 永续增长率 | 终值占比大 | 高 |
| 预测年限 | 影响准确性 | 低 |

**WACC（加权平均资本成本）**：
```
WACC = E/V × Re + D/V × Rd × (1 - Tc)

其中：
E = 股权市值
D = 债务市值
V = E + D
Re = 股权成本（CAPM）
Rd = 债务成本
Tc = 企业所得税率
```

**敏感性分析**：
- 调整折现率 ±1%
- 调整增长率 ±1%
- 观察估值变化范围

**局限性**：
- 假设对结果影响极大
- 预测期越长，不确定性越大
- 不适合高增长或亏损公司

---

### PEG（市盈率增长比）

**定义**：PE 除以预期增长率，调整高增长股票的估值。

**公式**：
```
PEG = PE / 预期增长率(%)

例：PE = 30，增长率 = 30%
PEG = 30 / 30 = 1.0
```

**适用场景**：
- 成长股估值
- 跨增速公司对比
- 区分真成长 vs 伪成长

**判断标准**（Peter Lynch 规则）：
- PEG < 1.0：低估
- PEG = 1.0：合理
- PEG > 1.5：高估
- PEG > 2.0：明显泡沫

**增长率选择**：
| 类型 | 数据来源 | 准确性 |
|------|---------|--------|
| 历史增长率 | 过去数据 | 高（但过去不代表未来） |
| 一致预期 | 分析师预测 | 中 |
| 自行预测 | 行业研究 | 低（需要深入研究） |

**注意事项**：
- 增长率预测不准会严重误导
- 短期高增长不可持续
- 需结合行业天花板判断

---

### EV/EBITDA（企业价值倍数）

**定义**：企业价值与息税折旧摊销前利润的比值。

**公式**：
```
EV/EBITDA = 企业价值 / EBITDA

企业价值 = 市值 + 净债务
EBITDA = 净利润 + 利息 + 税 + 折旧 + 摊销
```

**适用场景**：
- 跨资本结构对比（不受杠杆影响）
- 并购估值（收购方关注）
- 重资产行业（剔除折旧影响）

**优势**：
- 剔除资本结构差异
- 剔除折旧政策差异
- 适合跨国对比（税制不同）

**局限性**：
- 忽略资本支出差异
- 不适合高增长公司（EBITDA 可能为负）

**行业参考**：
| 行业 | EV/EBITDA 合理范围 |
|------|------------------|
| 公用事业 | 8-12x |
| 消费品 | 10-15x |
| 科技 | 15-25x |
| 周期股 | 5-10x |

---

## 行业估值方法总结

| 行业 | 首选方法 | 辅助方法 | 核心逻辑 |
|------|---------|---------|---------|
| 银行/金融 | PB | PE、股息率 | 资产价值是核心 |
| 消费/制造 | PE | PB、ROE | 利润稳定性 |
| 科技/成长 | PS + PEG | 毛利率 | 增速匹配估值 |
| 公用事业 | PCF/DCEV/EBITDA | 股息率 | 现金流稳定 |
| 周期股 | PB（底部） | PE（顶部反向） | 周期逆向思维 |
| 房地产 | NAV | PB、销售额 | 土地储备价值 |

---

## 应用原则

### 1. 行业适配

- 优先用行业通用方法
- 跨行业对比用 EV/EBITDA
- 单一指标有局限，多指标验证

### 2. 历史分位

- 纵向：与自身历史对比
- 横向：与同行业对比
- 极端值往往是机会或风险信号

### 3. 增长调整

- 高增长用 PEG 调整
- 周期股用 PB 底部判断
- 成长期公司关注收入增长

### 4. 风险调整

- 高杠杆公司折现率提高
- 政策敏感行业加风险溢价
- 现金流不稳定时用 PCF
