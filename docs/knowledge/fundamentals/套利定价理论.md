# 套利定价理论

## 定义

套利定价理论（Arbitrage Pricing Theory，APT）是由经济学家 Stephen Ross 于 1976 年提出的多因子资产定价模型。它基于这样一个理念：资产的预期收益可以通过资产预期收益与多个宏观经济变量之间的线性关系来预测，这些变量反映了系统性风险。

**核心思想**：
- 资产收益由多个因子驱动
- 市场偶尔会错误定价证券
- 套利者会利用价格偏离获利
- 最终价格回归公允价值

---

## 与 CAPM 的对比

| 维度 | CAPM | APT |
|------|------|-----|
| 因子数量 | 单因子（市场风险） | 多因子（多个系统性风险） |
| 假设 | 市场完美有效 | 市场可能暂时错误定价 |
| 灵活性 | 固定模型 | 可自定义因子 |
| 复杂度 | 简单 | 复杂（需确定因子） |
| 适用性 | 通用 | 需针对资产选择因子 |

**共同点**：
- 都基于系统性风险定价
- 都使用 Beta 系数度量敏感性
- 都假设无套利机会（长期）

**区别**：
- CAPM 是 APT 的特例（单因子）
- APT 更灵活，可包含多个因子
- APT 不需要假设市场组合存在

---

## APT 模型公式

### 基本公式

$$E(R_i) = R_f + \beta_{i1}(E(I_1) - R_f) + \beta_{i2}(E(I_2) - R_f) + ... + \beta_{in}(E(I_n) - R_f)$$

其中：
- $E(R_i)$ = 资产预期收益
- $R_f$ = 无风险利率
- $\beta_{in}$ = 资产对第 n 个因子的敏感性
- $E(I_n)$ = 第 n 个因子的预期收益
- $E(I_n) - R_f$ = 第 n 个因子的风险溢价

### 因子选择

**常用宏观经济因子**：

| 因子类型 | 示例 | 经济含义 |
|---------|------|---------|
| 经济增长 | GDP、GNP、工业产值 | 经济周期风险 |
| 通胀 | CPI、PPI、意外通胀 | 购买力风险 |
| 利率 | 收益率曲线、期限利差 | 利率风险 |
| 信用 | 公司债利差 | 信用风险 |
| 市场 | 市场指数收益 | 市场风险 |
| 商品 | 商品价格、油价、金价 | 商品风险 |
| 汇率 | 汇率变动 | 汇率风险 |

**因子数量**：
- 通常 4-5 个因子即可解释大部分收益
- 因子过多会导致过拟合
- 需平衡解释力和简洁性

---

## APT 的假设

### 核心假设

1. **无套利机会**：市场价格最终会回归均衡
2. **因子模型**：资产收益由有限个因子驱动
3. **分散化**：特异风险可通过分散化消除
4. **线性关系**：收益与因子呈线性关系

### 与 CAPM 的假设差异

| 假设 | CAPM | APT |
|------|------|-----|
| 市场组合 | 必须存在 | 不需要 |
| 效用函数 | 需假设 | 不需要 |
| 正态分布 | 需假设 | 不需要 |
| 完美市场 | 需要 | 不需要（允许暂时错误定价） |

---

## 计算步骤

### 1. 因子识别

选择影响资产收益的系统性因子：
- 宏观经济因子（GDP、通胀、利率）
- 风格因子（规模、价值、动量）
- 行业因子（行业暴露）

### 2. Beta 估计

使用线性回归估计资产对各因子的敏感性：

$$R_i = \alpha + \beta_{i1}F_1 + \beta_{i2}F_2 + ... + \beta_{in}F_n + \epsilon$$

其中：
- $R_i$ = 资产收益
- $F_n$ = 第 n 个因子收益
- $\beta_{in}$ = 因子敏感性
- $\epsilon$ = 特异收益

### 3. 风险溢价估计

估计每个因子的风险溢价：
- 使用历史数据
- 估计因子的预期超额收益

### 4. 预期收益计算

代入公式计算资产的预期收益

## 示例

假设某股票的因子敏感性如下：

| 因子 | Beta | 风险溢价 |
|------|------|---------|
| GDP 增长 | 0.6 | 4% |
| 通胀率 | 0.8 | 2% |
| 黄金价格 | -0.7 | 5% |
| 标普 500 指数 | 1.3 | 9% |

无风险利率 = 3%

**预期收益计算**：
$$E(R) = 3\% + (0.6 \times 4\%) + (0.8 \times 2\%) + (-0.7 \times 5\%) + (1.3 \times 9\%)$$
$$E(R) = 3\% + 2.4\% + 1.6\% - 3.5\% + 11.7\% = 15.2\%$$

## APT 的应用

### 1. 资产定价

- 识别错误定价的证券
- 套利机会发现
- 投资组合构建

### 2. 风险管理

- 识别风险来源
- 度量风险暴露
- 对冲策略设计

### 3. 绩效归因

- 分解收益来源
- 评估主动管理能力
- 因子暴露分析

### 4. 投资组合优化

- 构建因子中性组合
- 风险预算分配
- 多因子模型构建

---

## APT 的优势

1. **灵活性**：可根据资产特性选择因子
2. **多维度**：考虑多个风险源
3. **实证性**：基于数据确定因子
4. **无套利**：理论基础坚实

---

## APT 的局限性

### 1. 因子选择主观

- 理论不指定具体因子
- 需投资者自行判断
- 不同选择导致不同结果

### 2. 估计误差

- Beta 估计不稳定
- 风险溢价难以准确估计
- 历史数据可能不具代表性

### 3. 模型风险

- 线性假设可能不成立
- 因子关系可能变化
- 过拟合风险

### 4. 实施难度

- 需要大量研究
- 数据要求高
- 模型维护成本

## A 股应用

### 推荐因子

**宏观因子**：
- 经济增长：工业增加值、PMI
- 通胀：CPI、PPI
- 流动性：M2、社融
- 利率：10 年期国债收益率

**风格因子**：
- 规模、价值、动量、质量

**市场因子**：
- 沪深 300 指数
- 市场情绪指标

### 注意事项

1. **因子本土化**：选择 A 股市场特有的因子
2. **数据可得性**：确保因子数据持续可得
3. **结构性变化**：考虑市场制度变化的影响
4. **因子相关性**：避免因子高度相关

## 与多因子模型的关系

APT 是多因子模型的理论基础：

| 模型 | 关系 |
|------|------|
| Fama-French | APT 的特例（规模+价值+市场） |
| Barra 模型 | APT 的工业化应用 |
| 因子投资 | APT 的实践应用 |

**联系**：
- 都使用因子解释收益
- 都估计因子暴露
- 都构建因子组合

**区别**：
- APT 是理论框架
- 多因子模型是具体实现
- 因子选择和应用不同
