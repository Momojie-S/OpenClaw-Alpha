# 财务指标

## 定义

反映公司经营状况和财务健康的量化指标，用于评估盈利能力、运营效率、偿债能力。

---

## 盈利能力

### ROE（净资产收益率）

**定义**：净利润与净资产的比率，衡量股东资本回报效率。

**公式**：
```
ROE = 净利润 / 净资产 × 100%
```

**杜邦分解**：
```
ROE = 净利率 × 资产周转率 × 权益乘数
    = (净利润/营收) × (营收/总资产) × (总资产/净资产)
```

**应用场景**：
- 评估公司赚钱能力
- 筛选优质公司（ROE > 15%）
- 杜邦分析定位盈利驱动因素

**注意事项**：
- 高杠杆会拉高 ROE（需结合负债率看）
- 一次性收益会扭曲 ROE
- 不同行业 ROE 差异大

---

### ROA（总资产收益率）

**定义**：净利润与总资产的比率，衡量资产利用效率。

**公式**：
```
ROA = 净利润 / 总资产 × 100%
```

**应用场景**：
- 评估资产运营效率
- 重资产行业对比
- 与 ROE 结合判断杠杆水平

**注意事项**：
- 轻资产行业 ROA 通常较高
- 需结合行业特性判断

---

### 毛利率

**定义**：毛利与营业收入的比率，反映产品盈利空间。

**公式**：
```
毛利率 = (营业收入 - 营业成本) / 营业收入 × 100%
```

**应用场景**：
- 评估产品竞争力
- 成本控制能力
- 行业定价权判断

**注意事项**：
- 不同行业差异大（软件 > 制造 > 零售）
- 毛利率下降可能是价格战或成本上升
- 需结合净利率看费用控制

---

### 净利率

**定义**：净利润与营业收入的比率，反映最终盈利能力。

**公式**：
```
净利率 = 净利润 / 营业收入 × 100%
```

**应用场景**：
- 评估整体盈利效率
- 费用控制能力
- 与毛利率对比判断费用结构

**注意事项**：
- 受非经常性损益影响
- 需剔除非主业收入/支出

---

## 运营效率

### 资产周转率

**定义**：营业收入与总资产的比率，衡量资产运营效率。

**公式**：
```
资产周转率 = 营业收入 / 总资产
```

**应用场景**：
- 评估资产利用效率
- 轻资产 vs 重资产模式对比
- 杜邦分析组成部分

**注意事项**：
- 重资产行业周转率较低
- 季节性因素影响

---

### 存货周转率

**定义**：营业成本与平均存货的比率，衡量存货变现速度。

**公式**：
```
存货周转率 = 营业成本 / 平均存货
存货周转天数 = 365 / 存货周转率
```

**应用场景**：
- 评估库存管理效率
- 识别滞销风险
- 零售/制造行业关键指标

**注意事项**：
- 周转率下降可能是销售放缓或备货过多
- 行业差异大

---

### 应收账款周转率

**定义**：营业收入与平均应收账款的比率，衡量回款速度。

**公式**：
```
应收账款周转率 = 营业收入 / 平均应收账款
应收账款周转天数 = 365 / 应收账款周转率
```

**应用场景**：
- 评估客户质量
- 判断账期政策
- 识别坏账风险

**注意事项**：
- 周转天数增加可能是放宽账期或回款困难
- 需结合行业账期特点

---

## 偿债能力

### 资产负债率

**定义**：总负债与总资产的比率，衡量财务杠杆水平。

**公式**：
```
资产负债率 = 总负债 / 总资产 × 100%
```

**应用场景**：
- 评估财务风险
- 判断杠杆水平
- 行业对比

**注意事项**：
- 40%-60% 为常见区间
- 金融业天然高杠杆
- 需结合利率环境看

---

### 流动比率

**定义**：流动资产与流动负债的比率，衡量短期偿债能力。

**公式**：
```
流动比率 = 流动资产 / 流动负债
```

**应用场景**：
- 评估短期偿债能力
- 流动性风险预警

**注意事项**：
- > 2 通常认为安全
- 过高可能资金闲置
- 需结合速动比率看

---

### 速动比率

**定义**：速动资产与流动负债的比率，剔除存货后的短期偿债能力。

**公式**：
```
速动比率 = (流动资产 - 存货) / 流动负债
```

**应用场景**：
- 更严格的流动性测试
- 存货周转慢的行业

**注意事项**：
- > 1 通常认为安全
- 存货占比高的行业需重点关注

---

## 指标体系

| 维度 | 核心指标 | 辅助指标 |
|------|---------|---------|
| 盈利能力 | ROE | ROA、毛利率、净利率 |
| 运营效率 | 资产周转率 | 存货周转率、应收账款周转率 |
| 偿债能力 | 资产负债率 | 流动比率、速动比率 |
