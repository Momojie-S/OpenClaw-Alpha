# 资本资产定价模型

## 定义

资本资产定价模型（Capital Asset Pricing Model，CAPM）是现代金融市场价格理论的支柱，用于描述资产预期收益率与系统性风险之间的关系。由威廉·夏普（William Sharpe）、林特尔（John Lintner）、崔诺（Jack Treynor）和莫辛（Jan Mossin）等人于 1960 年代在现代投资组合理论的基础上发展。

**核心主张**：投资组合的期望报酬率只跟系统性风险有关，非系统性风险可通过分散化消除。

## 公式

### CAPM 主公式

$$E(R_i) = R_f + \beta_i \times [E(R_m) - R_f]$$

其中：
- $E(R_i)$ - 资产 $i$ 的期望报酬率（或普通股的资本成本率）
- $R_f$ - 无风险报酬率（通常用短期国债利率）
- $\beta_i$ - 资产 $i$ 的系统性风险系数
- $E(R_m)$ - 市场投资组合的期望报酬率
- $E(R_m) - R_f$ - 市场风险溢价

### 贝塔系数公式

$$\beta_i = \frac{Cov(R_i, R_m)}{Var(R_m)} = \frac{\sigma_{i,m}}{\sigma_m^2} = \rho_{i,m} \times \frac{\sigma_i}{\sigma_m}$$

其中：
- $Cov(R_i, R_m)$ - 证券收益与市场收益的协方差
- $Var(R_m)$ - 市场收益的方差
- $\rho_{i,m}$ - 证券与市场的相关系数
- $\sigma_i$ - 证券的标准差
- $\sigma_m$ - 市场的标准差

### 贝塔系数解读

| β 值 | 含义 | 风险水平 |
|------|------|---------|
| β = 0 | 无风险资产 | 无风险 |
| 0 < β < 1 | 波动低于市场 | 低风险 |
| β = 1 | 与市场同波动 | 市场风险 |
| β > 1 | 波动高于市场 | 高风险 |

**示例**：
- β = 0.5：市场上涨 10%，资产上涨 5%；市场下跌 10%，资产下跌 5%
- β = 2.0：市场上涨 10%，资产上涨 20%；市场下跌 10%，资产下跌 20%

## 模型假设

CAPM 建立在以下假设之上：

| 假设 | 说明 |
|------|------|
| 同质性预期 | 所有投资者对资产的均值、方差、协方差有相同预期 |
| 无风险借贷 | 可以无风险利率无限借入或贷出资金 |
| 单期优化 | 投资者追求单期财富期望效用最大化 |
| 完全市场 | 无交易成本、资产可细分、流动性充足 |
| 价格接受者 | 任何投资者的买卖行为不影响价格 |
| 资产固定 | 资产数量总和固定 |

## 资本市场线与证券市场线

### 资本市场线（CML）

- **定义**：无风险资产与市场投资组合的连线
- **坐标**：X 轴为标准差（风险），Y 轴为期望报酬率
- **含义**：若 CAPM 成立，所有投资者选择的投资组合必须在 CML 上

### 证券市场线（SML）

- **定义**：β = 1 时的市场报酬率与无风险利率的连线
- **坐标**：X 轴为 β 值，Y 轴为期望报酬率
- **含义**：若 CAPM 成立，所有投资组合必须位于 SML 上

## 与 APT 的对比

| 维度 | CAPM | APT |
|------|------|-----|
| 因子数量 | 单因子（市场风险） | 多因子（多个系统性风险） |
| 市场有效性 | 假设市场完美有效 | 市场可能暂时错误定价 |
| 因子选择 | 明确（市场组合） | 灵活（需自行识别） |
| 理论基础 | 均值-方差优化 | 无套利原理 |
| 实证表现 | 较弱 | 较好 |
| 实务应用 | 简单易用 | 更灵活 |

详见：[[套利定价理论]]

## 局限性

### 实证检验问题

1. **Roll 批判（1977）**：
   - CAPM 在实证中无法验证历史投资收益
   - 两种可能：模型无效 或 模型设定误差

2. **多因子模型的诞生**：
   - 基于模型设定误差的假设，APT 和 Fama-French 等多因子模型诞生
   - 详见：[[套利定价理论]]、[[Fama-French因子模型]]

### 理论假设过强

1. **同质性预期不现实**：投资者预期存在差异
2. **无交易成本不成立**：实际存在各种交易成本
3. **单期模型**：实际投资为多期决策
4. **市场组合无法观测**：真正的市场组合不可得

## 应用场景

### 1. 确定必要报酬率

CAPM 广泛用于估值分析中确定必要报酬率（Required Return of Equity）：

$$r_e = R_f + \beta \times (R_m - R_f)$$

### 2. 资本成本估算

用于计算公司的加权平均资本成本（WACC）：

$$WACC = w_e \times r_e + w_d \times r_d \times (1 - t)$$

其中 $r_e$ 通过 CAPM 计算。

### 3. 投资组合绩效评估

- 计算经风险调整的收益
- 评估基金经理的选股能力

### 4. 项目投资决策

- 计算项目的必要报酬率
- 评估项目是否创造价值

## A 股应用

### 无风险利率选择

- **推荐**：10 年期国债收益率
- **原因**：期限适中、流动性好、信用风险低

### 市场组合选择

- **推荐**：沪深 300 指数
- **原因**：覆盖面广、代表性强、流动性好

### 市场风险溢价

- **历史水平**：A 股市场风险溢价约 6-8%
- **计算方法**：历史平均法或前瞻法

### 贝塔系数计算

1. **数据选择**：
   - 期间：2-5 年
   - 频率：周度或月度数据
   - 市场指数：沪深 300 或中证 800

2. **计算方法**：
   - 线性回归：$R_i - R_f = \alpha + \beta \times (R_m - R_f)$
   - 公式法：$\beta = \frac{Cov(R_i, R_m)}{Var(R_m)}$

3. **调整方法**：
   - 布鲁姆调整：$\beta_{adjusted} = 0.33 + 0.67 \times \beta_{raw}$
   - 原因：贝塔系数有均值回归倾向

### A 股特殊情况

| 情况 | 处理方法 |
|------|---------|
| 涨跌停限制 | 使用复权价格，注意流动性风险 |
| 停牌 | 剔除停牌期间数据或使用替代指标 |
| 新股 | 上市不足 1 年的数据谨慎使用 |
| 市场剧烈波动 | 增加数据窗口或使用 GARCH 模型 |
