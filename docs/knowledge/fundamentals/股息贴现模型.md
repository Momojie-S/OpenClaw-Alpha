# 股息贴现模型

## 定义

股息贴现模型（Dividend Discount Model，DDM）是一种绝对估值方法，其核心思想是**股票的内在价值等于其未来所有预期股息的现值总和**。

投资者购买股票的最终回报来自于股息收入，因此将未来所有股息按适当折现率折现到当前时点，即可得到股票的合理价值。

## 基本公式

**一般形式**：

$$P_0 = \sum_{t=1}^{\infty} \frac{D_t}{(1+r)^t}$$

其中：
- $P_0$ = 股票内在价值
- $D_t$ = 第 t 年预期股息
- $r$ = 投资者要求的必要回报率（折现率）

由于预测无限期股息不现实，实际应用中采用简化形式。

## 模型变种

### 1. 戈登增长模型（Gordon Growth Model, GGM）

**适用场景**：股息以恒定比率永久增长的公司

**公式**：

$$P_0 = \frac{D_1}{r - g} = \frac{D_0 \times (1+g)}{r - g}$$

其中：
- $D_1$ = 下一年预期股息
- $D_0$ = 当前股息
- $g$ = 股息恒定增长率
- $r$ > $g$（必要条件）

**假设条件**：
- 股息永久支付
- 股息增长率恒定
- 折现率大于增长率
- 公司盈利和分红政策稳定

### 2. 两阶段 DDM

**适用场景**：公司先经历高增长期，再进入稳定增长期

**估值步骤**：
1. 计算高增长期（通常 3-5 年）股息现值
2. 计算高增长期结束时的终值（用 GGM）
3. 将终值折现到当前
4. 两部分现值相加

**公式**：

$$P_0 = \sum_{t=1}^{n} \frac{D_0 \times (1+g_s)^t}{(1+r)^t} + \frac{P_n}{(1+r)^n}$$

其中：
- $g_s$ = 高增长期增长率
- $n$ = 高增长期年数
- $P_n$ = 终值（用 GGM 计算）

### 3. 三阶段 DDM

**适用场景**：公司经历"高增长→过渡→稳定"三个阶段

**三个阶段**：
1. **高速增长期**：股息高增长
2. **过渡期**：增长率逐步下降
3. **稳定增长期**：增长率稳定，永续经营

**特点**：更符合公司成长规律，避免增长率突变

### 4. H 模型

**适用场景**：增长率从高到低线性下降

**公式**：

$$P_0 = \frac{D_0 \times (1+g_L)}{r - g_L} + \frac{D_0 \times H \times (g_S - g_L)}{r - g_L}$$

其中：
- $g_S$ = 短期高增长率
- $g_L$ = 长期稳定增长率
- $H$ = 高增长期半衰期（若高增长期 T 年，则 H = T/2）

**第一部分**：稳定增长价值
**第二部分**：高增长带来的额外价值

## 适用条件

| 条件 | 说明 |
|------|------|
| **稳定分红** | 公司有长期、稳定的股息支付历史 |
| **可预测增长** | 股息增长率相对可预测 |
| **成熟公司** | 公用事业、银行、大型消费品等成熟行业 |
| **高股息率** | 股息率较高，是投资回报重要组成部分 |

## 不适用场景

- 不分红或低分红公司（成长股）
- 股息波动大的公司
- 处于高速成长期的公司
- 利润再投资优于分红的公司

## 优缺点

### 优点

- **简单直观**：公式易于理解和计算
- **关注现金流**：强调实际获得的现金回报
- **理论严谨**：基于现值原理，逻辑清晰
- **适合价值投资**：与长期持有、关注分红的价值理念一致

### 缺点

- **假设严格**：恒定增长假设难以满足
- **参数敏感**：对增长率和折现率微小变化高度敏感
- **范围有限**：不适用于不分红或股息不稳的公司
- **忽略其他价值**：无法反映股票回购、利润再投资等价值

## A 股应用

### 适用行业

| 行业 | 原因 |
|------|------|
| **银行** | 分红稳定、股息率高、增长可预测 |
| **公用事业** | 现金流稳定、分红政策明确 |
| **高速公路** | 收入稳定、分红率高 |
| **电力** | 现金流充裕、股息稳定 |
| **部分央国企** | "中特估"背景下，高股息率受关注 |

### 应用要点

1. **选择合适模型**：根据公司增长特征选择 GGM、两阶段或三阶段模型
2. **参数估计谨慎**：
   - 增长率 (g)：参考历史股息增长率、行业增速、GDP 增速
   - 折现率 (r)：用 CAPM 计算，考虑无风险利率、β值、市场风险溢价
3. **敏感性分析**：对关键参数进行敏感性测试，了解估值区间
4. **结合其他方法**：与 PE、PB、DCF 等方法交叉验证

### 局限性

- A 股许多公司股息政策不稳定
- 市场情绪和资金偏好可能使股价与股息表现背离
- 投资者可能更关注资本利得而非股息收入

## 注意事项

1. **参数敏感性**：当 r 接近 g 时，估值结果波动剧烈
2. **增长率限制**：长期增长率不应超过 GDP 增速
3. **分红可持续性**：检查股息支付率是否合理（通常 < 80%）
4. **股息真实性**：警惕"高股息陷阱"，关注盈利质量和现金流
5. **与其他回报形式**：考虑股票回购等非股息回报
