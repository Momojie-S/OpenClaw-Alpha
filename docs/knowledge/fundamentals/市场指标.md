# 市场指标

## 定义

反映市场交易活跃度和价格波动的量化指标，用于判断流动性、情绪和风险。

---

## 交易活跃度

### 换手率

**定义**：成交量与流通股本的比率，反映股票交易活跃程度。

**公式**：
```
换手率 = 成交量 / 流通股本 × 100%
```

**应用场景**：
- 判断市场关注度
- 识别主力资金动向
- 评估流动性风险

**注意事项**：
- 高换手率可能是出货或建仓，需结合价格判断
- 不同股票流通盘差异大，横向对比需谨慎
- 次新股换手率天然较高

**参考标准**：
| 换手率 | 状态 | 含义 |
|--------|------|------|
| < 1% | 低 | 关注度低，流动性差 |
| 1%-3% | 正常 | 交易平稳 |
| 3%-7% | 活跃 | 关注度提升 |
| 7%-10% | 高度活跃 | 可能有大动作 |
| > 10% | 超高 | 密集换手，注意风险 |

---

### 量比

**定义**：当前成交量与过去 5 日平均成交量的比率，反映成交量相对变化。

**公式**：
```
量比 = 当前成交量 / 过去5日平均成交量
```

**应用场景**：
- 判断成交量异常放大
- 识别突破有效性
- 短线交易信号

**注意事项**：
- 量比 > 1 表示放量
- 需结合价格走势判断（放量涨 vs 放量跌）
- 开盘初期量比不稳定

**参考标准**：
| 量比 | 状态 | 含义 |
|------|------|------|
| < 0.5 | 极缩量 | 交易冷清 |
| 0.5-1 | 缩量 | 低于平均 |
| 1-2 | 正常 | 平稳交易 |
| 2-5 | 放量 | 明显放大 |
| > 5 | 巨量 | 异常波动 |

---

### 成交额

**定义**：成交金额总和，反映市场资金参与规模。

**公式**：
```
成交额 = 成交量 × 成交价格
```

**应用场景**：
- 评估市场资金规模
- 判断大盘资金流向
- 筛选流动性好的股票

**注意事项**：
- 大盘股成交额天然较高
- 需结合换手率看活跃度
- 关注成交额变化趋势

---

## 价格波动

### 振幅

**定义**：当日最高价与最低价的差额占前收盘价的百分比。

**公式**：
```
振幅 = (最高价 - 最低价) / 前收盘价 × 100%
```

**应用场景**：
- 评估价格波动程度
- 判断多空分歧
- 设置止损止盈参考

**注意事项**：
- 振幅大说明分歧大
- 需结合成交量看
- 涨跌停股票振幅失真

---

### 涨跌幅

**定义**：当前价格与前收盘价的变动百分比。

**公式**：
```
涨跌幅 = (当前价 - 前收盘价) / 前收盘价 × 100%
```

**应用场景**：
- 评估价格变化
- 判断趋势方向
- 设置涨跌停限制

**注意事项**：
- A股涨跌停限制：主板 10%、创业板/科创板 20%、ST 5%
- 北交所涨跌停 30%
- 新股上市前 5 日无涨跌停限制

---

### 委比

**定义**：委买量与委卖量的差额占委托总量的百分比。

**公式**：
```
委比 = (委买量 - 委卖量) / (委买量 + 委卖量) × 100%
```

**应用场景**：
- 判断买卖盘力量对比
- 短期价格方向参考
- 盘口分析

**注意事项**：
- 委托单可随时撤销，参考价值有限
- 需结合实际成交看
- 大单委托可能是假象

**参考标准**：
| 委比 | 状态 |
|------|------|
| > 50% | 强势买盘 |
| 20%-50% | 买盘占优 |
| -20%~20% | 相对均衡 |
| -50%~-20% | 卖盘占优 |
| < -50% | 强势卖盘 |

---

## 资金流向

### 主力资金净流入

**定义**：大单成交的买入金额减去卖出金额。

**公式**：
```
主力净流入 = 大单买入金额 - 大单卖出金额
```

**应用场景**：
- 判断主力动向
- 识别资金关注标的
- 辅助判断趋势持续性

**注意事项**：
- 大单定义因平台而异（通常 50 万以上）
- 主力可能拆单隐藏意图
- 需结合价格走势判断

---

### 北向资金

**定义**：通过沪港通/深港通流入 A 股的外资。

**应用场景**：
- 判断外资对 A 股态度
- 关注外资偏好标的
- 市场情绪参考

**注意事项**：
- 外资偏好龙头白马
- 长期趋势更有参考价值
- 短期波动可能是调仓

---

## 指标体系

| 维度 | 核心指标 | 辅助指标 |
|------|---------|---------|
| 交易活跃度 | 换手率 | 量比、成交额 |
| 价格波动 | 振幅 | 涨跌幅、委比 |
| 资金流向 | 主力净流入 | 北向资金、龙虎榜 |
