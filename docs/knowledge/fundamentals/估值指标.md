# 估值指标

## 定义

评估股票价格是否合理的指标体系，通过比较股价与公司财务数据，判断当前价格相对价值。

---

## 常见指标

### PE（市盈率）

**定义**：股价与每股收益的比率，反映投资回本年限。

**公式**：
```
PE = 股价 / 每股收益 (EPS)
PE = 总市值 / 净利润
```

**类型**：
- 静态 PE：用上一年年报数据
- 动态 PE：用当年预测数据
- TTM PE：用最近 12 个月数据（最常用）

**应用场景**：
- 同行业公司对比
- 判断是否被低估（低于历史分位）
- 评估成长股定价合理性（PEG = PE / 增长率）

**注意事项**：
- 亏损企业 PE 无意义
- 周期股在利润高点时 PE 反而低（陷阱）
- 不同行业 PE 差异大，跨行业对比无意义
- 需剔除非经常性损益影响

---

### PB（市净率）

**定义**：股价与每股净资产的比率，反映资产溢价程度。

**公式**：
```
PB = 股价 / 每股净资产
PB = 总市值 / 净资产
```

**应用场景**：
- 重资产行业估值（银行、地产、制造业）
- 判断是否跌破资产价值（PB < 1）
- 周期股底部识别

**注意事项**：
- 轻资产公司（科技、服务）PB 意义不大
- 资产质量影响 PB 可比性（需关注资产减值）
- 商誉高的公司 PB 会失真

---

### PS（市销率）

**定义**：股价与每股销售额的比率，反映市场对收入的定价。

**公式**：
```
PS = 股价 / 每股销售额
PS = 总市值 / 营业收入
```

**应用场景**：
- 亏损或微利成长股估值
- SaaS、互联网公司（早期不盈利）
- 同行业收入规模对比

**注意事项**：
- 不考虑盈利能力，高 PS 不代表高价值
- 收入质量影响 PS 意义（需关注毛利率）
- 低毛利业务会拉低 PS

---

### PCF（市现率）

**定义**：股价与每股现金流的比率，反映现金流支撑程度。

**公式**：
```
PCF = 股价 / 每股现金流
PCF = 总市值 / 经营现金流
```

**应用场景**：
- 评估公司造血能力
- 现金流稳定的公司估值（公用事业）
- 识别利润质量（利润高但现金流差的公司）

**注意事项**：
- 现金流波动大的公司 PCF 不稳定
- 资本支出大的公司需用自由现金流（FCF）
- 季节性行业需看全年数据

---

## 指标对比

| 指标 | 适用行业 | 适用场景 | 局限性 |
|------|----------|----------|--------|
| PE | 盈利稳定行业 | 价值判断、行业对比 | 亏损无效、周期陷阱 |
| PB | 重资产行业 | 资产价值判断、周期底部 | 轻资产无效、商誉影响 |
| PS | 成长型公司 | 亏损期估值、规模对比 | 忽略盈利能力 |
| PCF | 现金流稳定行业 | 利润质量验证 | 波动大、受资本支出影响 |

---

## 应用原则

### 行业适配

| 行业类型 | 首选指标 | 辅助指标 |
|---------|---------|---------|
| 银行/保险 | PB | PE |
| 消费/制造 | PE | PB |
| 科技成长 | PS + PEG | PE |
| 公用事业 | PCF | PE |
| 周期股 | PB（底部） | PE（顶部反向） |

### 历史分位

- **低估**：当前估值 < 历史 20% 分位
- **合理**：当前估值在历史 20%-80% 分位
- **高估**：当前估值 > 历史 80% 分位

### 组合使用

- PE + PB：综合判断估值
- PE + PEG：判断成长性定价
- PE + PCF：验证利润质量
