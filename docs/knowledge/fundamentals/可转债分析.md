# 可转债分析

## 定义

**可转换债券（Convertible Bond，简称可转债）**：一种可以在特定时期、按特定条件转换为发行公司普通股的债券。兼具债券的防御性和股票的进攻性，是"进可攻、退可守"的投资工具。

---

## 核心特征

### 双重属性

| 属性 | 说明 |
|------|------|
| **债性** | 固定利息、到期还本、保底价值 |
| **股性** | 转股权、上涨弹性、与正股联动 |

### 基本条款

| 条款 | 说明 |
|------|------|
| **转股价** | 转换为股票的价格（转股比例 = 面值 / 转股价） |
| **转股期** | 允许转股的时间窗口（通常发行后 6 个月） |
| **赎回条款** | 正股持续上涨时，发行人可强制赎回 |
| **回售条款** | 正股持续下跌时，投资者可回售给发行人 |
| **下修条款** | 正股大幅下跌时，发行人可下调转股价 |

---

## 价值构成

### 可转债价值公式

```
可转债价值 ≈ 纯债价值 + 期权价值（转股权）
```

| 组成部分 | 含义 | 影响因素 |
|----------|------|----------|
| **纯债价值** | 作为普通债券的价值 | 票息、剩余期限、信用利差、利率水平 |
| **期权价值** | 转股权的价值 | 正股价格、波动率、剩余期限、转股价 |

### 价值边界

```
可转债价格 ≥ max(纯债价值, 转股价值)
```

- **纯债价值**：可转债的"地板价"
- **转股价值**：立即转股可获得的价值

---

## 核心指标

### 转股价值

```
转股价值 = 正股价格 × (100 / 转股价)
```

**示例**：
- 正股价格：15 元
- 转股价：12 元
- 转股价值 = 15 × (100 / 12) = 125 元

**含义**：转股价值 > 100，说明转股有利可图。

### 转股溢价率

```
转股溢价率 = (可转债价格 - 转股价值) / 转股价值 × 100%
```

**示例**：
- 可转债价格：135 元
- 转股价值：125 元
- 转股溢价率 = (135 - 125) / 125 = 8%

**解读**：
| 溢价率 | 含义 |
|--------|------|
| < 0% | 折价，套利机会（罕见） |
| 0% ~ 10% | 低溢价，股性强 |
| 10% ~ 30% | 正常区间 |
| > 30% | 高溢价，股性弱 |
| > 50% | 极高溢价，偏离正股 |

**注意**：溢价率会随时间衰减（时间价值损耗）。

### 纯债溢价率

```
纯债溢价率 = (可转债价格 - 纯债价值) / 纯债价值 × 100%
```

**含义**：衡量可转债价格相对于纯债价值的溢价程度。

### 纯债到期收益率（YTM）

```
YTM = 计算持有到期的内部收益率
```

**解读**：
| YTM | 含义 |
|-----|------|
| > 0% | 债性保护强 |
| ≈ 0% | 债性保护一般 |
| < 0% | 债性保护弱，价格已反映期权价值 |

## 双低策略

### 定义

**双低**：低转股溢价率 + 低可转债价格

**逻辑**：
- 低溢价率：股性强，与正股联动紧密
- 低价格：债底保护，下跌空间有限

### 筛选标准（参考）

| 指标 | 筛选条件 |
|------|---------|
| 可转债价格 | < 115 元 |
| 转股溢价率 | < 30% |
| 剩余期限 | > 1 年 |
| 信用评级 | AA- 及以上 |
| 正股质地 | 排除 ST、亏损股 |

### 双低值计算

```
双低值 = 可转债价格 + 转股溢价率 × 100
```

**示例**：
- 可转债价格：110 元
- 转股溢价率：15%
- 双低值 = 110 + 15 = 125

**应用**：双低值越低，性价比越高。

## 风险与收益特征

### 收益来源

| 来源 | 条件 | 说明 |
|------|------|------|
| **正股上涨** | 正股涨幅 > 转股价 | 转股价值提升，转债跟涨 |
| **套利收益** | 转股溢价率 < 0% | 买入转债 + 转股 + 卖出股票 |
| **利息收入** | 持有期间 | 票息（通常较低） |
| **下修收益** | 转股价下修 | 转股价值提升 |

### 主要风险

| 风险 | 说明 |
|------|------|
| **正股下跌** | 转股价值下降，转债价格跟跌 |
| **溢价率压缩** | 时间流逝，溢价率自然下降 |
| **强制赎回** | 触发赎回条款，被迫转股或卖出 |
| **信用风险** | 发行人违约（低评级转债） |
| **流动性风险** | 部分转债成交清淡 |

### 不对称收益

| 情景 | 收益特征 |
|------|---------|
| 正股大涨 | 转债跟随上涨，收益接近正股（扣除溢价） |
| 正股大跌 | 跌至债底附近获得支撑，损失有限 |
| 正股横盘 | 溢价率衰减，转债小幅下跌 |

## 关键时点

### 发行阶段

| 时点 | 说明 |
|------|------|
| **发行公告** | 关注发行规模、转股价、票息 |
| **申购日** | 原股东配售 + 网上申购 |
| **上市日** | 新债上市，可能有炒作 |

### 存续期

| 时点 | 说明 |
|------|------|
| **进入转股期** | 转股开始，溢价率趋于收敛 |
| **触发赎回** | 正股持续高于转股价（通常 130%），强赎 |
| **触发下修** | 正股持续低于转股价，可能下修 |
| **触发回售** | 正股大幅下跌，可回售给发行人 |

### 到期

- 未转股部分还本付息
- 通常发行人会促成转股（下修或拉股价）

## 投资策略

### 低价策略

**逻辑**：买入价格接近债底的转债，防守为主。

**筛选**：
- 可转债价格 < 105 元
- YTM > 0%
- 剩余期限 > 2 年

**优点**：下行保护强，波动小
**缺点**：上涨弹性有限

### 低溢价策略

**逻辑**：买入低溢价率转债，跟随正股上涨。

**筛选**：
- 转股溢价率 < 10%
- 正股趋势向上
- 转债流动性好

**优点**：与正股联动紧密，进攻性强
**缺点**：正股下跌时保护有限

### 双低策略

**逻辑**：兼顾债底保护和上涨弹性。

**筛选**：
- 双低值排名前 20
- 正股基本面良好
- 分散持有（10-20 只）

**优点**：攻守平衡，适合中长期
**缺点**：收益弹性不如纯股票

### 条款博弈

**下修博弈**：
- 正股持续下跌，接近下修条件
- 买入等待下修公告
- 下修后转股价值提升

**赎回博弈**：
- 接近触发赎回（正股 > 转股价 × 130%）
- 溢价率收敛至 0
- 转股或卖出

---

## 注意事项

### 信用风险

- 低评级转债（AA- 以下）需关注发行人财务状况
- 违约时债底保护失效

### 流动性

- 小规模转债成交可能清淡
- 大额买卖注意冲击成本

### 强赎风险

- 触发赎回后，溢价率迅速归零
- 未及时操作可能亏损

### 溢价率陷阱

- 高溢价率转债随时间流逝自然贬值
- 即使正股不跌，转债也会跌

### 税收

- 利息收入按债券利息税率（20%）
- 转股后卖出股票按股票收益税率
