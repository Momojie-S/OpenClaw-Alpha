# DCF 估值法

## 定义

DCF（Discounted Cash Flow，现金流折现法）是将企业未来产生的自由现金流折现到当前，计算企业内在价值的绝对估值方法。

**核心思想**：一家公司的价值等于其未来产生的所有现金流折现值之和。

---

## 基本公式

### 通用形式

```
企业价值 = Σ (FCFₜ / (1 + WACC)ᵗ) + 终值 / (1 + WACC)ⁿ

其中：
FCFₜ = 第 t 年自由现金流
WACC = 加权平均资本成本（折现率）
n = 预测期年数
终值 = 预测期后的持续经营价值
```

### 两种 FCF 类型

| 类型 | 公式 | 适用场景 |
|------|------|---------|
| **FCFF**（企业自由现金流） | NOPAT + 折旧 - 资本支出 - 营运资本变化 | 估值整个企业 |
| **FCFE**（股权自由现金流） | FCFF - 利息支出 + 净债务变化 | 估值股权 |

**常用 FCFF 公式**：
```
FCFF = EBIT × (1 - 税率) + 折旧摊销 - 资本支出 - 营运资本增加

简化版：
FCFF = 经营现金流 - 资本支出
```

## DCF 计算步骤

### Step 1: 预测未来现金流

**预测期选择**：
- 一般 5-10 年
- 超过 10 年预测不可靠
- 成熟公司可用更长预测期

**关键假设**：
| 假设 | 影响 | 设定方法 |
|------|------|---------|
| 收入增长率 | 最关键 | 行业增速 + 公司份额变化 |
| EBITDA 利润率 | 决定盈利能力 | 历史趋势 + 行业对标 |
| 税率 | 相对稳定 | 法定税率 25% |
| 资本支出/折旧比 | 影响现金流 | 行业特征 |
| 营运资本占比 | 影响现金流 | 历史比例 |

### Step 2: 计算折现率（WACC）

```
WACC = E/V × Re + D/V × Rd × (1 - Tc)

Re = Rf + β × (Rm - Rf)  [CAPM 模型]

其中：
E/V = 股权占比
D/V = 债务占比
Re = 股权成本
Rd = 债务成本
Tc = 企业所得税率
Rf = 无风险利率（10年国债）
β = 公司贝塔值
Rm - Rf = 股权风险溢价
```

**参数设定**：
| 参数 | 中国市场常用值 |
|------|--------------|
| 无风险利率 | 2.5%-3.0% |
| 股权风险溢价 | 6%-8% |
| β 值 | 行业 β 或公司历史 β |
| 债务成本 | 银行贷款利率或债券收益率 |

### Step 3: 计算终值

**方法一：永续增长法**
```
终值 = FCFₙ₊₁ / (WACC - g)

其中：
FCFₙ₊₁ = 第 n+1 年自由现金流
g = 永续增长率（一般 2%-3%，不超过 GDP 增速）
```

**方法二：退出倍数法**
```
终值 = EBITDAₙ × 退出倍数

退出倍数 = 行业平均 EV/EBITDA
```

**两种方法对比**：
| 方法 | 优点 | 缺点 |
|------|------|------|
| 永续增长法 | 理论严谨 | 对 g 和 WACC 敏感 |
| 退出倍数法 | 简单直观 | 依赖市场估值 |

### Step 4: 折现求和

```
企业价值 = Σ (FCFₜ / (1 + WACC)ᵗ) + 终值 / (1 + WACC)ⁿ

股权价值 = 企业价值 - 净债务
每股价值 = 股权价值 / 总股本
```

## 实务要点

### 敏感性分析

**必须做的测试**：
1. WACC ±1% 对估值的影响
2. 永续增长率 ±0.5% 的影响
3. 增长率假设调整的影响

**示例输出**：
| WACC \ g | 2% | 2.5% | 3% |
|----------|-----|------|-----|
| 8% | 120 | 130 | 142 |
| 9% | 105 | 112 | 120 |
| 10% | 93 | 98 | 104 |

### 终值占比检查

**警示信号**：
- 终值占比 > 70%：预测期可能太短或终值假设过高
- 终值占比 < 40%：可能低估了持续经营价值

**合理范围**：50%-70%

### 分阶段预测

**三阶段模型**：
| 阶段 | 期限 | 特点 |
|------|------|------|
| 高增长期 | 3-5 年 | 增长率 15%+ |
| 过渡期 | 3-5 年 | 增长率逐步下降 |
| 成熟期 | 永续 | 增长率 2-3% |

---

## 适用场景

### 适合 DCF 的公司

- ✅ 现金流稳定可预测（公用事业、消费龙头）
- ✅ 成熟期公司，增长趋势明确
- ✅ 并购估值（绝对价值参考）

### 不适合 DCF 的公司

- ❌ 早期亏损公司（现金流为负）
- ❌ 高增长创业公司（预测极不稳定）
- ❌ 周期股（现金流波动大）
- ❌ 资产驱动型（银行、保险）

---

## 局限性与陷阱

### 假设敏感性

**问题**：微小参数变化导致估值大幅波动。

**示例**：
- WACC 从 9% 降到 8%：估值可能上升 20%+
- g 从 2% 升到 3%：终值可能上升 30%+

**应对**：
- 必须做敏感性分析
- 给出估值区间而非单一值
- 结合相对估值交叉验证

### 预测偏差

**问题**：未来难以预测，假设往往过于乐观。

**常见偏差**：
| 偏差 | 表现 | 应对 |
|------|------|------|
| 过度乐观 | 增长率假设过高 | 保守假设 |
| 忽略竞争 | 利润率假设不变 | 考虑均值回归 |
| 低估风险 | WACC 设定过低 | 加入风险溢价 |

### 终值依赖

**问题**：终值占比过大，实质是"终值游戏"。

**应对**：
- 检查终值占比
- 对比退出倍数法和永续增长法
- 用相对估值验证终值假设

### 垃圾进垃圾出

**问题**：DCF 只是计算框架，价值取决于输入假设。

**关键**：花时间研究业务，而非纠结于 DCF 计算本身。

## 与其他估值方法结合

### DCF + 相对估值

**方法**：
1. DCF 计算绝对价值区间
2. PE/PB/EV-EBITDA 计算相对价值
3. 两者交叉验证

**信号**：
- DCF 低估 + 相对低估 = 强买入信号
- DCF 高估 + 相对高估 = 强卖出信号
- 两者背离 = 需深入研究

### DCF 作为安全边际参考

**Warren Buffett 方法**：
- 用 DCF 估算内在价值
- 只有价格 < 内在价值 × 0.7 时才买入
- 留足 30% 安全边际

## 计算示例

**假设**：
- 当前 FCF = 10 亿
- 未来 5 年增长率：15%, 12%, 10%, 8%, 5%
- WACC = 9%
- 永续增长率 = 3%

**计算**：
```
第1年: 10 × 1.15 = 11.5 → 折现: 11.5 / 1.09 = 10.55
第2年: 11.5 × 1.12 = 12.88 → 折现: 12.88 / 1.09² = 10.84
第3年: 12.88 × 1.10 = 14.17 → 折现: 14.17 / 1.09³ = 10.94
第4年: 14.17 × 1.08 = 15.30 → 折现: 15.30 / 1.09⁴ = 10.84
第5年: 15.30 × 1.05 = 16.07 → 折现: 16.07 / 1.09⁵ = 10.44

预测期价值 = 10.55 + 10.84 + 10.94 + 10.84 + 10.44 = 53.61

终值 = 16.07 × 1.03 / (0.09 - 0.03) = 275.87
终值折现 = 275.87 / 1.09⁵ = 179.32

企业价值 = 53.61 + 179.32 = 232.93 亿
```
