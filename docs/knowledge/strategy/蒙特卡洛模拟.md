# 蒙特卡洛模拟

## 定义

蒙特卡洛模拟是一种通过重复随机抽样来估算复杂系统可能结果的统计方法。在金融领域，用于评估风险和不确定性对投资决策的影响。

## 核心原理

1. **随机抽样**：对不确定变量赋予随机值
2. **模型计算**：基于随机值计算结果
3. **重复模拟**：运行成千上万次
4. **统计汇总**：平均所有结果得到估计值

## 金融应用场景

### 1. [[VaR计算|风险价值计算]]
- 模拟资产价格路径
- 计算置信区间下的最大损失
- 比参数法更灵活，可处理非正态分布

### 2. [[期权定价]]
- 模拟标的资产价格路径
- 计算每条路径的期权收益
- 折现求平均得到期权价值
- 适用：欧式期权、奇异期权

### 3. 组合风险分析
- 模拟多个资产的联合分布
- 评估组合极端损失
- [[压力测试]]场景生成

### 4. 退休规划
- 模拟市场回报序列
- 评估退休资金耗尽概率
- 安全提取率测算

## 股价模拟公式

模拟股价路径的核心方程（几何布朗运动）：

```
S(t+dt) = S(t) × exp((μ - σ²/2)×dt + σ×√dt×Z)

其中：
- S(t)：当前股价
- μ：漂移率（预期收益率）
- σ：波动率
- dt：时间步长
- Z：标准正态随机数
```

## 漂移率计算

```
Drift = 平均日收益率 - 方差/2

（方差/2 是 Itô 校正项，确保几何布朗运动的期望正确）
```

## 模拟步骤

1. **收集历史数据**：计算收益率、波动率、方差
2. **计算漂移**：平均收益 - 方差/2
3. **生成随机冲击**：从标准正态分布抽样
4. **迭代模拟**：按时间步长推进价格
5. **重复路径**：生成数千条价格路径
6. **汇总分析**：计算统计量和风险指标

## 优势与局限

| 优势 | 局限 |
|------|------|
| 处理复杂非线性关系 | 依赖历史数据假设 |
| 可模拟极端情况 | 计算量大 |
| 灵活适应各种分布 | 对模型假设敏感 |
| 直观展示概率分布 | 无法预测黑天鹅 |

## 与其他方法对比

| 方法 | 适用场景 | 局限 |
|------|----------|------|
| 蒙特卡洛 | 复杂衍生品、非正态分布 | 计算慢 |
| 参数法VaR | 正态分布、简单组合 | 低估尾部风险 |
| 历史模拟法 | 有足够历史数据 | 无法预测新情况 |

## 实践注意事项

1. **模拟次数**：通常 10,000 次以上收敛
2. **时间步长**：步长越小越精确，但计算量越大
3. **相关性处理**：多资产需用 Cholesky 分解保持相关性
4. **方差缩减**：对偶变量法、控制变量法提高效率
