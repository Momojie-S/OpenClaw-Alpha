# 因子择时

## 定义

因子择时（Factor Timing）指根据市场状态、经济周期或其他信号，动态调整各因子的权重配置，以期获得比静态因子配置更高的风险调整后收益。

**核心假设**：因子收益具有时变性，可预测

**与因子轮动的区别**：
- 因子择时：连续调整权重（如 30%→25%→20%）
- 因子轮动：离散切换（如动量→价值→质量）

---

## 择时信号来源

### 1. 因子动量（Factor Momentum）

**原理**：过去表现好的因子，短期内可能继续表现好

**方法**：
```
因子权重_t = f(因子收益_t-1 到 t-12)
```

**参数选择**：
- 回看窗口：3-12个月
- 权重函数：线性、指数衰减、排名分位

**实证发现**：
- 短期（1-3个月）动量较强
- 中长期（6-12个月）动量也有效
- 但动量本身也会崩溃

### 2. 宏观对冲（Macro Hedging）

**原理**：不同因子在不同宏观环境下表现不同

| 经济周期 | 表现较好的因子 | 表现较差的因子 |
|----------|---------------|---------------|
| 扩张期 | 动量、小盘、高Beta | 低波动、质量 |
| 衰退期 | 质量、低波动、价值 | 动量、小盘 |
| 通胀期 | 价值、商品因子 | 成长、长久期 |
| 通缩期 | 质量、低波动 | 价值、周期 |

**常用宏观指标**：
- 经济增长：GDP、PMI、工业增加值
- 通胀：CPI、PPI、核心通胀
- 流动性：M2、社融、信用利差
- 利率：国债收益率、收益率曲线斜率

### 3. 估值择时（Valuation Timing）

**原理**：估值便宜的因子未来收益更高

**方法**：
```
因子估值价差 = 高因子组合估值 - 低因子组合估值
因子权重 = f(估值价差的历史分位)
```

**估值指标选择**：
- BP（市净率倒数）
- EP（市盈率倒数）
- CP（现金流率）
- 综合估值分数

### 4. 拥挤度择时（Crowding Timing）

**原理**：拥挤度高的因子未来收益下降

**方法**：
```
因子权重 = base_weight * (1 - crowding_adjustment)
```

**拥挤度信号**：
- 资金流入
- 配对相关性
- 换手率
- 综合拥挤度指数

详见 [[因子拥挤度]]

### 5. 技术指标择时

**原理**：因子指数的技术形态反映供需关系

**常用指标**：
- 移动均线（MA）
- 相对强弱指标（RSI）
- 布林带
- 趋势强度

### 6. 风险平价择时（Risk Parity Timing）

**原理**：根据因子波动率动态调整权重

**方法**：
```
因子权重_i = (1/σ_i) / Σ(1/σ_j)
```

**变体**：
- 波动率倒数
- 风险贡献相等
- 条件波动率（GARCH）

## 择时策略类型

### 1. 独立择时

每个因子独立生成择时信号

```
w_i,t = base_i * signal_i,t
```

**优点**：简单，易理解
**缺点**：忽略因子间相关性

### 2. 联合优化

考虑所有因子，联合优化权重

```
max Σ w_i * E[R_i] - λ * Σ Σ w_i * w_j * Cov(R_i, R_j)
s.t. Σ w_i = 1, w_i ≥ 0
```

**优点**：考虑相关性
**缺点**：对输入敏感，过拟合风险

### 3. 状态切换

根据市场状态切换因子配置

```
状态 = f(宏观变量, 市场变量)
if 状态 == 衰退:
    w = w_recession
elif 状态 == 扩张:
    w = w_expansion
...
```

**方法**：
- 隐马尔可夫模型（HMM）
- 门槛模型
- 机器学习分类

### 4. 多信号融合

结合多个择时信号

```
综合信号 = Σ s_k * signal_k
w = f(综合信号)
```

**融合方法**：
- 等权平均
- 历史绩效加权（IC加权）
- 机器学习（XGBoost、Random Forest）

## 绩效评估

### 择时收益

```
择时收益 = Σ (w_t - w_benchmark) * R_t
```

### 信息系数（IC）

```
IC = corr(预测信号, 实际因子收益)
```

- IC > 0.05：较好
- IC > 0.10：优秀
- IC < 0.02：接近噪音

### 信息比率（IR）

```
IR = E[择时收益] / σ[择时收益]
```

- IR > 0.5：有价值的择时
- IR > 1.0：优秀的择时

### 统计显著性

- t统计量：检验择时收益是否显著不为零
- 单边检验：择时是否显著优于静态配置

## 实证发现与争议

### 乐观观点

- 因子动量在多个市场有效
- 宏观对冲能改善组合表现
- 估值择时在长期有效

### 怀疑观点

- 因子择时收益不稳定，不同时期差异大
- 交易成本可能抵消择时收益
- 数据挖掘风险：回测有效≠实盘有效
- 因子收益预测难度高于股票收益预测

### 学术共识

1. **择时很难**：IC通常较低（0.02-0.08）
2. **长期无效**：择时优势随时间衰减
3. **特定因子有效**：动量、价值因子的择时相对可靠
4. **风险调整后**：扣除交易成本和风险，优势有限

---

## 实践建议

### 保守策略

1. **不择时**：静态等权或风险平价配置
2. **轻度择时**：仅做风险控制（减仓极端拥挤因子）
3. **规则化**：设定明确规则，避免主观判断

### 积极策略

1. **多信号结合**：不依赖单一信号
2. **控制换手**：月度或季度调整，避免过度交易
3. **成本意识**：考虑交易成本、市场冲击
4. **分散化**：同时使用多个因子，降低单一因子择时失败风险

### A股 特点

- 散户占比高，因子收益波动大
- 政策敏感，宏观择时相对有效
- 机构行为（北向、公募）是重要信号
- 因子拥挤度变化快，需要更高频率监控

---

## 注意事项

### 过拟合风险

- 择时参数多，容易过度拟合历史数据
- 解决：样本外检验、Walk-Forward、参数稳定性检验

### 交易成本

- 因子择时需要调整持仓，产生交易成本
- A股交易成本：印花税0.1%（卖出）、佣金0.03%、冲击成本
- 高频择时可能被成本吞噬

### 模型风险

- 择时模型本身可能失效
- 解决：模型监控、多模型备份

### 行为偏差

- 过度自信：高估择时能力
- 确认偏差：只关注支持择时的证据
- 解决：纪律化执行、客观评估
