# 多因子模型

## 定义

将资产收益分解为多个系统性因子（行业、风格）和特质风险的贡献，用于解释和预测投资组合风险来源的金融模型。

---

## 核心思想

**风险分解**：股票收益 = 因子收益 + 特质收益

$$r_i = \sum_{k} \beta_{ik} \cdot f_k + \epsilon_i$$

其中：
- $r_i$ = 股票 i 的收益
- $\beta_{ik}$ = 股票 i 对因子 k 的暴露
- $f_k$ = 因子 k 的收益
- $\epsilon_i$ = 特质收益（无法被因子解释的部分）

---

## Barra 因子体系

Barra 模型是多因子风险模型的代表，包含三大类因子：

### 风格因子（Style Factors）

| 因子 | 英文 | 定义 | 计算指标 |
|------|------|------|----------|
| 规模 | Size | 市值大小 | ln(总市值) |
| 价值 | Value | 估值水平 | B/P、E/P、CF/P |
| 动量 | Momentum | 过去表现 | 过去 12M 收益（剔除最近 1M） |
| 波动率 | Volatility | 价格波动 | 历史波动率、Beta |
| 成长 | Growth | 增长潜力 | 营收/利润增速 |
| 流动性 | Liquidity | 交易活跃度 | 换手率、成交额 |
| 杠杆 | Leverage | 财务杠杆 | 负债率、财务杠杆 |
| 质量 | Quality | 盈利质量 | ROE、盈利稳定性、应计项目 |
| 红利 | Yield | 分红能力 | 股息率 |

### 行业因子

按行业分类（如申万一级行业）捕捉同行业股票"同涨同跌"的特征。

### 市场/国家因子

捕捉宏观经济或国家层面的系统性风险。

---

## 因子暴露（Factor Exposure）

**定义**：股票对某因子的敏感程度，即因子系数 $\beta$

**计算方法**：
- 风格因子暴露 = 标准化后的因子值
- 行业因子暴露 = 0/1（是否属于该行业）

**标准化步骤**：
1. 去极值（MAD 法或 3σ 法）
2. 标准化（Z-score）
3. 行业中性化（回归剔除行业影响）

---

## 因子收益（Factor Return）

**定义**：因子的边际收益，即因子暴露增加 1 个单位带来的收益增量

**估计方法**：
- 截面回归：$r_t = X_t \cdot f_t + \epsilon_t$
- $X_t$ = 因子暴露矩阵
- $f_t$ = 因子收益向量

**用途**：
- 判断哪种风格近期表现好
- 风格轮动信号

---

## 应用场景

### 风险管理

| 功能 | 说明 |
|------|------|
| 风险分解 | 将总风险分解为因子风险 + 特质风险 |
| 风险预测 | 通过因子协方差矩阵预测组合波动 |
| 风险控制 | 限制特定因子暴露，避免风格偏离 |

### 绩效归因

**分解超额收益来源**：
$$r_p - r_b = \sum_k (w_p - w_b)_k \cdot f_k + \alpha$$

- 因子收益贡献 = 权重差 × 因子收益
- Alpha = 无法被因子解释的选股能力

### 组合构建

| 策略 | 方法 |
|------|------|
| Smart Beta | 主动暴露特定因子（如红利、低波） |
| 因子中性 | 控制因子暴露，只追求 Alpha |
| 因子轮动 | 根据因子收益预测动态调整暴露 |

## A 股本土化考量

### 行业分类

- GICS（全球）vs 申万行业（本土）
- 申万分类更符合 A 股投资者习惯

### 因子有效性

| 因子 | A 股特点 |
|------|----------|
| 规模 | 小盘效应明显，但波动大 |
| 价值 | 长期有效，但近年价值陷阱多 |
| 动量 | A 股反转效应更强 |
| 质量 | 逐步受到重视 |

### 数据处理

- **去极值**：MAD 法（中位数绝对偏差）
- **缺失值**：行业均值填充
- **标准化**：Z-score，可选行业中性

## 局限性

- **因子时变**：因子有效性会变化
- **过度拟合**：历史有效不代表未来有效
- **交易成本**：因子轮动带来换手成本
- **模型风险**：模型本身可能失效
