# 商品周期与通胀对冲

## 定义

商品周期是指大宗商品价格的长期波动规律，通常与经济周期、库存周期高度相关。通胀对冲是在通胀环境下，通过配置抗通胀资产保护购买力的策略。

## 商品周期理论

### 超级周期（Super Cycle）

- **周期长度**：15-25 年
- **驱动因素**：
  - 基础设施建设浪潮（中国城镇化、印度工业化）
  - 技术革命带来的新材料需求
  - 供给端长期投资不足
- **典型商品**：铜、铁矿石、原油

### 库存周期（基钦周期）

- **周期长度**：3-4 年
- **阶段划分**：
  | 阶段 | 库存 | 价格 | 需求 | 策略 |
  |------|------|------|------|------|
  | 主动补库 | ↑ | ↑ | ↑ | 做多商品 |
  | 被动补库 | ↑ | ↓ | ↓ | 观望 |
  | 主动去库 | ↓ | ↓ | ↓ | 做空商品 |
  | 被动去库 | ↓ | ↑ | ↑ | 逐步做多 |

### 季节性周期

- **农产品**：播种期、收获期、消费旺季
- **能源**：冬季取暖油、夏季汽油需求
- **贵金属**：印度婚季、中国春节

## 通胀传导机制

```
货币扩张 → 资产价格上涨 → 商品价格上涨 → CPI 上升
     ↓              ↓              ↓
  流动性溢出   生产成本上升   生活成本上升
```

### 通胀敏感资产

| 资产 | 通胀敏感度 | 传导逻辑 | 滞后期 |
|------|-----------|----------|--------|
| 商品期货 | 高 | 直接定价 | 0-3月 |
| 能源股 | 高 | 盈利改善 | 1-3月 |
| 黄金 | 中 | 实物需求 | 3-6月 |
| TIPS | 中 | 本金调整 | 0-1月 |
| REITs | 中 | 租金上涨 | 6-12月 |
| 价值股 | 低-中 | 估值重估 | 6-12月 |

## 通胀对冲策略

### 1. 大宗商品配置

**直接配置**：
- **商品期货**：直接做多商品期货
- **商品 ETF**：DBC、GSG 等指数基金
- **商品股**：能源、矿业、农业股

**间接配置**：
- **资源国货币**：澳元、加元、挪威克朗
- **资源出口国股市**：澳大利亚、加拿大

### 2. TIPS（通胀保值债券）

- **机制**：本金随 CPI 调整，利息按调整后本金计算
- **实际收益率** = 名义收益率 - 预期通胀率
- **优势**：无信用风险，直接对冲 CPI
- **劣势**：流动性较差，实际收益率可能为负

### 3. 黄金配置

**抗通胀逻辑**：
- 实物资产，购买力稳定
- 零息资产，通胀环境下相对价值提升
- 避险属性，对冲货币贬值风险

**配置建议**：
- 正常时期：组合的 5-10%
- 高通胀时期：组合的 10-20%

### 4. 行业配置

**受益行业**：
| 行业 | 受益逻辑 | 敏感度 |
|------|----------|--------|
| 能源 | 油价上涨 → 盈利改善 | 高 |
| 材料 | 商品涨价 → 毛利扩张 | 高 |
| 金融 | 加息预期 → 净息差扩大 | 中 |
| 房地产 | 资产重估 → 估值提升 | 中 |

**受损行业**：
- 消费品（购买力下降）
- 公用事业（成本上升，价格管制）

## 实战框架

### 通胀周期定位

**四阶段框架**：
1. **通胀初期**：商品触底反弹 → 配置商品股、能源
2. **通胀中期**：CPI 持续上升 → 加仓商品、TIPS、黄金
3. **通胀后期**：央行收紧 → 减商品、加现金
4. **通缩期**：需求萎缩 → 减商品、加债券

### 信号指标

**领先指标**：
- M2 增速
- PPI 同比
- 商品指数（CRB）
- 通胀预期（5年盈亏平衡通胀率）

**同步指标**：
- CPI 同比
- 核心通胀率
- 工资增速

**滞后指标**：
- 央行加息
- 长端利率上行

### 配置权重

**基于通胀预期的动态配置**：
```python
# 示例逻辑（非代码）
if 通胀预期 < 2%:
    商品权重 = 5%
    TIPS权重 = 0%
    黄金权重 = 5%
elif 2% <= 通胀预期 < 3%:
    商品权重 = 10%
    TIPS权重 = 5%
    黄金权重 = 10%
elif 3% <= 通胀预期 < 5%:
    商品权重 = 15%
    TIPS权重 = 10%
    黄金权重 = 15%
else:  # 高通胀 > 5%
    商品权重 = 20%
    TIPS权重 = 15%
    黄金权重 = 20%
```

## A 股应用

### 抗通胀板块

1. **上游资源**：煤炭、有色、石油石化
2. **农产品**：种植、养殖、农产品加工
3. **化工**：化纤、农药、化肥

### 传导链条

```
PPI 上升 → 上游资源股盈利改善 → 中游利润承压 → 下游提价能力
```

**投资时序**：
1. **PPI 触底回升**：买上游资源股
2. **PPI 高位**：减上游，关注中游成本控制能力
3. **CPI 上升**：关注下游提价能力强的消费股

### 风险提示

- **政策风险**：价格管制（如煤炭、电力）
- **供给冲击**：地缘政治、天气灾害
- **需求萎缩**：经济衰退导致通胀快速回落
