# 对冲策略

## 定义

对冲策略是通过建立相反或相关的头寸，来降低或消除投资组合风险的策略。

**核心思想**：不完全追求收益最大化，而是在收益和风险之间找到平衡。

## 对冲类型

### 1. 完全对冲

- **目标**：消除所有风险
- **代价**：也消除所有收益
- **应用**：套利交易、锁定收益

### 2. 部分对冲

- **目标**：降低部分风险，保留部分收益
- **代价**：仍承担部分风险
- **应用**：风险管理、波动率控制

### 3. 动态对冲

- **目标**：根据市场变化调整对冲比例
- **代价**：交易成本高
- **应用**：期权做市商、Delta 中性策略

## 常见对冲工具

| 工具 | 优点 | 缺点 | 适用场景 |
|------|------|------|----------|
| 股指期货 | 流动性好、成本低 | 基差风险 | 系统性风险对冲 |
| 股票期权 | 灵活、可定制 | 时间价值损耗 | 下行保护 |
| ETF | 简单、透明 | 追踪误差 | 行业对冲 |
| 反向 ETF | 无需期货账户 | 费用高、长期偏差 | 短期对冲 |
| 可转债 | 保底+上行空间 | 复杂性高 | 个股对冲 |

## 对冲策略类型

### 1. 市场中性

**方法**：做多股票 + 做空股指期货

**公式**：
```
对冲比例 = β × (组合市值 / 期货合约价值)
```

**A 股实践**：
- 用 IF/IC/IM 对冲
- 注意：期货贴水时对冲成本高

### 2. Beta 中性

**方法**：调整多头和空头的 Beta，使组合 Beta = 0

**优点**：更精确的市场中性
**难点**：需要准确的 Beta 估计

### 3. 行业中性

**方法**：在每个行业内做多做空，保持行业权重相同

**优点**：消除行业轮动风险
**缺点**：选股能力要求高

### 4. 配对交易

**方法**：做多股票 A + 做空股票 B（两者高度相关）

**核心逻辑**：
- 价差扩大 → 做多价差小的、做空价差大的
- 价差回归 → 平仓获利

**风险**：
- 相关性失效
- 价差不回归

详见：[[统计套利]]

### 5. 期权对冲

#### 保护性看跌

- **操作**：持有股票 + 买入看跌期权
- **效果**：限定最大损失 = 执行价 - 买入价 - 权利金
- **代价**：权利金

#### 备兑看涨

- **操作**：持有股票 + 卖出看涨期权
- **效果**：增强收益，但限制上行
- **风险**：股价大涨时被迫卖出

详见：[[期权策略组合]]

## 对冲成本

### 1. 显性成本

- 交易手续费
- 买卖价差
- 融资成本（做空）
- 期权权利金

### 2. 隐性成本

- 期货贴水（对冲时支付）
- 追踪误差
- 流动性成本
- 机会成本（放弃上行）

### 3. A 股特有成本

- 融券成本高（年化 8-15%）
- 期货长期贴水（年化 5-10%）
- 涨跌停限制（无法及时对冲）

## A 股对冲实践

### 1. 股指期货对冲

**合约选择**：
- IF（沪深 300）：大盘股对冲
- IC（中证 500）：中盘股对冲
- IM（中证 1000）：小盘股对冲

**对冲比例**：
```
期货手数 = (组合市值 × β) / (期货价格 × 合约乘数)
```

**注意事项**：
- 期货贴水时，对冲成本高
- 季月合约流动性最好
- 展期时有价差风险

### 2. 期权对冲

**适用场景**：
- 大跌风险对冲（保护性看跌）
- 波动率对冲（跨式组合）
- 增强收益（备兑看涨）

**注意事项**：
- 流动性最好的是上证 50ETF 期权
- 深度虚值期权对冲效果差
- 时间价值损耗

### 3. ETF 对冲

**优点**：无需期货账户，操作简单
**缺点**：追踪误差，成本高

**实践**：
- 用沪深 300ETF 对冲大盘股
- 用行业 ETF 对冲行业风险

## 对冲比率优化

### 1. 固定比率

- 简单、易执行
- 缺点：无法适应市场变化

### 2. Beta 加权

```python
对冲比率 = β × (组合市值 / 对冲工具市值)
```

- 更精确
- 缺点：Beta 估计不准确

### 3. 最小方差

**目标**：找到使组合方差最小的对冲比率

**方法**：回归分析
```
组合收益 = α + β × 对冲工具收益 + ε
```

**最优对冲比率** = β

### 4. 动态对冲

- 根据 Delta、Gamma 动态调整
- 需要高频交易
- 成本高

## 风险管理

### 1. 基差风险

- 对冲工具与标的价格不一致
- 解决：选择高度相关的对冲工具

### 2. 相关性风险

- 相关性在危机时失效
- 解决：压力测试 + 分散对冲工具

### 3. 流动性风险

- 对冲工具流动性不足
- 解决：选择流动性好的合约

### 4. 操作风险

- 对冲比例计算错误
- 解决：定期检查 + 自动化系统

## 绩效评估

### 1. 对冲效率

```
对冲效率 = 1 - (对冲后组合波动率 / 原组合波动率)
```

### 2. 成本效益比

```
成本效益比 = 对冲成本 / 降低的风险
```

### 3. 信息比率

```
IR = (对冲后收益 - 基准收益) / 跟踪误差
```
