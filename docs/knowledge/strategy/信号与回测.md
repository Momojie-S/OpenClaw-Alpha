# 信号与回测

## 定义

**信号**：基于特定条件生成的买卖指令，包含时间点、动作（买入/卖出）、强度等信息。

**回测**：用历史数据验证交易策略表现的方法，评估策略的收益、风险和稳定性。

---

## 信号基础

### 信号类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 技术信号 | 基于技术指标 | 均线金叉、RSI 超卖 |
| 资金信号 | 基于资金流向 | 北向资金持续流入 |
| 情绪信号 | 基于市场情绪 | 恐慌指数极端值 |
| 基本面信号 | 基于财务数据 | PE 低于历史分位 |
| 组合信号 | 多维度综合 | 技术 + 资金 + 情绪 |

### 信号结构

```json
{
  "date": "2026-03-08",
  "action": "buy",
  "score": 1,
  "reason": "MA5 上穿 MA20",
  "metadata": {
    "fast_ma": 12.50,
    "slow_ma": 12.30
  }
}
```

**字段说明**：
- `date`：信号触发日期
- `action`：动作类型（buy/sell/hold）
- `score`：信号强度（-1 到 1，正数看多，负数看空）
- `reason`：信号原因（用于复盘）
- `metadata`：原始数据（可选）

### 信号生成

**单条件信号**：
```
IF 条件满足 THEN 生成信号
```

**多条件信号**：
```
IF 条件1 AND 条件2 THEN 生成信号
IF 条件1 OR 条件2 THEN 生成信号
IF 条件1 AND (条件2 OR 条件3) THEN 生成信号
```

**信号过滤**：
- 连续信号去重（避免频繁交易）
- 信号强度阈值（过滤弱信号）
- 时间窗口限制（同一窗口只取最强）

## 回测框架

### 核心要素

| 要素 | 说明 |
|------|------|
| 数据 | 历史价格、成交量等 |
| 信号 | 买卖信号的时序序列 |
| 规则 | 仓位、止损、止盈等 |
| 成本 | 手续费、滑点、冲击成本 |
| 评估 | 收益、风险、稳定性指标 |

### 回测流程

```
历史数据 → 信号生成 → 交易执行 → 持仓计算 → 绩效评估
```

### 数据要求

**必需数据**：
- 开盘价、最高价、最低价、收盘价（OHLC）
- 成交量
- 复权因子（处理分红、拆股）

**可选数据**：
- 财务指标
- 资金流向
- 情绪指标

## 回测评估

### 收益指标

**累计收益率**：
```
累计收益 = (期末资产 - 期初资产) / 期初资产
```

**年化收益率**：
```
年化收益 = (1 + 累计收益)^(252/交易天数) - 1
```

**超额收益**：
```
超额收益 = 策略收益 - 基准收益
```

### 风险指标

**最大回撤**：
```
最大回撤 = max(peak - trough) / peak
```

**波动率**：
```
年化波动率 = 日收益率标准差 × √252
```

**下行波动率**：
```
下行波动率 = 负收益的标准差 × √252
```

### 风险调整收益

**夏普比率**：
```
夏普 = (年化收益 - 无风险利率) / 年化波动率
```
- > 1：良好
- > 2：优秀
- > 3：卓越

**卡玛比率**：
```
卡玛 = 年化收益 / 最大回撤
```
- > 3：优秀

**索提诺比率**：
```
索提诺 = (年化收益 - 无风险利率) / 下行波动率
```
- 关注下行风险，对策略评价更公平

### 交易统计

| 指标 | 说明 |
|------|------|
| 总交易次数 | 买卖信号触发的次数 |
| 胜率 | 盈利交易 / 总交易 |
| 盈亏比 | 平均盈利 / 平均亏损 |
| 持仓天数 | 平均每笔交易持有天数 |
| 最大连续亏损 | 最大回撤期间的连续亏损次数 |

## 回测陷阱

### 前视偏差

**问题**：使用了当时不可得的数据。

**示例**：
- 使用当天收盘价生成信号，当天交易
- 财务数据公告日之前使用

**解决**：
- 使用前一日数据生成信号
- 明确数据可用时间点

### 幸存者偏差

**问题**：只测试现存的股票，忽略已退市的。

**影响**：高估策略表现。

**解决**：
- 使用包含退市股票的全量数据
- 模拟真实投资环境

### 过拟合

**问题**：策略过度适应历史数据，未来表现差。

**表现**：
- 训练集表现极佳
- 测试集表现骤降
- 参数敏感度过高

**解决**：
- 样本外测试
- 参数数量控制
- 交叉验证

### 交易成本低估

**问题**：忽略或低估手续费、滑点、冲击成本。

**影响**：高频策略尤其受影响。

**解决**：
- 设置合理费率（A股双边约 0.1%）
- 考虑滑点（流动性差的股票）
- 大资金考虑冲击成本

### 理想化假设

**问题**：假设总能以收盘价成交、无涨跌停限制等。

**影响**：实际交易难以达到。

**解决**：
- 涨跌停时无法成交的处理
- 流动性不足时的滑点模拟
- 分仓、限仓等实际限制

## 信号质量评估

### 信号有效性

**胜率分析**：
- 买入后 N 日上涨概率
- 信号后收益分布

**分组测试**：
- 按信号强度分组
- 各组收益对比

### 信号衰减

**时间衰减**：
- 信号当天 vs 次日 vs 3 日收益
- 判断信号时效性

**强度衰减**：
- 强信号 vs 弱信号表现差异
- 优化信号阈值

### 信号组合

**独立性检验**：
- 信号 A 和信号 B 相关性
- 避免高度相关信号叠加

**权重优化**：
- 各信号贡献度分析
- 动态调整权重
