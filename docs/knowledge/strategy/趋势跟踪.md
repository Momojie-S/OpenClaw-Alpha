# 趋势跟踪

## 定义

识别并跟随市场趋势的投资策略，核心是"截断亏损，让利润奔跑"。

**核心理念**：
- 趋势一旦形成，倾向于延续
- 不预测市场，只跟随市场
- 小亏大赚，靠少数大趋势盈利

---

## 理论基础

### 行为金融学解释

| 偏差 | 表现 | 趋势来源 |
|------|------|----------|
| **反应不足** | 信息逐步消化 | 价格缓慢调整，形成趋势 |
| **羊群效应** | 跟风买入卖出 | 趋势自我强化 |
| **处置效应** | 过早止盈，过晚止损 | 趋势延续时间更长 |
| **锚定效应** | 参考历史价格 | 趋势中价格持续偏离 |

### 动量效应

**定义**：过去表现好的资产，未来一段时间继续表现好

**时间周期**：
- 短期动量：1-12 个月
- 长期动量：3-5 年（可能反转）

**学术证据**：
- Jegadeesh & Titman (1993)：3-12 个月动量显著
- Moskowitz et al. (2012)：时间序列动量跨资产有效

---

## 趋势识别方法

### 1. 移动平均

**简单均线**：
```
价格 > MA(n) → 上升趋势
价格 < MA(n) → 下降趋势
```

**双均线交叉**：
```
MA(快) > MA(慢) → 金叉，买入
MA(快) < MA(慢) → 死叉，卖出
```

**常见周期**：
| 策略 | 快线 | 慢线 | 特点 |
|------|------|------|------|
| 短线 | 5 | 20 | 信号多，噪音大 |
| 中线 | 10 | 50 | 平衡 |
| 长线 | 20 | 200 | 信号少，捕捉大趋势 |

### 2. 价格突破

**Donchian Channel**：
```
价格 > max(High, n) → 突破上轨，买入
价格 < min(Low, n) → 突破下轨，卖出
```

**典型参数**：n = 20（海龟交易法则）

### 3. 时间序列动量（TSM）

**公式**：
$$Signal_t = sign(r_{t-n:t})$$

**规则**：
- 过去 n 期收益为正 → 买入
- 过去 n 期收益为负 → 卖出（或做空）

**常见周期**：1、3、6、12 个月

### 4. 趋势强度指标

**ADX（Average Directional Index）**：
- ADX > 25：趋势明显
- ADX < 20：无趋势或震荡

**斜率法**：
- 回归线的斜率 > 0：上升趋势
- 回归线的斜率 < 0：下降趋势

## 交易规则

### 入场规则

**趋势确认**：
1. 价格突破前高
2. 均线金叉
3. 动量为正
4. 成交量放大（可选）

**过滤条件**：
- 趋势强度指标确认
- 多周期共振
- 基本面不恶化

### 出场规则

**止损**：
| 止损类型 | 方法 | 特点 |
|---------|------|------|
| 固定止损 | 亏损 N% | 简单直接 |
| ATR 止损 | N × ATR | 适应波动 |
| 移动止损 | 最高价回撤 N% | 保护利润 |
| 均线止损 | 跌破 MA | 趋势反转信号 |

**止盈**：
- 通常不主动止盈
- 让利润奔跑
- 靠移动止损出场

## 仓位管理

### 波动率目标

**原理**：根据目标波动率动态调整仓位

**公式**：
$$Position = \frac{TargetVol}{\sigma_{realized}}$$

**作用**：
- 高波动时降仓位
- 低波动时加仓位
- 保持风险敞口稳定

### 金字塔加仓

**规则**：
- 盈利后才加仓
- 每次加仓量递减
- 最大仓位有限制

**示例**：
```
初始：1 手
盈利 1R 后：+0.5 手
盈利 2R 后：+0.25 手
总仓位：1.75 手（最大）
```

### 风险平价

**原理**：不同资产分配相同的风险贡献

**公式**：
$$w_i = \frac{1/\sigma_i}{\sum_j 1/\sigma_j}$$

**应用**：多资产趋势跟踪组合

## 收益特征

### 正偏态分布

| 特征 | 说明 |
|------|------|
| **胜率低** | 40-50%，甚至更低 |
| **盈亏比高** | 盈利交易平均是亏损的 2-3 倍 |
| **正偏态** | 少数大盈利弥补多次小亏损 |
| **肥尾** | 极端盈利概率高于正态分布 |

### 危机 Alpha

**特点**：危机时可能正收益

**原因**：
- 可以做空
- 危机时趋势明确
- 与股票负相关

**历史案例**：
| 危机 | 趋势跟踪表现 |
|------|-------------|
| 2008 金融危机 | +10% ~ +20% |
| 2022 通胀危机 | +15% ~ +25%（商品牛市） |

---

## 风险

### 震荡市亏损

**问题**：无趋势时反复止损

**表现**：
- 买入后价格回落，止损
- 反向操作后又止损
- "来回打脸"

**应对**：
- 趋势过滤（ADX）
- 降低交易频率
- 多周期确认

### 趋势反转

**问题**：趋势末期入场被套

**表现**：
- 追高买入后趋势反转
- 移动止损来不及
- 单笔大亏损

**应对**：
- 严格止损
- 避免过度加仓
- 结合估值判断

### 滑点成本

**问题**：突破时滑点巨大

**原因**：
- 突破时波动剧烈
- 流动性下降
- 挤兑交易

**应对**：
- 使用限价单（可能错过）
- 分批建仓
- 选择流动性好的标的

## 与均值回归的对比

| 维度 | 趋势跟踪 | 均值回归 |
|------|----------|----------|
| **假设** | 趋势延续 | 极端不可持续 |
| **操作** | 顺势 | 逆势 |
| **买入时机** | 突破、金叉 | 超跌、触及下轨 |
| **卖出时机** | 跌破、死叉 | 超涨、触及上轨 |
| **胜率** | 低（40-50%） | 高（60-70%） |
| **盈亏比** | 高（2-3） | 低（0.5-1） |
| **收益分布** | 正偏态 | 负偏态 |
| **适用市场** | 趋势市 | 震荡市 |
| **心理挑战** | 追涨杀跌 | 逆向操作 |

**组合使用**：
- 判断市场状态（趋势 vs 震荡）
- 趋势期用趋势跟踪
- 震荡期用均值回归
- 通过 ADX、波动率等指标判断

## A 股应用

### 适用场景

| 场景 | 效果 | 说明 |
|------|------|------|
| 大盘趋势 | 中 | A 股牛短熊长，趋势时间短 |
| 个股动量 | 好 | 个股动量效应明显 |
| 商品期货 | 好 | 商品趋势性强 |
| 股指期货 | 中 | 做空受限，贴水成本 |

### 实践建议

1. **多周期组合**：短中长周期信号结合
2. **严格止损**：A 股波动大，止损不能松
3. **控制仓位**：避免单笔大亏
4. **选择标的**：流动性好、趋势性强的标的
5. **结合基本面**：避开基本面恶化的趋势
