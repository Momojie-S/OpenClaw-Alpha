# 均值回归

## 定义

价格或指标偏离长期均值后，倾向于回归均值的现象。是统计套利、逆向投资等策略的理论基础。

**核心假设**：
- 极端状态不可持续
- 长期存在均衡水平
- 偏离会修正

---

## 理论基础

### 统计学原理

**平稳时间序列**：
- 均值恒定
- 方差恒定
- 自相关结构稳定

**均值回归过程**（Ornstein-Uhlenbeck）：
$$dX_t = \theta(\mu - X_t)dt + \sigma dW_t$$

其中：
- $\theta$ = 回归速度
- $\mu$ = 长期均值
- $\sigma$ = 波动率

### 行为金融学解释

| 偏差 | 表现 | 回归机制 |
|------|------|----------|
| 过度反应 | 价格过度偏离 | 信息充分消化后修正 |
| 羊群效应 | 极端情绪推高/压低 | 理性投资者反向操作 |
| 损失厌恶 | 恐慌性抛售 | 价值投资者抄底 |

---

## 应用场景

### 1. 价格均值回归

**相对价值**：
- 两只相关股票价差回归
- 同行业股票相对估值回归

**绝对价格**：
- 个股超跌反弹
- 指数长期均值回归

### 2. 估值均值回归

**常见指标**：

| 指标 | 均值 | 回归周期 |
|------|------|----------|
| PE | 行业/历史均值 | 3-5 年 |
| PB | 1-2 倍 | 3-5 年 |
| 股息率 | 2-4% | 1-3 年 |
| 风险溢价 | 3-6% | 1-3 年 |

### 3. 因子均值回归

| 因子 | 回归现象 |
|------|----------|
| 动量 | 强势股长期回落 |
| 价值 | 低估值回归合理 |
| 波动率 | 高波动回归正常 |

### 4. 情绪均值回归

**情绪指标**：
- 恐慌贪婪指数
- VIX 波动率
- 换手率

**极端情绪**：
- 恐慌极值 → 后续反弹
- 贪婪极值 → 后续调整

## 量化检验

### ADF 检验

**目的**：检验时间序列是否平稳

**原假设**：存在单位根（不平稳，无均值回归）
**备择假设**：不存在单位根（平稳，有均值回归）

**p 值 < 0.05**：拒绝原假设，存在均值回归

### Hurst 指数

$$H = \frac{\log(R/S)}{\log(n/2)}$$

| H 值 | 含义 |
|------|------|
| H = 0.5 | 随机游走 |
| H < 0.5 | 均值回归 |
| H > 0.5 | 趋势延续 |

**理想范围**：0.3-0.45（明显均值回归）

### 半衰期

**定义**：偏离均值衰减一半所需时间

$$Half-life = \frac{\ln(2)}{\theta}$$

**应用**：
- 半衰期短 → 回归快，适合交易
- 半衰期长 → 回归慢，需耐心

## 交易策略

### 布林带策略

**原理**：价格触及布林带边界后回归

**规则**：
- 价格触及下轨 → 买入
- 价格触及上轨 → 卖出
- 价格回归中轨 → 平仓

**参数**：
- 周期：20 日
- 标准差倍数：2

### Z-Score 策略

**计算**：
$$Z = \frac{X - \mu}{\sigma}$$

**交易规则**：

| Z 值 | 操作 |
|------|------|
| Z < -2 | 买入（极端低估） |
| Z > 2 | 卖出（极端高估） |
| |Z| < 0.5 | 平仓 |

### 配对交易

详见 [[统计套利]]

**核心**：
1. 找到协整的股票对
2. 计算价差的 Z-Score
3. 价差极端时建仓
4. 价差回归时平仓

## 风险

### 均值漂移

**问题**：均值本身发生变化

**原因**：
- 公司基本面改变
- 行业格局变化
- 宏观环境变化

**后果**：历史均值失效，回归失败

### 回归时间不确定

**问题**：不知道何时回归

**风险**：
- 偏离可能持续数年
- 资金成本侵蚀收益
- 心理压力

### 极端值更极端

**问题**：偏离可能继续扩大

**原因**：
- 趋势延续
- 信息未充分消化
- 流动性危机

**应对**：严格止损

## A 股应用

### 适用场景

| 场景 | 效果 | 说明 |
|------|------|------|
| 超跌反弹 | 好 | A 股波动大 |
| 风格轮动 | 好 | 价值/成长轮动明显 |
| 行业轮动 | 好 | 行业热度均值回归 |
| 估值修复 | 中 | 需要耐心 |

### 不适用场景

| 场景 | 原因 |
|------|------|
| 强趋势股 | 动量效应更强 |
| 基本面恶化 | 均值已漂移 |
| ST/退市股 | 可能归零 |

### 实践建议

1. **结合基本面**：避免均值漂移
2. **控制仓位**：防止极端更极端
3. **设置止损**：限制单笔损失
4. **分散标的**：降低单一风险
5. **耐心等待**：回归需要时间

## 与趋势的对比

| 维度 | 均值回归 | 趋势跟踪 |
|------|----------|----------|
| 假设 | 极端不可持续 | 趋势延续 |
| 操作 | 逆势 | 顺势 |
| 买入时机 | 超跌 | 突破 |
| 卖出时机 | 超涨 | 跌破 |
| 收益分布 | 高胜率低盈亏比 | 低胜率高盈亏比 |
| 适用市场 | 震荡市 | 趋势市 |
| 心理挑战 | 逆向操作 | 追涨杀跌 |

**组合使用**：
- 震荡期用均值回归
- 趋势期用趋势跟踪
- 通过波动率/趋势强度判断市场状态
