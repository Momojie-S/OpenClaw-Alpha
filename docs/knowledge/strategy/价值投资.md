# 价值投资

## 定义

以低于内在价值的价格买入资产，等待市场修正定价错误的长期投资策略。核心是"用 50 美分买 1 美元的东西"。

**核心理念**：
- 价格 ≠ 价值
- 市场短期是投票机，长期是称重机
- 安全边际是投资的基石

---

## 理论基础

### 格雷厄姆三原则

1. **安全边际**：买入价格显著低于内在价值
2. **市场先生**：市场报价波动，投资者应理性利用
3. **能力圈**：只投资自己理解的领域

### 内在价值

**定义**：资产未来现金流的现值

**DCF 公式**：
$$V = \sum_{t=1}^{\infty} \frac{CF_t}{(1+r)^t}$$

**简化方法**：
- 市盈率倒数（E/P）> 无风险利率
- 市净率 < 行业均值
- 股息率 > 债券收益率

---

## 价值因子的度量

### 常见估值指标

| 指标 | 公式 | 适用场景 |
|------|------|----------|
| P/E | 股价 / 每股收益 | 盈利稳定的公司 |
| P/B | 股价 / 每股净资产 | 资产密集型行业 |
| P/S | 股价 / 每股营收 | 亏损或周期性行业 |
| P/CF | 股价 / 每股现金流 | 盈利波动大的公司 |
| EV/EBITDA | 企业价值 / 息税折旧摊销前利润 | 跨行业比较 |
| 股息率 | 年股息 / 股价 | 成熟企业 |

### 复合价值指标

**Fama-French 价值因子**：
$$HML = \frac{1}{2}(Small\ Value + Big\ Value) - \frac{1}{2}(Small\ Growth + Big\ Growth)$$

**多指标复合**：
$$Value\ Score = w_1 \cdot B/P + w_2 \cdot E/P + w_3 \cdot CF/P$$

## 价值陷阱

### 定义

低估值但基本面持续恶化的股票，买入后继续下跌。

### 识别信号

| 信号 | 说明 |
|------|------|
| PE 低但盈利下滑 | 行业衰退、竞争加剧 |
| PB 低但资产质量差 | 存货积压、坏账风险 |
| 高股息但不可持续 | 分红率超 100% |
| 低估值但高负债 | 财务风险 |
| 历史低估值但无催化剂 | 长期被市场忽视 |

### 避免策略

1. **结合质量因子**：低估值 + 高 ROE
2. **关注盈利趋势**：避开盈利下滑
3. **检查资产负债表**：负债率 < 60%
4. **分析行业前景**：避开夕阳产业
5. **寻找催化剂**：重组、分红政策变化

## A 股价值投资

### 适用行业

| 类型 | 行业 | 原因 |
|------|------|------|
| **高股息** | 银行、电力、高速公路 | 现金流稳定 |
| **周期底部** | 钢铁、煤炭、化工 | 周期低点估值极低 |
| **低估值蓝筹** | 地产、保险 | 估值长期偏低 |
| **困境反转** | ST 股、重组股 | 催化剂明确 |

### A 股特点

| 特点 | 说明 |
|------|------|
| 散户占比高 | 价值发现时间更长 |
| 政策敏感 | 价值股可能受政策压制 |
| 风格轮动快 | 价值与成长轮动明显 |
| 分红文化弱 | 高股息策略相对稀缺 |

### 实践建议

1. **耐心等待**：A 股价值回归可能需要 2-3 年
2. **分散持有**：避免单一价值陷阱
3. **结合风格轮动**：在价值风格占优时加仓
4. **关注机构持仓**：北向资金偏好价值股

## 价值 vs 成长

| 维度 | 价值 | 成长 |
|------|------|------|
| 估值 | 低（PE < 15） | 高（PE > 30） |
| 增长 | 低（< 10%） | 高（> 20%） |
| 分红 | 高 | 低或无 |
| 波动 | 低 | 高 |
| 回撤控制 | 好 | 差 |
| 牛市表现 | 一般 | 优秀 |
| 熊市表现 | 抗跌 | 跌幅大 |

**长期收益**：
- 价值因子长期存在溢价（年化 3-5%）
- 但 A 股近年成长风格占优

## 投资流程

### 自上而下

```
1. 宏观环境 → 判断价值风格是否占优
2. 行业筛选 → 找到低估值的行业
3. 个股选择 → 在行业内选最优标的
4. 组合构建 → 分散持有 10-20 只
5. 动态调整 → 根据估值变化调仓
```

### 自下而上

```
1. 筛选低估股票 → PE < 行业均值 70%
2. 基本面分析 → ROE > 10%，负债率 < 60%
3. 排除价值陷阱 → 检查盈利趋势
4. 估值合理性 → DCF 验证
5. 等待买入 → 价格低于内在价值 20%+
```

## 风险管理

### 主要风险

| 风险 | 说明 | 应对 |
|------|------|------|
| 价值陷阱 | 基本面恶化 | 结合质量因子 |
| 估值陷阱 | 便宜但更便宜 | 设置止损 |
| 时间成本 | 价值回归漫长 | 耐心或结合择时 |
| 风格错配 | 成长风格占优 | 风格轮动配置 |

### 仓位管理

**分批建仓**：
- 初始仓位：20-30%
- 下跌加仓：每跌 10% 加 10%
- 最大仓位：单股 < 10%

**动态调整**：
- 估值修复 50% → 减仓 1/3
- 估值修复完成 → 清仓
