# 有效前沿

## 定义

**有效前沿 (Efficient Frontier)** 是在给定风险水平下，能够获得最高预期收益的投资组合集合。由 Harry Markowitz 于 1952 年提出，是现代投资组合理论 (MPT) 的核心概念。

**核心思想**：通过分散投资，可以在不降低收益的情况下减少风险，或在不增加风险的情况下提高收益。

---

## 均值方差模型

### 基本假设

1. 投资者是理性的，追求效用最大化
2. 投资者根据预期收益和风险（方差）做决策
3. 投资者风险厌恶：同等收益偏好低风险
4. 单期投资
5. 资产收益服从正态分布

### 数学表述

**目标**：在给定风险下最大化收益，或在给定收益下最小化风险

**优化问题**：

```
min  w'Σw               (最小化组合方差)
s.t. w'μ = r_target     (目标收益)
     w'1 = 1            (权重和为1)
     w_i >= 0           (不允许卖空，可选)
```

其中：
- `w` = 权重向量
- `Σ` = 协方差矩阵
- `μ` = 预期收益向量
- `r_target` = 目标收益

## 有效前沿的形状

### 可行集 vs 有效前沿

```
收益
  ^
  |         * 有效前沿
  |      *     （上边界）
  |   *  ·  ·  ·  可行集
  | *   ·  ·  ·  （所有可能组合）
  |*____·__·__·__> 风险
```

**可行集**：所有可能的投资组合

**有效前沿**：可行集的上边界（最优组合）

### 有效组合的特征

- 给定风险下收益最高
- 给定收益下风险最低
- 不存在"支配"它的组合

### 关键点

| 点 | 特征 |
|---|------|
| 最小方差组合 (MVP) | 有效前沿最左端，风险最小 |
| 切点组合 | 与无风险资产线相切，夏普最高 |
| 市场组合 | CAPM 理论中所有投资者持有的组合 |

---

## 两基金分离定理

### 定理内容

任何有效组合都可以由两个有效组合的线性组合构成。

**意义**：
- 只需找到两个有效组合
- 通过调整权重构造任意有效组合
- 简化了投资决策

### 实践应用

1. 选择两个基准组合（如股票指数 + 债券指数）
2. 根据风险偏好调整两者比例
3. 实现任意有效组合

## 加入无风险资产

### 资本配置线 (CAL)

**无风险资产 + 风险组合**

```
E(R) = r_f + (σ_p / σ_r) × (E(R_r) - r_f)
```

其中：
- `r_f` = 无风险利率
- `E(R_r)` = 风险组合预期收益
- `σ_r` = 风险组合波动率

### 切点组合

**夏普比率最高的风险组合**

```
max Sharpe = (E(R_p) - r_f) / σ_p
```

**资本市场线 (CML)**：

```
E(R) = r_f + (E(R_m) - r_f) / σ_m × σ
```

当切点组合 = 市场组合时，CAL 变为 CML。

## 实践步骤

### 1. 估计参数

**预期收益**：
- 历史平均收益
- 分析师预测
- 风险溢价模型

**协方差矩阵**：
- 历史协方差
- 收缩估计（shrinkage）
- 因子模型

### 2. 构建优化

**方法**：
- 二次规划
- 数值优化（如 SLSQP）
- 闭式解（特殊情况）

### 3. 权重约束

常见约束：
- 上下限：`0 <= w_i <= 0.3`
- 行业中性：`sum(w_i in industry) = target`
- 因子暴露限制

---

## 局限性

### 参数敏感性

**问题**：对预期收益和协方差估计非常敏感

**表现**：
- 输入微小变化 → 权重剧烈变化
- 优化结果不直观、不分散
- 实际表现往往不如等权

**解决方法**：
- 收缩估计
- Black-Litterman 模型
- 风险平价（不依赖收益预测）

### 假设问题

| 假设 | 问题 |
|------|------|
| 正态分布 | 实际收益有"厚尾" |
| 单期 | 忽略多期复利效应 |
| 稳定参数 | 协方差矩阵时变 |
| 无交易成本 | 实际有滑点、冲击成本 |

### 估计误差放大

**误差最大化**：优化过程会放大输入误差

**现象**：
- 历史表现好的资产被高配
- 历史低波动资产被高配
- 样本外表现差

---

## 改进方法

### 收缩估计

**协方差矩阵收缩**：

```
Σ_shrink = λ × Σ_sample + (1-λ) × Σ_target
```

- `Σ_sample`：样本协方差
- `Σ_target`：目标结构（如单位阵）
- `λ`：收缩强度

### Black-Litterman 模型

**结合市场均衡收益 + 投资者观点**

详见 [[Black-Litterman模型]]

**优势**：
- 不依赖历史收益
- 观点不确定性可量化
- 权重更合理、分散

### 重抽样有效前沿

**方法**：
1. 从历史数据 Bootstrap 生成多组样本
2. 对每组样本计算有效前沿
3. 平均所有有效前沿的权重

**效果**：减少估计误差影响

### 风险预算/风险平价

**不预测收益，只管理风险**

详见 [[风险平价]]

## A 股应用

### 简化版有效前沿

**资产池**：
- 沪深 300（大盘）
- 中证 500（中盘）
- 创业板指（成长）
- 国债 ETF
- 黄金 ETF

**步骤**：
1. 计算过去 3 年的收益和协方差
2. 对协方差矩阵做收缩处理
3. 设置权重约束（单资产不超过 40%）
4. 求解有效前沿
5. 根据风险偏好选择组合

### 实践建议

1. **保守估计收益**：使用风险溢价模型而非历史平均
2. **协方差收缩**：减少极端权重
3. **加入约束**：限制单资产权重
4. **定期再平衡**：季度调整
5. **样本外验证**：回测表现

## 与其他概念的关系

```
有效前沿（核心理论）
    │
    ├── 无风险资产 ──→ 资本配置线(CAL)
    │                    └── 切点组合 ──→ 夏普最优
    │
    ├── 市场均衡 ──→ Black-Litterman
    │
    └── 等风险贡献 ──→ 风险平价
```
