# 期权策略组合

## 定义

将看涨期权（Call）和看跌期权（Put）按不同行权价、到期日组合，构建具有特定风险收益特征的投资策略。

---

## 策略分类

| 类别 | 适用场景 | 代表策略 |
|------|---------|---------|
| 方向性策略 | 看涨/看跌 | 牛市价差、熊市价差 |
| 波动率策略 | 预期波动变化 | 跨式、宽跨式、蝶式 |
| 收益增强策略 | 持仓增强收益 | 备兑看涨、现金担保卖Put |
| 保护性策略 | 风险对冲 | 保护性看跌、领口策略 |

---

## 方向性策略

### 牛市看涨价差（Bull Call Spread）

**构建**：
- 买入低行权价 Call（K1）
- 卖出高行权价 Call（K2）
- K1 < K2，同期到期

**收益图**：
```
收益
  ↑
  |      ________
  |     /        
  |____/__________
  |   K1   K2   → 标的价格
```

**特征**：
| 项目 | 说明 |
|------|------|
| 最大收益 | K2 - K1 - 净权利金 |
| 最大亏损 | 净权利金（Call买价 - Call卖价） |
| 盈亏平衡 | K1 + 净权利金 |
| 适用场景 | 温和看涨，预期涨幅有限 |

**优点**：
- 降低买入成本（卖出高行权价Call抵消部分成本）
- 风险收益明确

**缺点**：
- 限制上涨收益（超过K2不再获利）

---

### 牛市看跌价差（Bull Put Spread）

**构建**：
- 卖出高行权价 Put（K2）
- 买入低行权价 Put（K1）
- K1 < K2，同期到期

**特征**：
| 项目 | 说明 |
|------|------|
| 最大收益 | 净权利金（Put卖价 - Put买价） |
| 最大亏损 | K2 - K1 - 净权利金 |
| 适用场景 | 温和看涨或横盘 |

**特点**：权利金净收入策略，适合震荡偏强市场。

---

### 熊市看跌价差（Bear Put Spread）

**构建**：
- 买入高行权价 Put（K2）
- 卖出低行权价 Put（K1）
- K1 < K2，同期到期

**收益图**：
```
收益
  ↑
  |______
  |      \
  |       \_______
  |   K1   K2   → 标的价格
```

**特征**：
| 项目 | 说明 |
|------|------|
| 最大收益 | K2 - K1 - 净权利金 |
| 最大亏损 | 净权利金 |
| 适用场景 | 温和看跌，预期跌幅有限 |

---

### 熊市看涨价差（Bear Call Spread）

**构建**：
- 卖出低行权价 Call（K1）
- 买入高行权价 Call（K2）
- K1 < K2，同期到期

**特征**：
| 项目 | 说明 |
|------|------|
| 最大收益 | 净权利金 |
| 最大亏损 | K2 - K1 - 净权利金 |
| 适用场景 | 温和看跌或横盘 |

---

## 波动率策略

### 跨式组合（Straddle）

**构建**：
- 买入 1 份 Call（K）
- 买入 1 份 Put（K）
- 同行权价，同期到期

**收益图**：
```
收益
  ↑
  |    \    /
  |     \  /
  |______\/______
  |       K    → 标的价格
```

**特征**：
| 项目 | 说明 |
|------|------|
| 最大收益 | 无上限（方向涨跌均可） |
| 最大亏损 | 总权利金（Call + Put） |
| 盈亏平衡上 | K + 总权利金 |
| 盈亏平衡下 | K - 总权利金 |
| 适用场景 | 预期大幅波动，方向不确定 |

**适用事件**：
- 财报发布前后
- 重大政策决议
- 并购结果公布

**风险**：
- 时间价值双倍损耗
- 波动率下降时亏损

---

### 宽跨式组合（Strangle）

**构建**：
- 买入 1 份 Call（K2）
- 买入 1 份 Put（K1）
- K1 < K2，同期到期

**收益图**：
```
收益
  ↑
  |      \    /
  |       \  /
  |________\/______
  |    K1   K2  → 标的价格
```

**特征**：
| 项目 | 说明 |
|------|------|
| 最大收益 | 无上限 |
| 最大亏损 | 总权利金（比Straddle便宜） |
| 盈亏平衡上 | K2 + 总权利金 |
| 盈亏平衡下 | K1 - 总权利金 |
| 适用场景 | 预期极度波动，降低成本 |

**vs Straddle**：
- 成本更低（虚值期权便宜）
- 需要更大的波动才能盈利

---

### 卖出跨式（Short Straddle）

**构建**：
- 卖出 1 份 Call（K）
- 卖出 1 份 Put（K）

**收益图**：
```
收益
  ↑
  |______/\
  |      /  \
  |     /    \
  |    K     → 标的价格
```

**特征**：
| 项目 | 说明 |
|------|------|
| 最大收益 | 总权利金 |
| 最大亏损 | 无上限 |
| 适用场景 | 预期波动率下降，价格稳定 |

**风险**：
- 无限亏损可能
- 需严格止损

---

### 蝶式价差（Butterfly Spread）

**构建**（看涨蝶式）：
- 买入 1 份 Call（K1）
- 卖出 2 份 Call（K2）
- 买入 1 份 Call（K3）
- K1 < K2 < K3，K2 = (K1 + K3) / 2

**收益图**：
```
收益
  ↑
  |      /\
  |_____/  \______
  |   K1 K2 K3 → 标的价格
```

**特征**：
| 项目 | 说明 |
|------|------|
| 最大收益 | K2 - K1 - 净权利金 |
| 最大亏损 | 净权利金 |
| 适用场景 | 预期价格稳定在 K2 附近 |

**特点**：
- 低成本、有限风险
- 收益集中在中间行权价

---

### 铁蝶式（Iron Butterfly）

**构建**：
- 买入 1 份 Put（K1）
- 卖出 1 份 Put（K2）
- 卖出 1 份 Call（K2）
- 买入 1 份 Call（K3）
- K1 < K2 < K3

**特征**：
- 类似蝶式，但初始收取权利金
- 最大收益 = 净权利金
- 适用场景：预期价格稳定

---

## 收益增强策略

### 备兑看涨（Covered Call）

**构建**：
- 持有正股 100 股
- 卖出 1 份 Call（K）

**收益图**：
```
收益
  ↑
  |      ________
  |     /
  |____/__________
  |   成本  K   → 标的价格
```

**特征**：
| 项目 | 说明 |
|------|------|
| 最大收益 | (K - 成本) + 权利金 |
| 最大亏损 | 成本 - 权利金（股价跌到0） |
| 适用场景 | 持仓不动，增强收益 |

**适用情况**：
- 持有正股且短期看平
- 愿意在 K 价位卖出
- 增加现金流

**风险**：
- 上涨收益被封顶
- 下跌时保护有限

---

### 现金担保卖Put（Cash-Secured Put）

**构建**：
- 卖出 1 份 Put（K）
- 准备 K × 100 现金作为担保

**特征**：
| 项目 | 说明 |
|------|------|
| 最大收益 | 权利金 |
| 最大亏损 | K - 权利金（股价跌到0） |
| 适用场景 | 愿意在 K 价位买入 |

**两种结果**：
1. 股价 > K：赚取权利金，不买股
2. 股价 < K：被行权，以 K - 权利金 的成本买入

**用途**：
- 在目标价位"挂单"买入
- 赚取等待期间的权利金

---

### 备兑组合（Buy-Write）

**构建**：
- 买入正股的同时卖出 Call

**特点**：
- 与备兑看涨相同
- 强调"买入时即卖出期权"

**适用**：
- 新建仓位时降低成本
- 不追求最大上涨收益

---

## 保护性策略

### 保护性看跌（Protective Put）

**构建**：
- 持有正股 100 股
- 买入 1 份 Put（K）

**收益图**：
```
收益
  ↑
  |      /
  |_____/________
  |   成本  K   → 标的价格
      /
  |__/
```

**特征**：
| 项目 | 说明 |
|------|------|
| 最大收益 | 无上限 |
| 最大亏损 | (成本 - K) + 权利金 |
| 适用场景 | 持仓规避下跌风险 |

**特点**：
- 类似保险，支付权利金换取保护
- 锁定最大亏损

**成本考量**：
- 长期持有成本高
- 可只在关键时点购买（财报前）

---

### 领口策略（Collar）

**构建**：
- 持有正股 100 股
- 买入 1 份 Put（K1）
- 卖出 1 份 Call（K2）
- K1 < 当前价 < K2

**收益图**：
```
收益
  ↑
  |      ________
  |     /
  |____/__________
  |  K1  成本  K2 → 标的价格
```

**特征**：
| 项目 | 说明 |
|------|------|
| 最大收益 | K2 - 成本 + 净权利金 |
| 最大亏损 | 成本 - K1 + 净权利金 |
| 适用场景 | 低成本保护，限制两端 |

**特点**：
- 卖出Call的权利金抵消买入Put的成本
- 可能零成本（权利金相抵）

**适用**：
- 持仓保护，不追求大涨
- 担心下跌但不想花钱买保险

## 策略选择指南

### 按市场观点

| 观点 | 推荐策略 |
|------|---------|
| 强烈看涨 | 买入Call |
| 温和看涨 | Bull Call Spread / Bull Put Spread |
| 横盘震荡 | Short Straddle / Iron Butterfly |
| 温和看跌 | Bear Put Spread / Bear Call Spread |
| 强烈看跌 | 买入Put |
| 波动加大 | Long Straddle / Long Strangle |
| 波动减小 | Short Straddle / Short Strangle |

### 按风险偏好

| 风险偏好 | 推荐策略 |
|---------|---------|
| 保守 | Collar / Protective Put |
| 中等 | Spreads / Covered Call |
| 激进 | Naked Short / Straddles |

### 按持仓情况

| 持仓 | 目标 | 推荐策略 |
|------|------|---------|
| 有股票 | 保护下跌 | Protective Put / Collar |
| 有股票 | 增强收益 | Covered Call |
| 无股票 | 低价买入 | Cash-Secured Put |
| 无股票 | 方向押注 | Spreads / Straddles |

## 期权策略风险

### 希腊字母风险

| 希腊字母 | 风险 | 敏感策略 |
|---------|------|---------|
| Delta | 方向判断错误 | 所有方向性策略 |
| Gamma | 价格剧烈变动 | 卖出期权策略 |
| Theta | 时间价值损耗 | 买入期权策略 |
| Vega | 波动率变化 | Straddle / Strangle |

### 常见陷阱

| 陷阱 | 说明 |
|------|------|
| 裸卖期权 | 无限亏损可能 |
| 忽略保证金 | 强制平仓风险 |
| 过度杠杆 | 小资金大风险 |
| 忽略流动性 | 无法平仓 |
