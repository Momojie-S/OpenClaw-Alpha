# 多信号融合与权重优化

## 定义

将多个独立的交易信号（技术、资金、情绪、基本面等）按照一定规则组合，形成综合决策信号的方法。核心问题是如何分配各信号的权重，以及如何处理信号间的相关性。

---

## 核心问题

### 为什么需要融合？

| 问题 | 单信号局限 | 多信号优势 |
|------|-----------|-----------|
| 噪声 | 单一信号噪声大 | 多信号相互对冲噪声 |
| 失效 | 单信号可能失效 | 多信号分散风险 |
| 信息量 | 单维度信息 | 多维度捕捉市场 |
| 稳定性 | 波动大 | 更平滑稳定 |

### 融合的前提

**信号独立性**：
- 理想情况：信号之间不相关
- 现实情况：多数信号有一定相关性
- 需要检验并处理相关性

**信号有效性**：
- 每个信号单独有预测能力
- 无效信号会稀释组合效果

**信号一致性**：
- 各信号的预测目标一致（如下期收益）
- 时间尺度一致

---

## 融合方法

### 1. 等权组合（Equal Weight）

**公式**：
$$Score = \frac{1}{N} \sum_{i=1}^{N} Signal_i$$

**优点**：
- 简单，无需估计
- 避免估计误差
- 对信号顺序不敏感

**缺点**：
- 忽略信号质量差异
- 相关信号会重复计算

**适用场景**：
- 信号质量相近
- 缺乏历史数据估计权重

---

### 2. IC 加权（IC Weighting）

**公式**：
$$w_i = \frac{IC_i}{\sum_j |IC_j|}$$

$$Score = \sum_i w_i \cdot Signal_i$$

**IC 取绝对值原因**：
- 正 IC 信号正向使用
- 负 IC 信号反向使用

**优点**：
- 根据历史预测能力分配权重
- 质量高的信号权重更大

**缺点**：
- IC 是历史估计，未来可能失效
- 忽略 IC 的稳定性

---

### 3. IR 加权（Information Ratio Weighting）

**公式**：
$$w_i = \frac{IR_i}{\sum_j |IR_j|} = \frac{IC_i / \sigma_{IC,i}}{\sum_j |IC_j / \sigma_{IC,j}|}$$

**优点**：
- 同时考虑预测能力和稳定性
- 稳定的信号权重更高

**适用场景**：
- 信号稳定性差异大
- 追求长期稳定收益

---

### 4. 均值方差优化（Mean-Variance Optimization）

**目标**：最大化组合 IC，控制 IC 波动

**目标函数**：
$$\max_w \quad w^T \mu - \frac{\lambda}{2} w^T \Sigma w$$

其中：
- $\mu$ = 各信号 IC 均值向量
- $\Sigma$ = 信号 IC 协方差矩阵
- $\lambda$ = 风险厌恶系数

**约束**：
- $\sum w_i = 1$（权重和为 1）
- $w_i \geq 0$（非负权重）

**优点**：
- 理论最优
- 考虑信号相关性

**缺点**：
- 估计误差放大（协方差矩阵）
- 对输入参数敏感
- 可能出现极端权重

**改进方法**：
- 收缩估计（Shrinkage）
- 正则化（L1/L2）
- 约束权重范围

---

### 5. 风险平价（Risk Parity）

**核心思想**：各信号对组合风险的贡献相等

**风险贡献**：
$$RC_i = w_i \cdot \frac{(\Sigma w)_i}{\sqrt{w^T \Sigma w}}$$

**目标**：
$$\min \sum_i \sum_j (RC_i - RC_j)^2$$

**优点**：
- 不依赖收益预测
- 自动调整相关性影响
- 稳健性高

**缺点**：
- 忽略信号质量差异
- 可能分配权重给弱信号

---

### 6. 主成分分析（PCA）

**目的**：降维，提取独立信号

**步骤**：
1. 标准化各信号
2. 计算协方差矩阵
3. 特征值分解
4. 取前 K 个主成分

**权重**：
$$Signal_{PC} = \sum_i v_i \cdot Signal_i$$

其中 $v_i$ 为主成分载荷

**优点**：
- 消除信号共线性
- 自动提取主要信号

**缺点**：
- 主成分难以解释
- 可能丢失信息

## 权重优化实践

### 约束优化

**常见约束**：

| 约束类型 | 公式 | 目的 |
|---------|------|------|
| 权重和 | $\sum w_i = 1$ | 标准化 |
| 非负 | $w_i \geq 0$ | 避免反向 |
| 上限 | $w_i \leq w_{max}$ | 分散化 |
| 下限 | $w_i \geq w_{min}$ | 防止零权重 |
| 换手约束 | $\sum |w_t - w_{t-1}| \leq \tau$ | 控制交易成本 |

### 动态权重

**滚动窗口**：
- 使用过去 N 期数据估计权重
- 定期更新（月度/季度）

**衰减加权**：
- 近期数据权重更大
- 指数衰减：$w_{decay} = e^{-\lambda t}$

**自适应权重**：
- 根据信号近期表现调整
- 表现好 → 增加权重

## 信号相关性处理

### 相关性检验

**Pearson 相关系数**：
$$\rho_{ij} = corr(Signal_i, Signal_j)$$

**阈值**：
- |ρ| > 0.7：高度相关，考虑合并或剔除
- |ρ| < 0.3：低相关，可独立使用

### 正交化

**方法**：回归剔除共性

$$Signal'_i = Signal_i - \beta \cdot Signal_j$$

其中 $\beta = \frac{cov(Signal_i, Signal_j)}{var(Signal_j)}$

**效果**：
- 去除信号间共线性
- 每个信号贡献更独立

### 聚类选择

**步骤**：
1. 计算信号相关性矩阵
2. 层次聚类
3. 每类选择代表性信号（IC 最高）

## 信号强度与方向

### 信号标准化

**Z-score**：
$$Signal'_i = \frac{Signal_i - \mu_i}{\sigma_i}$$

**分位数标准化**：
$$Signal'_i = Rank(Signal_i) / N$$

### 方向一致性

**问题**：不同信号方向可能冲突

**处理方法**：
- 多数投票
- 加权投票
- 设置一致性阈值

**示例**：
```
信号1: +1 (买入)
信号2: +1 (买入)
信号3: -1 (卖出)
综合: +1 (2:1 多数)
```

## 评估指标

### 组合 IC

$$IC_{combined} = corr(Score_t, Return_{t+1})$$

**目标**：组合 IC > 单信号 IC

### IC 衰减

$$Decay = \frac{IC_{test} - IC_{train}}{IC_{train}}$$

**判断**：
- Decay < 30%：良好
- Decay > 50%：可能过拟合

### 多空收益

**分组回测**：
- 按组合信号分 5 组
- 多空收益 = Top 组 - Bottom 组

## A 股实践建议

### 信号组合原则

| 原则 | 说明 |
|------|------|
| 简单优先 | 从等权开始，逐步优化 |
| 类别分散 | 技术 + 资金 + 情绪 + 基本面 |
| 相关性控制 | 同类信号避免过高相关 |
| 动态调整 | 定期评估权重有效性 |

### 常见组合模式

**技术 + 资金**：
- 技术信号捕捉趋势
- 资金信号验证主力意愿

**情绪 + 技术**：
- 情绪极端 + 技术超卖 → 反弹
- 情绪极端 + 技术超买 → 回调

**多周期组合**：
- 短期信号（日频）+ 长期信号（周/月频）
- 不同时间尺度信号互补

### 换手控制

**信号平滑**：
$$Signal_{smooth} = \alpha \cdot Signal_t + (1-\alpha) \cdot Signal_{t-1}$$

**最小调整阈值**：
- 权重变化 < 5% 时不调整
