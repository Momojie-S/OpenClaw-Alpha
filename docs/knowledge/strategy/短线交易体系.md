# 短线交易体系

## 定义

短线交易是指在较短时间周期内（通常为 1-5 个交易日）完成买卖的交易方式。A 股短线交易以"打板"（涨停板买入）和"接力"（连板股接力）为核心模式，是 A 股特有的交易生态。

---

## 核心框架

短线交易体系由三个维度构成：**情绪周期 × 题材分析 × 龙头识别**

```
情绪周期（判断时机）
    ↓
题材分析（选择方向）
    ↓
龙头识别（确定标的）
```

---

## 一、情绪周期

情绪周期是短线交易的"天气预报"，决定是否参与、仓位大小。

### 四阶段模型

| 阶段 | 特征 | 仓位 | 操作重点 |
|------|------|------|---------|
| 发酵期 | 冰点复苏，新题材萌芽 | 3-5 成 | 试错 1-2 板 |
| 上升期 | 周期龙确立，赚钱效应好 | 7 成 | 龙头 + 中军 |
| 退潮期 | 龙头断板，亏钱效应 | 1-2 成或空仓 | 防守为主 |
| 混沌期 | 反复拉锯，方向不明 | 视情况 | 把握时间节点 |

### 情绪指标

| 指标 | 观察方法 | 意义 |
|------|---------|------|
| 连板数量 | 涨停股连续涨停数量 | 情绪强度 |
| 涨跌停比 | 涨停数 / 跌停数 | 多空对比 |
| 炸板率 | 炸板数 / 涨停总数 | 封板质量 |
| 昨日涨停表现 | 昨日涨停今日平均涨幅 | 修复能力 |

详见 [[情绪周期理论]]

## 二、题材分析

题材是短线交易的"故事"，决定持续性和空间。

### 题材生命周期

**启动 → 发酵 → 高潮 → 分歧 → 退潮**

| 阶段 | 特征 | 参与价值 |
|------|------|---------|
| 启动期 | 首板出现，无人关注 | 低（不确定性高） |
| 发酵期 | 2-3 板，市场关注 | 高（上车机会） |
| 高潮期 | 一字板，买不到 | 观望 |
| 分歧期 | 炸板、断板 | 谨慎（龙头可能反包） |
| 退潮期 | 普遍回调 | 空仓 |

### 题材评估维度

| 维度 | 问题 | 高分特征 |
|------|------|---------|
| 新颖性 | 是否新题材？ | 首次出现 |
| 确定性 | 催化明确？ | 政策/事件落地 |
| 容量 | 资金容量？ | 大盘股可参与 |
| 持续性 | 能持续多久？ | 有后续催化 |
| 共识度 | 市场认同？ | 机构+游资共振 |

### 强题材特征

> "新、大、真、持续"

- **新**：首次出现，无历史包袱
- **大**：影响范围广，涉及股票多
- **真**：有实质内容，非纯概念
- **持续**：有后续催化事件

详见 [[题材炒作理论]]

## 三、龙头识别

龙头是短线交易的"标的"，决定收益空间。

### 龙头股特征

| 维度 | 特征 |
|------|------|
| 连板高度 | 板块内最高 |
| 封板时间 | 最早（< 10:30） |
| 封单强度 | 最大（> 流通盘 5%） |
| 市值 | 适中（20-100 亿） |
| 人气 | 市场关注度最高 |

### 龙头类型

| 类型 | 定义 | 特点 |
|------|------|------|
| 空间龙 | 板块内连板最高 | 代表板块强度 |
| 周期龙 | 突破市场压制 + 市场认可 | 代表市场情绪 |
| 补涨龙 | 龙头见顶后启动 | 高度有限 |
| 妖股 | 无题材纯情绪炒作 | 波动极大 |

### 周期龙形成条件

1. 突破市场连板压制
2. 经历充分换手
3. 市场广泛认可
4. 整体情绪良好

## 四、买入策略

### 首板挖掘

**条件**：
- 首次涨停
- 封板时间早（< 10:30）
- 有题材支撑
- 市值适中（20-100 亿）

**风险**：首板失败率高，需结合情绪周期判断

### 连板接力

**参与条件**：
- 2-3 板确认趋势
- 题材持续发酵
- 龙头地位确立
- 封板坚决

**参与方式**：
| 位置 | 方式 | 风险 |
|------|------|------|
| 竞价 | 集合竞价买入 | 高（可能高开低走） |
| 打板 | 涨停价买入 | 中（可能炸板） |
| 排板 | 挂单排队 | 低（可能买不到） |

### 弱转强

**定义**：昨日弱势（烂板、炸板回封），今日强势（高开、快速涨停）

**逻辑**：分歧转一致，市场认可度提升

**条件**：
- 竞价高开或平开
- 开盘后快速拉升
- 封板时间早
- 封单量大

### 烂板回封

**定义**：涨停打开后又封回

**机会**：
- 强势股洗盘
- 上车机会

**风险**：
- 可能是主力出货
- 真炸板

详见 [[涨停追踪]]

## 五、卖出策略

### 止盈信号

| 信号 | 操作 |
|------|------|
| 达到目标价 | 按计划止盈 |
| 情绪退潮 | 分批减仓 |
| 龙头断板 | 清仓离场 |
| 放量滞涨 | 减仓观望 |

### 止损信号

| 信号 | 操作 |
|------|------|
| 跌破关键支撑 | 立即止损 |
| 炸板无法回封 | 次日离场 |
| 题材证伪 | 不计成本离场 |
| 买错标的 | 认错出局 |

### 卖出时机

| 时机 | 说明 |
|------|------|
| 竞价卖出 | 开盘即出，避免日内波动 |
| 冲高卖出 | 盘中冲高分批减仓 |
| 尾盘卖出 | 收盘前卖出，避免过夜风险 |

## 六、仓位管理

### 总仓位控制

| 情绪阶段 | 总仓位 | 说明 |
|---------|--------|------|
| 发酵期 | 3-5 成 | 试错为主 |
| 上升期 | 7 成 | 重仓参与 |
| 退潮期 | 1-2 成 | 防守为主 |
| 混沌期 | 0-5 成 | 视情况 |

### 单票仓位

| 类型 | 仓位 | 说明 |
|------|------|------|
| 龙头 | 2-3 成 | 重仓核心 |
| 中军 | 1-2 成 | 辅助配置 |
| 跟风 | 0.5-1 成 | 轻仓试错 |

### 金字塔加仓

```
第一笔：2 成
第二笔：1.5 成（盈利后加）
第三笔：1 成（继续盈利加）
```

**原则**：
- 盈利才加仓
- 越加越少
- 总仓位不超过 7 成

## 七、风险控制

### 高风险信号

| 信号 | 含义 | 操作 |
|------|------|------|
| 天地板 | 极端风险 | 立即离场 |
| 连续炸板 | 多头力竭 | 减仓观望 |
| 放量滞涨 | 出货嫌疑 | 止盈离场 |
| 监管关注 | 可能停牌 | 提前离场 |

### 常见陷阱

| 陷阱 | 特征 | 应对 |
|------|------|------|
| 诱多涨停 | 尾盘偷袭，次日低开 | 不追尾盘板 |
| 对倒涨停 | 成交异常 | 查看龙虎榜 |
| 假龙头 | 跟风涨停，无持续性 | 等待确认 |
| 一日游 | 游资快进快出 | 关注龙虎榜 |

### 回撤控制

- 单日回撤 > 3%：降低仓位
- 单周回撤 > 5%：暂停交易
- 连续亏损 3 笔：休息复盘

## 八、复盘体系

### 每日复盘

1. **周期定位**：确定情绪阶段
2. **题材梳理**：识别主线和支线
3. **龙头分析**：分析龙头表现和竞争
4. **涨停复盘**：统计涨停数据
5. **龙虎榜**：查看资金动向
6. **制定预案**：为次日做推演

### 每周复盘

1. **盈亏统计**：计算周收益率
2. **交易回顾**：分析每笔交易
3. **策略优化**：改进交易系统
4. **情绪回顾**：总结情绪周期演变

---

## 九、心态管理

### 核心原则

> "不是你认为对的就是对的，而是市场认为对的才是对的。"

1. **顺势而为**：不与市场作对
2. **知行合一**：执行交易计划
3. **及时止损**：认错出局
4. **保持冷静**：不被情绪左右

### 常见心理陷阱

| 陷阱 | 表现 | 应对 |
|------|------|------|
| 贪婪 | 该止盈不止盈 | 设定目标价 |
| 恐惧 | 该买入不敢买 | 制定交易计划 |
| 后悔 | 卖飞后追高 | 接受不完美 |
| 报复 | 亏损后重仓博回 | 强制休息 |
