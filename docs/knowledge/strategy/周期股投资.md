# 周期股投资

## 定义

周期股是指业绩和股价与经济周期密切相关的股票，通常在经济扩张期表现优异，在衰退期大幅下跌。

**典型周期行业**：
- 资源类：煤炭、有色金属、钢铁、石油
- 金融类：银行、保险、券商
- 地产类：房地产开发、建材
- 可选消费：汽车、家电、航空

## 周期股特征

### 业绩特征

| 特征 | 说明 |
|------|------|
| 高波动 | 盈利随经济周期大幅波动 |
| 强杠杆 | 经营杠杆高，利润弹性大 |
| 业绩滞后 | 业绩改善滞后于价格信号 |
| 周期分化 | 不同行业周期不同步 |

### 估值特征

| 指标 | 周期股特点 | 投资含义 |
|------|-----------|----------|
| PE | 高峰时低、低谷时高 | 低 PE 可能是卖出信号 |
| PB | 谷底接近 1 甚至破净 | PB < 1 时关注机会 |
| 股息率 | 高股息，但不稳定 | 提供安全边际 |

**⚠️ 周期股估值陷阱**：
- PE 最低时往往是周期顶点（盈利最高）
- PE 最高时可能是周期底部（盈利最差）
- 传统价值投资方法可能失效

## 投资框架

### 周期定位四象限

```
            景气度上升
                ↑
    ② 衰退后期  |  ① 复苏前期
    逢低布局    |  持有待涨
←─────────────────────────→ 流动性/政策
    ③ 衰退前期  |  ④ 景气顶部
    及时止损    |  逐步退出
                ↓
            景气度下降
```

### 买入信号

| 信号 | 说明 |
|------|------|
| PB 破净 | PB < 1，接近历史低位 |
| 产品价格企稳 | 大宗商品价格止跌 |
| 库存见顶回落 | 行业去库存接近尾声 |
| 政策利好 | 供给侧改革、刺激政策 |
| 机构增持 | 公募、社保开始建仓 |

### 卖出信号

| 信号 | 说明 |
|------|------|
| PE 极低 | PE 处于历史低位（盈利高点） |
| 产品价格见顶 | 大宗商品价格回落 |
| 库存高企 | 行业库存创历史新高 |
| 市场狂热 | 全市场看好，估值泡沫 |
| 机构减持 | 公募、外资开始减仓 |

## 周期股 vs 成长股

| 维度 | 周期股 | 成长股 |
|------|--------|--------|
| 估值方法 | PB、EV/EBITDA | PE、PEG、DCF |
| 买入时机 | 景气底部 | 业绩加速期 |
| 卖出时机 | 景气顶点 | 成长逻辑破坏 |
| 持仓周期 | 1-3 年（周期长度） | 3-5 年或更长 |
| 风险来源 | 周期误判 | 估值过高 |
| 核心能力 | 周期判断 | 成长识别 |

## 细分周期投资

### 资源周期（煤炭、有色）

**核心指标**：
- 期货价格（动力煤、铜、铝等）
- 库存水平（港口库存、社会库存）
- 下游需求（电力、地产、汽车）

**投资要点**：
- 供给侧约束是关键（环保、限产）
- 关注期货升贴水结构
- 全球定价品种看美元周期

### 金融周期（银行、券商）

**核心指标**：
- 利率曲线（息差）
- 信用利差
- 资产质量（不良率）

**投资要点**：
- 银行看息差和不良
- 券商看成交量和两融
- 金融股适合左侧布局

### 地产周期

**核心指标**：
- 销售：商品房销售面积、销售额
- 投资：新开工、施工、竣工
- 库存：待售面积、去化周期

**投资要点**：
- 政策是最大变量
- 销售领先投资 6 个月
- 关注龙头集中度提升

### 汽车周期

**核心指标**：
- 汽车销量（乘联会数据）
- 库存系数
- 优惠政策

**投资要点**：
- 政策刺激影响大
- 新能源替代传统车
- 关注车型周期

## 风险与陷阱

### 周期误判风险

| 风险 | 说明 |
|------|------|
| 抄底过早 | 在下跌途中买入，继续亏损 |
| 逃顶过晚 | 在下跌初期没走，利润回吐 |
| 周期延长 | 等待周期反转时间过长 |

### 估值陷阱

- **低 PE 陷阱**：PE 最低时往往是卖点
- **破净陷阱**：破净不代表见底，可能继续下跌
- **高股息陷阱**：高股息可能是盈利高点的信号

### 流动性风险

- 周期股往往在市场恐慌时暴跌
- 下跌时流动性急剧收缩
- 小盘周期股可能无量跌停

## 实战策略

### 左侧交易（逆向）

- 适合：有周期研究能力的投资者
- 方法：在底部区域分批建仓
- 风险：可能抄底过早
- 要点：控制仓位，分批买入

### 右侧交易（趋势）

- 适合：趋势投资者
- 方法：等周期反转信号确认后入场
- 风险：可能错过部分涨幅
- 要点：果断进场，设置止损

### 对冲策略

- 多周期股 + 空商品期货
- 多强势周期股 + 空弱势周期股
- 用期权保护下行风险
