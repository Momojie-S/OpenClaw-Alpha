# 因子有效性检验

## 定义

验证一个因子能否持续、稳定地预测股票未来收益的系统性方法。是构建多因子模型前的关键步骤。

---

## 核心指标

### IC（Information Coefficient）

**定义**：因子值与未来收益的相关系数，衡量因子的预测能力。

**计算方法**：
$$IC_t = corr(Rank(Factor_t), Rank(Return_{t+1}))$$

使用**斯皮尔曼秩相关系数**（Spearman），而非皮尔逊：
- 对异常值不敏感
- 只关注排序关系
- 更适合选股因子

**取值范围**：[-1, 1]

| IC 值 | 含义 |
|-------|------|
| IC = 1 | 完美正向预测（因子高的股票收益最高） |
| IC = -1 | 完美反向预测（因子高的股票收益最低） |
| IC = 0 | 无预测能力 |

**有效性判断**（A 股经验）：

| IC 均值 | 有效性 |
|---------|--------|
| |IC| > 0.10 | 优秀因子 |
| 0.05 < |IC| < 0.10 | 强因子 |
| 0.03 < |IC| < 0.05 | 中等因子 |
| |IC| < 0.03 | 弱因子 |

---

### IR（Information Ratio）

**定义**：衡量因子预测能力的稳定性。

**计算**：
$$IR = \frac{IC_{mean}}{IC_{std}} = \frac{\bar{IC}}{\sigma_{IC}}$$

**意义**：
- IR 高 = IC 稳定，因子可靠
- IR 低 = IC 波动大，因子不稳定

**有效性判断**：

| IR 值 | 稳定性 |
|-------|--------|
| IR > 0.80 | 优质因子，稳定性强 |
| 0.50 < IR < 0.80 | 良好因子 |
| IR < 0.50 | 稳定性不足 |

**注意**：IC 均值相同时，标准差越小，IR 越高。

---

## 检验方法

### 1. IC/IR 法

**步骤**：
1. 计算每期因子值（T 期）
2. 计算每期股票收益（T+1 期）
3. 计算两者的秩相关系数 → IC_t
4. 重复多期，得到 IC 序列
5. 计算 IC 均值、标准差、IR

**分析维度**：

| 指标 | 分析内容 |
|------|----------|
| IC 均值 | 预测方向和强度 |
| IC 标准差 | 预测稳定性 |
| IR | 综合评价 |
| IC > 0 占比 | 正向预测频率 |
| 累计 IC 曲线 | 时间衰减、失效时间点 |

**累计 IC 曲线**：
$$CumIC_t = \sum_{i=1}^{t} IC_i$$

- 曲线稳定向上 = 因子持续有效
- 曲线走平 = 因子开始衰减
- 曲线下降 = 因子可能失效

---

### 2. 分层检验法（Grouping Test）

**原理**：将股票按因子值分组，比较各组未来收益。

**步骤**：
1. 每期按因子值排序
2. 分成 N 组（通常 5 或 10 组）
3. 计算各组下期平均收益
4. 重复多期，统计各组平均收益

**判断标准**：

| 现象 | 含义 |
|------|------|
| 单调性 | 各组收益从低到高递增/递减 |
| Top-Bottom 差距 | 多空收益差显著（>2%/年） |
| 中间组表现 | 可验证线性关系 |

**示例输出**：

| 分组 | 年化收益 |
|------|----------|
| Q1（因子最高） | 18% |
| Q2 | 14% |
| Q3 | 12% |
| Q4 | 9% |
| Q5（因子最低） | 6% |
| **多空收益** | **12%** |

---

### 3. 回归法（Regression Test）

**原理**：通过截面回归检验因子系数的显著性。

**模型**：
$$r_{i,t+1} = \alpha + \beta \cdot Factor_{i,t} + \epsilon_{i,t+1}$$

**检验内容**：

| 统计量 | 含义 |
|--------|------|
| $\beta$ 系数 | 因子边际收益 |
| t 统计量 | 系数显著性（|t| > 2 显著） |
| $R^2$ | 因子解释力 |

**扩展**：加入其他因子做控制变量

$$r_{i,t+1} = \alpha + \beta_1 Factor_{new} + \beta_2 Size + \beta_3 Value + ... + \epsilon$$

检验新因子在已有因子基础上是否仍有增量贡献。

## A 股实践要点

### 数据要求

| 维度 | 要求 |
|------|------|
| 股票数量 | 每期至少 100 只（样本充足） |
| 时间跨度 | 至少 3-5 年（跨牛熊周期） |
| 调仓频率 | 通常月频或周频 |
| 数据质量 | 去除 ST、停牌、新股 |

### 特殊处理

**去极值**：
$$Factor' = \text{clip}(Factor, Median - 3 \times MAD, Median + 3 \times MAD)$$

**行业中性**：
- 先按行业分组
- 组内计算 IC
- 避免 IC 被行业效应主导

**市值中性**：
- 市值大的股票 IC 权重更高？
- 需要控制市值影响

### 常见陷阱

| 陷阱 | 表现 | 解决方法 |
|------|------|----------|
| 前视偏差 | 用未来数据计算因子 | 严格时间对齐 |
| 幸存者偏差 | 只看现存股票 | 包含退市股票 |
| 交易成本 | 忽略换手成本 | 多空收益减去成本 |
| 过度拟合 | 历史 IC 高，实盘失效 | 样本外测试 |

## 因子衰减监控

**监控指标**：

| 指标 | 信号 |
|------|------|
| 滚动 IC 均值 | 连续 6 个月 IC < 0.03 |
| 累计 IC 曲线 | 曲线走平或下降 |
| 多空收益 | 持续收窄 |
| 因子拥挤度 | 拥挤度上升时警惕 |

**应对策略**：
- 降低因子权重
- 暂时剔除因子
- 寻找替代因子

---

## 检验流程总结

```
1. 数据准备
   ├── 获取因子数据（T 期）
   ├── 获取收益数据（T+1 期）
   └── 数据清洗（去极值、缺失值）

2. IC 分析
   ├── 计算每期 IC
   ├── IC 均值、标准差、IR
   └── 绘制累计 IC 曲线

3. 分层回测
   ├── 分组收益计算
   ├── 单调性检验
   └── 多空收益分析

4. 回归检验
   ├── 单因子回归
   └── 多因子控制

5. 稳定性检验
   ├── 分时段 IC
   ├── 滚动 IC
   └── 衰减监控

6. 结论
   ├── 是否有效
   ├── 有效性强弱
   └── 稳定性评估
```
