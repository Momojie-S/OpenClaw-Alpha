# 龙头战法

## 定义

龙头战法是指在题材炒作或行业轮动中，识别并交易板块内表现最强、具有引领作用的"龙头股"的策略。龙头股通常是板块内涨幅最大、人气最高、资金流入最多的股票。

## 核心原理

### 龙头股的特征

| 特征 | 说明 | 识别指标 |
|------|------|----------|
| **率先启动** | 板块内第一个涨停或大涨 | 涨停时间早、涨幅领先 |
| **连板高度** | 连续涨停天数最多 | 3板以上为强龙头 |
| **封单强度** | 涨停后封单金额大 | 封单/流通市值 > 5% |
| **换手充分** | 成交活跃但不过度 | 换手率 10-25% |
| **市值适中** | 便于资金推动 | 流通市值 50-300亿 |
| **题材纯正** | 与热点题材高度相关 | 主营业务契合度高 |

### 龙头的分类

1. **空间龙头**：涨幅最大、连板最高
2. **时间龙头**：率先启动、最先涨停
3. **换手龙头**：充分换手后继续走强
4. **中军龙头**：市值较大、走势稳健

## 龙头识别方法

### 一板定龙头
- 板块启动首日，第一个涨停的股票
- 关注涨停时间：越早越强（9:30-10:00 最强）
- 关注封单强度：封单量大、未炸板

### 二板确认龙头
- 次日继续涨停或大幅高开
- 换手充分但不放量滞涨
- 板块内其他股跟风上涨

### 三板成妖
- 连续三个涨停，市场认可度高
- 开始吸引场外资金关注
- 龙头地位基本确立

### 四板以上成妖股
- 市场情绪极度亢奋
- 风险与收益并存
- 需警惕监管问询和停牌风险

## 买卖时机

### 买入时机

| 时机 | 说明 | 风险等级 |
|------|------|----------|
| **首板打板** | 第一涨停时买入 | 中（可能炸板） |
| **二板接力** | 次日高开或回踩买入 | 中高（成本较高） |
| **三板确认** | 龙头地位确认后买入 | 高（位置已高） |
| **分歧转一致** | 盘中分歧后再次上攻时买入 | 中（需快速判断） |

### 卖出时机

| 信号 | 说明 |
|------|------|
| **炸板无法回封** | 涨停板被打开，多次尝试无法回封 |
| **高位放量滞涨** | 成交量放大但股价不涨 |
| **板块退潮** | 板块内个股集体下跌 |
| **龙头切换** | 出现新的更强龙头 |
| **监管问询** | 收到交易所问询函 |
| **涨幅过大** | 短期涨幅超过 50-100% |

## 仓位管理

### 金字塔建仓法
```
首板：30% 仓位
二板：+30% 仓位（确认后加仓）
三板：+20% 仓位（龙头确立）
预留：20% 现金（应对风险）
```

### 止损策略
- 首板炸板：次日低开 > 5% 止损
- 龙头失败：跌破 5 日线止损
- 板块退潮：无论盈亏全部清仓

## 风险提示

### 高风险因素
1. **追高风险**：龙头股位置已高，回调风险大
2. **炸板风险**：涨停板被打开，无法回封
3. **监管风险**：涨幅过大可能被停牌或问询
4. **情绪退潮**：题材炒作结束，龙头也会大跌
5. **龙头切换**：新龙头出现，旧龙头被抛弃

### 识别假龙头
- 被动跟风涨停，非主动拉升
- 涨停时间晚、封单弱
- 次日低开或大幅回调
- 基本面与题材无关

## A股适用性

### 适用场景
- 市场情绪高涨时期
- 题材炒作初期
- 资金面宽松时期

### 不适用场景
- 市场低迷时期
- 题材炒作末期
- 监管趋严时期

### A股特有风险
- 涨跌停板制度导致流动性风险
- T+1 制度无法当日止损
- 监管停牌风险
- 游资操纵风险

## 实战要点

1. **快进快出**：龙头战法是短线策略，不宜长期持有
2. **设置止损**：入场前设置好止损位
3. **控制仓位**：单只龙头不超过总仓位 30%
4. **顺势而为**：板块退潮时及时止损
5. **心态管理**：接受亏损，不扛单
6. **持续学习**：总结成功和失败案例
