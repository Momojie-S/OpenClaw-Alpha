# 可转债套利策略

## 定义

利用可转换公司债券（Convertible Bonds）与标的股票之间的定价差异，构建对冲组合以获取低风险收益的投资策略。

## 核心原理

可转债价值 = 纯债价值 + 股票看涨期权价值

套利机会来源：
1. **定价偏差** - 股债双重属性导致价格不一致
2. **Delta对冲** - 对冲股价波动风险，锁定波动率收益
3. **折价套利** - 转股溢价率为负时存在套利空间
4. **条款博弈** - 赎回/回售/下修条款创造机会

## 主要策略类型

### 1. Delta对冲套利

**原理**：做多可转债 + 做空正股，对冲股价波动风险

**操作**：
- 计算可转债的Delta值（0-1之间）
- 按Delta比例融券卖出正股
- 组合对股价变动呈中性，享受波动率上升收益

**适用**：
- 波动率较低的标的
- 融券成本可控

**风险**：
- 融券限制（券源、成本）
- Delta值动态变化需调整
- Gamma风险（大幅波动时对冲失效）

### 2. 转股套利（折价转股）

**原理**：转债价格 < 转股价值（转股溢价率为负）

**操作**：
1. 买入折价可转债
2. 收盘前转股
3. T+1日卖出股票

**套利空间**：折价 ≥ 2% 时有较大空间

**风险**：
- **T+1风险**：转股后次日才能卖出，面临隔夜价格波动
- 流动性不足导致滑点
- 交易成本侵蚀收益

### 3. 强制赎回套利

**原理**：博弈公司促成转股的意愿

**触发条件**：
- 正股价 ≥ 转股价 × 130%
- 持续15/30个交易日

**操作**：
- 关注即将满足强赎条件的转债
- 股价回调但公司促转股意愿强时买入
- 博弈公司拉升股价触发强赎

**风险**：
- 公司可能不下修转股价
- 市场环境不支持股价上涨
- 时间成本

### 4. 回售保护套利

**原理**：利用回售条款的价格下限保护

**触发条件**：
- 股价 < 转股价 × 70%
- 持续30个交易日

**操作**：
- 熊市中转债接近回售价时买入
- 作为纯债持有，等待回售或反弹

**风险**：
- 公司可能下修转股价避免回售
- 机会成本
- 信用风险

### 5. 配债套利

**原理**：利用原股东优先配售权

**操作**：
- 股权登记日持有股票获得配售权
- 认购新债，上市后卖出

**风险**：
- 持股期间股价下跌
- 新债上市破发
- 机会成本

### 6. 纯债价值套利

**原理**：转债价格 < 纯债价值

**操作**：
- 寻找跌破纯债价值的转债
- 持有到期获取高收益

**风险**：
- 信用风险
- 流动性风险
- 机会成本

## A股实战要点

### 交易制度限制

| 限制 | 影响 |
|------|------|
| T+1交易 | 转股套利面临隔夜风险 |
| 涨跌停板 | 流动性危机时无法交易 |
| 融券限制 | Delta对冲难度大 |

### 风险控制

1. **流动性风险** - 部分转债成交量小，大资金进出困难
2. **融券风险** - 券源不足、成本高、随时被召回
3. **波动风险** - 市场整体波动导致套利空间消失
4. **政策风险** - 监管变化影响市场
5. **条款风险** - 理解不充分导致损失
6. **交易成本** - 佣金、印花税侵蚀收益

### "双低"策略

选择低价格、低溢价率的转债：
- 低价格：接近纯债价值，下行保护强
- 低溢价率：接近正股价值，弹性好

## 策略组合

### 保守型
- 主配纯债价值套利 + 回售保护套利
- 年化目标：5-8%
- 最大回撤：<5%

### 平衡型
- 转股套利 + 双低转债轮动
- 年化目标：8-15%
- 最大回撤：<10%

### 激进型
- Delta对冲 + 强赎博弈
- 年化目标：15-25%
- 最大回撤：10-20%

## 与其他套利策略的对比
