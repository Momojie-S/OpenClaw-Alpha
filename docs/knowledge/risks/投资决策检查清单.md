# 投资决策检查清单

## 定义

投资决策检查清单是一套标准化的决策流程，帮助投资者在买入/卖出前系统性检查，避免认知偏差和情绪干扰，提高决策质量。

**核心理念**：事前预防 > 事后补救

---

## 买入前检查清单

### 1. 基本面检查

**财务健康**：
- [ ] ROE > 12% 或符合行业特征
- [ ] 经营现金流 > 净利润（盈利质量）
- [ ] 资产负债率 < 60% 或符合行业特征
- [ ] 近 3 年盈利稳定或增长
- [ ] 无重大财务造假嫌疑（红旗指标）

**估值合理**：
- [ ] PE 低于行业均值或历史中位数
- [ ] PB-ROE 匹配（低 PB 高 ROE 更优）
- [ ] 估值分位 < 70%（避免追高）
- [ ] 相对估值与同行比较

**行业前景**：
- [ ] 行业景气度向上或稳定
- [ ] 公司在行业中有竞争优势
- [ ] 非夕阳行业或被颠覆风险低

### 2. 技术面检查

**趋势确认**：
- [ ] 价格在关键均线上方（如 20/60 日线）
- [ ] 均线多头排列（短期 > 中期 > 长期）
- [ ] 近期有放量上涨（资金认可）
- [ ] 未处于明显下降趋势

**买点选择**：
- [ ] 回调至支撑位附近
- [ ] 突破关键阻力位并确认
- [ ] 技术指标未超买（RSI < 70）
- [ ] 量价配合良好

### 3. 情绪面检查

**市场情绪**：
- [ ] 市场情绪未过热（情绪指数 < 80）
- [ ] 板块拥挤度适中（非高拥挤）
- [ ] 北向资金流向未大幅流出
- [ ] 非涨停板上追高

**资金动向**：
- [ ] 主力资金近期流入
- [ ] 北向资金持股稳定或增加
- [ ] 融资余额未异常飙升

### 4. 风险评估

**风险识别**：
- [ ] 无重大利空（业绩暴雷、监管处罚等）
- [ ] 无大规模解禁（未来 1 个月）
- [ ] 无退市风险（ST、财务问题）
- [ ] 行业政策风险可控

**仓位规划**：
- [ ] 单只股票仓位 < 20%
- [ ] 单一行业仓位 < 40%
- [ ] 总仓位符合风险承受能力
- [ ] 预留现金应对加仓或机会

### 5. 认知偏差自查

**避免常见错误**：
- [ ] 不是因为"跌多了"而买入（赌徒谬误）
- [ ] 不是因为"别人都买"而买入（羊群效应）
- [ ] 不是因为"熟悉"而买入（可得性偏差）
- [ ] 不是因为"刚卖掉又涨了"而追回（后悔厌恶）
- [ ] 有明确的投资逻辑，不是"感觉要涨"

**反向思考**：
- [ ] 谁在卖？为什么卖？
- [ ] 如果买错了，最大亏损是多少？
- [ ] 如果 3 个月不涨，能接受吗？

## 卖出检查清单

### 1. 止盈检查

**达到目标**：
- [ ] 达到预设目标价
- [ ] 估值明显高于历史水平
- [ ] 板块拥挤度显著升高
- [ ] 市场情绪过热

**基本面恶化**：
- [ ] 公司业绩低于预期
- [ ] 行业景气度下行
- [ ] 竞争格局恶化
- [ ] 管理层出现问题

### 2. 止损检查

**技术面破位**：
- [ ] 跌破关键支撑位（如 60 日线）
- [ ] 均线空头排列
- [ ] 放量下跌
- [ ] 技术形态破坏

**基本面恶化**：
- [ ] 业绩暴雷或下修
- [ ] 行业逻辑证伪
- [ ] 出现重大利空
- [ ] 财务造假嫌疑

**情绪面极端**：
- [ ] 市场恐慌（情绪指数 < 20）
- [ ] 板块持续杀跌
- [ ] 资金大幅流出

### 3. 认知偏差自查

**避免常见错误**：
- [ ] 不是因为"亏多了"而死扛（损失厌恶）
- [ ] 不是因为"刚解套"就急着卖（处置效应）
- [ ] 不是因为"都跌了"而恐慌卖出（羊群效应）
- [ ] 有明确的卖出理由，不是"感觉要跌"

## 持仓管理检查清单

### 定期检视（每周/每月）

**基本面跟踪**：
- [ ] 查看最新公告和财报
- [ ] 关注行业动态和政策
- [ ] 检查分析师预期变化

**技术面跟踪**：
- [ ] 趋势是否仍然健康
- [ ] 是否出现技术预警信号
- [ ] 支撑/阻力位变化

**组合风险**：
- [ ] 持仓集中度是否过高
- [ ] 行业相关性是否过强
- [ ] 总仓位是否合适

### 异常情况处理

**突发利空**：
1. 评估影响程度（短期 vs 长期）
2. 检查是否影响核心逻辑
3. 如核心逻辑破坏，果断止损
4. 如短期影响，评估是否加仓机会

**市场极端**：
1. 市场恐慌：检查持仓质量，考虑加仓优质股
2. 市场过热：检查持仓估值，考虑逐步减仓

## 交易日程检查清单

### 盘前（9:00-9:25）

**信息收集**：
- [ ] 查看隔夜外盘表现
- [ ] 浏览重大新闻和政策
- [ ] 查看自选股公告
- [ ] 查看北向资金动向

**计划确认**：
- [ ] 回顾昨日交易计划
- [ ] 确认今日操作策略
- [ ] 准备好买入/卖出清单
- [ ] 检查可用资金

### 盘中（9:30-15:00）

**监控要点**：
- [ ] 关注板块轮动
- [ ] 监控持仓异动
- [ ] 跟踪北向资金流向
- [ ] 观察市场情绪变化

**纪律执行**：
- [ ] 按计划执行，不冲动交易
- [ ] 严格止损，不死扛
- [ ] 不追涨停，不抄跌停
- [ ] 记录临时想法，盘后复盘

### 盘后（15:00-17:00）

**复盘总结**：
- [ ] 记录今日交易和理由
- [ ] 分析对错和改进点
- [ ] 更新持仓状态
- [ ] 制定明日计划

## 心理状态检查清单

### 不适合交易的情况

**情绪状态**：
- [ ] 刚经历重大亏损（情绪不稳）
- [ ] 刚经历重大盈利（过度自信）
- [ ] 心情烦躁或焦虑（判断力下降）
- [ ] 身体不适或疲劳（注意力不集中）

**外部环境**：
- [ ] 有重要事情分心
- [ ] 时间紧迫（匆忙决策）
- [ ] 受他人强烈建议影响

### 应对措施

1. **暂停交易**：状态不佳时不交易
2. **降低仓位**：不确定时降低风险
3. **寻求意见**：与他人讨论或等待
4. **记录反思**：记录情绪状态，事后复盘

---

## 使用建议

### 如何使用

1. **打印或保存**：将检查清单打印或保存在手机
2. **强制执行**：每次交易前强制过一遍清单
3. **定期回顾**：每周/每月回顾清单执行情况
4. **持续优化**：根据自己的经验不断完善清单

### 注意事项

- 检查清单是工具，不是保证
- 重点在于避免低级错误，不是预测市场
- 坚持执行比完美更重要
- 根据个人情况定制化
