# 流动性风险

## 定义

流动性风险是指投资者无法在合理时间内、以合理价格买入或卖出资产的风险。

**两个维度**：
- **市场流动性风险**：市场交易活跃度不足，导致难以成交或价格偏离
- **资金流动性风险**：无法及时筹集资金履行支付义务

## 流动性度量指标

### 市场流动性指标

| 指标 | 公式/含义 | 说明 |
|------|----------|------|
| 买卖价差 | 卖一价 - 买一价 | 交易即时成本，越小流动性越好 |
| 换手率 | 成交量 / 流通股本 | 交易活跃度，越高流动性越好 |
| Amihud 非流动性比率 | \|收益率\| / 成交额 | 价格冲击，单位交易量引起的价格变动 |
| 市场深度 | 买卖盘口挂单量 | 大单成交能力 |

### Amihud 非流动性比率

最常用的流动性代理指标，由 Yakov Amihud 于 2002 年提出：

$$ILLIQ = \frac{|R_t|}{VOLD_t}$$

- $R_t$：日收益率
- $VOLD_t$：日成交额
- 高比率 = 低流动性（少量交易引起大幅价格变动）

### 企业财务流动性指标

| 指标 | 公式 | 健康标准 |
|------|------|---------|
| 流动比率 | 流动资产 / 流动负债 | > 1，理想 > 2 |
| 速动比率 | (现金 + 应收) / 流动负债 | > 1 |
| 现金比率 | 现金及等价物 / 流动负债 | > 0.5 |
| 经营现金流比率 | 经营现金流 / 流动负债 | > 1 |

## A股市场流动性特征

### 涨跌停板影响

| 板块 | 涨跌幅限制 | 流动性影响 |
|------|----------|-----------|
| 主板 | ±10% | 涨停难买入，跌停难卖出 |
| ST 股 | ±5% | 流动性更差 |
| 科创板/创业板 | ±20% | 首周无限制，波动更大 |

**流动性枯竭场景**：
- 跌停板上封单巨大 → 无法卖出
- 连续跌停 → 流动性完全丧失
- 小盘股日常成交稀少 → 冲击成本高

### 交易制度限制

- **T+1 制度**：当日买入次日才能卖出，降低日内流动性
- **100 股起买**：小额投资者交易成本较高
- **限价单为主**：缺乏做市商，极端行情下流动性差

### 散户特征

- 散户占比高，交易行为非理性
- 追涨杀跌加剧流动性波动
- 恐慌时集体出逃导致流动性枯竭

## 流动性风险类型

### 1. 交易流动性风险

**场景**：
- 小盘股日成交额低于 1000 万
- 跌停板无法卖出
- 大单交易冲击成本高

**应对**：
- 避免低成交额股票（日成交 < 5000 万）
- 分批建仓/减仓
- 设置流动性筛选条件

### 2. 市场流动性风险

**场景**：
- 市场恐慌时流动性集体枯竭
- 2015 股灾、2020 疫情开盘
- 千股跌停

**应对**：
- 保持现金头寸
- 分散投资
- 使用股指期货对冲

### 3. 资金流动性风险

**场景**：
- 融资盘强平
- 基金赎回压力
- 保证金不足

**应对**：
- 控制杠杆比例
- 预留应急资金
- 避免集中赎回

## 流动性溢价

### 定义

流动性差的资产要求更高的预期收益，以补偿投资者承担的流动性风险。

**实证研究**：
- A股存在显著的流动性风险溢价
- 非流动性股票平均收益更高
- 流动性风险是独立的定价因子

### 应用

| 策略 | 逻辑 |
|------|------|
| 小盘股策略 | 流动性差 → 溢价补偿 |
| 微盘股策略 | 极低流动性 → 高溢价 |
| 低换手率策略 | 被忽视股票 → 价值发现 |

### 注意事项

- 流动性溢价需长期持有才能实现
- 流动性差意味着退出困难
- 极端情况下溢价变为折价

## 流动性调整的 VaR（LVaR）

### 传统 VaR 的局限

传统 VaR 假设可以无限流动性下平仓，低估实际损失。

### LVaR 计算

$$LVaR = VaR + LC$$

- **VaR**：传统风险价值
- **LC（流动性成本）**：买卖价差 + 市场冲击成本

### LC 估算方法

1. **买卖价差法**：$LC = \frac{1}{2} \times Spread \times Position$
2. **价格冲击法**：$LC = \alpha \times \sqrt{Volume}$
3. **Amihud 法**：基于历史价格冲击数据

## 风险管理策略

### 事前预防

| 措施 | 说明 |
|------|------|
| 流动性筛选 | 日成交额 > 5000 万 |
| 仓位限制 | 单只股票 < 5% |
| 分散投资 | 行业、市值分散 |
| 现金预留 | 10-20% 现金应对赎回 |

### 事中控制

| 措施 | 说明 |
|------|------|
| 分批交易 | 大单拆小，降低冲击 |
| VWAP 算法 | 按成交量加权平均价执行 |
| 止损纪律 | 跌破阈值立即减仓 |
| 避免追涨杀跌 | 恐慌时保持理性 |

### 事后复盘

- 记录交易冲击成本
- 评估流动性筛选有效性
- 优化交易执行策略
