# 风险平价

## 定义

风险平价（Risk Parity）是一种投资组合配置策略，核心思想是**让每个资产的风险贡献相等**，而非资金分配相等。

**对比**：

| 策略 | 资金分配 | 风险分配 |
|------|---------|---------|
| 等权重 | 各 50% | 高波动资产主导风险 |
| 风险平价 | 低波动资产更多 | 各 50% |

---

## 核心原理

### 问题：等权重不等于等风险

**示例**：
- 股票：预期波动率 20%
- 债券：预期波动率 5%
- 50/50 配置

**实际风险分布**：
- 股票风险贡献 = 50% × 20% = 10%
- 债券风险贡献 = 50% × 5% = 2.5%
- **股票贡献了 80% 的风险**

### 解决：按风险贡献配置

让每个资产的风险贡献相等：
- 股票风险贡献 = 债券风险贡献
- 需要降低股票权重、提高债券权重

---

## 数学表达

### 风险贡献

资产 i 的风险贡献：
```
RC_i = w_i × σ_i × Σw_j×σ_j×ρ_ij / σ_p
```

其中：
- w_i：资产 i 的权重
- σ_i：资产 i 的波动率
- ρ_ij：资产 i 和 j 的相关系数
- σ_p：组合波动率

### 等风险贡献

优化目标：
```
min Σ(RC_i - RC_j)^2
s.t. Σw_i = 1, w_i ≥ 0
```

### 简化方法：波动率倒数

```
w_i = (1/σ_i) / Σ(1/σ_j)
```

**示例**：
- 股票波动率 20%：权重 = (1/20) / (1/20 + 1/5) = 20%
- 债券波动率 5%：权重 = (1/5) / (1/20 + 1/5) = 80%

## 经典案例：全天候策略

### 桥水全天候（All Weather）

**理念**：在不同经济环境下都能获得稳定收益

**经济环境划分**：

| 环境 | 特征 | 受益资产 |
|------|------|---------|
| 增长超预期 | 通胀稳定、增长上升 | 股票、信用债 |
| 增长低于预期 | 通胀稳定、增长下降 | 国债、通胀挂钩债 |
| 通胀超预期 | 增长稳定、通胀上升 | 商品、通胀挂钩债 |
| 通胀低于预期 | 增长稳定、通胀下降 | 股票、国债 |

**配置**：
- 每种环境分配 25% 的风险
- 总共 4 种环境，风险平分

### 典型配置

| 资产 | 权重 | 风险贡献 |
|------|------|---------|
| 股票 | 15-30% | ~30% |
| 长期国债 | 40-50% | ~30% |
| 中期国债 | 15% | ~15% |
| 商品/黄金 | 10-15% | ~15% |
| 通胀挂钩债 | 5-10% | ~10% |

## 实现方式

### 1. 直接计算

```python
def risk_parity_weights(volatilities):
    """波动率倒数法"""
    inv_vol = 1 / volatilities
    return inv_vol / inv_vol.sum()
```

### 2. 优化求解

```python
def risk_parity_optimize(cov_matrix):
    """等风险贡献优化"""
    from scipy.optimize import minimize
    
    n = len(cov_matrix)
    
    def objective(w):
        port_vol = np.sqrt(w @ cov_matrix @ w)
        marginal_contrib = cov_matrix @ w / port_vol
        risk_contrib = w * marginal_contrib
        # 目标：风险贡献相等
        target_rc = port_vol / n
        return np.sum((risk_contrib - target_rc)**2)
    
    constraints = [{'type': 'eq', 'fun': lambda w: np.sum(w) - 1}]
    bounds = [(0, 1) for _ in range(n)]
    
    result = minimize(objective, x0=np.ones(n)/n,
                      bounds=bounds, constraints=constraints)
    return result.x
```

### 3. 使用杠杆

**问题**：风险平价组合收益较低（债券占比高）

**解决**：对整体组合加杠杆
- 提高收益至目标水平
- 保持风险分散的特性

## 优缺点

### 优点

| 优点 | 说明 |
|------|------|
| 真正的风险分散 | 不被高波动资产主导 |
| 降低单一资产依赖 | 任何单一资产崩盘影响有限 |
| 更稳定的收益曲线 | 波动更均匀分布 |
| 适应不同市场环境 | 不押注单一情景 |

### 缺点

| 缺点 | 说明 |
|------|------|
| 收益可能较低 | 债券占比高 |
| 依赖相关性假设 | 相关性变化影响效果 |
| 杠杆风险 | 加杠杆放大流动性风险 |
| 实现成本 | 频繁再平衡产生交易成本 |

## 实践要点

### 波动率估计

**方法**：
- 历史波动率：简单但滞后
- EWMA：对近期数据加权
- GARCH：考虑波动聚集
- 隐含波动率：前瞻性

**建议**：长期投资用历史+EWMA，短期用隐含波动率

### 相关性处理

**问题**：危机时相关性会上升

**应对**：
- 使用动态相关性（DCC-GARCH）
- 压力测试极端情况
- 配置真正低相关资产（如黄金）

### 再平衡频率

| 频率 | 优点 | 缺点 |
|------|------|------|
| 月度 | 及时调整 | 成本高 |
| 季度 | 平衡 | 推荐 |
| 半年度 | 成本低 | 可能偏离 |

### 杠杆管理

**风险**：
- 流动性风险：紧急时难平仓
- 融资成本：侵蚀收益
- 强制平仓：极端波动时被迫卖出

**控制**：
- 杠杆率 < 2x
- 保持流动性缓冲
- 监控融资成本

## A 股 应用

### 资产选择

**适合作为风险平价组合的资产**：
- 沪深300/中证500（股票）
- 利率债/信用债（债券）
- 黄金 ETF
- 商品 ETF

### 特殊考虑

| 因素 | 影响 |
|------|------|
| 散户占比高 | 波动率估计困难 |
| 政策敏感 | 相关性不稳定 |
| 杠杆工具少 | 加杠杆成本高 |
| 股债相关性 | 可能比成熟市场高 |

### 简化方案

**股债风险平价**：
- 股票：15-25%（根据波动率调整）
- 债券：75-85%

**定期调整**：每季度根据波动率重新计算权重
