# 组合保险策略

## 定义

**组合保险策略**：通过特定的资产配置或衍生品组合，在保证本金安全的前提下，参与市场上涨收益的投资策略。

**核心目标**：下行有底、上行可期。

---

## 两大主流策略

### CPPI - 固定比例投资组合保险

**全称**：Constant Proportion Portfolio Insurance

**核心思想**：将资产分为安全资产和风险资产，动态调整比例。

**运作机制**：
1. **设定最低保险额度**：确定期末希望保留的最低价值
2. **计算安全垫**：当前资产价值 - 最低保险额度
3. **动态配置**：风险资产 = 安全垫 × 风险乘数
4. **剩余配置无风险资产**

**公式**：
```
风险资产金额 = Max(0, 安全垫 × 风险乘数)
安全资产金额 = 总资产 - 风险资产金额
```

**示例**：
- 初始资金：100 万
- 最低保险额度：90 万
- 安全垫：10 万
- 风险乘数：2
- 风险资产配置：10 × 2 = 20 万
- 安全资产配置：80 万

**风险乘数选择**：

| 乘数 | 特点 | 适用场景 |
|------|------|---------|
| 1-2 | 保守，安全垫厚 | 市场波动大、本金保护优先 |
| 2-3 | 平衡 | 一般市场环境 |
| 3-5 | 激进，收益潜力大 | 牛市、风险承受能力强 |

**优点**：
- 操作简单，参数直观
- 自动纪律性调整仓位
- 不依赖衍生品

**缺点**：
- 追涨杀跌特性明显
- 大跌时可能触及底线
- 乘数设定影响大

**市场下跌时的风险**：
- 连续下跌 → 安全垫缩小 → 被迫减仓
- 极端情况：安全垫归零 → 全部转入安全资产
- 称为"仓位冻结"，失去反弹参与机会

---

### OBPI - 期权保险策略

**全称**：Option-Based Portfolio Insurance

**核心思想**：通过购买看跌期权锁定下行风险。

**运作机制**：
1. **直接购买看跌期权**：持有组合 + 买入保护性看跌期权
2. **合成看跌期权**：通过 Delta 对冲动态复制

**保护性看跌期权（Protective Put）**：
```
持有资产 + 买入看跌期权 = 锁定最大损失
最大损失 = 执行价格 - 买入价格 + 权利金
```

**Delta 对冲复制**：
- 计算期权的 Delta 值
- 持有 Delta × 组合价值的风险资产
- 剩余持有无风险资产
- 动态调整

**优点**：
- 上涨时收益可能高于 CPPI
- 锁定明确的下行风险
- 参与上涨无限制

**缺点**：
- 依赖期权市场流动性
- 权利金成本（持续支付）
- 波动率估计复杂
- Delta、Gamma、Vega 等希腊字母敏感

**A股应用挑战**：
- 期权品种有限（主要是 50ETF、300ETF 期权）
- 场外期权门槛高
- 成本相对较高

---

## CPPI vs OBPI 对比

| 维度 | CPPI | OBPI |
|------|------|------|
| **执行方式** | 动态调整资产比例 | 期权或 Delta 对冲 |
| **操作复杂度** | 简单 | 复杂 |
| **衍生品依赖** | 不依赖 | 依赖 |
| **牛市表现** | 参与上涨 | 可能更好 |
| **熊市表现** | 相对稳健 | 明确锁定损失 |
| **震荡市表现** | 可能频繁调仓 | 权利金损耗 |
| **成本** | 交易成本 | 权利金 + 交易成本 |
| **参数敏感性** | 风险乘数 | Delta/Gamma/Vega |

## A股应用场景

### 保本基金 / 避险策略基金

**典型结构**：
- CPPI 策略为主
- 安全资产：高等级债券
- 风险资产：股票、可转债

**历史演变**：
- 早期保本基金广泛使用 CPPI
- 监管趋严后转型为"避险策略基金"

### 固收+产品

**CPPI 模式**：
- 大部分配置债券
- 小部分仓位动态参与权益

**固收+期权模式（OBPI 变体）**：
- 90%+ 配置固收
- 用固收收益购买期权博取增强收益
- 期权挂钩标的：沪深300、中证500、黄金等

### 券商收益凭证

**结构**：
- 保本型：CPPI 思路
- 挂钩型：OBPI 思路

## 风险与局限性

### CPPI 风险

1. **缺口风险**：市场连续跌停，无法及时减仓
2. **仓位冻结**：触及底线后无法参与反弹
3. **追涨杀跌**：高买低卖的内在特性

### OBPI 风险

1. **波动率风险**：隐含波动率变化影响对冲成本
2. **流动性风险**：极端市场期权可能无法买入
3. **模型风险**：Delta 对冲依赖模型假设

### 共同局限

1. **交易成本**：频繁调整带来成本
2. **机会成本**：安全资产收益有限
3. **不适用短线**：策略价值需要时间体现

---

## 实施要点

### CPPI 参数设置

1. **风险乘数**：2-3 为常见，根据风险偏好调整
2. **调仓频率**：日度/周度/月度，平衡成本与精度
3. **调仓阈值**：偏离度超过一定比例才调仓

### OBPI 执行建议

1. **期权选择**：
   - 执行价格：根据保护需求选择
   - 到期日：与投资期限匹配
   - 权利金：不超过组合的 2-3%

2. **Delta 对冲**：
   - 定期再平衡（日度/周度）
   - 关注 Gamma 风险
