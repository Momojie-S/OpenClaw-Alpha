# 价值陷阱

## 定义

表面估值偏低（低 PE/PB），但因公司经营困难、产业衰退等深层问题，导致股价持续低迷或下跌的投资标的。

**核心特征**：便宜有便宜的原因，低估值反映的是基本面恶化预期。

---

## 识别信号

### 财务指标

| 信号 | 说明 | 阈值参考 |
|------|------|----------|
| ROIC < WACC | 资本回报低于成本，损毁价值 | ROIC < 8% |
| 营收/利润下滑 | 增长潜力丧失 | 连续 2 年负增长 |
| 毛利率下降 | 定价权或成本控制弱化 | 同比下降 > 3pct |
| 自由现金流为负 | 盈利质量差 | FCF < 0 |
| 高负债 | 财务压力大 | 资产负债率 > 70% |
| 股息不可持续 | 靠借债分红 | 股息率 > 现金收益率 |

### 非财务信号

- **行业衰退**：技术替代、政策冲击、需求萎缩
- **份额流失**：市场份额持续下降
- **管理层问题**：频繁变动、决策失误、治理缺陷
- **产品老化**：缺乏创新，竞争力下降

## 量化筛选模型

### PEG 陷阱

$$PEG = \frac{P/E}{g}$$

**问题**：
- 依赖增长预测，可能过于乐观
- 低 PEG + 增长不及预期 = 陷阱
- 不适用负增长公司

### ROIC 检验

$$ROIC = \frac{NOPAT}{Invested Capital}$$

**判断**：
- ROIC > WACC → 创造价值
- ROIC < WACC → 损毁价值，可能是陷阱

### 综合评分

```python
# 价值陷阱风险评分示例
def value_trap_score(公司):
    score = 0
    
    # ROIC 低于 WACC
    if ROIC < WACC:
        score += 2
    
    # 营收下滑
    if revenue_growth < 0:
        score += 1
    
    # 毛利率下降
    if gross_margin_change < -0.03:
        score += 1
    
    # 负自由现金流
    if FCF < 0:
        score += 1
    
    # 高负债
    if debt_ratio > 0.7:
        score += 1
    
    return score  # > 3 高风险
```

## A 股特点

1. **"赚股息赔价差"**：高股息但股价持续下跌
2. **周期股低谷**：周期顶点低 PE，实际是周期顶
3. **ST/壳价值**：低市值不代表低估，退市风险
4. **行业政策**：政策变化导致估值体系重构

---

## 避免策略

| 策略 | 说明 |
|------|------|
| 质量筛选 | 结合 ROIC、现金流等质量指标 |
| 行业中性 | 同行业对比，避免行业衰退 |
| 动态跟踪 | 季度财报跟踪基本面变化 |
| 分散投资 | 单只股票仓位控制 |
