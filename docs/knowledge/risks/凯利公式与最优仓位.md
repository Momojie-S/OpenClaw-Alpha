# 凯利公式与最优仓位

## 定义

凯利公式（Kelly Criterion）由 John L. Kelly Jr. 于 1956 年提出，用于计算在给定胜率和赔率下的最优投注比例，以实现长期财富最大化增长率。

---

## 核心公式

### 基本形式

$$f^* = \frac{bp - q}{b}$$

其中：
- $f^*$ = 最优仓位比例
- $b$ = 赔率（盈利/亏损比）
- $p$ = 胜率
- $q$ = 败率（$1 - p$）

### 变换形式

$$f^* = p - \frac{q}{b} = p - \frac{1 - p}{b}$$

### 连续收益形式（用于投资）

$$f^* = \frac{\mu - r}{\sigma^2}$$

其中：
- $\mu$ = 预期收益率
- $r$ = 无风险利率
- $\sigma^2$ = 收益率方差

---

## 直觉理解

凯利公式本质是**增长最优**策略：

1. **仓位过小**：错失复利增长机会
2. **仓位过大**：波动剧烈，可能爆仓
3. **凯利最优**：长期增长率最大化

**关键洞察**：
- 胜率和赔率决定最优仓位
- 不是"梭哈"，也不是"满仓"
- 高胜率 + 高赔率 = 大仓位

---

## 计算示例

### 示例 1：简单赌局

| 参数 | 值 |
|------|---|
| 胜率 (p) | 60% |
| 赔率 (b) | 1:1（赢 1 元，输 1 元） |

$$f^* = \frac{1 \times 0.6 - 0.4}{1} = 0.2$$

**结论**：最优仓位 20%

### 示例 2：投资场景

| 参数 | 值 |
|------|---|
| 预期收益 | 15% |
| 波动率 | 25% |
| 无风险利率 | 3% |

$$f^* = \frac{0.15 - 0.03}{0.25^2} = \frac{0.12}{0.0625} = 1.92$$

**结论**：理论最优仓位 192%（可加杠杆）

## 风险与调整

### 凯利仓位的特征

| 特征 | 说明 |
|------|------|
| 最大增长 | 长期几何增长率最大 |
| 高波动 | 短期可能大幅回撤 |
| 参数敏感 | 胜率/赔率估计误差影响大 |

### 实践调整：分数凯利

实践中常用**分数凯利**（Fractional Kelly）降低风险：

| 比例 | 说明 | 适用场景 |
|------|------|---------|
| 100% 凯利 | 理论最优 | 参数确定时 |
| 50% 凯利 | 半凯利 | 保守，推荐 |
| 25% 凯利 | 四分之一凯利 | 高度不确定 |

**半凯利的优势**：
- 回撤减少约 50%
- 增长率仅损失约 25%
- 更好的风险调整收益

### 安全准则

1. **永远不超过凯利**：超过凯利会降低增长率
2. **参数要保守估计**：高估胜率是致命的
3. **使用分数凯利**：除非非常确定
4. **留出安全边际**：黑天鹅无法预测

## 多资产扩展

### 多资产凯利公式

对于 $n$ 个资产，最优权重向量：

$$\mathbf{f}^* = \Sigma^{-1}(\boldsymbol{\mu} - r\mathbf{1})$$

其中：
- $\Sigma$ = 协方差矩阵
- $\boldsymbol{\mu}$ = 预期收益向量
- $r$ = 无风险利率
- $\mathbf{1}$ = 全 1 向量

### 与均值方差的关系

凯利公式等价于**对数效用最大化**，与均值方差优化有联系：

| 模型 | 目标函数 | 特点 |
|------|----------|------|
| 均值方差 | $E[R] - \frac{\lambda}{2}\sigma^2$ | 二次效用 |
| 凯利 | $E[\ln(1+R)]$ | 对数效用 |
| 近似关系 | $\ln(1+R) \approx R - \frac{R^2}{2}$ | 小收益时近似 |

## 与其他仓位策略对比

| 策略 | 仓位计算 | 优点 | 缺点 |
|------|----------|------|------|
| 固定比例 | 固定比例（如 50%） | 简单 | 不考虑风险收益 |
| 波动率目标 | 目标波动率 / 实际波动率 | 风险稳定 | 忽略收益预期 |
| 凯利公式 | $(\mu - r) / \sigma^2$ | 理论最优 | 参数敏感 |
| 风险平价 | $1/\sigma_i$ | 风险均衡 | 忽略收益差异 |
| 风险预算 | 按预算分配风险 | 灵活 | 需要设定预算 |

## 实践应用

### A 股应用考量

1. **做空限制**：凯利仓位可能为负（做空），A 股受限
2. **涨跌停**：流动性丧失影响执行
3. **杠杆限制**：融资成本和限额约束
4. **参数估计**：A股 历史 US 数据可能偏差大

### 实施步骤

1. **估计参数**：历史收益、波动率、协方差
2. **调整预期**：加入主观判断
3. **计算凯利权重**：使用连续形式
4. **应用分数凯利**：保守起见
5. **加入约束**：杠杆上限、集中度限制
6. **定期再平衡**：保持目标仓位

### 与其他框架结合

```
资产配置
    ├── 战略配置（长期）
    │   └── 风险平价 / 均值方差
    ├── 战术调整（中期）
    │   └── 凯利公式（分数凯利）
    └── 动态风险管理
        └── 波动率目标
```

## 局限性

### 理论局限

| 问题 | 说明 |
|------|------|
| 参数估计误差 | 历史数据不等于未来 |
| 单期假设 | 实际是连续投资 |
| 对数效用 | 不适合所有投资者 |
| 忽略尾部风险 | 正态分布假设 |

### 实践局限

| 问题 | 说明 |
|------|------|
| 交易成本 | 频繁再平衡的摩擦 |
| 流动性约束 | 大额交易冲击成本 |
| 杠杆限制 | 无法完全执行 |
| 心理承受 | 凯利仓位波动太大 |
