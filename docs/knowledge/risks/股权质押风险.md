# 股权质押风险

## 定义

股权质押是指上市公司股东将其持有的股票质押给金融机构（银行、券商等）以获取贷款的融资行为。当股价下跌触及特定阈值时，可能引发强制平仓，对股东和公司造成重大影响。

## 核心概念

### 质押率

融资金额与质押股票市值的比例，通常为 3-6 折：
- 主板：约 50%
- 中小板：约 40%
- 创业板：约 30%

### 警戒线

金融机构设定的预警价格水平。触及警戒线时，质权人会要求出质人追加抵押物或提前还款。

**计算公式**：
```
警戒价 = 质押价格 × (1 + 融资成本) × 警戒比例
```

**示例**：
- 质押价格：10 元
- 质押率：50%（融资 5 元）
- 融资成本：10%
- 警戒比例：160%
- 警戒价 = 5 × (1 + 10%) × 160% = **8.80 元**

### 平仓线

强制平仓价格水平。跌破平仓线且未能及时补仓时，质权人有权强制卖出质押股票。

**计算公式**：
```
平仓价 = 质押价格 × (1 + 融资成本) × 平仓比例
```

**示例**：
- 平仓比例：140%
- 平仓价 = 5 × (1 + 10%) × 140% = **7.70 元**

## 风险类型

| 风险类型 | 说明 | 影响 |
|----------|------|------|
| 市场风险 | 股价大幅下跌触及警戒线/平仓线 | 质权人权益受损 |
| 强制平仓风险 | 股价跌破平仓线且无法补仓 | 股价暴跌、恶性循环 |
| 控制权变更风险 | 控股股东高比例质押后爆仓 | 公司控制权易主 |
| 流动性风险 | 高质押公司面临巨大资金压力 | 经营困难、财务危机 |
| 道德风险 | 股东可能掏空公司 | 公司价值受损 |

## 风险传导机制

```
股价下跌
    ↓
触及警戒线 → 补仓压力
    ↓
继续下跌
    ↓
触及平仓线 → 强制平仓
    ↓
大量股票抛售
    ↓
股价进一步下跌
    ↓
更多质押触及平仓线（恶性循环）
```

## 应对措施

### 股东层面
1. **追加抵押物** - 补充质押其他股票或现金
2. **提前赎回** - 偿还贷款，解除质押
3. **协商展期** - 延长质押期限

### 公司层面
1. **临时停牌** - 避免股价进一步下跌
2. **引入战略投资者** - 获取资金解决危机
3. **回购/增持** - 稳定股价

### 监管层面
1. **纾困基金** - 政府资金支持
2. **政策放宽** - 调整质押规则

## 投资者识别要点

| 关注指标 | 风险信号 | 说明 |
|----------|----------|------|
| 大股东质押比例 | > 50% | 高质押比例风险较高 |
| 质押笔数 | 多笔分散质押 | 资金链紧张 |
| 质押方类型 | 高利贷/非持牌机构 | 融资成本高、风控严格 |
| 股价距离警戒线 | < 20% | 接近风险区域 |
| 公司经营状况 | 业绩下滑/亏损 | 难以补充质押 |

## A股特点

1. **高质押比例普遍** - 部分公司大股东质押超 90%
2. **市场波动敏感** - 股价下跌容易触发连锁反应
3. **监管逐步加强** - 要求披露质押细节、限制新增质押
4. **纾困机制完善** - 政府和金融机构提供纾困支持
