# 回撤分析

## 定义

**回撤 (Drawdown)**：投资组合从历史最高净值到当前净值的跌幅，衡量投资者在持有期间可能遭受的最大账面损失。

**回撤分析**：系统性研究回撤的深度、持续时间、恢复周期等特征，评估投资策略的风险特性和心理承受要求。

---

## 核心概念

### 回撤深度 (Drawdown Depth)

**定义**：从峰值到谷值的最大跌幅

$$DD_t = \frac{V_{peak} - V_t}{V_{peak}}$$

- $V_{peak}$：历史最高净值
- $V_t$：当前净值

**最大回撤 (Maximum Drawdown, MDD)**：

$$MDD = \max_{t \in T} DD_t$$

**示例**：
- 净值从 1.0 涨到 1.5，再跌到 1.2
- 回撤深度 = (1.5 - 1.2) / 1.5 = 20%

---

### 回撤持续时间 (Drawdown Duration)

**定义**：从开始回撤到恢复到前期高点的时间长度

**阶段划分**：
1. **下跌期**：从峰值到谷值的时间
2. **恢复期**：从谷值回到前期峰值的时间
3. **完整周期**：下跌期 + 恢复期

**意义**：
- 持续时间越长，心理压力越大
- 影响资金的机会成本
- 长期回撤可能导致投资者中途放弃

---

### 回撤恢复时间 (Recovery Time)

**定义**：从谷值恢复到前期峰值所需的时间

**影响因素**：
- 回撤深度：回撤越深，恢复越难
- 策略特性：趋势策略恢复慢，均值回归策略恢复快
- 市场环境：牛市恢复快，熊市恢复慢

**经验法则**：

| 回撤深度 | 所需涨幅恢复 | 预估恢复时间（年） |
|---------|------------|-----------------|
| -10% | +11% | 0.5-1 |
| -20% | +25% | 1-2 |
| -30% | +43% | 2-3 |
| -50% | +100% | 3-5 |
| -70% | +233% | 5-10 |

## 回撤指标体系

### 深度指标

| 指标 | 定义 | 用途 |
|------|------|------|
| 最大回撤 | 历史最大跌幅 | 风险边界 |
| 平均回撤 | 所有回撤的平均值 | 典型风险水平 |
| 回撤中位数 | 回撤的中位数 | 排除极端值 |
| 回撤标准差 | 回撤的波动程度 | 回撤稳定性 |

### 时间指标

| 指标 | 定义 | 用途 |
|------|------|------|
| 最大回撤持续时间 | 最长处于回撤状态的时间 | 心理承受极限 |
| 平均回撤持续时间 | 所有回撤的平均时长 | 典型等待时间 |
| 最大恢复时间 | 最长恢复周期 | 流动性规划 |
| 回撤频率 | 单位时间内发生回撤的次数 | 策略稳定性 |

### 综合指标

**Pain Index（痛苦指数）**：

$$Pain Index = \frac{1}{T}\sum_{t=1}^{T} DD_t$$

- 含义：平均回撤深度
- 意义：衡量整体"痛苦程度"

**Pain Ratio（痛苦比率）**：

$$Pain Ratio = \frac{R_{annual}}{Pain Index}$$

- 含义：单位痛苦获得的收益
- 类似卡玛比率，但分母用平均回撤而非最大回撤

**Calmar Ratio（卡玛比率）**：

$$Calmar = \frac{R_{annual}}{MDD}$$

- 含义：年化收益与最大回撤的比值
- 详见：[[绩效评估指标]]

## 回撤与投资者心理

### 心理阶段

**1. 否认期（-5% 以内）**
- 心态：正常波动，不用在意
- 行为：持仓观望
- 风险：忽视早期风险信号

**2. 焦虑期（-5% 到 -15%）**
- 心态：开始担心，频繁看盘
- 行为：犹豫是否止损
- 风险：情绪化决策

**3. 恐慌期（-15% 到 -30%）**
- 心态：恐惧加剧，怀疑策略
- 行为：考虑清仓
- 风险：底部割肉

**4. 绝望期（-30% 以上）**
- 心态：彻底崩溃，不想看账户
- 行为：逃避或冲动交易
- 风险：做出非理性决策

### 回撤持续时间的影响

| 持续时间 | 心理影响 | 应对难度 |
|---------|---------|---------|
| < 1 个月 | 轻微不适 | 容易 |
| 1-3 个月 | 明显焦虑 | 中等 |
| 3-6 个月 | 信心动摇 | 较难 |
| 6-12 个月 | 怀疑人生 | 困难 |
| > 12 个月 | 放弃策略 | 极难 |

**关键认知**：大多数投资者不是被回撤深度打败，而是被回撤持续时间打败。

## 回撤分析方法

### 1. 回撤曲线分析

**方法**：绘制净值曲线和回撤曲线

**观察要点**：
- 回撤的频率和深度
- 回撤的聚集性（是否有连续回撤）
- 恢复速度（V型 vs L型）

**模式识别**：
- **V型回撤**：快速下跌快速恢复，短期冲击
- **U型回撤**：缓慢下跌缓慢恢复，市场底部震荡
- **L型回撤**：快速下跌长期横盘，熊市特征
- **锯齿型回撤**：反复震荡，策略不稳定

---

### 2. 回撤分布分析

**方法**：统计所有回撤的分布

**指标**：
- 回撤分布直方图
- 不同深度区间的频率
- 回撤深度与持续时间的关系

**判断标准**：
- 集中在浅回撤（<10%）：策略稳定
- 尾部有深回撤（>30%）：存在尾部风险
- 分布均匀：策略可能过于激进

---

### 3. 滚动回撤分析

**方法**：计算不同时间窗口的回撤

**窗口选择**：
- 1 个月滚动回撤
- 3 个月滚动回撤
- 1 年滚动回撤

**用途**：
- 识别策略在不同市场环境下的表现
- 评估回撤的时间稳定性
- 发现策略退化信号

---

### 4. 条件回撤分析

**方法**：在特定条件下分析回撤

**条件类型**：
- **市场环境**：牛市/熊市/震荡市
- **波动率水平**：高波动/低波动
- **宏观事件**：加息/降息/危机

**用途**：
- 识别策略的脆弱环境
- 制定环境适配规则
- 优化风控参数

## 回撤管理策略

### 事前控制

**1. 策略设计**
- 控制单笔风险敞口
- 设置止损规则
- 分散风险来源

**2. 回撤预算**
- 设定可接受的最大回撤
- 制定回撤触发规则
- 预留安全边际

**3. 心理准备**
- 了解策略历史回撤
- 预演极端情景
- 制定应对计划

### 事中控制

**1. 回撤监控**
- 实时跟踪回撤深度
- 预警关键阈值
- 记录回撤原因

**2. 动态调整**
- 回撤触及阈值时减仓
- 暂停开新仓
- 提高开仓标准

**3. 情绪管理**
- 减少看盘频率
- 回顾投资逻辑
- 避免冲动决策

### 事后复盘

**1. 回撤归因**
- 市场因素 vs 策略因素
- 系统性风险 vs 特质风险
- 可避免 vs 不可避免

**2. 策略优化**
- 调整风控参数
- 改进选股/择时逻辑
- 增加对冲机制

**3. 经验积累**
- 记录心理变化
- 总结应对方法
- 更新应急预案

## 回撤指标与其他风险指标的关系

| 指标 | 与回撤的关系 | 互补性 |
|------|-------------|--------|
| 波动率 | 高波动率往往伴随深回撤 | 波动率反映日常风险，回撤反映极端风险 |
| VaR | VaR 是概率化的回撤预期 | VaR 关注阈值，回撤关注实际发生 |
| 夏普比率 | 深回撤会显著降低夏普 | 夏普惩罚所有波动，回撤只关注下跌 |
| 索提诺比率 | 用下行波动率，类似回撤思想 | 都只关注负面风险 |

## A 股回撤特征

### 市场特征

- **波动大**：A 股年化波动率 25-30%，高于美股
- **回撤深**：历史最大回撤常超 30%
- **持续时间长**：熊市回撤可持续 1-2 年
- **恢复慢**：深回撤后恢复周期长

### 典型回撤场景

| 场景 | 回撤深度 | 持续时间 | 恢复时间 |
|------|---------|---------|---------|
| 2015 年股灾 | -45% | 3 个月 | 2-3 年 |
| 2018 年熊市 | -30% | 12 个月 | 1-2 年 |
| 2020 年疫情 | -15% | 1 个月 | 3 个月 |
| 2022 年调整 | -25% | 10 个月 | 进行中 |

### 实践建议

**风险预算**：
- 激进策略：最大回撤 30%
- 平衡策略：最大回撤 20%
- 保守策略：最大回撤 10%

**心理准备**：
- 预期至少经历一次 20% 回撤
- 预期回撤可能持续 3-6 个月
- 制定回撤应对计划并严格执行

## 实现要点

### 最大回撤计算

```python
def max_drawdown(nav_series):
    """
    计算最大回撤
    nav_series: 净值序列
    """
    peak = nav_series.expanding(min_periods=1).max()
    drawdown = (nav_series - peak) / peak
    return drawdown.min()
```

### 回撤持续时间计算

```python
def drawdown_duration(nav_series):
    """
    计算当前回撤持续时间（天）
    """
    peak = nav_series.expanding(min_periods=1).max()
    is_drawdown = nav_series < peak
    
    # 找到最近一次创新高的位置
    last_peak_idx = nav_series[nav_series == peak.iloc[-1]].index[0]
    current_idx = nav_series.index[-1]
    
    return (current_idx - last_peak_idx).days
```

### Pain Index 计算

```python
def pain_index(nav_series):
    """
    计算 Pain Index（平均回撤）
    """
    peak = nav_series.expanding(min_periods=1).max()
    drawdown = (peak - nav_series) / peak
    return drawdown.mean()
```

## 注意事项

### 回撤指标的局限

1. **后视偏差**：历史最大回撤不代表未来极限
2. **样本依赖**：不同时期回撤特征差异大
3. **低估极端风险**：极端事件样本不足

### 常见误区

| 误区 | 问题 | 正确认知 |
|------|------|---------|
| 只看最大回撤 | 忽视回撤持续时间和频率 | 多维度评估 |
| 过度追求低回撤 | 牺牲收益，策略平庸化 | 平衡收益与风险 |
| 忽视心理承受 | 高估自己的忍耐力 | 实事求是评估 |
| 回撤就止损 | 频繁止损增加成本 | 区分正常回撤和策略失效 |
