# 产业链分析

## 定义

产业链分析是从原材料到终端产品的全链条研究方法，通过分析上下游关系、价格传导、利润分配，判断行业景气位置和投资机会。

---

## 产业链结构

### 基本架构

```
上游（原材料）→ 中游（制造加工）→ 下游（终端消费）
```

| 环节 | 特点 | 关注点 | 代表行业 |
|------|------|--------|---------|
| 上游 | 资源垄断、周期性强 | 产能、库存、价格 | 矿产、能源、化工原料 |
| 中游 | 竞争激烈、毛利受压 | 成本控制、产能利用率 | 制造、加工、组装 |
| 下游 | 品牌溢价、渠道为王 | 需求、品牌、渠道 | 消费品、零售、服务 |

---

### 典型产业链

**新能源车产业链**：
```
锂矿/镍钴 → 正极/负极/电解液 → 电池 → 整车 → 充电桩
（上游资源）   （中游材料）      （中游核心） （下游） （配套）
```

**光伏产业链**：
```
硅料 → 硅片 → 电池片 → 组件 → 电站
（上游） （中游） （中游）  （下游）（终端）
```

**半导体产业链**：
```
硅片/化学品 → 设备 → 设计 → 制造 → 封测 → 终端产品
（材料）    （设备） （Fabless）（Foundry）（OSAT）
```

## 价格传导机制

### 传导路径

```
上游涨价 → 中游成本上升 → 下游价格调整 → 终端需求变化
```

### 传导类型

| 类型 | 特点 | 行业示例 |
|------|------|---------|
| 顺畅传导 | 成本可完全转嫁 | 品牌消费品、高端制造 |
| 部分传导 | 只能转嫁部分成本 | 一般制造业 |
| 无法传导 | 成本无法转嫁，毛利压缩 | 竞争激烈的中游行业 |

### 传导时滞

- **即时传导**：大宗商品（原油、金属）
- **短期传导**（1-3 个月）：制造业
- **长期传导**（3-6 个月）：终端消费品

## 利润分配

### 利润池分析

**核心问题**：产业链各环节的利润占比如何变化？

| 阶段 | 上游利润占比 | 中游利润占比 | 下游利润占比 |
|------|-------------|-------------|-------------|
| 供需平衡 | 30% | 40% | 30% |
| 上游紧缺 | 50%+ | 30% | 20% |
| 下游景气 | 25% | 35% | 40%+ |

### 利润转移信号

| 信号 | 含义 | 投资启示 |
|------|------|---------|
| 上游价格暴涨 | 上游利润扩张 | 布局上游资源股 |
| 中游毛利下滑 | 成本压力传导难 | 回避中游 |
| 下游量价齐升 | 终端需求旺盛 | 关注下游龙头 |
| 库存堆积 | 供需失衡 | 谨慎看待全链条 |

## 追踪方法

### 数据来源

| 环节 | 高频数据 | 低频数据 |
|------|---------|---------|
| 上游 | 大宗商品价格、期货 | 矿企产量、库存 |
| 中游 | 开工率、产能利用率 | 财报毛利率 |
| 下游 | 销量、渠道调研 | 零售数据、消费者信心 |

### 关键指标

**上游**：
- 大宗商品价格（期货/现货）
- 库存天数
- 产能利用率
- 资本开支

**中游**：
- 加工费（如铜加工费 TC/RC）
- 开工率
- 毛利率变化
- 订单量

**下游**：
- 销量/销售额
- 终端价格
- 渠道库存
- 消费者信心

## 分析框架

### 四步法

1. **画图**：梳理产业链结构
2. **找瓶颈**：识别供需紧张环节
3. **看利润**：分析利润在哪个环节
4. **追踪**：持续跟踪关键指标

### 瓶颈识别

| 信号 | 瓶颈位置 |
|------|---------|
| 该环节价格持续上涨 | 供不应求，是瓶颈 |
| 该环节毛利率显著高于上下游 | 利润集中，是瓶颈 |
| 该环节产能利用率接近上限 | 产能受限，是瓶颈 |
| 该环节订单排期延长 | 供给紧张，是瓶颈 |

## 投资应用

### 链式投资法

**核心思路**：沿着产业链寻找投资机会

```
发现瓶颈 → 布局瓶颈环节 → 向上下游扩散
```

**示例**：
1. 发现硅料紧缺 → 布局硅料企业
2. 硅料价格传导到硅片 → 布局硅片龙头
3. 下游装机需求爆发 → 布局组件/逆变器

### 对冲策略

**上下游对冲**：
- 做多上游 + 做空下游（上游涨价周期）
- 做空上游 + 做多下游（上游降价周期）

---

## 注意事项

### 信息滞后

- 高频数据：价格、产量（日度/周度）
- 中频数据：开工率、库存（月度）
- 低频数据：财报、行业报告（季度）

### 传导不畅

- 价格传导可能受阻
- 利润转移需要时间
- 供需错配可能加剧

### 行业差异

- 周期行业：价格传导快、波动大
- 消费行业：传导慢、品牌溢价影响大
- 科技行业：技术迭代可能颠覆产业链
