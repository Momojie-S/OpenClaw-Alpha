# 利率周期与资产配置

## 定义

利率周期是指市场利率水平随经济周期和央行政策变化而呈现的周期性波动，通过分析收益率曲线的形态和变化，可以判断经济阶段并指导资产配置。

## 核心概念

### 利率期限结构

利率期限结构（Term Structure）是指某一时点不同期限的即期利率与到期期限的关系，通常用**收益率曲线**表示。

### 收益率曲线形态

| 形态 | 特征 | 经济含义 |
|------|------|----------|
| **正常（向上）** | 长期利率 > 短期利率 | 经济扩张，通胀预期上升 |
| **平坦** | 长短期利率接近 | 经济转折点，政策变动 |
| **倒挂（向下）** | 短期利率 > 长期利率 | 经济衰退信号 |
| **隆起** | 中期利率最高 | 市场分割，特定期限供需失衡 |

### 三大理论

| 理论 | 核心观点 | 局限性 |
|------|----------|--------|
| **预期假说** | 长期利率 = 未来短期利率预期的平均值 | 假设预期确定，市场完全有效 |
| **市场分割理论** | 不同期限市场独立，供需决定利率 | 无法解释不同期限利率的联动 |
| **流动性偏好理论** | 长期利率 = 预期短期利率 + 流动性溢价 | 投资者偏好短期，需补偿才持长期 |

## 利率周期四阶段

### 周期定位

```
经济复苏 → 经济过热 → 滞胀 → 衰退 → 复苏
    ↑           ↑         ↑        ↑
 降息周期   加息周期   加息末期  降息末期
```

| 阶段 | 利率特征 | 收益率曲线 | 资产配置建议 |
|------|----------|-----------|-------------|
| **复苏期** | 利率低位，开始回升 | 陡峭向上 | 股票 > 债券 > 现金 |
| **扩张期** | 利率逐步上升 | 正常向上 | 股票、商品、高收益债 |
| **过热期** | 利率高位，紧缩加剧 | 趋于平坦 | 现金、短债、防御股 |
| **衰退期** | 利率见顶回落 | 倒挂 → 陡峭 | 长债、防御股、现金 |

## 关键指标

### 利差指标

| 指标 | 计算方式 | 信号意义 |
|------|----------|----------|
| **10Y-2Y利差** | 10年期 - 2年期国债收益率 | 倒挂预示衰退（领先6-24个月） |
| **10Y-3M利差** | 10年期 - 3个月国债收益率 | 更敏感的衰退指标 |
| **期限溢价** | 长期利率 - 预期短期利率路径 | 反映风险偏好 |

### 股债相关性

| 利率环境 | 股债相关性 | 原因 |
|----------|-----------|------|
| **利率低位** | 正相关 | 经济复苏驱动股债同涨 |
| **利率上行** | 负相关 | 利率上升伤债，利好股市 |
| **高通胀** | 负相关 | 通胀同时打压股债 |
| **危机时刻** | 正相关（避险） | 资金同时逃离股债 |

## 应用场景

### 宏观择时

1. **观察收益率曲线形态**
   - 倒挂：减持风险资产，增加防御
   - 陡峭：增加股票配置
   - 平坦：观望或降低仓位

2. **跟踪央行政策**
   - 加息周期：短债吸引力上升
   - 降息周期：长债资本利得机会

3. **结合经济指标**
   - [[宏观拐点识别]]：信用底 → 利率底 → 经济底
   - [[美林投资时钟]]：利率周期与经济周期对应

### 行业配置

| 利率阶段 | 受益行业 | 受压行业 |
|----------|---------|---------|
| **降息** | 地产、基建、高负债企业 | 银行（息差收窄） |
| **加息** | 银行、保险 | 地产、高负债企业 |
| **利率稳定** | 成长股、消费 | - |

## 注意事项

### 指标滞后性

- 收益率曲线倒挂领先衰退6-24个月，但**时滞不固定**
- 需结合其他指标（PMI、就业、信贷）综合判断

### 结构性因素

- 央行量化宽松（QE）压低长端利率，可能扭曲曲线形态
- 全球资本流动影响本国利率，不完全反映国内经济

### 中美差异

- 美国收益率曲线信号较强，A股相关性较低
- 中国利率市场化程度不同，需关注政策利率传导

### 非线性关系

- 利率变化对资产的影响非对称
- 利率从低位上升初期利好股市，后期则相反
