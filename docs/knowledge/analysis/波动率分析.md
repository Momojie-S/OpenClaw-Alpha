# 波动率分析

## 定义

波动率是金融资产价格的波动程度，是对资产收益率不确定性的衡量，用于反映金融资产的风险水平。

**核心特征**：
- 波动率越高，价格波动越剧烈，收益率不确定性越强
- 波动率越低，价格波动越平缓，收益率确定性越强
- 通常用收益率的标准差来衡量

## 波动率类型

| 类型 | 英文 | 说明 | 特点 |
|------|------|------|------|
| **实际波动率** | Realized Volatility | 期权有效期内投资回报率波动程度 | 永远未知，只能估计 |
| **历史波动率** | Historical Volatility (HV) | 基于过去数据计算的波动率 | 反映历史表现，可精确计算 |
| **预测波动率** | Forecasted Volatility | 对未来波动率的预测值 | 用于期权定价 |
| **隐含波动率** | Implied Volatility (IV) | 从期权价格反推的波动率 | 反映市场预期，期权交易核心 |

### 历史波动率 vs 隐含波动率

| 维度 | 历史波动率 | 隐含波动率 |
|------|-----------|-----------|
| 来源 | 历史价格数据 | 期权市场价格 |
| 方向 | 向后看 | 向前看 |
| 数量 | 唯一（同一时间窗口） | 多个（不同执行价格） |
| 用途 | 风险评估、波动率预测 | 期权定价、市场情绪 |

## 计算方法

### 1. 标准差法（推荐）

**公式**：
```
波动率 = σ × √T
```

其中：
- σ = 收益率标准差
- T = 年化时间因子（如日波动率年化：√252）

**计算步骤**：
1. 获取固定时间间隔的价格序列 {P₁, P₂, ..., Pₙ}
2. 计算对数收益率：Xᵢ = ln(Pᵢ₊₁ / Pᵢ)
3. 计算收益率标准差 σ
4. 年化：σ × √252（假设一年 252 个交易日）

### 2. 对数价格变动法（Black-Scholes 模型适用）

**公式**：
```
Xᵢ = ln(Pᵢ₊₁ / Pᵢ)
σ = √[Σ(Xᵢ - μ)² / N] × √252
```

其中：
- Xᵢ = 对数收益率
- μ = 对数收益率平均值
- N = 观察值数量
- 252 = 年交易日数（中国 A 股）

### 3. 百分比价格变动法

**公式**：
```
Xᵢ = (Pᵢ₊₁ - Pᵢ) / Pᵢ
```

**注意**：适用于离散价格变化，不适合连续模型。

## 波动率产生原因

1. **系统性风险**：宏观经济因素对产业部门的影响
2. **非系统性风险**：特定事件对企业的冲击
3. **投资者心理**：心理状态或预期变化对股价的作用

## 应用场景

### 1. 风险管理

| 应用 | 说明 |
|------|------|
| **风险评估** | 高波动率 = 高风险，需谨慎 |
| **仓位调整** | 波动率上升时降低仓位 |
| **止损设置** | 根据波动率设置动态止损位 |
| **VaR 计算** | 波动率是 VaR 的核心参数 |

### 2. 期权定价

- **Black-Scholes 模型**：波动率是五个核心参数之一
- **期权价格**：波动率越高，期权价格越高
- **IV 交易**：做期权就是做预期的波动率

**逻辑**：
- 波动率↑ → 标的价格偏离执行价概率↑ → 期权买方收益可能↑ → 愿付更高权利金
- 波动率↑ → 期权卖方风险↑ → 要求更高权利金

### 3. 投资决策

| 场景 | 策略 |
|------|------|
| **低波动率** | 适合买入持有，长期投资 |
| **高波动率** | 适合短线交易，快进快出 |
| **波动率突破** | 波动率突然放大，可能趋势反转 |
| **波动率收敛** | 波动率持续缩小，可能变盘在即 |

### 4. A 股特殊应用

- **新股波动率**：上市初期波动率高，需观望
- **涨跌停板**：10%/20% 限制影响波动率计算
- **停牌影响**：停牌期间波动率为 0，需特殊处理
- **板块效应**：同一板块个股波动率相关性高

## 波动率估计注意事项

### 1. 数据频率选择

| 频率 | 优点 | 缺点 |
|------|------|------|
| 日数据 | 数据多，统计误差小 | 噪声大 |
| 周数据 | 平滑噪声 | 数据少 |
| 月数据 | 更稳定 | 数据太少 |

**建议**：优先使用交易日数据（非日历日）

### 2. 样本期选择

| 期限 | 适用场景 |
|------|---------|
| 短期（30 日） | 近期波动率预测 |
| 中期（90 日） | 季度持仓 |
| 长期（1 年） | 长期风险评估 |
| 匹配法 | 样本期 = 预测期 |

### 3. 经济日 vs 正常日

- **正常日**：波动率低，价格变化平稳
- **经济日**：波动率高，重要事件发生

**预测方法**：
```
预期波动率 = (正常日天数 × 正常波动率 + 经济日天数 × 经济波动率) / 总天数
```

### 4. 价格选择

- **最常用**：收盘价
- **外汇市场**：最高价/最低价（无收盘价）
- **高频数据**：日内价格（更精确）

## 波动率的均值回归特性

波动率具有均值回归特性：
- 高波动率时期 → 逐渐回归均值
- 低波动率时期 → 逐渐回升至均值

**应用**：
- 波动率极端高 → 做空波动率（卖期权）
- 波动率极端低 → 做多波动率（买期权）

## A 股应用建议

1. **优先用历史波动率**：A 股期权市场不如美股发达，隐含波动率可靠性较低
2. **结合板块波动率**：同板块个股波动率趋同，可参考板块均值
3. **关注极端值**：波动率突然放大往往是信号
4. **区分趋势和震荡**：
   - 趋势行情：波动率持续放大
   - 震荡行情：波动率收敛
5. **避免单一指标**：波动率需结合成交量、价格趋势等综合判断

## 注意事项

1. **波动率 ≠ 风险**：波动率描述价格变化幅度，风险是损失概率，两者不完全等同
2. **历史不代表未来**：历史波动率仅供参考，未来可能不同
3. **模型假设**：Black-Scholes 假设波动率恒定，实际波动率是时变的
4. **数据质量**：停牌、涨跌停等特殊事件影响波动率计算
5. **年化因子**：不同市场年交易日数不同（A股 252，美股 252，港股 245）
