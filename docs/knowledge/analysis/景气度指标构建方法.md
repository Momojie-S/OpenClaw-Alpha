# 景气度指标构建方法

## 定义

将多维度行业数据转化为可比、可追踪的景气度指标的技术方法，包括数据标准化、权重确定、指标合成、回测验证等环节。

---

## 构建流程

```
原始数据 → 标准化处理 → 权重分配 → 指标合成 → 回测验证
```

---

## 一、数据标准化

### 目的

消除量纲影响，使不同指标可比、可加权。

### 常用方法

| 方法 | 公式 | 特点 |
|------|------|------|
| **Z-Score** | `(x - μ) / σ` | 标准正态分布，异常值敏感 |
| **Min-Max** | `(x - min) / (max - min)` | 映射到 [0, 1]，受极值影响 |
| **分位数标准化** | 映射到分位数排名 | 抗异常值，适合非正态分布 |
| **环比标准化** | `(x_t - x_{t-1}) / |x_{t-1}|` | 关注边际变化 |

### 行业横向可比

将不同行业的指标统一标准化后，可横向比较景气强弱：

```
行业A景气度 = Σ(wi × 标准化指标i)
行业B景气度 = Σ(wi × 标准化指标i)
```

## 二、权重确定

### 常用方法

| 方法 | 说明 | 适用场景 |
|------|------|---------|
| **等权法** | 各指标权重相同 | 指标重要性相近 |
| **专家赋权** | 基于业务经验 | 主观性强、有领域知识 |
| **主成分分析** | 按解释方差加权 | 指标相关性高 |
| **回测优化** | 按历史超额收益加权 | 面向投资决策 |
| **层次分析法（AHP）** | 专家打分+一致性检验 | 多层次指标体系 |

### 回测优化权重

**步骤**：
1. 计算各指标分组的多头收益
2. 以未来 N 期超额收益为目标
3. 回归或优化求解权重

**示例**：方正证券中观景气指数
- 以**未来 6 个月超额收益**确定二级行业权重
- 再加权得到一级行业景气指数

### 国家统计局方法

企业景气指数权重：
- **即期景气指数**：40%
- **预期景气指数**：60%

体现指数的前瞻判断功能。

## 三、量-价-库存体系

### 三维指标

| 维度 | 含义 | 典型指标 |
|------|------|---------|
| **量** | 需求/产出规模 | 销量、产量、订单 |
| **价** | 供需价格信号 | 产品价格、原材料价格 |
| **库存** | 供需平衡状态 | 产成品库存、原材料库存 |

### 构建步骤

1. **拆分行业**：梳理一级行业的关键二级行业
2. **优选指标**：为每个二级行业选择量-价-库存指标
3. **标准化**：对各指标进行标准化处理
4. **加权合成**：按权重合成二级行业指数
5. **再加权**：按回测表现合成一级行业指数

### 上中下游差异

| 位置 | 核心指标 | 周期特点 |
|------|---------|---------|
| **上游** | 价格 > 库存 | 价格周期主导 |
| **中游** | 量+库存 | 与投资周期同步 |
| **下游** | 需求+量 | 与消费紧密相关 |

详见 [[库存周期]]

## 四、指标合成

### 线性加权

```
景气指数 = Σ(wi × 标准化指标i)
```

### 扩散指数法

```
DI = 扩张指标数 / 总指标数 × 100
```

**临界值**：
- DI > 50：扩张
- DI < 50：收缩

详见 [[行业景气度#扩散指数法]]

## 五、回测验证

### 目的

验证景气指数改善的行业是否获得超额收益。

### 验证方法

| 方法 | 说明 |
|------|------|
| **分组回测** | 按景气度分 5 组，比较收益 |
| **IC 分析** | 景气度与未来收益的相关性 |
| **多空策略** | 做多高景气、做空低景气 |
| **相对强弱** | 景气改善 vs 景气恶化的超额对比 |

### 验证指标

| 指标 | 说明 |
|------|------|
| **IC** | 信息系数，景气度与收益相关性 |
| **IR** | 信息比率，超额收益/跟踪误差 |
| **胜率** | 高景气行业跑赢基准的比例 |
| **分组单调性** | 景气度排序与收益排序的一致性 |

### 常见问题

| 问题 | 原因 | 解决 |
|------|------|------|
| **IC 不稳定** | 单一指标噪音大 | 多指标复合 |
| **过拟合** | 权重过度优化 | Walk-Forward 验证 |
| **滞后性** | 财报数据延迟 | 加入高频/预期指标 |

## 六、复合景气度指标

### 构建原则

选择**高收益、低相关**的单项指标复合：

```
复合景气度 = Σ(wi × 标准化指标i)
```

### 常用组合

| 组合 | 适用场景 | 优势 |
|------|---------|------|
| 财务+预期 | 通用 | 兼顾历史与前瞻 |
| 量+价+库存 | 周期行业 | 捕捉库存周期 |
| 订单+产能+价格 | 制造业 | 跟踪产能利用率 |
| 政策+订单+盈利 | 政策驱动行业 | 捕捉政策红利 |

## 注意事项

### 数据质量

- **缺失值处理**：插值、前值填充或剔除
- **异常值处理**：Winsorize 或剔除
- **频率对齐**：统一为月度或季度

### 实时性

- **财报滞后**：加入预期、高频指标
- **修订问题**：宏观数据可能回溯修订
- **发布时点**：考虑数据公布的实际时间

### 行业特性

- **周期行业**：量价库存敏感
- **消费行业**：渠道、动销数据重要
- **科技行业**：订单、政策影响大
