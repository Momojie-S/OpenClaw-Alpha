# 宏观拐点识别

## 定义

通过领先指标、同步指标、滞后指标的组合分析，识别经济周期的转折点，为投资决策提供前瞻信号。

---

## 指标分类

### 领先指标（Leading）

**特点**：提前经济 3-6 个月变化

| 指标 | 含义 | 获取方式 |
|------|------|----------|
| PMI 新订单 | 需求前瞻 | 月度发布 |
| 社融增速 | 信用扩张 | 月度发布 |
| M1-M2 剪刀差 | 活期存款意愿 | 月度发布 |
| 利率曲线斜率 | 预期变化 | 日度 |
| 股价指数 | 市场预期 | 实时 |

**拐点信号**：
- PMI 新订单连续 3 个月回升 → 复苏信号
- 社融增速触底反弹 → 信用底
- M1-M2 剪刀差收窄 → 活跃度提升

---

### 同步指标（Coincident）

**特点**：与经济同步变化

| 指标 | 含义 | 用途 |
|------|------|------|
| 工业增加值 | 生产活动 | 当前经济状态 |
| 发电量 | 实体经济活跃度 | 高频验证 |
| PMI 综合 | 采购经理判断 | 快速参考 |

**应用**：确认拐点，验证领先指标信号

---

### 滞后指标（Lagging）

**特点**：落后经济 3-6 个月

| 指标 | 含义 | 用途 |
|------|------|------|
| CPI | 消费者物价 | 确认通胀 |
| PPI | 生产者物价 | 确认周期 |
| 失业率 | 就业市场 | 确认衰退/复苏 |

**应用**：事后验证，不做前瞻判断

---

## 核心拐点识别框架

### 信用底 → 市场底 → 经济底 → 盈利底

```
时间轴 →
─────────────────────────────────────────
 信用底    市场底    经济底    盈利底
   │         │         │         │
 社融增速   股价触底   GDP/PMI   企业盈利
 触底反弹   政策转向   回升      改善
   
 间隔：约 2-3 个月 → 约 3-6 个月 → 约 6-9 个月
```

**关键指标**：
- **信用底**：社融增速触底 + M1-M2 剪刀差收窄
- **市场底**：估值极值 + 政策转向 + 成交量萎缩
- **经济底**：PMI 回升 + 工业增加值反弹
- **盈利底**：工业企业利润增速转正

---

### 顶部拐点

**特征组合**：

| 信号 | 典型表现 |
|------|----------|
| 政策转向 | 央行收紧、监管加强 |
| 估值极值 | 沪深 300 PE > 历史均值 +2σ |
| 情绪过热 | 新开户数激增、换手率高企 |
| 技术背离 | 指数新高 + 成交量萎缩 |

**领先信号**：
- 社融增速见顶回落
- PMI 新订单下降
- 商品价格见顶（PPI 前瞻）

## 量化识别方法

### 1. 移动平均交叉

**方法**：指标 3 月 MA 上穿 12 月 MA

```python
# 伪代码
signal = MA(3) > MA(12)  # 布尔值
turning_point = signal.diff()  # 信号变化点
```

**适用**：PMI、工业增加值等

---

### 2. 增速变化率

**方法**：同比增速的二阶导

```
二阶导数 = (增速_t - 增速_{t-1}) - (增速_{t-1} - 增速_{t-2})
```

**解读**：
- 二阶导转正 → 加速期
- 二阶导转负 → 减速期

---

### 3. 历史分位数

**方法**：指标处于历史分位数的位置

| 分位数 | 含义 |
|--------|------|
| < 10% | 极度低迷（底部信号） |
| > 90% | 极度过热（顶部信号） |

**适用**：估值、情绪、信用利差

---

### 4. 多指标合成

**方法**：加权合成多个指标

```
综合信号 = w1 × 信用信号 + w2 × PMI信号 + w3 × 估值信号
```

**权重示例**：
- 信用信号：40%
- PMI 信号：30%
- 估值信号：30%

**决策规则**：
- 综合信号 > 0.7 → 看多
- 综合信号 < 0.3 → 看空
- 0.3-0.7 → 震荡

## 注意事项

### 伪信号

- 单一指标容易产生伪信号
- 需要多指标共振确认
- 结合政策环境判断

### 滞后修正

- 官方数据可能修正
- 初值与终值有差异
- 关注修正方向

### 外部冲击

- 疫情、战争等黑天鹅
- 海外央行政策溢出
- 可能打破原有周期规律

### 市场预期

- 拐点常领先于数据确认
- 市场交易的是预期差
- 关注预期 vs 实际
