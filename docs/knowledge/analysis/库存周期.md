# 库存周期

## 定义

库存周期（又称基钦周期）是 3-4 年的短期经济周期，反映企业库存的被动/主动调整过程，是判断行业景气度拐点的重要工具。

---

## 四个阶段

```
需求变化 → 企业库存调整 → 景气度变化

┌─────────────────────────────────────────┐
│                                         │
│   主动补库存 ←───── 被动补库存           │
│   (景气上行)         (景气下行开始)      │
│       ↑                   ↓             │
│   需求旺盛           需求回落           │
│   企业主动扩产       库存被动堆积        │
│                                         │
│   主动去库存 ─────→ 被动去库存           │
│   (景气下行)         (景气拐点)         │
│       ↓                   ↑             │
│   需求疲软           需求回暖           │
│   企业主动清仓       库存被动下降        │
│                                         │
└─────────────────────────────────────────┘
```

### 各阶段特征

| 阶段 | 需求 | 库存 | 价格 | 企业行为 | 景气度 |
|------|------|------|------|---------|--------|
| 主动补库存 | ↑↑ | ↑ | ↑ | 主动扩产 | 高景气 |
| 被动补库存 | ↓ | ↑↑ | →/↓ | 销售放缓 | 景气见顶 |
| 主动去库存 | ↓↓ | ↓ | ↓ | 主动减产 | 低景气 |
| 被动去库存 | ↑ | ↓↓ | →/↑ | 销售好转 | 景气拐点 |

### 投资机会

| 阶段 | 配置策略 | 关注信号 |
|------|---------|---------|
| 被动去库存 | **左侧布局** | 库存快速下降、价格企稳 |
| 主动补库存 | **超配** | 订单回暖、产能利用率提升 |
| 被动补库存 | **减仓** | 库存累积、价格滞涨 |
| 主动去库存 | **回避** | 库存高企、价格下跌 |

## 核心指标

### 量价库存三角

| 指标 | 说明 | 数据来源 |
|------|------|---------|
| 库存增速 | 产成品存货同比 | 统计局 |
| 价格指数 | PPI / 行业价格 | 统计局 |
| 销量增速 | 营收 / 销量同比 | 财报 |

### 判断方法

**量-价-库存 三维定位**：

```
价格 ↑ + 库存 ↓ + 销量 ↑ = 被动去库存（拐点）
价格 ↑ + 库存 ↑ + 销量 ↑ = 主动补库存（高景气）
价格 →/↓ + 库存 ↑ + 销量 ↓ = 被动补库存（见顶）
价格 ↓ + 库存 ↓ + 销量 ↓ = 主动去库存（低景气）
```

## 量化跟踪方法

### 景气指数构建（方正证券）

**步骤**：
1. **指标筛选**：选择行业相关的量、价、库存指标
2. **标准化**：消除量纲差异
3. **加权合成**：构建综合景气指数
4. **周期定位**：确定当前所处阶段

**核心公式**：
```
景气指数 = w1 × 量指标 + w2 × 价指标 + w3 × 库存指标

其中：
- w1, w2, w3 为权重（可等权或根据历史回测确定）
- 指标均需标准化处理
```

## 行业应用

### 周期行业（强周期）

**典型行业**：钢铁、煤炭、有色、化工、建材

| 指标 | 数据源 | 频率 |
|------|--------|------|
| 社会库存 | 行业协会 | 周度 |
| 价格 | 现货/期货 | 日度 |
| 开工率 | 钢联/百川 | 周度 |

### 消费行业（弱周期）

**典型行业**：家电、汽车、白酒、服装

| 指标 | 数据源 | 频率 |
|------|--------|------|
| 渠道库存 | 公司调研/财报 | 季度 |
| 终端销量 | 乘联会/中汽协 | 月度 |
| 价格 | 渠道调研 | 月度 |

### 科技行业（创新周期）

**典型行业**：半导体、光伏、新能源车

| 指标 | 数据源 | 频率 |
|------|--------|------|
| 晶圆厂产能利用率 | SEMI | 月度 |
| 库存周转天数 | 财报 | 季度 |
| BB值（订单/出货） | SEMI | 月度 |

**BB 值判断**：
- BB > 1：需求旺盛，景气上行
- BB < 1：需求疲软，景气下行

## 注意事项

### 周期长度

- **短周期**：3-4 年（基钦周期）
- **中周期**：8-10 年（朱格拉周期，设备更新）
- **长周期**：50-60 年（康德拉季耶夫周期，技术革命）

**实际应用中主要关注 3-4 年的短周期。**

### 行业差异

- **强周期行业**：库存周期明显，信号清晰
- **弱周期行业**：库存波动小，需结合其他指标
- **成长行业**：受创新周期影响更大

### 假信号

- **政策干扰**：供给侧改革、限产等政策影响库存
- **外部冲击**：疫情、贸易战等突发事件
- **季节因素**：春节、淡旺季需剔除

### 指标滞后

- **官方数据滞后**：统计局数据通常滞后 1-2 月
- **财报数据滞后**：季度财报滞后约 1 月
- **解决方案**：使用高频数据（价格、开工率）前瞻判断
