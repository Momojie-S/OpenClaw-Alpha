# 融资融券分析

## 定义

**融资融券**是投资者向证券公司借入资金买入证券（融资）或借入证券卖出（融券）的行为。融资融券余额反映了市场的杠杆水平和多空情绪，是重要的市场情绪指标。

## 基本概念

### 融资交易

| 概念 | 说明 |
|------|------|
| 融资买入 | 向券商借钱买股票 |
| 融资余额 | 融资买入后未偿还的金额 |
| 融资净买入 | 融资买入 - 融资偿还 |

### 融券交易

| 概念 | 说明 |
|------|------|
| 融券卖出 | 向券商借股票卖出（做空） |
| 融券余量 | 融券卖出后未买回的股票数量 |
| 融券余额 | 融券余量 × 市价 |

### 综合指标

| 指标 | 公式 | 意义 |
|------|------|------|
| 两融余额 | 融资余额 + 融券余额 | 市场总杠杆水平 |
| 融资占比 | 融资余额 / 两融余额 | 多空力量对比 |
| 融资融券比 | 融资余额 / 融券余额 | 市场情绪偏向 |

## 市场信号

### 融资余额变化

| 现象 | 解读 | 风险 |
|------|------|------|
| 融资余额持续上升 | 市场看多情绪升温 | 杠杆过高，下跌时加速 |
| 融资余额快速增加 | 情绪亢奋，可能过热 | 短期见顶风险 |
| 融资余额持续下降 | 市场悲观，去杠杆 | 下跌趋势延续 |
| 融资余额快速下降 | 强制平仓压力大 | 可能加速见底 |

### 融券余额变化

| 现象 | 解读 | 注意 |
|------|------|------|
| 融券余额上升 | 看空力量增加 | 可能被逼空 |
| 融券余额快速上升 | 做空情绪浓厚 | 反转信号 |
| 融券余额下降 | 空头回补 | 可能推动上涨 |

### 融资融券比

| 比值 | 情绪状态 | 应对 |
|------|---------|------|
| > 100 | 极度看多 | 警惕过热 |
| 50-100 | 偏多 | 正常 |
| 20-50 | 多空均衡 | 观望 |
| < 20 | 偏空 | 关注反弹机会 |

## 个股分析

### 融资买入信号

| 信号 | 含义 | 验证方法 |
|------|------|---------|
| 融资余额占比高 | 杠杆资金关注 | 占流通市值 > 5% |
| 融资净买入连续为正 | 持续加杠杆 | 连续 3 天以上 |
| 融资买入额占比高 | 当日主力行为 | 占成交额 > 20% |

### 融资偿还信号

| 信号 | 含义 | 风险 |
|------|------|------|
| 融资净偿还 | 杠杆资金撤退 | 可能下跌 |
| 大额偿还 | 集中离场 | 警惕风险 |
| 强制平仓 | 融资客爆仓 | 加速下跌 |

### 融券信号

| 信号 | 含义 | 注意 |
|------|------|------|
| 融券余量大幅增加 | 做空力量增强 | 可能下跌或逼空 |
| 融券余量快速减少 | 空头回补 | 推动上涨 |
| 融券余额占比高 | 市场分歧大 | 关注方向选择 |

## 风险预警

### 强制平仓风险

**触发条件**：
- 维持担保比例 < 130%（通常）

**影响**：
- 被动卖出加剧下跌
- 连锁平仓效应
- 市场流动性危机

**预警信号**：
- 股价快速下跌 30% 以上
- 融资余额占比过高（> 10%）
- 市场整体融资余额快速下降

### 杠杆风险

| 融资比例 | 杠杆倍数 | 下跌 20% 后 |
|:--------:|:--------:|:-----------:|
| 1:0.5 | 1.5x | 亏损 30% |
| 1:1 | 2x | 亏损 40% |
| 1:1.5 | 2.5x | 亏损 50% |

**结论**：高杠杆放大亏损，需严格控制融资比例。

## 与其他指标的结合

### + 价格走势

| 组合 | 解读 |
|------|------|
| 融资增 + 股价涨 | 健康上涨，可持续 |
| 融资增 + 股价滞涨 | 资金分歧，警惕见顶 |
| 融资减 + 股价跌 | 去杠杆，下跌延续 |
| 融资减 + 股价企稳 | 去杠杆完成，可能见底 |

### + 成交量

| 组合 | 解读 |
|------|------|
| 融资增 + 放量 | 主力进场，强势 |
| 融资增 + 缩量 | 吸筹，可能启动 |
| 融资减 + 放量 | 出货，下跌 |
| 融资减 + 缩量 | 阴跌，观望 |

### + 北向资金

| 组合 | 解读 |
|------|------|
| 融资增 + 北向增 | 内外资一致看多 |
| 融资增 + 北向减 | 内资看多外资看空 |
| 融资减 + 北向增 | 内资看空外资看多 |
| 融资减 + 北向减 | 内外资一致看空 |

## 注意事项

### 滞后性

**问题**：T+1 披露

**对策**：
- 结合盘中走势
- 关注连续趋势

### 数据解读

**问题**：融资余额上升不等于股价上涨

**原因**：
- 融资客可能被套
- 分散买入多个标的
- 对冲操作

### 市场环境

**牛市**：融资余额持续上升，信号可靠
**熊市**：融资余额下降，去杠杆
**震荡市**：融资余额波动大，信号不稳定
