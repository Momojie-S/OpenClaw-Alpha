# 相对强度

## 定义

相对强度（Relative Strength, RS）衡量个股或板块相对于基准指数（如大盘）的表现。**不是**衡量股票的绝对涨幅，而是其"相对"表现。

**与 RSI 的区别**：
- **相对强度（RS）**：股票 vs 基准指数的相对表现
- **相对强弱指数（RSI）**：单一股票的动量震荡指标

---

## 核心逻辑

**为什么重要**：
- 强者恒强：相对强度高的股票往往继续跑赢
- 提前识别：在绝对涨幅出现前，RS 已显示强势
- 板块轮动：识别领涨板块和弱势板块

**温斯坦的定义**（Stan Weinstein）：
> 相对强度是股票（或板块）相对于整体市场的表现。例如，股票上涨10%而市场上涨20%，这是弱相对强度；股票下跌10%而市场下跌20%，这是强相对强度。

---

## 计算方法

### 标准相对表现（Standard Relative Performance）

**公式**：
```
RP = (股票收盘价 / 指数收盘价) × 100
零线 = RP 的 52 周移动平均
```

**解读**：
- RP 上升：股票跑赢指数
- RP 下降：股票跑输指数
- RP > 零线：长期相对强度强
- RP < 零线：长期相对强度弱

---

### 曼斯菲尔德相对表现（Mansfield Relative Performance）

**公式**：
```
MRP = ((RP(today) / SMA(RP, n)) - 1) × 100

其中：
- RP = 标准相对表现
- SMA = 简单移动平均
- n = 52（周线）或 200（日线）
```

**特点**：
- 零线是"扁平化"的52周移动平均
- 显示为震荡器风格（围绕零线波动）
- 更容易看出相对强度的极端值

**解读**：
- MRP > 0：股票跑赢指数
- MRP < 0：股票跑输指数
- MRP 创新高：相对强度增强
- MRP 创新低：相对强度减弱

---

### 相对强度评分（RS Rating）

**定义**：将所有股票的相对强度排序，转换为0-100的评分。

**示例**：
- RS 评分 = 95：该股票的表现优于95%的股票
- RS 评分 = 20：该股票的表现仅好于20%的股票

**CAN SLIM 方法**：
- William O'Neil 强调买入 RS 评分 > 80 的股票
- 避免买入 RS 评分 < 50 的股票

---

## 应用场景

### 1. 选股筛选

**强势股特征**：
- RS 评分 > 80（优于80%股票）
- RS 持续上升
- RS 在市场回调时保持强势（横盘代替下跌）

**筛选步骤**：
1. 计算 RS 评分
2. 筛选 RS > 80 的股票
3. 结合基本面、技术面进一步筛选
4. 等待合适的买入时机

---

### 2. 板块轮动

**识别领涨板块**：
- 计算板块指数相对于大盘的 RS
- RS 上升的板块：资金流入，关注
- RS 下降的板块：资金流出，回避

**轮动信号**：
- 板块 RS 创新高：可能成为新热点
- 板块 RS 跌破零线：可能走弱
- 板块 RS 与价格背离：注意反转

---

### 3. 温斯坦阶段分析

**阶段2突破 + RS 确认**：
- 股价从阶段1（筑底）突破到阶段2（上涨）
- **必须伴随 RS 上升**（关键确认信号）
- RS 提前突破：股票已相对走强
- RS 同步突破：确认股票领导地位

**四阶段 RS 特征**：

| 阶段 | 股价 | RS 特征 |
|------|------|---------|
| 1 筑底 | 横盘 | RS 可能提前走强 |
| 2 上涨 | 上升 | RS 持续走强 |
| 3 派发 | 横盘 | RS 开始走弱 |
| 4 下跌 | 下降 | RS 持续走弱 |

---

### 4. 卖出信号

**RS 走弱信号**：
- RS 创新低
- RS 跌破零线（52周均线）
- RS 与股价出现看跌背离
- RS 评分从高位大幅下降

**结合阶段分析**：
- 阶段2末期 + RS 走弱 → 减仓
- 进入阶段3 + RS 负值 → 退出

## A 股应用

### 计算基准

**大盘基准**：
- 沪深300：适合大盘股
- 中证500：适合中盘股
- 创业板指：适合成长股

**行业基准**：
- 申万一级行业指数
- 板块相对于行业平均的 RS

---

### 注意事项

**市场环境**：
- 牛市：高 RS 股票更强势
- 熊市：关注抗跌的股票（相对强度强）
- 震荡：RS 选股效果更好

**数据周期**：
- 长期 RS（52周）：趋势判断
- 中期 RS（13周）：轮动信号
- 短期 RS（4周）：短线机会

**避免误区**：
- RS 高 ≠ 立即买入（需结合趋势、基本面）
- RS 低 ≠ 立即卖出（可能是暂时调整）
- RS 是筛选工具，不是独立信号

---

## 实践建议

### 选股流程

1. **计算 RS 评分**（1年周期）
2. **筛选 RS > 80** 的股票池
3. **技术面确认**（趋势、形态）
4. **基本面验证**（业绩、估值）
5. **等待买入信号**（突破、回调）

### 组合管理

1. **监控持仓 RS** 变化
2. **RS 持续走弱**：考虑减仓或换股
3. **定期再平衡**：增持 RS 强的标的
4. **设置止损**：RS 跌破关键水平

---

## 局限性

1. **滞后性**：基于历史数据，有滞后
2. **市场环境**：极端行情下可能失效
3. **基准选择**：不同基准导致不同结果
4. **短期噪音**：短期 RS 波动大，需平滑
