# 恐慌指数（VIX）

## 定义

芝加哥期权交易所（CBOE）波动率指数，基于标普500指数期权的隐含波动率计算，反映市场对未来30天波动率的预期。通常被称为"恐慌指数"。

---

## 核心概念

### 隐含波动率

VIX 不是历史波动率，而是**隐含波动率**——从期权价格反推出的市场预期。

**期权与波动率的关系**：
- 标的证券波动率越高 → 期权价值越高（更大可能进入价内）
- 期权价格上涨 → 隐含波动率上升

### 预测区间

VIX 以年化百分比表示，可转换为30天预期波动区间：

```
30天预期波动 = VIX / √12
```

**示例**：
- VIX = 15%
- 30天预期波动 = 15% / 3.46 = 4.33%
- 含义：标普500未来30天在正负4.33%区间波动的概率约68%

---

## 计算方法

### 理论基础

VIX 是**方差互换**的波动率，不是波动率互换的波动率。

- 方差互换：可静态复制（看涨+看跌期权组合）
- 波动率互换：需要动态对冲

### 计算步骤

1. 选择近月和次近月的标普500期权（看涨+看跌）
2. 选择一系列行权价（覆盖一定范围）
3. 使用期权价格计算方差
4. 对两个月份进行时间加权插值
5. 年化处理得到 VIX 值

### 关键特性

- 使用**虚值期权**价格（最能反映尾部风险）
- 涵盖一系列行权价（而非单一期权）
- 实时更新，高频发布

## 解读与应用

### VIX 水平解读

| VIX 值 | 市场状态 | 含义 |
|--------|---------|------|
| < 15 | 平静 | 市场稳定，投资者乐观 |
| 15-20 | 正常 | 正常波动水平 |
| 20-30 | 波动加剧 | 有所担忧，需关注风险 |
| 30-40 | 恐慌 | 市场恐慌，可能超卖 |
| > 40 | 极端恐慌 | 历史性危机，反向机会？ |

### VIX 的误区

**高 VIX ≠ 熊市**：
- VIX 衡量的是波动率（包括上涨和下跌）
- 投资者预期大涨时，VIX 也会上升
- 高 VIX 代表市场预期剧烈变动，方向不确定

**低 VIX ≠ 安全**：
- 可能是暴风雨前的宁静
- 过度 complacency 往往预示变盘

## 交易策略

### VIX 相关产品

| 产品类型 | 代码示例 | 特点 |
|---------|---------|------|
| VIX 期货 | - | 杠杆高，需要移仓 |
| VIX 期权 | - | 对冲尾部风险 |
| VIX ETF | VXX, VIXY | 方便但有跟踪误差 |
| VIX ETN | VXZ | 中期期货，波动较小 |

### 策略应用

**对冲策略**：
- 买入 VIX 期货/期权对冲股市下跌风险
- VIX 与股市负相关（危机时飙升）

**波动率交易**：
- 预期波动率上升：做多 VIX
- 预期波动率下降：做空 VIX
- 需要考虑期货升贴水

**均值回归**：
- VIX 长期均值约 20
- 极端高值后通常回归
- 但回归时间不确定

## 与其他指标的关系

### VIX vs 股市

- **负相关**：股市下跌时 VIX 通常上升
- **不对称**：股市大涨时 VIX 上升幅度小于大跌
- **领先/滞后**：有时领先股市，有时滞后

### VIX 期限结构

**正常（升水）**：
- 远月 VIX > 近月 VIX
- 表示市场预期波动率逐步上升

**倒挂（贴水）**：
- 近月 VIX > 远月 VIX
- 表示短期恐慌，市场预期恢复平静
- 通常是危机模式

---

## 局限性

1. **仅反映期权市场预期** - 不是实际波动率
2. **美股特有** - 其他市场需要本地化版本
3. **可操纵** - 期权价格理论上可被影响
4. **均值回归速度不确定** - 极端值可能持续
5. **衍生品跟踪误差** - VIX ETF 与 VIX 现货有差异
