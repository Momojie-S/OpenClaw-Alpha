# 资金流向分析

## 定义

资金流向分析是通过追踪市场资金进出情况，判断市场趋势和投资机会的分析方法。在 A 股市场，资金流向是重要的分析维度，因为资金是推动股价上涨的直接动力。

## 分类

### 按资金来源

| 类型 | 说明 | 数据源 |
|------|------|--------|
| 北向资金 | 外资通过沪港通/深港通流入 A 股的资金 | 每日公布，实时性强 |
| 主力资金 | 大单、超大单交易的资金（通常指机构和游资） | 交易所数据，需估算 |
| 散户资金 | 中小单交易的资金 | 交易所数据，需估算 |
| 融资资金 | 融资买入的资金 | 证监会数据，每日更新 |

### 按分析对象

| 类型 | 说明 | 用途 |
|------|------|------|
| 大盘资金 | 整个市场的资金流向 | 判断市场整体趋势 |
| 板块资金 | 行业/概念板块的资金流向 | 板块轮动分析 |
| 个股资金 | 单只股票的资金流向 | 个股选股和择时 |

## 核心指标

### 净流入/净流出

**公式**：
```
净流入 = 买入金额 - 卖出金额
净流入占比 = 净流入 / 成交额 × 100%
```

**意义**：
- 净流入 > 0：资金流入，买盘强于卖盘
- 净流入 < 0：资金流出，卖盘强于买盘

### 资金流向强度

**公式**：
```
资金流向强度 = 净流入 / 流通市值 × 100%
```

**意义**：标准化指标，便于跨股票/板块对比

### 连续流入/流出天数

**应用**：
- 连续 3 天以上流入：趋势确认信号
- 连续流入后转流出：趋势转折信号

### 主力资金占比

**公式**：
```
主力资金占比 = (大单 + 超大单) / 总成交额 × 100%
```

**意义**：
- 高占比（> 40%）：机构主导
- 低占比（< 20%）：散户主导

## 应用场景

### 1. 趋势判断

**大盘资金流向**：
- 持续流入 + 指数上涨：趋势向上，可加仓
- 持续流出 + 指数下跌：趋势向下，减仓或观望
- 流入减少 + 指数滞涨：可能见顶

**北向资金**：
- 外资持续流入：看好 A 股，中长期乐观
- 外资持续流出：避险情绪，注意风险

### 2. 板块轮动

**板块资金排名**：
- 资金流入 TOP 3 板块：当前热点，可关注
- 资金流出 TOP 3 板块：资金撤离，回避

**资金流向变化**：
- 流出转流入：趋势反转，关注
- 流入转流出：趋势见顶，警惕

### 3. 个股选股

**选股条件**：
- 近 3 日主力资金净流入
- 主力资金占比 > 30%
- 资金流入强度 > 1%

**避雷条件**：
- 连续 5 日主力资金净流出
- 大单持续流出但股价未跌（出货嫌疑）

### 4. 择时信号

**买入信号**：
- 连续 3 天流入 + 单日大幅流入（> 阈值）
- 流出转流入（趋势转折）

**卖出信号**：
- 连续 3 天流出 + 单日大幅流出（< 阈值）
- 流入转流出（趋势转折）

## 注意事项

### 1. 滞后性

**问题**：资金流向数据通常是 T+1 日公布，存在滞后。

**对策**：
- 结合盘中实时数据（如有）
- 使用北向资金（实时性强）作为先行指标
- 不要单独依赖资金流向做决策

### 2. 数据估算

**问题**：主力资金、散户资金是估算值，不同数据源口径不同。

**对策**：
- 选择权威数据源（如东方财富、同花顺）
- 了解数据源的计算方法
- 关注趋势变化，而非绝对值

### 3. 假信号

**问题**：资金流入不代表股价一定上涨，可能是主力出货。

**假信号特征**：
- 资金流入 + 股价滞涨：主力吸筹或出货
- 资金流出 + 股价抗跌：散户恐慌或主力护盘

**对策**：
- 结合价格走势、成交量综合判断
- 关注资金流入后的股价表现
- 设置止损位

### 4. 市场环境

**问题**：资金流向的有效性受市场环境影响。

**牛市**：
- 资金流入信号更可靠
- 资金流出可能是调仓

**熊市**：
- 资金流入可能是反弹
- 资金流出信号更可靠

**震荡市**：
- 资金流向频繁变化，信号有效性降低
- 关注趋势而非单日数据

### 5. 资金性质

**问题**：不同性质的资金有不同的投资风格和持仓周期。

**北向资金**：
- 偏好蓝筹、白马股
- 持仓周期长（月度/季度）
- 适合中长期投资者参考

**游资**：
- 偏好题材、小盘股
- 持仓周期短（日/周）
- 适合短线投资者参考

**融资资金**：
- 杠杆资金，风险偏好高
- 大幅增加或减少时需警惕

## 与其他分析方法的结合

### + 技术分析

- 资金流入 + 均线多头排列 → 加仓
- 资金流出 + 均线空头排列 → 减仓

### + 基本面分析

- 资金流入 + 业绩增长 → 中长期持有
- 资金流出 + 业绩下滑 → 及时止损

### + 情绪分析

- 北向资金流入 + 恐慌指数下降 → 市场见底
- 北向资金流出 + 恐慌指数上升 → 市场见顶
