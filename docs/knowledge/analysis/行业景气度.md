# 行业景气度

## 定义

行业景气度是衡量行业整体发展状况、趋势及市场预期的综合性指标，反映行业内企业盈利增长、需求扩张、政策支持等多维度状态。

**景气指数**：以 100 为临界值，范围 0-200：
- **景气指数 > 100**：行业处于景气状态，趋于上升或改善
- **景气指数 < 100**：行业处于不景气状态，趋于下降或恶化

---

## 核心指标体系

### 需求端指标

| 指标 | 说明 | 数据来源 |
|------|------|---------|
| 销售额/量 | 行业产品销售变化 | 行业协会、财报 |
| 订单量 | 新增订单趋势 | 公司公告、调研 |
| 产能利用率 | 产能利用程度 | 统计局 |

### 盈利端指标

| 指标 | 说明 | 特点 |
|------|------|------|
| 利润增速 | 行业整体利润变化 | 核心指标 |
| ROE | 净资产收益率变化 | 边际改善信号 |
| 毛利率/净利率 | 盈利能力变化 | 成本/定价能力 |

### 宏观层面指标

| 指标 | 说明 | 频率 |
|------|------|------|
| PMI 行业分项 | 采购经理人指数 | 月度 |
| 工业增加值 | 行业产出变化 | 月度 |
| 社融增速 | 信用环境 | 月度 |

---

## 扩散指数法（DI）

### 计算公式

```
DI = 扩张系列数 / 采用系列数 × 100
```

### 判断标准

| DI 值 | 含义 | 行业状态 |
|-------|------|---------|
| DI > 50% | 过半数指标上升 | 扩张/景气 |
| DI = 50% | 扩张与收缩平衡 | 转折点 |
| DI < 50% | 过半数指标下降 | 收缩/不景气 |

### 指标分类

**按时间关系**：
- **先行系列**：订货、库存投资、股价（领先景气变化）
- **一致系列**：生产活动指标（同步景气变化）
- **迟行系列**：库存余额、成本（滞后景气变化）

## 景气度前瞻方法

### 驱动力筛选（两步法）

1. **相关性分析**：初选与行业景气相关的宏观/行业指标
2. **正交回归**：剔除不显著变量，确定最终驱动力

### 常用前瞻指标

| 类型 | 指标 | 适用行业 |
|------|------|---------|
| 政策驱动 | 补贴政策、产业规划 | 新能源、科技 |
| 消费驱动 | 居民收入、消费倾向 | 消费、零售 |
| 投资驱动 | 固定资产投资、基建 | 周期、建材 |
| 出口驱动 | 海外订单、汇率 | 出口导向行业 |

## 景气度与投资

### 轮动逻辑

```
景气上行 → 业绩改善 → 估值扩张 → 股价上涨
```

### 配置原则

1. **高景气行业**：超配，享受业绩+估值双击
2. **景气拐点**：左侧布局，等待业绩验证
3. **景气下行**：低配或回避，避免业绩杀

### 验证指标

| 验证维度 | 指标 | 说明 |
|---------|------|------|
| 业绩验证 | 财报超预期 | 实际 vs 预期 |
| 资金验证 | 北向/公募增仓 | 机构认可度 |
| 估值验证 | PEG、估值分位 | 是否透支 |

## 注意事项

### 时效性

- **财报滞后**：定期财报公布滞后 1-2 个月
- **先行指标**：PMI、订单等高频数据更及时
- **一致预期**：分析师预期变化可前瞻

### 行业差异

- **周期行业**：关注价格、库存、产能利用率
- **消费行业**：关注渠道、销量、居民收入
- **科技行业**：关注订单、研发投入、政策

### 假信号

- **短期波动**：单月数据噪音大，需连续观察
- **季节因素**：剔除季节性后再判断
- **基数效应**：同比数据受基数影响
