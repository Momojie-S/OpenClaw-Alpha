# 军工行业景气度追踪

## 定义

军工行业景气度追踪是通过监测国防预算、订单交付、产能利用等指标，判断军工行业周期位置和投资机会的分析方法。是[[行业景气度]]和[[产业链分析]]在国防军工领域的具体应用。

**核心特点**：
- **计划性强**：受"五年规划"驱动，呈"前松后紧"节奏
- **需求刚性**：国防投入稳定，受经济周期影响小
- **保密性高**：信息披露受限，需间接指标推断
- **政策驱动**：地缘政治、军事战略影响大

---

## 一、产业链结构

```
原材料/元器件 → 分系统/零部件 → 整机/主机厂 → 军方用户
（上游材料）     （中游配套）      （下游总装）    （最终客户）
```

### 各环节代表企业

| 环节 | 代表企业 | 特点 |
|------|---------|------|
| 原材料 | 抚顺特钢、宝钛股份、西部超导 | 特殊钢材、钛合金、高温合金 |
| 元器件 | 紫光国微、振华科技、鸿远电子 | 军工电子、被动元件 |
| 分系统 | 航发控制、中航机电、航天电器 | 子系统配套 |
| 整机 | 中航沈飞、中航西飞、航发动力 | 主机厂，核心资产 |

### 产业链特点

| 特点 | 说明 |
|------|------|
| 整机厂核心地位 | 话语权强，订单分配者 |
| 配套体系封闭 | 资质壁垒高，竞争有限 |
| 长期合作关系 | 上下游绑定深，替换成本高 |

## 二、军工周期（五年规划周期）

### 2.1 周期特征

**周期长度**：5 年一个完整周期（与国家五年规划同步）

```
规划初期 → 规划中期 → 规划后期 → 规划收官
（1-2年）   （2-3年）   （4年）     （5年）
订单启动    批产爬坡    加速交付    集中交付
```

### 2.2 周期驱动因素

| 因素 | 作用机制 | 领先/滞后 |
|------|---------|----------|
| 国防预算 | 年度军费增长 | 领先 |
| 五年规划 | 装备采购节奏 | 领先 |
| 地缘政治 | 紧急需求触发 | 同步 |
| 订单节奏 | 采购计划落地 | 同步 |

### 2.3 历史周期回顾

| 周期 | 时间 | 特点 |
|------|------|------|
| "十二五" | 2011-2015 | 机械化装备为主 |
| "十三五" | 2016-2020 | 信息化装备加速 |
| "十四五" | 2021-2025 | 智能化、新质战斗力 |

## 四、核心指标解读

### 4.1 国防预算

**构成**：人员生活费 + 训练维持费 + 装备费

**增速意义**：

| 增速 | 含义 |
|------|------|
| >7% | 高增长，装备采购加速 |
| 5-7% | 稳健增长，符合预期 |
| <5% | 增速放缓，需关注原因 |

**注意**：装备费占比提升才是军工企业真正的利好

### 4.2 合同负债（预收款）

**定义**：已签合同但尚未确认收入的预付款

**判断标准**：

| 合同负债/营收 | 含义 |
|--------------|------|
| >50% | 订单充足，景气度高 |
| 30-50% | 订单正常 |
| <30% | 订单可能不足 |

### 4.3 存货

**存货构成**：原材料 + 在产品 + 产成品

**存货高企意味着**：
- 积极备产 → 预期订单增加（好事）
- 交付延迟 → 需求或生产问题（坏事）

需结合合同负债综合判断

## 五、下游需求追踪

### 5.1 需求结构（按装备类型）

| 装备类型 | 占比 | 周期性 | 驱动因素 |
|---------|------|--------|---------|
| 航空装备 | ~35% | 中 | 新机型列装、更新换代 |
| 航天装备 | ~20% | 低 | 卫星互联网、导弹 |
| 兵器装备 | ~15% | 低 | 陆军装备更新 |
| 舰船装备 | ~15% | 低 | 海军建设 |

## 六、细分赛道追踪

### 6.1 军工电子

**特点**：信息化核心，增长确定性强

**追踪指标**：
- 军工芯片国产化率
- 元器件订单增长
- 下游整机厂需求

**代表企业**：紫光国微、振华科技、鸿远电子、火炬电子

### 6.2 航空发动机

**特点**："长坡厚雪"赛道，技术壁垒最高

**追踪指标**：
- 主力型号批产进度（WS-10等）
- 新型号研制进展
- 维修后市场规模

**代表企业**：航发动力、航发控制

**投资逻辑**：
- 批产放量：主力型号产能爬坡
- 新品突破：新型号定型批产
- 维修市场：存量发动机维修

### 6.3 导弹

**特点**：消耗品属性，实战化训练受益

**追踪指标**：
- 军演频率和规模
- 导弹订单情况
- 远程火箭炮等新型号

**代表企业**：航天彩虹、高德红外、国科军工

### 6.4 军机

**特点**：主机厂核心，订单风向标

**追踪指标**：
- 新机型列装进度
- 合同负债变化
- 交付数量

**代表企业**：中航沈飞、中航西飞、中直股份

## 七、景气度综合判断

### 7.1 景气度评分卡

| 维度 | 权重 | 指标 | 评分标准 |
|------|------|------|---------|
| 国防预算 | 20% | 预算增速 | >7%=5分, 5-7%=3分, <5%=1分 |
| 订单 | 30% | 合同负债/营收 | >50%=5分, 30-50%=3分, <30%=1分 |
| 业绩 | 25% | 净利润增速 | >20%=5分, 10-20%=3分, <10%=1分 |
| 产能 | 15% | 产能利用率 | >90%=5分, 70-90%=3分, <70%=1分 |
| 政策 | 10% | 地缘紧张度 | 升级=5分, 稳定=3分, 缓和=1分 |

**总分判断**：
- 20-25 分：高景气
- 15-19 分：景气平稳
- < 15 分：景气下行

### 7.2 周期定位

| 信号 | 规划初期 | 规划中期 | 规划后期 | 收官期 |
|------|---------|---------|---------|--------|
| 订单 | 启动 | 稳定增长 | 加速 | 集中释放 |
| 业绩 | 低基数 | 稳定增长 | 高增长 | 业绩兑现 |
| 合同负债 | 上升 | 高位 | 维持 | 下降 |
| 估值 | 较高 | 中等 | 较低 | 修复 |

## 八、投资应用

### 8.1 周期投资法

**底部布局**：
- 信号：人事调整结束、订单重启、合同负债回升
- 标的：核心主机厂、高弹性配套

**高点谨慎**：
- 信号：合同负债持续下降、业绩不及预期
- 操作：减配或切换防御

### 8.2 成长投资法

**穿越周期的公司特征**：
- 核心技术（航空发动机、军工电子）
- 独家供应（细分领域垄断）
- 民品拓展（军民融合）
- 军贸出口（海外市场）

### 8.3 细分赛道选择

| 周期阶段 | 首选赛道 | 逻辑 |
|---------|---------|------|
| 订单启动期 | 主机厂 | 订单最先落地 |
| 批产爬坡期 | 配套企业 | 业绩弹性大 |
| 高景气期 | 军工电子 | 信息化需求 |
| 调整期 | 航发、电子 | 确定性高 |

## 九、特殊情况追踪

### 9.1 人事调整影响

**现象**：军工集团领导层调整期间，订单可能延迟

**应对**：
- 关注人事变动公告
- 预期调整期 6-12 个月
- 调整期结束后订单加速

### 9.2 军品降价

**现象**：军品价格调整，影响企业利润

**判断**：
- 降价幅度 vs 量增幅度
- 规模效应能否对冲
- 核心环节降价压力小

### 9.3 军民融合

**方向**：
- 军用技术民用（卫星互联网、低空经济）
- 民用技术军用（芯片、AI）

**追踪**：
- C919 大飞机进展
- 商业航天发展
- 低空经济政策

## 十一、注意事项

### 局限性

- **信息不透明**：核心数据保密，需间接推断
- **政策敏感**：政策变化影响大
- **估值难**：缺乏可参照的估值锚

### 常见误区

| 误区 | 事实 |
|------|------|
| 地缘冲突=军工必涨 | 冲突持续性、订单落地才是关键 |
| 国防预算增=业绩增 | 装备费占比、订单分配更重要 |
| 军工=成长股 | 大部分是周期股，估值方法不同 |
| 合同负债高=业绩高 | 需看交付节奏和利润率 |
