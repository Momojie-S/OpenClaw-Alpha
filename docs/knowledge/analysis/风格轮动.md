# 风格轮动

## 定义

股票市场中具有对立分类属性的股票池走势相对强弱随市场状况变化而变化的现象。最常见的风格分类为：
- **价值 vs 成长**：估值水平 vs 增长潜力
- **大盘 vs 小盘**：市值规模差异

---

## 核心逻辑

**轮动本质**：
- 不同风格对宏观环境敏感度不同
- 资金在不同风格间流动，追逐相对收益
- 风格相对强弱存在均值回归特征

**底层驱动**：
- 经济周期：复苏期价值占优，衰退期成长占优
- 流动性：宽松利好小盘，紧缩利好大盘
- 风险偏好：高涨期成长/小盘占优，低迷期价值/大盘占优
- 估值收敛：极端估值差会触发风格切换

---

## 风格定义

### 价值 vs 成长

| 维度 | 价值股 | 成长股 |
|------|--------|--------|
| 特征 | 低估值、高分红、稳定盈利 | 高增长、高估值、低分红 |
| 指标 | 低 PE/PB、高股息率 | 高营收/利润增速、高 PEG |
| 代表行业 | 银行、地产、公用事业 | 科技、医药、新能源 |
| 风险 | 价值陷阱、行业衰退 | 估值泡沫、增速不及预期 |

### 大盘 vs 小盘

| 维度 | 大盘股 | 小盘股 |
|------|--------|--------|
| 特征 | 流动性好、业绩稳定 | 弹性大、成长性强 |
| 风险 | 增速放缓、弹性不足 | 流动性差、波动大 |
| 代表指数 | 上证50、沪深300 | 中证1000、国证2000 |

---

## 轮动规律

### 经济周期驱动

| 周期阶段 | 占优风格 | 逻辑 |
|----------|----------|------|
| 复苏初期 | 价值、大盘 | 盈利改善、估值修复 |
| 繁荣期 | 成长、小盘 | 风险偏好上升、追逐弹性 |
| 滞胀期 | 价值、大盘 | 防御属性、流动性收紧 |
| 衰退期 | 成长、防御 | 利率下降、确定性溢价 |

### 流动性驱动

**宽松环境**：
- 利率下行 → 成长股估值弹性更大
- 资金充裕 → 小盘股流动性溢价
- 信用扩张 → 周期/价值受益

**紧缩环境**：
- 利率上行 → 价值股现金流更确定
- 资金收紧 → 大盘股流动性优势
- 信用收缩 → 防御属性受青睐

### 风险偏好驱动

| 风险偏好 | 占优风格 | 市场特征 |
|----------|----------|----------|
| 高涨 | 成长、小盘 | 追逐高弹性、容忍高估值 |
| 中性 | 均衡 | 风格分化不明显 |
| 低迷 | 价值、大盘 | 追求确定性、防御优先 |

## 判断指标

### 估值差

**相对估值比率**：
```
价值/成长估值比 = 价值指数PE / 成长指数PE
大盘/小盘估值比 = 大盘指数PE / 小盘指数PE
```

**解读**：
- 比率处于历史高位：价值/大盘相对昂贵，风格可能切换
- 比率处于历史低位：成长/小盘相对昂贵，风格可能切换
- 极端估值差是风格反转的领先指标

### 相对强弱

**相对收益率**：
```
风格相对收益 = 风格A收益率 - 风格B收益率
相对强度 = 风格A累计收益 / 风格B累计收益
```

**均线系统**：
- 相对强度上穿均线：风格A转强
- 相对强度下穿均线：风格A转弱
- 结合趋势确认避免假信号

### 动量指标

**相对动量**：
```
风格动量 = 风格指数N日收益率
相对动量差 = 风格A动量 - 风格B动量
```

**应用**：
- 动量差转正并扩大：风格A趋势加强
- 动量差转负并扩大：风格B趋势加强
- 动量差收窄：风格可能切换

### 宏观信号

| 指标 | 价值占优信号 | 成长占优信号 |
|------|-------------|-------------|
| 利率趋势 | 上行 | 下行 |
| 信用利差 | 扩大 | 收窄 |
| PMI | 回升 | 回落 |
| 盈利增速 | 加速 | 放缓 |

## A股风格特征

### 风格指数

**价值/成长**：
- 国证价值（399371）/ 国证成长（399370）
- 沪深300价值 / 沪深300成长
- 中证800价值 / 中证800成长

**大盘/小盘**：
- 上证50（大盘蓝筹）
- 沪深300（大盘代表）
- 中证500（中盘代表）
- 中证1000（小盘代表）
- 国证2000（微盘代表）

### 历史轮动规律

**年度级别轮动**：
- A股风格轮动周期通常1-2年
- 极端风格行情后往往伴随反转
- 风格切换通常由政策或宏观事件触发

**典型场景**：
- 2016-2017：大盘价值占优（供给侧改革）
- 2019-2020：成长占优（科技牛市）
- 2021：小盘成长占优（新能源）
- 2022-2023：价值防御占优（熊市防御）

### A股特殊性

**政策敏感**：
- 产业政策直接影响成长风格
- 金融监管影响大盘金融股
- 并购重组政策影响小盘股

**机构行为**：
- 外资偏好大盘价值/消费
- 公募偏成长，仓位影响风格
- 量化基金加剧风格波动

## 实战应用

### 风格判断流程

1. **宏观定位**：判断经济周期位置
2. **估值比较**：计算风格估值分位
3. **趋势确认**：相对强度 + 动量
4. **信号交叉验证**：多指标共振

### 配置策略

**战术风格轮动**：
- 根据风格信号调整仓位
- 偏向占优风格，不押注单一风格
- 保留基础仓位控制风险

**组合构建**：
- 核心：风格中性配置
- 卫星：风格轮动增强
- 动态调整权重

### 风险提示

**轮动失败风险**：
- 风格切换时点难以精确把握
- 假信号导致频繁调仓亏损
- 建议采用渐进式调仓

**极端行情**：
- 单一风格长期占优（如2019-2020成长）
- 轮动策略可能持续跑输
- 需设置止损或风格暴露上限

---

## 局限性

1. **择时难度** - 风格切换时点难以精确预测
2. **交易成本** - 频繁调仓增加成本
3. **信号滞后** - 技术指标可能滞后于实际切换
4. **结构性变化** - 市场结构变化影响历史规律有效性
