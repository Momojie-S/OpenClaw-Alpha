# 涨停追踪

## 定义

涨停追踪是通过监控股票涨停板的情况，分析市场热点、资金动向和情绪变化的分析方法。涨停板是 A 股市场特有的制度，限制了股票单日最大涨幅（普通股 10%，ST 股 5%，创业板/科创板 20%）。

## 涨停板基础

### 涨停类型

| 类型 | 涨幅限制 | 适用范围 |
|------|---------|---------|
| 主板涨停 | 10% | 沪市主板、深市主板 |
| ST 涨停 | 5% | ST、*ST 股票 |
| 创业板涨停 | 20% | 创业板股票 |
| 科创板涨停 | 20% | 科创板股票 |
| 北交所涨停 | 30% | 北交所股票 |

### 涨停术语

| 术语 | 定义 |
|------|------|
| 首板 | 当日首次涨停 |
| 连板 | 连续多个交易日涨停 |
| 炸板 | 涨停后打开，收盘未封死 |
| 天板 | 从跌停到涨停（地天板） |
| 地板 | 从涨停到跌停（天地板） |
| 一字板 | 开盘即涨停，无成交 |

## 核心指标

### 封单量

**定义**：涨停价位的买单总量

**意义**：
- 封单量大（> 流通盘 5%）：封板坚决，次日高开概率高
- 封单量小（< 流通盘 1%）：封板不稳，易炸板

### 封板时间

| 时间段 | 含义 |
|--------|------|
| 9:30-10:00 | 早盘封板，强势信号 |
| 10:00-11:00 | 上午封板，较强 |
| 13:00-14:00 | 下午封板，中等 |
| 14:00-14:30 | 尾盘封板，偏弱 |
| 14:30-15:00 | 临收盘封板，弱势，可能是偷袭 |

### 开板次数

| 次数 | 含义 |
|------|------|
| 0 次 | 一字板或开盘封死，最强 |
| 1-2 次 | 封板较稳，强势 |
| 3-5 次 | 反复震荡，分歧较大 |
| > 5 次 | 多空激烈博弈，风险较大 |

### 连板高度

| 连板数 | 市场含义 |
|--------|---------|
| 2 板 | 确认趋势，龙头候选 |
| 3 板 | 市场龙头，高度关注 |
| 4-5 板 | 妖股特征，高风险高收益 |
| 6+ 板 | 极端情绪，见顶风险大 |

## 涨停分类

### 按原因分类

| 类型 | 特征 | 持续性 |
|------|------|--------|
| 题材涨停 | 受政策/事件驱动 | 取决于题材热度 |
| 业绩涨停 | 业绩超预期 | 中长线 |
| 超跌反弹 | 跌幅大、估值低 | 短期反弹 |
| 次新涨停 | 新上市股票 | 高波动 |
| 复牌涨停 | 停牌后复牌 | 不确定 |

### 按形态分类

| 形态 | 描述 | 信号 |
|------|------|------|
| 一字板 | 开盘涨停无成交 | 最强，但难买入 |
| T 字板 | 涨停开盘，中间打开 | 强势，有分歧 |
| 实体板 | 盘中拉升至涨停 | 正常，可参与 |
| 尾盘板 | 收盘前拉涨停 | 较弱，可能是诱多 |

## 涨停与资金

### 涨停资金结构

| 资金类型 | 特征 | 后续走势 |
|---------|------|---------|
| 机构主导 | 机构买入占比高 | 走势相对稳健 |
| 游资主导 | 知名游资买入 | 波动大，快进快出 |
| 北向参与 | 外资买入 | 中长线看好 |
| 散户行情 | 无主力资金 | 持续性差 |

### 涨停与龙虎榜

涨停板股票通常会上龙虎榜，通过龙虎榜可以：
- 识别主导资金类型
- 判断游资操作意图
- 评估后续持续性

详见 [[龙虎榜与游资]]

## 涨停追踪策略

### 首板挖掘

**选股条件**：
- 首次涨停
- 封板时间早（< 10:30）
- 封单量大
- 有题材支撑
- 市值适中（20-100 亿）

**风险**：首板失败率高，需结合题材和资金判断

### 连板接力

**参与条件**：
- 2-3 板确认趋势
- 题材持续发酵
- 龙头地位确立
- 封板坚决

**退出信号**：
- 炸板无法回封
- 放量滞涨
- 龙虎榜显示资金撤退

### 炸板回封

**定义**：涨停打开后又封回

**机会**：
- 强势股洗盘
- 上车机会

**风险**：
- 可能是主力出货
- 真炸板

### 跌停抄底

**条件**：
- 基本面无重大问题
- 跌停原因是情绪恐慌
- 有抄底资金介入

**风险**：极高，需谨慎

## 涨停统计指标

### 市场情绪指标

| 指标 | 计算方法 | 意义 |
|------|---------|------|
| 涨停家数 | 当日涨停股票数量 | 市场热度 |
| 跌停家数 | 当日跌停股票数量 | 市场恐慌 |
| 涨跌比 | 涨停家数 / 跌停家数 | 多空对比 |
| 封板率 | 未炸板数 / 涨停总数 | 封板质量 |

### 个股统计指标

| 指标 | 说明 |
|------|------|
| 连板天数 | 连续涨停天数 |
| 累计涨幅 | 连板期间累计涨幅 |
| 上榜次数 | 近期涨停次数 |
| 炸板次数 | 涨停后打开次数 |

## 风险警示

### 高风险信号

| 信号 | 含义 |
|------|------|
| 天地板 | 涨停到跌停，极端风险 |
| 连续炸板 | 多头力竭 |
| 放量滞涨 | 出货嫌疑 |
| 高位涨停 | 追高风险大 |
| 监管关注 | 可能停牌核查 |

### 常见陷阱

| 陷阱 | 特征 |
|------|------|
| 诱多涨停 | 尾盘偷袭，次日低开 |
| 对倒涨停 | 成交异常，主力出货 |
| 假龙头 | 跟风涨停，无持续性 |
| 一日游 | 游资快进快出 |

## 与其他分析方法结合

### + 龙虎榜分析

涨停 + 龙虎榜数据 → 判断资金意图

### + 题材分析

涨停 + 题材热度 → 判断持续性

### + 技术分析

涨停 + 技术形态 → 判断买点卖点

### + 基本面分析

涨停 + 业绩支撑 → 判断是否价值发现
