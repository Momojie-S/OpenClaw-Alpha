# 多指标组合策略

## 定义

将多个技术指标结合使用，通过指标间的相互确认提高信号可靠性，减少假信号的分析方法。

---

## 为什么需要组合

**单一指标的局限性**：
| 问题 | 说明 |
|------|------|
| 假信号多 | 每个指标都会产生噪音信号 |
| 滞后性 | 大多数指标基于历史数据 |
| 适用场景受限 | 某些指标在特定市场更有效 |
| 互相矛盾 | 不同指标可能给出相反信号 |

**组合的优势**：
- **过滤假信号**：多指标同时确认，可靠性更高
- **互补性强**：趋势指标 + 震荡指标，覆盖不同市场
- **多重验证**：信号得到多维确认

---

## 指标分类

### 趋势类指标

**特点**：追踪趋势方向，适合趋势市场

| 指标 | 核心信号 | 适用场景 |
|------|---------|---------|
| MA（均线） | 金叉/死叉、多头/空头排列 | 明显趋势市场 |
| MACD | DIF/DEA 交叉、柱状图 | 趋势强度判断 |
| ADX | 趋势强度（>25 趋势存在） | 区分趋势/震荡 |

### 震荡类指标

**特点**：识别超买超卖，适合震荡市场

| 指标 | 核心信号 | 适用场景 |
|------|---------|---------|
| RSI | >70 超买，<30 超卖 | 震荡区间 |
| KDJ | K/D 交叉，超买超卖区 | 短期波动 |
| Stochastic | %K/%D 交叉 | 震荡市场 |

### 波动类指标

**特点**：衡量价格波动范围

| 指标 | 核心信号 | 适用场景 |
|------|---------|---------|
| 布林带 | 触及上下轨、带宽变化 | 波动率判断 |
| ATR | 真实波动幅度 | 止损位设置 |

## 经典组合策略

### 组合一：MACD + RSI

**原理**：MACD 判断趋势，RSI 判断超买超卖

**买入信号**：
1. MACD 金叉（DIF 上穿 DEA）
2. RSI < 50（非超买区）
3. 趋势确认：ADX > 25（可选）

**卖出信号**：
1. MACD 死叉（DIF 下穿 DEA）
2. RSI > 70（超买区）
3. 或出现顶背离（价格新高，RSI 不新高）

**优势**：
- 趋势确认 + 位置判断
- 避免 MACD 在超买区的金叉陷阱

---

### 组合二：KDJ + 布林带

**原理**：布林带提供支撑/压力，KDJ 提供买卖时机

**买入信号**：
1. 价格触及布林下轨
2. KDJ 的 K 值 < 20，K 上穿 D
3. 或 J 值 < 0 后回升

**卖出信号**：
1. 价格触及布林上轨
2. KDJ 的 K 值 > 80，K 下穿 D
3. 或 J 值 > 100 后回落

**优势**：
- 布林带提供价格边界
- KDJ 提供精确入场点

---

### 组合三：均线系统 + RSI + MACD（三重确认）

**原理**：多维度确认，可靠性最高

**买入信号**（需同时满足）：
1. 均线：价格站上 MA20，或短期均线上穿长期均线
2. RSI：从超卖区回升，或 RSI > 50
3. MACD：金叉或柱状图由负转正

**卖出信号**（满足 2 个以上）：
1. 均线：跌破 MA20，或死叉
2. RSI：进入超买区后回落
3. MACD：死叉或柱状图由正转负

**优势**：
- 三重确认，假信号大幅减少
- 覆盖趋势、动量、位置三个维度

---

### 组合四：MACD + 布林带 + ADX

**原理**：先判断趋势，再寻找入场点

**使用流程**：
1. **判断趋势**：ADX > 25 说明趋势存在
2. **确认方向**：MACD 金叉做多，死叉做空
3. **寻找入场**：布林带回调至中轨或下轨（做多）

**优势**：
- 先判断市场状态（趋势/震荡）
- 避免在震荡市使用趋势指标

## 信号确认机制

### 共振信号

**定义**：多个指标在同一时间发出相同方向信号

**强度分级**：
| 共振数量 | 信号强度 | 可靠性 |
|---------|---------|--------|
| 2 个指标 | 中等 | 中等 |
| 3 个指标 | 强 | 高 |
| 4+ 个指标 | 很强 | 很高 |

### 时间确认

**多周期确认**：
1. 大周期（周线/月线）判断趋势方向
2. 中周期（日线）确认趋势
3. 小周期（60分钟/30分钟）寻找入场点

**示例**：
- 周线：上升趋势
- 日线：MACD 金叉
- 60分钟：RSI 回调至 40-50 区间
→ 大周期顺势，小周期入场

### 背离确认

**定义**：价格与指标走势不一致

**类型**：
- **顶背离**：价格创新高，指标不创新高 → 看跌
- **底背离**：价格创新低，指标不创新低 → 看涨

**多指标背离**：
- MACD + RSI 同时背离 → 信号更强
- 单一指标背离需谨慎

## 参数优化

### 默认参数

| 指标 | 默认参数 | 说明 |
|------|---------|------|
| MACD | 12, 26, 9 | 日线标准 |
| RSI | 14 | 中等敏感度 |
| KDJ | 9, 3, 3 | 9 日周期 |
| 布林带 | 20, 2 | 20 日均线，2 倍标准差 |

### 参数调整原则

**短线交易**：
- 缩短周期（如 RSI 用 6-9）
- 提高敏感度，但噪音增加

**中线交易**：
- 使用默认参数
- 平衡敏感度和稳定性

**长线交易**：
- 延长周期（如 RSI 用 21-28）
- 降低噪音，但信号滞后

### 避免"参数拟合"

**问题**：过度优化参数以适应历史数据，导致实盘失效

**原则**：
1. 参数在多个品种上都要有效
2. 参数在多个时间段都要有效
3. 参数有逻辑支撑，而非纯数据拟合

## 实战应用

### 信号过滤流程

```
1. 判断市场状态（趋势/震荡）
   ↓
2. 选择适合的指标组合
   ↓
3. 等待共振信号
   ↓
4. 多周期确认
   ↓
5. 设置止损止盈
   ↓
6. 执行交易
```

### 止损设置

**技术止损**：
- 固定点数：买入价 - N 倍 ATR
- 均线止损：跌破关键均线
- 指标止损：MACD 死叉或 RSI 跌破关键位

### 风险控制

| 原则 | 说明 |
|------|------|
| 不重仓单一信号 | 即使多指标共振，也控制仓位 |
| 接受假信号 | 技术分析不是 100% 准确 |
| 严格止损 | 执行纪律比分析更重要 |

## 常见陷阱

### 1. 指标过多

**问题**：同时使用太多指标，信号混乱
**解决**：2-4 个指标足够，选择互补的指标

### 2. 确认偏误

**问题**：只关注支持自己观点的信号
**解决**：建立客观的信号规则，机械执行

### 3. 忽视市场环境

**问题**：在震荡市用趋势指标，或反之
**解决**：先用 ADX 等判断市场状态

### 4. 过度优化

**问题**：历史回测很好，实盘亏损
**解决**：样本外测试，参数保持稳健
