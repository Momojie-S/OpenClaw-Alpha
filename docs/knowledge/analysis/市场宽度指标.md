# 市场宽度指标

## 定义

衡量市场整体参与度的指标体系，用于判断指数涨跌是否得到大多数股票的支持，识别趋势健康程度和潜在反转信号。

---

## 核心逻辑

**趋势健康判断**：
- 指数上涨 + 宽度改善 = 健康上涨（多数股票参与）
- 指数上涨 + 宽度恶化 = 背离（少数权重股拉升）
- 指数下跌 + 宽度恶化 = 健康下跌
- 指数下跌 + 宽度改善 = 背离（可能反弹）

**背离信号**：
- 顶背离：指数创新高但宽度指标下降 → 上涨动力衰减
- 底背离：指数创新低但宽度指标上升 → 抛压减轻

---

## 核心指标

### 腾落线（ADL）

**定义**：累积上涨家数减下跌家数的连续曲线，用于判断市场趋势的健康程度。

**公式**：
```
ADL = 前日ADL + (今日上涨家数 - 今日下跌家数)
```

**解读**：
- ADL 上升：多数股票上涨，趋势健康
- ADL 下降：多数股票下跌，趋势走弱
- ADL 与指数背离：趋势可能反转

**背离信号**：
- 看跌背离：指数上涨但 ADL 下降 → 上涨由少数权重股推动，趋势不健康
- 看涨背离：指数下跌但 ADL 上升 → 下跌股票减少，可能接近底部

**A股应用**：
- 使用全市场或沪深300成分股
- 排除停牌股票

**时间周期**：主要用于中长期趋势分析

---

### 涨跌比（A/D Ratio）

**定义**：上涨家数与下跌家数的比值。

**公式**：
```
涨跌比 = 上涨家数 / 下跌家数
```

**解读**：
- > 2：强势市场，情绪亢奋
- 1-2：健康上涨
- 0.5-1：弱势调整
- < 0.5：恐慌抛售

---

### 阿姆氏指标（TRIN）

**定义**：结合涨跌家数和涨跌成交量的复合指标，比较涨跌比例与涨跌成交量比例的关系。

**公式**：
```
涨跌比 = 上涨家数 / 下跌家数
涨跌量比 = 上涨成交量 / 下跌成交量
TRIN = 涨跌比 / 涨跌量比
```

**解读**：
- TRIN < 1：资金流向上涨股，多头强势（伴随价格上涨）
- TRIN = 1：多空平衡
- TRIN > 1：资金流向下跌股，空头强势（伴随价格下跌）
- TRIN > 2：极端恐慌（反向买入信号）
- TRIN < 0.5：极端贪婪（警惕见顶）

**注意**：
- 与涨跌比相反，TRIN 低是看多信号
- TRIN 与价格走势相反：价格大涨时 TRIN 下降，价格大跌时 TRIN 上升
- 适合日内交易或短期分析
- 与 A/D Line 配合使用：A/D Line 用于中长期，TRIN 用于短期

---

### 新高新低指标（NHNL）

**定义**：创N日新高股票数减创新低股票数。

**公式**：
```
NHNL = 创新高家数 - 创新低家数
```

**参数**：常用 52 周（约 250 个交易日）

**解读**：
- NHNL > 0 且上升：市场强势
- NHNL < 0 且下降：市场弱势
- NHNL 与指数背离：趋势可能反转

---

### 麦克莱伦震荡指标（McClellan Oscillator）

**定义**：对涨跌家数差进行指数平滑处理。

**公式（原始）**：
```
涨跌差 = 上涨家数 - 下跌家数
EMA19 = 涨跌差的19日指数平滑（系数0.10）
EMA39 = 涨跌差的39日指数平滑（系数0.05）
McClellan = EMA19 - EMA39
```

**公式（调整后）**：
用于长期比较，消除上市股票数量变化的影响：
```
ANA = (上涨家数 - 下跌家数) / (上涨家数 + 下跌家数)
EMA19 = ANA的19日指数平滑
EMA39 = ANA的39日指数平滑
调整后McClellan = EMA19 - EMA39
```

**解读**：
- > +100：超买
- < -100：超卖
- 零轴穿越：趋势转换信号
- 指标背离：趋势反转预警

**宽度脉冲（Breadth Thrust）**：
- McClellan 从负值快速穿越到正值，幅度超过100点
- 表示市场情绪急剧转变，强烈的趋势反转信号
- 通常出现在重要底部

---

### 麦克莱伦总和指数（MSI）

**定义**：麦克莱伦震荡指标的累积值，用于识别主要趋势反转。

**公式**：
```
MSI = MSI(前日) + 今日McClellan
```

**解读**：
- MSI > 0：看涨，上涨股票多于下跌股票
- MSI < 0：看跌，下跌股票多于上涨股票

**极值信号**：
| 信号 | 阈值 | 含义 |
|------|------|------|
| 主要底部 | MSI < -1300 | 可能是重要底部 |
| 主要顶部 | MSI > +1600 + 背离 | 警惕见顶风险 |
| 牛市启动 | 从低点上涨3600点后突破+1900 | 新牛市开始信号 |

**与 McClellan 的关系**：
- McClellan 用于短期波动分析
- MSI 用于中长期趋势确认
- MSI 平滑了日常波动，信号更可靠

---

### 200日均线以上股票比例

**定义**：股价位于200日均线之上的股票占全市场的比例。

**公式**：
```
200日均线比例 = 股价 > 200日均线的股票数 / 总股票数 × 100%
```

**解读**：
| 比例 | 市场状态 |
|------|---------|
| > 80% | 极度强势，警惕过热 |
| 60-80% | 牛市特征 |
| 40-60% | 震荡市 |
| 20-40% | 熊市特征 |
| < 20% | 极度弱势，关注底部机会 |

**交易信号**：
- 比例从低位突破 50%：趋势转强
- 比例从高位跌破 50%：趋势转弱
- 与指数背离：趋势可能反转

**优点**：
- 直观反映市场整体位置
- 不受权重股影响
- 适合中长期趋势判断

---

### 成交量腾落线（ADVL）

**定义**：结合成交量的累积指标，用涨跌成交量替代涨跌家数。

**公式**：
```
涨跌量差 = 上涨成交量 - 下跌成交量
ADVL = ADVL(前日) + 涨跌量差
```

**与 ADL 的区别**：
- ADL：只看股票数量，每只股票权重相同
- ADVL：考虑成交量，大成交量的股票权重更大

**解读**：
- ADVL 上升：资金流向上涨股，买盘强劲
- ADVL 下降：资金流向下跌股，卖压沉重
- ADVL 与 ADL 背离：需关注成交量验证

---

## 指标对比

| 指标 | 类型 | 适用周期 | 信号特点 |
|------|------|---------|---------|
| ADL | 累积 | 中长期 | 趋势背离 |
| 涨跌比 | 即时 | 短期 | 当日强度 |
| TRIN | 即时 | 短期 | 资金方向 |
| NHNL | 即时 | 中期 | 趋势强度 |
| McClellan | 震荡 | 短中期 | 超买超卖 |
| MSI | 累积 | 中长期 | 主要趋势反转 |
| 200日均线比例 | 即时 | 中长期 | 整体位置 |
| ADVL | 累积 | 中长期 | 资金流向验证 |

## 实战应用

### 趋势确认

**上涨趋势健康**：
- 指数上涨 + ADL 上升 + NHNL 正值
- McClellan 在零轴上方

**下跌趋势确认**：
- 指数下跌 + ADL 下降 + NHNL 负值
- McClellan 在零轴下方

---

### 背离交易

**顶背离**：
1. 指数创新高
2. ADL 或 MSI 未创新高
3. 成交量萎缩
4. 减仓或止盈

**底背离**：
1. 指数创新低
2. ADL 或 MSI 未创新低
3. 恐慌抛售（TRIN > 2）
4. 寻找买入机会

---

### 极端值交易

**恐慌极值**：
- TRIN > 2
- 涨跌比 < 0.3
- McClellan < -150
- 200日均线比例 < 20%
- 可能是短线买入点

**贪婪极值**：
- TRIN < 0.5
- 涨跌比 > 3
- McClellan > +150
- 200日均线比例 > 80%
- 警惕回调风险

### 宽度脉冲（Breadth Thrust）

**定义**：市场宽度在极短时间内从超卖转向超买。

**典型特征**：
- McClellan 从负值快速穿越到正值，幅度超过100点
- 涨跌比从 < 0.5 快速反弹到 > 2
- 通常出现在重要底部后

**交易含义**：
- 强烈的趋势反转信号
- 不应等待回调，直接跟进
- 历史上宽度脉冲后通常有可观涨幅

---

## A股特殊性

**数据来源**：
- 沪深两市分开统计
- 常用沪深300或中证800作为基准
- 排除ST、停牌股票

**板块效应**：
- 权重股影响指数但不算"宽度"
- 可单独统计中小盘宽度
- 关注上证50 vs 中证1000 的宽度差异

**涨跌停限制**：
- 涨跌停股票无法继续参与当日宽度
- 涨停家数多时宽度指标可能失真
- 建议结合涨跌停统计使用

---

## 局限性

1. **数据依赖** - 需要全市场涨跌数据
2. **参数敏感** - EMA周期选择影响信号
3. **假信号** - 单一指标可能误判
4. **市场结构变化** - IPO、退市影响历史可比性
