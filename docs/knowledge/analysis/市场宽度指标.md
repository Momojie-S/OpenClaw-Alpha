# 市场宽度指标

## 定义

衡量市场整体参与度的指标体系，用于判断指数涨跌是否得到大多数股票的支持，识别趋势健康程度和潜在反转信号。

---

## 核心逻辑

**趋势健康判断**：
- 指数上涨 + 宽度改善 = 健康上涨（多数股票参与）
- 指数上涨 + 宽度恶化 = 背离（少数权重股拉升）
- 指数下跌 + 宽度恶化 = 健康下跌
- 指数下跌 + 宽度改善 = 背离（可能反弹）

**背离信号**：
- 顶背离：指数创新高但宽度指标下降 → 上涨动力衰减
- 底背离：指数创新低但宽度指标上升 → 抛压减轻

---

## 核心指标

### 腾落线（ADL）

**定义**：累积上涨家数减下跌家数的连续曲线。

**公式**：
```
ADL = 前日ADL + (今日上涨家数 - 今日下跌家数)
```

**解读**：
- ADL 上升：多数股票上涨，趋势健康
- ADL 下降：多数股票下跌，趋势走弱
- ADL 与指数背离：趋势可能反转

**A股应用**：
- 使用全市场或沪深300成分股
- 排除停牌股票

---

### 涨跌比（A/D Ratio）

**定义**：上涨家数与下跌家数的比值。

**公式**：
```
涨跌比 = 上涨家数 / 下跌家数
```

**解读**：
- > 2：强势市场，情绪亢奋
- 1-2：健康上涨
- 0.5-1：弱势调整
- < 0.5：恐慌抛售

---

### 阿姆氏指标（TRIN）

**定义**：结合涨跌家数和涨跌成交量的复合指标。

**公式**：
```
TRIN = (上涨家数/下跌家数) / (上涨成交量/下跌成交量)
```

**解读**：
- TRIN < 1：资金流向上涨股，多头强势
- TRIN = 1：多空平衡
- TRIN > 1：资金流向下跌股，空头强势
- TRIN > 2：极端恐慌（反向买入信号）
- TRIN < 0.5：极端贪婪（警惕见顶）

**注意**：与涨跌比相反，TRIN 低是看多信号。

---

### 新高新低指标（NHNL）

**定义**：创N日新高股票数减创新低股票数。

**公式**：
```
NHNL = 创新高家数 - 创新低家数
```

**参数**：常用 52 周（约 250 个交易日）

**解读**：
- NHNL > 0 且上升：市场强势
- NHNL < 0 且下降：市场弱势
- NHNL 与指数背离：趋势可能反转

---

### 麦克莱伦震荡指标（McClellan Oscillator）

**定义**：对涨跌家数差进行指数平滑处理。

**公式**：
```
涨跌差 = 上涨家数 - 下跌家数
EMA19 = 涨跌差的19日指数平滑
EMA39 = 涨跌差的39日指数平滑
McClellan = EMA19 - EMA39
```

**解读**：
- > +100：超买
- < -100：超卖
- 零轴穿越：趋势转换信号
- 指标背离：趋势反转预警

---

### 麦克莱伦总和指数（MSI）

**定义**：麦克莱伦震荡指标的累积值。

**公式**：
```
MSI = MSI(前日) + 今日McClellan
```

**解读**：
- MSI 持续上升：中期趋势向上
- MSI 持续下降：中期趋势向下
- MSI 与指数背离：重要反转信号

---

## 指标对比

| 指标 | 类型 | 适用周期 | 信号特点 |
|------|------|---------|---------|
| ADL | 累积 | 中长期 | 趋势背离 |
| 涨跌比 | 即时 | 短期 | 当日强度 |
| TRIN | 即时 | 短期 | 资金方向 |
| NHNL | 即时 | 中期 | 趋势强度 |
| McClellan | 震荡 | 短中期 | 超买超卖 |
| MSI | 累积 | 中长期 | 趋势确认 |

---

## 实战应用

### 趋势确认

**上涨趋势健康**：
- 指数上涨 + ADL 上升 + NHNL 正值
- McClellan 在零轴上方

**下跌趋势确认**：
- 指数下跌 + ADL 下降 + NHNL 负值
- McClellan 在零轴下方

---

### 背离交易

**顶背离**：
1. 指数创新高
2. ADL 或 MSI 未创新高
3. 成交量萎缩
4. 减仓或止盈

**底背离**：
1. 指数创新低
2. ADL 或 MSI 未创新低
3. 恐慌抛售（TRIN > 2）
4. 寻找买入机会

---

### 极端值交易

**恐慌极值**：
- TRIN > 2
- 涨跌比 < 0.3
- McClellan < -150
- 可能是短线买入点

**贪婪极值**：
- TRIN < 0.5
- 涨跌比 > 3
- McClellan > +150
- 警惕回调风险

---

## A股特殊性

**数据来源**：
- 沪深两市分开统计
- 常用沪深300或中证800作为基准
- 排除ST、停牌股票

**板块效应**：
- 权重股影响指数但不算"宽度"
- 可单独统计中小盘宽度
- 关注上证50 vs 中证1000 的宽度差异

**涨跌停限制**：
- 涨跌停股票无法继续参与当日宽度
- 涨停家数多时宽度指标可能失真
- 建议结合涨跌停统计使用

---

## 局限性

1. **数据依赖** - 需要全市场涨跌数据
2. **参数敏感** - EMA周期选择影响信号
3. **假信号** - 单一指标可能误判
4. **市场结构变化** - IPO、退市影响历史可比性
