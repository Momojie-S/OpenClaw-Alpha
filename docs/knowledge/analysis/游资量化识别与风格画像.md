# 游资量化识别与风格画像

## 定义

## 二、量化识别指标

### 2.1 游资强度指标

**单席位影响力**：
```
影响力 = 单席位买入金额 / 当日总成交额 × 100%
```

**判定标准**：
- > 5%：强控盘，游资主导
- 2-5%：中等影响
- < 2%：跟风参与

### 2.2 游资协同度

**多游资共振**：
```
协同度 = 知名游资席位数量 / 上榜买卖席位总数
```

**判定**：
- 协同度 > 0.4 + 净买入为正：多游资共振，看好
- 协同度 > 0.4 + 净卖出为正：多游资撤退，看空

### 2.3 游资持续性

**N日净买入**：
```
持续性 = ∑(近N日游资净买入) / 流通市值
```

**应用**：
- 连续 3 日净流入 + 持续性 > 1%：游资建仓，可能拉升
- 连续流入后转流出：游资出货

### 2.4 游资成本估算

**加权成本**：
```
成本价 = ∑(买入金额 × 买入均价) / 总买入金额
```

**应用**：
- 当前价 > 成本价 + 10%：游资盈利，注意出货
- 当前价 < 成本价 - 5%：游资被套，可能自救

## 三、行为模式识别

### 3.1 封板模式

**特征**：
- 尾盘（14:30后）大单买入
- 买入金额 > 当日成交额的 5%
- 股价封死涨停至收盘

**量化**：
```
封板强度 = 尾盘30分钟买入 / 总买入
封板确认：封板强度 > 0.6 且 收盘价 = 涨停价
```

### 3.2 引导模式

**特征**：
- 早盘（9:30-10:30）快速拉升
- 买入集中在开盘后1小时
- 目的是吸引跟风盘

**量化**：
```
引导强度 = 早盘60分钟买入 / 总买入
引导确认：引导强度 > 0.5 且 早盘涨幅 > 3%
```

### 3.3 低吸模式

**特征**：
- 股价大幅下跌时买入
- 可能是跌停板吸货
- 买入分散在全天

**量化**：
```
低吸特征 = 日内低点附近买入占比 > 40%
且 股价日内跌幅 > 5%
```

### 3.4 做T模式

**特征**：
- 同一席位同时出现在买入和卖出榜
- 买入金额 ≈ 卖出金额

**量化**：
```
做T确认：同一席位双向上榜
且 min(买入,卖出) / max(买入,卖出) > 0.5
```

## 四、游资生命周期识别

### 4.1 建仓期

**信号**：
- 连续 2-3 日小幅净买入
- 买入金额逐步增加
- 股价涨幅不大（< 5%）

**特征值**：
```
建仓信号 = 连续N日净买入 且 涨幅 < 5% 且 买入递增
```

### 4.2 拉升期

**信号**：
- 单日大幅净买入（> 流通市值 2%）
- 可能出现涨停
- 换手率放大

**特征值**：
```
拉升信号 = 单日净买入/流通市值 > 2% 且 涨幅 > 7%
```

### 4.3 出货期

**信号**：
- 净卖出转正
- 高位震荡
- 换手率异常放大

**特征值**：
```
出货信号 = 净卖出 且 高位（近期高点附近）且 换手率 > 15%
```

## 五、实战应用

### 5.1 每日监控清单

1. **龙虎榜净买入 TOP 10**
   - 关注知名游资席位
   - 记录买入金额和比例

2. **新游资介入**
   - 之前未关注的席位大额买入
   - 可能是新资金入场

3. **游资撤退信号**
   - 知名游资从净买入转净卖出
   - 连续 2 日以上净卖出

### 5.2 个股跟踪模板

| 日期 | 席位 | 买入(万) | 卖出(万) | 净额 | 累计成本 | 阶段判断 |
|------|------|---------|---------|------|---------|---------|
| 03-01 | 章盟主 | 5000 | 0 | +5000 | - | 建仓 |
| 03-02 | 章盟主 | 3000 | 0 | +3000 | - | 加仓 |
| 03-03 | 章盟主 | 2000 | 8000 | -6000 | 12.5 | 出货 |

### 5.3 风险信号

| 信号 | 含义 | 应对 |
|------|------|------|
| 游资买入后股价不涨 | 被套或诱多 | 不追 |
| 多游资同时撤退 | 集体看空 | 及时止损 |
| 高位换手率 > 20% | 派发嫌疑 | 减仓 |
| 知名游资做T | 不看好后市 | 观望 |

## 六、注意事项

### 6.1 数据局限

- **滞后性**：T+1 披露，无法实时跟踪
- **不完整**：只披露前五名，看不到全貌
- **马甲问题**：游资可能使用多个席位

### 6.2 假信号识别

| 假信号 | 特征 | 原因 |
|--------|------|------|
| 游资买入后暴跌 | 隔日低开低走 | 抄底被套 |
| 游资卖出后暴涨 | 隔日继续涨停 | 卖飞/洗盘 |
| 名单无游资 | 但股价连续涨停 | 散户行情或隐秘操作 |

### 6.3 不建议盲目跟风

- 游资有信息优势，散户跟风易被收割
- 应作为分析工具，而非唯一决策依据
- 结合基本面、技术面综合判断
