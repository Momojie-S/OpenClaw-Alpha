# 半导体行业景气度追踪

## 定义

半导体行业景气度追踪是通过监测产业链各环节的关键指标，判断"硅周期"位置、供需状况和投资机会的分析方法。是[[行业景气度]]和[[产业链分析]]在高科技周期行业的具体应用。

**核心特点**：
- **强周期性**：3-4 年一个完整周期（硅周期）
- **全球联动**：全球定价，受海外需求影响大
- **技术驱动**：摩尔定律推进，技术迭代快
- **资本密集**：晶圆厂投资巨大，扩产周期长

---

## 一、产业链结构

```
硅片/化学品/设备 → 晶圆制造 → 封装测试 → 终端应用
（上游材料/设备）  （晶圆厂）   （封测厂）  （下游需求）
      Fabless ← ─ ─ ─ IC设计 ─ ─ ─ → 终端厂商
```

### 各环节代表企业

| 环节 | 全球代表 | 国内代表 | 特点 |
|------|---------|---------|------|
| 硅片 | 信越化学、胜高 | 沪硅产业、立昂微 | 日企垄断 |
| 设备 | ASML、AMAT、LAM | 北方华创、中微公司 | 美荷日主导 |
| 晶圆制造 | TSMC、三星、Intel | 中芯国际、华虹 | 投资大、周期长 |
| 封装测试 | 日月光、安靠 | 长电科技、通富微电 | 劳动密集 |
| IC设计 | NVIDIA、高通、AMD | 海思、韦尔、兆易创新 | 轻资产、设计密集 |

### 商业模式

| 模式 | 说明 | 代表 |
|------|------|------|
| IDM | 设计+制造+封测一体化 | Intel、三星、TI |
| Foundry | 纯代工 | TSMC、中芯国际 |
| Fabless | 纯设计 | NVIDIA、高通 |
| OSAT | 纯封测 | 日月光、长电 |

## 二、硅周期（半导体周期）

### 2.1 周期特征

**周期长度**：3-4 年一个完整周期

```
复苏期 → 扩张期 → 顶峰期 → 衰退期 → 底部期 → 复苏期
（需求回暖）（产能扩张）（供需失衡）（库存去化）（产能出清）
```

### 2.2 周期驱动因素

| 因素 | 作用机制 | 领先/滞后 |
|------|---------|----------|
| 终端需求 | 手机/PC/服务器需求变化 | 领先 |
| 库存周期 | 产业链库存去化/累积 | 同步 |
| 产能周期 | 晶圆厂扩产/投产 | 滞后 |
| 价格周期 | 芯片价格涨跌 | 同步 |

### 2.3 历史周期回顾

| 周期 | 时间 | 触发因素 |
|------|------|---------|
| PC周期 | 1990s-2000s | 个人电脑普及 |
| 手机周期 | 2000s-2010s | 智能手机爆发 |
| AI周期 | 2023-至今 | AI算力需求爆发 |

## 四、下游需求追踪

### 4.1 需求结构（按终端）

| 终端 | 占比 | 周期性 | 驱动因素 |
|------|------|--------|---------|
| 通信（手机） | ~30% | 中 | 换机周期、5G渗透 |
| 计算机（PC/服务器） | ~25% | 高 | 企业IT支出、AI服务器 |
| 消费电子 | ~15% | 高 | 居民消费 |
| 汽车 | ~15% | 低 | 新能源车渗透 |
| 工业 | ~10% | 低 | 自动化升级 |
| 政府/其他 | ~5% | 低 | 国防、航天 |

### 4.2 各终端追踪方法

**手机**：
- 全球手机出货量（IDC、Canalys）
- 新机发布节奏
- 库存水平

**PC/服务器**：
- PC出货量（IDC、Gartner）
- 服务器出货量
- 云厂商资本开支

**汽车**：
- 新能源车销量
- 汽车电子渗透率

## 五、库存周期追踪

### 5.1 库存指标

| 指标 | 计算 | 健康区间 |
|------|------|---------|
| 库存周转天数 | 365/周转率 | 行业差异大 |
| 库存/营收比 | 库存/季度营收 | < 50% |
| 原材料库存 | 绝对值+同比 | 需结合订单 |

### 5.2 产业链库存位置

```
终端厂商 ← 渠道商 ← 方案商 ← 设计公司 ← 代工厂 ← 设备商
（终端库存）（渠道库存）（中间库存）（IC库存）（晶圆库存）（设备订单）
```

**库存传导**：
- 下行周期：库存从终端向上游传导（终端先去库存）
- 上行周期：补库存从终端向上游传导

### 5.3 库存周期判断

| 阶段 | 特征 | 投资信号 |
|------|------|---------|
| 被动补库存 | 需求下滑、库存上升 | 谨慎 |
| 主动去库存 | 削减订单、库存下降 | 观察 |
| 被动去库存 | 需求回暖、库存下降 | 布局 |
| 主动补库存 | 增加订单、库存上升 | 积极参与 |

## 六、产能周期追踪

### 6.1 产能指标

| 指标 | 频率 | 说明 |
|------|------|------|
| 产能利用率 | 季 | >85% 供需紧张 |
| 资本开支 | 季 | 扩产意愿 |
| 设备订单 | 月 | BB值 > 1 扩张 |
| 晶圆厂投产计划 | 年 | 未来产能 |

### 6.2 BB值（Book-to-Bill Ratio）

**定义**：设备订单金额/设备出货金额

| BB值 | 含义 | 预示 |
|------|------|------|
| > 1.0 | 订单 > 出货 | 扩张周期，未来景气向上 |
| = 1.0 | 订单 = 出货 | 平衡 |
| < 1.0 | 订单 < 出货 | 收缩周期，未来景气向下 |

### 6.3 制程结构

| 制程 | 应用 | 产能状况 |
|------|------|---------|
| 先进制程（<7nm） | 高端CPU/GPU、AI | 紧张，TSMC垄断 |
| 成熟制程（28nm-90nm） | 物联网、汽车、电源管理 | 相对宽松 |
| 特殊工艺 | CIS、功率器件、MCU | 结构性紧张 |

## 七、细分赛道追踪

### 7.1 存储（DRAM/NAND）

**周期性最强**，是半导体周期的风向标。

| 指标 | 意义 |
|------|------|
| DRAM价格 | 景气度核心指标 |
| NAND价格 | 存储景气 |
| 位元增长 | 供给端 |
| 库存天数 | 库存压力 |

**周期规律**：
- 价格上涨 + 库存下降 → 复苏期
- 价格高位 + 库存低位 → 顶峰期
- 价格下跌 + 库存上升 → 衰退期
- 价格低位 + 库存去化 → 底部期

### 7.2 模拟芯片

**周期性较弱**，业绩稳定性高。

**追踪指标**：
- 下游需求（汽车、工业）
- 产品价格（相对稳定）
- 订单能见度

### 7.3 数字芯片（CPU/GPU/MCU）

**受终端需求影响大**。

| 类型 | 周期驱动 |
|------|---------|
| CPU | PC/服务器需求 |
| GPU | 游戏+AI训练 |
| MCU | 工控、汽车、消费 |

### 7.4 功率半导体

**新能源驱动**，成长性强。

**追踪指标**：
- 新能源车销量
- 光伏装机量
- SiC/GaN 渗透率

## 八、景气度综合判断

### 8.1 景气度评分卡

| 维度 | 权重 | 指标 | 评分标准 |
|------|------|------|---------|
| 价格 | 25% | DRAM价格趋势 | 上涨=5分, 稳定=3分, 下跌=1分 |
| 库存 | 25% | 库存周转天数 | 下降=5分, 稳定=3分, 上升=1分 |
| 产能 | 20% | 产能利用率 | >85%=5分, 70-85%=3分, <70%=1分 |
| 需求 | 20% | 终端需求增速 | >15%=5分, 5-15%=3分, <5%=1分 |
| 订单 | 10% | BB值 | >1.1=5分, 0.9-1.1=3分, <0.9=1分 |

**总分判断**：
- 20-25 分：高景气
- 15-19 分：景气平稳
- < 15 分：景气下行

### 8.2 周期定位

| 信号 | 复苏期 | 扩张期 | 顶峰期 | 衰退期 | 底部期 |
|------|--------|--------|--------|--------|--------|
| 价格 | 底部回升 | 持续上涨 | 高位震荡 | 下跌 | 底部企稳 |
| 库存 | 去化加速 | 正常 | 累积 | 累积 | 去化完成 |
| 产能 | 利用率回升 | 扩产 | 满产+扩产 | 利用率下降 | 出清 |
| BB值 | >1 | >1 | ≈1 | <1 | <1 |

## 九、投资应用

### 9.1 周期投资法

**底部布局**：
- 信号：价格触底、库存去化完成、BB值触底反弹
- 标的：周期弹性大的存储、设计公司

**顶峰回避**：
- 信号：价格高位、库存累积、BB值回落
- 操作：减配周期股，转向防御

### 9.2 成长投资法

**穿越周期的公司特征**：
- 技术壁垒（先进制程、独特IP）
- 客户粘性（大客户绑定）
- 产品组合（多终端分散）
- 国产替代（国内份额提升）

### 9.3 细分赛道轮动

| 周期阶段 | 首选赛道 | 逻辑 |
|---------|---------|------|
| 复苏初期 | 设计公司 | 业绩弹性大 |
| 扩张期 | 代工、设备 | 产能扩张受益 |
| 顶峰期 | 模拟、功率 | 防御属性 |
| 衰退期 | 现金流好的龙头 | 生存能力强 |

## 十、国产替代追踪

### 10.1 国产化率

| 环节 | 国产化率 | 说明 |
|------|---------|------|
| IC设计 | 较高 | 手机芯片、安防芯片等 |
| 晶圆制造 | 中等 | 成熟制程较高，先进制程较低 |
| 封装测试 | 较高 | 技术门槛相对较低 |
| 半导体设备 | 较低 | 高端设备依赖进口 |
| 半导体材料 | 较低 | 硅片、光刻胶等 |

### 10.2 替代逻辑

**短期**：成熟制程替代（28nm及以上）
**中期**：设备、材料突破
**长期**：先进制程追赶

### 10.3 追踪指标

- 国产设备中标率
- 国产材料认证进度
- 客户导入情况

## 十二、注意事项

### 局限性

- **地缘政治**：贸易限制、出口管制影响大
- **技术变革**：新技术可能颠覆格局
- **周期判断滞后**：数据滞后，拐点难把握

### 常见误区

| 误区 | 事实 |
|------|------|
| 景气高点=股价高点 | 股价通常领先基本面见顶 |
| 国产替代=高成长 | 需看具体环节的竞争力和替代空间 |
| 周期股用PE估值 | 周期高点PE低、周期低点PE高 |
