# 信用周期

## 定义

信用周期是经济中信贷可用性的扩张和收缩的周期性模式。信用周期通常比商业周期更长，因为经济基本面的弱化和资产价值的下跌需要时间显现。

---

## 周期阶段

### 扩张期（Expansion）

**特征**：
- 利率较低
- 借贷标准宽松
- 信贷规模扩张
- 投资活跃

**机制**：
```
低利率 + 宽松标准 → 信贷扩张 → 资产价格上涨 → 借款人信用改善 → 银行更愿意放贷
```

**风险积累**：
- 过度借贷
- 资产价格泡沫
- 债务负担加重

---

### 收缩期（Contraction）

**特征**：
- 利率上升
- 借贷标准收紧
- 信贷规模收缩
- 去杠杆

**触发因素**：
- 资产价格下跌
- 违约率上升
- 银行风险厌恶增强
- 监管收紧

**机制**：
```
经济下行 → 资产贬值 → 借款人信用恶化 → 银行收紧放贷 → 信贷收缩
```

---

## 与商业周期的关系

| 维度 | 商业周期 | 信用周期 |
|------|---------|---------|
| 驱动因素 | 产出、就业、通胀 | 信贷、杠杆、资产价格 |
| 周期长度 | 3-7 年 | 10-20 年 |
| 观察指标 | GDP、CPI、失业率 | 社融、信贷、债务率 |
| 政策工具 | 财政政策、货币政策 | 宏观审慎政策 |

**信用周期滞后性**：
- 信贷过度扩张的恶果需要时间显现
- 资产价格下跌对信贷的影响有滞后
- 去杠杆过程缓慢

## 核心指标

### 中国信用周期指标

| 指标 | 说明 | 数据来源 |
|------|------|---------|
| 社融存量增速 | 广义信用扩张速度 | 央行 |
| M2 增速 | 货币供应量 | 央行 |
| 信贷脉冲 | 新增信贷/GDP 的变化 | 计算 |
| 宏观杠杆率 | 债务/GDP | 社科院 |
| 居民杠杆率 | 居民债务/GDP | 计算 |

### 信贷脉冲

**定义**：新增信贷占 GDP 比重的变化

**公式**：
$$CreditPulse_t = \frac{\Delta Credit_t}{GDP_t} - \frac{\Delta Credit_{t-4}}{GDP_{t-4}}$$

**解读**：
- 信贷脉冲 > 0：信用加速扩张
- 信贷脉冲 < 0：信用减速或收缩

**与资产价格的关系**：
- 信贷脉冲领先房价 6-12 个月
- 信贷脉冲领先股市 3-6 个月

## 周期定位

### 信号判断

| 阶段 | 社融增速 | 利率 | 资产质量 | 政策导向 |
|------|---------|------|---------|---------|
| 扩张初期 | ↑ | 低 | 稳定 | 宽松 |
| 扩张后期 | ↑↑ | 逐渐上升 | 风险积累 | 逐渐收紧 |
| 收缩初期 | ↓ | 高 | 恶化 | 紧缩 |
| 收缩后期 | ↓↓ | 下降 | 出清 | 逐渐宽松 |

### 关键拐点

**信用底部特征**：
- 社融增速见底回升
- 政策开始宽松（降准、降息）
- 信用利差收窄
- 企业融资环境改善

**信用顶部特征**：
- 社融增速见顶回落
- 政策开始收紧
- 信用违约增加
- 资产价格见顶

## 资产配置应用

### 信用周期与股债配置

| 阶段 | 股票 | 债券 | 现金 | 说明 |
|------|------|------|------|------|
| 扩张初期 | ↑↑ | ↓ | ↓ | 信用扩张，股票牛市 |
| 扩张后期 | ↑ | ↓↓ | - | 估值已高，警惕风险 |
| 收缩初期 | ↓↓ | ↑↑ | ↑ | 避险需求，债券受益 |
| 收缩后期 | - | ↑ | ↑↑ | 等待信用见底信号 |

### 行业轮动

**信用扩张期**：
- 金融地产：受益于信贷投放
- 周期行业：需求旺盛
- 可选消费：财富效应

**信用收缩期**：
- 必需消费：防御属性
- 公用事业：稳定现金流
- 医疗：刚性需求

---

## A 股特殊性

### 政策驱动

A 股信用周期受政策影响大：
- 房地产调控影响居民信贷
- 地方政府债务管控影响基建投资
- 金融监管影响影子银行

### 典型传导路径

```
政策宽松 → 社融回升 → 经济改善 → 企业盈利 ↑ → 股市上涨
         ↘ 利率下降 → 估值提升 → 股市上涨
```

| 居民中长期贷款 | ⭐⭐ | 月度 |

## 历史案例

### 2008 金融危机

**背景**：
- 次贷危机前：美国长期低利率、信贷过度扩张
- 房地产泡沫：次级贷款大量发放
- 杠杆积累：金融机构高杠杆

**崩溃**：
- 房价下跌 → 次贷违约激增
- 银行资产负债表恶化
- 信贷急剧收缩（Credit Crunch）

### 中国 2016-2018

**扩张期（2016-2017）**：
- 供给侧改革 + 房地产去库存
- 社融增速回升
- 经济企稳

**收缩期（2018）**：
- 去杠杆 + 影子银行治理
- 社融增速下滑
- 股市下跌

### 2020-2021

**扩张期（2020）**：
- 疫情后宽松政策
- 信贷大幅扩张
- 资产价格上涨

**收缩期（2021）**：
- 政策正常化
- 信用收紧
- 房地产下行

## 风险信号

### 过热预警

| 信号 | 阈值 | 说明 |
|------|------|------|
| 宏观杠杆率增速 | > 10%/年 | 债务积累过快 |
| 居民杠杆率 | > 60% | 居民债务负担重 |
| 信贷/GDP 比率 | 超过趋势 10% | 信贷过度扩张 |
| 信用利差 | 历史低位 | 风险定价不足 |

### 收缩预警

| 信号 | 说明 |
|------|------|
| 社融增速连续下滑 | 信用收缩信号 |
| 信用违约增加 | 企业融资困难 |
| 利率上行 + 信贷下滑 | 被动收紧 |
| M1-M2 剪刀差扩大 | 活钱减少 |
