# 分析师预期

## 定义

分析师预期是指券商研究员对上市公司未来业绩的预测，包括营收、利润、增长率等。一致预期（Consensus Estimate）是多位分析师预测的加权平均值，反映市场对公司的共同看法。

---

## 核心指标

### 预测指标

| 指标 | 说明 | 用途 |
|------|------|------|
| EPS 预测 | 每股收益预测 | 估值、业绩对比 |
| 营收预测 | 营业收入预测 | 成长性判断 |
| ROE 预测 | 净资产收益率预测 | 盈利能力趋势 |
| 增速预测 | 利润/营收同比增速 | 成长性评估 |

### 变化指标

| 指标 | 说明 | 信号含义 |
|------|------|---------|
| 预测上调 | 分析师调高预期 | 业绩超预期概率大 |
| 预测下调 | 分析师调低预期 | 业绩不及预期风险 |
| 预期修正幅度 | 调整幅度大小 | 变化强度 |

---

## 前瞻性应用

### 景气度前瞻

**核心逻辑**：分析师预测变化先于财报公布，是行业景气度的前瞻信号。

**有效指标**：
- **一致预期 ROE 环比变化**：反映盈利能力边际改善
- **净利润预测调整比例**：上调 vs 下调的比例
- **预期增速差**：当年 vs 次年预期增速变化

### 应用场景

| 场景 | 使用方法 |
|------|---------|
| 行业景气判断 | 行业内公司预期调整比例 |
| 拐点识别 | 预期从下调转为上调 |
| 验证信号 | 财报公布后验证预期准确性 |

---

## 量化指标

### 预期修正比例

```
预期修正比例 = 预测上调公司数 / (预测上调 + 预测下调) 公司数
```

**解读**：
- > 60%：行业景气上行，超配
- 40%-60%：景气平稳，标配
- < 40%：景气下行，低配

### 预期偏离度

```
预期偏离度 = (一致预期 EPS - 实际 EPS) / 实际 EPS
```

**解读**：
- 正偏离：预期过于乐观，注意下修风险
- 负偏离：预期过于悲观，存在超预期机会

### 分析师覆盖度

```
覆盖度 = 覆盖该公司的分析师数量
```

**含义**：
- 高覆盖：市场关注度高，预期更准确
- 低覆盖：存在信息不对称，可能有预期差

## 注意事项

### 预期偏差

- **过度乐观**：分析师倾向于高估，尤其在牛市
- **羊群效应**：预测趋同，缺乏独立判断
- **利益冲突**：券商与上市公司关系影响客观性

### 时效性

- **更新滞后**：重大事件后预测更新不及时
- **季节因素**：财报季前后预测密集更新

### 覆盖偏差

- **大公司偏好**：分析师更愿意覆盖大市值公司
- **冷门股盲区**：小公司缺乏分析师覆盖
