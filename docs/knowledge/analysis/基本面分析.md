# 基本面分析

## 定义

通过研究公司财务状况、行业地位、宏观经济等因素，评估股票内在价值的分析方法。

---

## 分析框架

```
宏观环境
    ↓
行业分析
    ↓
公司分析
    ↓
估值判断
```

---

## 宏观分析

### 经济周期

**四个阶段**：

| 阶段 | 特征 | 利好行业 |
|------|------|---------|
| 复苏 | GDP 增速上升，通胀低 | 周期股、可选消费 |
| 繁荣 | GDP 高增，通胀上升 | 商品、金融 |
| 滞胀 | GDP 放缓，通胀高 | 能源、必需消费 |
| 衰退 | GDP 下降，通胀回落 | 防御板块、债券 |

### 货币政策

**利率影响**：
- 降息：利好股市，降低融资成本
- 加息：利空股市，资金成本上升

**流动性影响**：
- M2 增速：货币供应量
- 社融数据：实体经济融资需求
- 北向资金：外资流入流出

## 行业分析

### 行业生命周期

| 阶段 | 特征 | 投资策略 |
|------|------|---------|
| 导入期 | 高增长，高不确定性 | 风险投资，分散布局 |
| 成长期 | 高增长，竞争激烈 | 优选龙头 |
| 成熟期 | 增长放缓，格局稳定 | 关注分红 |
| 衰退期 | 负增长，产能出清 | 谨慎参与 |

### 竞争格局

**波特五力模型**：
1. 现有竞争者竞争程度
2. 新进入者威胁
3. 替代品威胁
4. 供应商议价能力
5. 买方议价能力

**护城河要素**：
- 品牌优势
- 规模效应
- 网络效应
- 转换成本
- 专利/牌照

### 护城河评估

| 类型 | 特征 | 案例 |
|------|------|------|
| 无形资产 | 品牌、专利、牌照 | 茅台、片仔癀 |

## 估值方法

### 绝对估值

**DCF（现金流折现）**：
```
企业价值 = Σ 未来现金流 / (1 + 折现率)^n
```

**适用场景**：
- 现金流稳定的公司
- 增长可预测的公司

**注意事项**：
- 对假设高度敏感
- 周期股不适用
- 参数调整可操纵结果

---

### 相对估值

**PE 估值**：参见 [[估值指标#PE]]

**PB 估值**：参见 [[估值指标#PB]]

**PS 估值**：参见 [[估值指标#PS]]

**PEG（市盈率相对盈利增长比率）**：
```
PEG = PE / 盈利增长率
```
- PEG < 1：低估
- PEG = 1：合理
- PEG > 1：高估

## 分析流程

### 自上而下

1. 宏观经济判断
2. 选择景气行业
3. 筛选行业内优质公司
4. 估值确认买卖点

### 自下而上

1. 筛选财务指标优秀的公司
2. 分析商业模式和竞争力
3. 判断行业趋势
4. 估值确认买卖点

---

## 价值陷阱

**高股息但盈利下滑**：
- 股息率 = 派息 / 股价
- 盈利下滑导致无法维持分红

**低 PE 但周期见顶**：
- 周期股利润高点 PE 最低
- 随后业绩下滑，股价下跌

**低 PB 但资产质量差**：
- 资产可能虚高或减值
- PB 低不代表安全

**识别方法**：
- 看盈利趋势而非单一数值
- 关注现金流匹配度
- 分析行业周期位置

---

## 局限性

1. **信息滞后** - 财报有延迟
2. **信息不完整** - 仅公开信息
3. **预测不确定** - 未来难以准确预测
4. **市场非理性** - 价格可能长期偏离价值
