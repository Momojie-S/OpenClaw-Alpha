# 医药行业景气度追踪

## 定义

医药行业景气度追踪是通过监测政策环境、研发进展、销售数据和竞争格局等关键指标，判断各细分领域景气度和投资机会的分析方法。是[[行业景气度]]和[[产业链分析]]的具体应用。

---

## 一、行业结构

```
原料药 → 制剂生产 → 流通配送 → 终端销售
（上游）   （中游）    （流通）    （下游）

创新药 ─────────────────────────→ 患者
仿制药 ─────────────────────────→ 患者
医疗器械 ──────────────────────→ 医院/患者
CXO（研发/生产外包）←─────────── 药企
```

### 细分领域

| 细分 | 特点 | 代表企业 |
|------|------|---------|
| 创新药 | 高研发投入、高风险高回报 | 恒瑞医药、百济神州 |
| 仿制药 | 集采降价压力、规模效应 | 华东医药、科伦药业 |
| CXO | 服务外包、受益研发浪潮 | 药明康德、凯莱英 |
| 医疗器械 | 国产替代、集采分化 | 迈瑞医疗、联影医疗 |
| 中药 | 政策扶持、消费属性 | 片仔癀、云南白药 |
| 生物制品 | 技术壁垒高、增长快 | 智飞生物、长春高新 |
| 医疗服务 | 消费升级、连锁扩张 | 爱尔眼科、通策医疗 |

## 二、政策环境追踪

### 2.1 集采政策

**核心指标**：
- 集采批次与覆盖范围
- 中标价格降幅
- 企业中标数量

**影响评估**：
| 集采类型 | 降价幅度 | 受影响程度 |
|---------|---------|-----------|
| 化药集采 | 50-90% | 仿制药承压 |
| 高值耗材 | 60-80% | 中标企业份额提升 |
| 中药集采 | 30-50% | 相对温和 |

**投资启示**：
- 集采品种：回避或等待价格触底
- 非集采品种：寻找政策避风港
- 中标企业：关注量价平衡

### 2.2 医保谈判

**关键指标**：
- 谈判品种数量
- 纳入医保的降价幅度
- 医保谈判成功率

**观察点**：
- 创新药医保谈判：降价换量
- 续约规则：降幅收窄利好存量
- 医保目录动态调整频率

### 2.3 审批政策

**指标**：
- 新药审批速度
- CDE（药审中心）积压情况
- 突破性治疗认定数量

**信号**：
- 审批加速 → 创新药友好
- 审批趋严 → 门槛提高，龙头受益

## 三、创新药追踪

### 3.1 研发管线

**核心指标**：
| 阶段 | 成功率 | 价值贡献 |
|------|--------|---------|
| 临床前 | 5-10% | 低 |
| 临床 I 期 | 15-20% | 中 |
| 临床 II 期 | 25-35% | 中高 |
| 临床 III 期 | 50-70% | 高 |
| NDA/BLA | 80-90% | 很高 |
| 上市 | 100% | 最高 |

**追踪方法**：
- 定期查看公司公告的临床进展
- 关注 ClinicalTrials.gov 注册情况
- 跟踪学术会议（ASCO、ESMO 等）数据披露

### 3.2 关键催化

| 事件 | 时间节点 | 影响程度 |
|------|---------|---------|
| 临床数据读出 | 不定期 | 高 |
| NDA/BLA 受理 | 不定期 | 中 |
| 获批上市 | 不定期 | 很高 |
| 医保谈判 | 年底 | 高 |
| 海外授权 | 不定期 | 很高 |

### 3.3 商业化追踪

**指标**：
- 上市后销售爬坡曲线
- 医院准入数量
- 处方量/患者数

**判断标准**：
- 年销售峰值预期（10 亿/50 亿/100 亿级）
- 竞争格局（First-in-class / Best-in-class / Me-too）
- 医保准入进度

## 四、CXO 追踪

### 4.1 订单与产能

**核心指标**：
| 指标 | 说明 | 判断标准 |
|------|------|---------|
| 在手订单 | 反映未来收入 | 同比增长 > 20% 健康 |
| 产能利用率 | 供需匹配 | > 80% 健康扩张 |
| 人员扩张 | 交付能力 | 持续扩张 |
| 资本开支 | 扩产意愿 | 高增速预示信心 |

### 4.2 客户结构

**观察维度**：
- 国内 vs 海外客户占比
- 大药企 vs Biotech 占比
- 前五大客户集中度

**趋势判断**：
- 海外客户占比高 → 全球竞争力
- Biotech 客户多 → 创新活跃度
- 客户集中度低 → 风险分散

### 4.3 估值锚

**常用指标**：
- PE（市盈率）：成熟 CXO
- PEG：高增长 CXO
- EV/EBITDA：国际化程度高的 CXO

**历史区间**：
| 时期 | PE 区间 | 背景 |
|------|--------|------|
| 高景气期 | 50-80x | 订单饱满、扩产积极 |
| 正常期 | 30-50x | 稳定增长 |
| 低谷期 | 20-30x | 行业出清 |

## 五、医疗器械追踪

### 5.1 国产替代

**指标**：
- 国产化率
- 进口替代空间
- 技术差距

**替代进程**：
| 器械类型 | 国产化率 | 替代难度 |
|---------|---------|---------|
| 低值耗材 | > 80% | 已完成 |
| IVD（体外诊断） | 60-70% | 进行中 |
| 医学影像 | 30-50% | 中等 |
| 高值耗材 | 30-50% | 中等 |
| 手术机器人 | < 20% | 较难 |

### 5.2 集采分化

**影响差异**：
| 类型 | 集采影响 | 投资策略 |
|------|---------|---------|
| 已集采品种 | 价格见底，关注份额 | 龙头份额提升 |
| 即将集采 | 降价预期压制估值 | 回避 |
| 不集采品种 | 政策避风港 | 优先配置 |

### 5.3 创新器械

**追踪指标**：
- 创新器械特别审查通道数量
- 获批创新器械数量
- 国产 vs 进口获批比例

## 六、中药追踪

### 6.1 政策环境

**利好政策**：
- 中药传承创新发展意见
- 中药饮片不纳入集采（部分省份试点除外）
- 中药配方颗粒试点结束

### 6.2 消费属性

**核心指标**：
- 渠道扩张（药店/电商）
- 提价能力
- 品牌力（OTC 占比）

**判断标准**：
- 提价 > 成本上涨 → 品牌力强
- OTC 占比高 → 消费属性强
- 渠道下沉能力强 → 增长可持续

## 七、生物制品追踪

### 7.1 疫苗

**核心指标**：
| 指标 | 说明 |
|------|------|
| 批签发量 | 供给能力 |
| 接种率 | 渗透空间 |
| 新品上市 | 增长驱动 |
| 竞品格局 | 价格压力 |

**重点关注**：
- 大品种（HPV、带状疱疹、肺炎结合等）
- 新品爬坡曲线
- 竞争格局变化

### 7.2 血制品

**核心指标**：
- 采浆量
- 浆站数量
- 产品价格

**供需关系**：
- 供给受限（浆站审批严）
- 需求刚性
- 价格相对稳定

### 7.3 生长激素

**关注点**：
- 新患入组数
- 存量患者续费率
- 竞争格局（长效 vs 短效）
- 安徽联盟集采影响

## 八、医疗服务追踪

### 8.1 连锁扩张

**核心指标**：
| 指标 | 说明 |
|------|------|
| 门店数量 | 扩张速度 |
| 单店爬坡周期 | 盈利能力 |
| 同店增长 | 成熟门店增长 |
| 毛利率 | 商业模式验证 |

### 8.2 消费医疗

**细分领域**：
- 眼科（近视防控、屈光手术）
- 口腔（种植、正畸）
- 医美（注射、光电）
- 辅助生殖

**追踪指标**：
- 客流量/手术量
- 客单价
- 复购率

## 九、景气度综合判断

### 9.1 评分卡

| 细分 | 权重 | 核心指标 | 评分维度 |
|------|------|---------|---------|
| 创新药 | 25% | 研发进展 + 商业化 | 临床数据、销售爬坡 |
| CXO | 20% | 订单 + 产能 | 在手订单、产能利用率 |
| 医疗器械 | 20% | 国产替代 + 集采 | 国产化率、集采影响 |
| 中药 | 15% | 政策 + 提价 | 政策友好度、提价幅度 |
| 生物制品 | 10% | 批签发 + 新品 | 批签发量、新品放量 |
| 医疗服务 | 10% | 扩张 + 同店 | 门店增长、同店增速 |

### 9.2 景气度排序（动态更新）

**高景气**：
- 创新器械（国产替代 + 避风港）
- 中药 OTC（政策友好 + 消费属性）

**中等景气**：
- CXO（订单恢复中）
- 生物制品（大品种放量）
- 医疗服务（扩张稳健）

**低景气**：
- 仿制药（集采持续）
- 集采耗材（价格压力）

## 十、投资应用

### 10.1 政策避风港策略

**逻辑**：寻找不受集采影响的细分领域

| 类型 | 代表 | 避风港程度 |
|------|------|-----------|
| 消费医疗 | 医美、眼科、口腔 | 高 |
| OTC 中药 | 片仔癀、云南白药 | 高 |
| 创新器械 | 手术机器人 | 高 |
| 服务类 | 体检、连锁医院 | 高 |

### 10.2 创新兑现策略

**逻辑**：追踪研发管线，在关键节点布局

**关键节点**：
- 临床数据读出前 1-2 个月
- NDA 受理后
- 海外授权谈判期间

**风险**：
- 临床失败（概率 30-80%）
- 数据不及预期
- 获批延迟

### 10.3 困境反转策略

**逻辑**：在利空出尽时布局

**典型案例**：
- 集采品种价格触底后
- 行业负面事件后的龙头
- 业绩底部确认后

## 十一、数据来源

| 数据类型 | 来源 | 频率 |
|---------|------|------|
| 药品批签发 | 中检院 | 月 |
| 临床试验 | ClinicalTrials.gov、CDE | 实时 |
| 集采信息 | 联采办 | 不定期 |
| 医保目录 | 国家医保局 | 年 |
| 公司业绩 | 财报 | 季 |
| 行业数据 | 米内网、PDB | 月/季 |

## 十二、注意事项

### 行业特殊性

- **政策敏感**：集采、医保谈判影响巨大
- **研发风险**：创新药临床失败率高
- **估值体系**：不同阶段适用不同估值方法
- **信息不对称**：专业性强，需要持续学习

### 常见误区

| 误区 | 事实 |
|------|------|
| 创新药=好投资 | 研发失败率高，商业化难度大 |
| 集采=毁灭性打击 | 中标企业可获得以量补价 |
| CXO 无周期 | 也受投融资环境、订单波动影响 |
| 医疗服务=稳增长 | 扩张期盈利波动，集采也有影响 |
