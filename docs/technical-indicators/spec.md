# 技术指标分析 - 需求文档

> 更新日期：2026-03-07

---

## 一、背景

技术分析是股票投资的重要方法，通过历史价格和成交量数据，计算各类技术指标，判断股票走势和买卖时机。

现有 skill 提供了基本面（个股分析）、资金面（北向资金、龙虎榜）、情绪面（市场情绪）等分析维度，但缺少技术面分析，无法形成完整的分析体系。

---

## 二、目标

实现技术指标分析 skill，提供常用技术指标的计算和信号判断，帮助用户从技术面判断股票走势。

---

## 三、业务概念

### 3.1 支持的技术指标

| 指标 | 类型 | 说明 | 信号 |
|------|------|------|------|
| **MACD** | 趋势 | 指数平滑异同移动平均线 | 金叉/死叉 |
| **RSI** | 动量 | 相对强弱指数 | 超买/超卖 |
| **KDJ** | 动量 | 随机指标 | 金叉/死叉、超买/超卖 |
| **布林带** | 波动 | 价格通道 | 突破上轨/下轨 |
| **均线** | 趋势 | 移动平均线 | 多头/空头排列 |

### 3.2 指标参数

| 指标 | 默认参数 | 可选参数 |
|------|----------|---------|
| MACD | 12, 26, 9 | 自定义快线、慢线、信号线 |
| RSI | 14 | 自定义周期 |
| KDJ | 9, 3, 3 | 自定义 K、D、J 周期 |
| 布林带 | 20, 2 | 自定义周期、标准差倍数 |
| 均线 | 5, 10, 20, 60 | 自定义周期列表 |

### 3.3 信号判断规则

**MACD 信号**：
- 金叉：MACD 线上穿信号线，买入信号
- 死叉：MACD 线下穿信号线，卖出信号
- 柱状图：正值转负值，卖出信号；负值转正值，买入信号

**RSI 信号**：
- 超买：RSI > 80，卖出信号
- 超卖：RSI < 20，买入信号
- 中性：20 ≤ RSI ≤ 80，持有

**KDJ 信号**：
- 金叉：K 线上穿 D 线，买入信号
- 死叉：K 线下穿 D 线，卖出信号
- 超买：K > 80，卖出信号
- 超卖：K < 20，买入信号

**布林带信号**：
- 突破上轨：价格 > 上轨，卖出信号
- 突破下轨：价格 < 下轨，买入信号
- 回归中轨：价格回归中轨附近，持有

**均线信号**：
- 多头排列：短期 > 中期 > 长期，买入信号
- 空头排列：短期 < 中期 < 长期，卖出信号
- 金叉：短期均线上穿长期均线
- 死叉：短期均线下穿长期均线

### 3.4 综合评分

基于多个指标的信号，计算综合评分：

| 信号 | 得分 |
|------|------|
| 强烈买入 | +2 |
| 买入 | +1 |
| 中性 | 0 |
| 卖出 | -1 |
| 强烈卖出 | -2 |

**综合判断**：
- 总分 ≥ 3：强烈买入
- 总分 ≥ 1：买入
- 总分 ≤ -3：强烈卖出
- 总分 ≤ -1：卖出
- 其他：中性

## 四、验收标准

### 4.1 功能验收

- [ ] 能获取指定股票的历史行情数据（至少 60 天）
- [ ] 能计算 MACD、RSI、KDJ、布林带、均线等指标
- [ ] 能根据指标值判断买卖信号
- [ ] 能输出综合评分和操作建议
- [ ] 支持自定义指标参数

### 4.2 数据验收

- [ ] 指标计算准确（与主流软件对比）
- [ ] 信号判断符合规则
- [ ] 综合评分逻辑合理

### 4.3 用户体验

- [ ] 输出精简，重点突出（指标值 + 信号 + 建议）
- [ ] 默认参数合理，开箱即用
- [ ] 支持自定义参数，灵活配置

---

## 五、范围边界

### 5.1 在范围内

- 日线级别技术指标计算
- 常用技术指标（MACD、RSI、KDJ、布林带、均线）
- 买卖信号判断
- 综合评分

### 5.2 不在范围内

- 分时、周线、月线级别指标（后续扩展）
- 复杂技术指标（威廉指标、ATR、CCI 等）
- 自动交易
- 图表绘制
- 回测验证
