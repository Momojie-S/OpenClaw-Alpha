# 需求文档：北向资金追踪

> 版本: v1.0
> 创建时间: 2026-03-07

---

## 一、问题背景

北向资金（外资通过沪深港通流入 A 股的资金）是市场重要的风向标。投资者需要了解：
- 今天外资是买还是卖？
- 哪些股票被外资大笔买入/卖出？
- 外资近期流向趋势如何？

## 二、目标用户

A 股投资者，希望了解外资动向，作为投资决策参考。

## 三、业务概念

### 3.1 北向资金

通过沪深港通机制，从香港流入内地股市的资金。包括：
- **沪股通**：通过上海证券交易所买入的资金
- **深股通**：通过深圳证券交易所买入的资金
- **合计**：沪股通 + 深股通

### 3.2 关键指标

| 指标 | 说明 | 单位 |
|------|------|------|
| 净买入额 | 买入金额 - 卖出金额 | 亿元 |
| 资金余额 | 当日可用额度 | 亿元 |
| 持股变化 | 个股持仓市值变化 | 万元 |

### 3.3 资金状态

| 状态 | 判断规则 |
|------|----------|
| 大幅流入 | 净买入 > 50 亿 |
| 流入 | 净买入 > 10 亿 |
| 平衡 | 净买入在 -10 ~ 10 亿之间 |
| 流出 | 净买入 < -10 亿 |
| 大幅流出 | 净买入 < -50 亿 |

## 四、功能范围

### V1.0 功能

1. **每日净流入统计**：查询当日北向资金净买入情况
2. **个股资金流向**：查询当日外资买入/卖出 Top 股票
3. **近 N 日趋势**：查询近 5/10/20 日资金流向趋势

### 后续版本

- 个股持股明细
- 行业资金分布
- 与指数涨跌的相关性分析

## 五、验收标准

- [ ] 能查询当日北向资金净流入（沪股通、深股通、合计）
- [ ] 能查询当日个股资金流入 Top N
- [ ] 能查询当日个股资金流出 Top N
- [ ] 能查询近 N 日资金流向趋势
- [ ] 资金状态判断正确

## 六、非功能需求

- **响应时间**：单次查询 < 5 秒
- **数据延迟**：盘中数据有 15 分钟延迟
- **可用性**：支持非交易日查询最近交易日数据
