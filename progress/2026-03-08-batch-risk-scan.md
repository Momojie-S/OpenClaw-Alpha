# 任务：添加批量风险扫描功能

## 需求

risk_alert 目前只支持单个股票的风险检查，需要添加批量扫描功能，方便用户一次性检查多只股票（如自选股）的风险。

**分析思路**：
- 添加 `--batch` 参数支持批量检查
- 支持从文件或命令行读取股票列表
- 支持从 watchlist_monitor 读取自选股
- 输出汇总报告，按风险等级分组

## 完成总结

### 新增功能

1. **批量风险扫描**
   - `--batch "000001,600000,002475"` - 逗号分隔的股票列表
   - `--batch-file stocks.txt` - 从文件读取（每行一个代码）
   - `--watchlist` - 从自选股列表读取

2. **批量检查输出**
   - 汇总统计（各风险等级数量）
   - 按风险等级分组显示详情
   - 支持保存到 JSON 文件

### 参数说明

| 参数 | 说明 |
|------|------|
| `--batch` | 批量检查（逗号分隔） |
| `--batch-file` | 从文件读取股票列表 |
| `--watchlist` | 从自选股列表读取 |
| `--top-n` | 每个风险等级显示数量（默认 10） |
