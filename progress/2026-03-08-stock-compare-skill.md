# 任务：个股对比分析功能

## 需求

投资决策时，经常需要在多只股票中进行选择。个股对比功能可以帮助用户横向对比多只股票的关键指标，快速做出决策。

**分析思路**：
- 支持输入多只股票代码（2-10 只）
- 获取关键指标（估值、动量、流动性等）
- 对比展示，高亮优劣
- 生成综合评分

## 完成总结

### 新增 Skill

**stock_compare** - 个股对比分析

**功能**：
1. 横向对比 2-10 只股票
2. 四维度评分：估值、动量、流动性、资金
3. 综合排名和建议
4. 文本和 JSON 输出

**命令**：
```bash
uv run --env-file .env python -m skills.stock_compare.scripts.compare_processor.compare_processor "000001,600000,002475"
```

**评分体系**：
- 估值得分（25%）：PE 越低越好
- 动量得分（30%）：涨跌幅贡献
- 流动性得分（25%）：换手率适中最佳
- 资金得分（20%）：主力资金方向
