# 任务：为已有 Skill 适配信号输出

## 需求

信号驱动回测框架已完成 Phase 1，`technical_indicators` 已支持 `--signal-only` 输出。需要为其他适合回测的 Skill 也添加信号输出能力。

**分析思路**：识别哪些 Processor 能产生明确的买卖信号（有历史数据、有时间点、有动作）

- [ ] 记录每个 Processor 可能产生的信号类型

## Phase 2: 实现
- [ ] 按优先级逐个实现

### 不适合输出信号

| Processor | 原因 |
|-----------|------|
| alert_processor | 预警不是交易信号 |
| announcement_processor | 公告分析，无明确动作 |
| overview_processor | 市场概览，聚合数据 |
| sentiment_processor | 情绪指标，无明确买卖点 |
| breadth_processor | 市场宽度，需衍生 |
| compare_processor | 对比分析，不是信号 |
| screener_processor | 筛选，一次性结果 |
