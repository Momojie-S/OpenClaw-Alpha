### 新增功能

**量价关系分析** - `volume_price_processor`

**功能**：
1. OBV 能量潮分析（趋势判断）
2. 量价相关系数计算（关联性分析）
3. 成交量状态分析（放量/缩量）
4. 量比分析（活跃度判断）
5. 综合量价关系判断

**命令**：
```bash
uv run --env-file .env python -m skills.technical_indicators.scripts.volume_price_processor.volume_price_processor 000001
```

### 量价关系判断逻辑

| 模式 | 信号 | 说明 |
|------|------|------|
| 量价齐升 | 看涨 | 价涨量增，趋势健康 |
| 量价齐跌 | 看跌 | 价跌量缩，抛压减轻 |
| 量价背离 | 偏空 | 价涨量缩/价跌量增，趋势可能反转 |
| 放量滞涨 | 偏空 | 量大价平，可能见顶 |
| 缩量止跌 | 偏多 | 量小价稳，可能见底 |
