- 复用 market_sentiment 的 equity_bond_ratio_processor
- 根据股债性价比确定定投倍数

## 完成总结

### 实现内容

**智能定投策略** - `dip_advice_processor`

**功能**：
1. 获取股债性价比数据（复用 equity_bond_ratio_processor）
2. 根据估值区间确定定投倍数
3. 计算建议金额
4. 提供历史平均倍数参考

**股债性价比驱动策略**：

| 股债性价比 | 市场状态 | 定投倍数 | 操作建议 |
|------------|----------|----------|----------|
| > 3% | 极度低估 | 2.0x | 大幅加仓 |
| 2% - 3% | 低估 | 1.5x | 增加定投 |
| 0% - 2% | 合理 | 1.0x | 正常定投 |
| -1% - 0% | 高估 | 0.5x | 减半定投 |
| < -1% | 极度高估 | 0x | 暂停定投 |

**命令**：
```bash
uv run --env-file .env python -m skills.smart_dip.scripts.dip_advice_processor.dip_advice_processor --base-amount 2000
```

**输出示例**：
```
=== 智能定投建议 ===
日期: 2026-03-06
基准金额: 2000 元

【估值分析】
股债性价比: 4.95%
市场状态: 极度低估

【定投建议】
倍数: 2.0x
建议金额: 4000 元
操作: 大幅加仓

【历史参考】
近3月平均倍数: 1.5x
近6月平均倍数: 1.3x
```
