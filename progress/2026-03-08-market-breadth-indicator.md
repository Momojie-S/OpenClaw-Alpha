## 完成总结

### 新增功能

**市场宽度指标** - `breadth_processor`

**功能**：
1. 获取 20/60/120 日新高新低股票数量
2. 计算市场宽度比率
3. 健康度评估（健康/正常/偏弱/危险）
4. 趋势分析（上升/下降/平稳）
5. 背离信号检测（顶背离/底背离）

**命令**：
```bash
uv run --env-file .env python -m skills.market_sentiment.scripts.breadth_processor.breadth_processor --symbol all --days 30
```

**输出示例**：
```
==================================================
市场宽度分析报告
==================================================
日期: 2026-03-05
收盘点位: 4124.19

【宽度指标】
  20日宽度比率: 85.1%
  60日宽度比率: 93.6%
  120日宽度比率: 94.2%

【新高新低统计】
  20日: 新高 541 / 新低 95

【健康度评估】
  等级: 健康
  得分: 85.1

【趋势分析】

【市场信号】
  ⚠️ 市场宽度极度健康，大部分股票走强
  ⚠️ 底背离信号：指数下跌但市场宽度改善，关注机会
```

### 健康度判断

| 宽度比率 | 健康度 | 市场含义 |
|----------|--------|----------|
| > 70% | 健康 | 大部分股票走强 |
| 50-70% | 正常 | 市场分化 |
| 30-50% | 偏弱 | 大部分股票走弱 |
| < 30% | 危险 | 极端弱势 |

### 背离信号

- **顶背离**：指数上涨但宽度收窄 → 可能见顶
- **底背离**：指数下跌但宽度改善 → 可能见底
